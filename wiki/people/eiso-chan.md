# Eiso Chan

## 概要

Eiso Chan は、CJKV Components proposal、`kIRG_SGSource`、FS / SC guidelines、ideographic punctuation proposals、Kana proposals / feedback など複数の CJK / ideographic 文書に関わる contributor である。`IRG N2799` では Jianzi Musical Notation などの追加 data source を提供する author として現れ、`IRG N2878R3` の final proposal author list にも含まれる。Kana 関連では、Taiwanese Kana proposal への feedback、Ryukyuan superscript Katakana proposal への feedback、明治期 alternate Katakana NE / WI の introduction、後続の revised proposal に関係している。
