# U+23326 𣌦

## 要約

Unicode 18.0 delta chartではExtension BのU+23326が**G-source reference and glyph update**の対象になっている。G-sourceを`GHZ-21496.06`から`GCESI-00560`へ変更し、参照資料に対応する代表字形へ更新する。

## 変更内容

G-sourceを`GHZ-21496.06`から`GCESI-00560`へ変更し、参照資料に対応する代表字形へ更新する。

## Unihan dataの差分

| Property | Unicode 17.0 | Unicode 18.0 beta |
| --- | --- | --- |
| `kIRG_GSource` | `GHZ-21496.06` | `GCESI-00560` |

表はmojidataが固定しているUnicode 17.0 UnihanとUnicode 18.0 beta Unihanの比較である。値が不変でもchart fontの字形変更は別途存在しうる。

## 経緯

ChinaによるG-source evidenceとbulk source referencesの再点検を経てUnicode 18.0へ反映された。

## 影響

- plain textのcode pointはU+23326のままであり、文字列の置換を要求する変更ではない。
- source property、IDS、地域別glyphを単一の「標準字形」と混同しない。
- 実装ではUnicode 18.0 chartとUnihan beta dataを同じ版として更新する。
