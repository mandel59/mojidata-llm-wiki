
# Unicode 18.0 CJK glyph delta

Unicode 18.0 Delta Charts Indexが字形更新として列挙するCJK統合漢字37字とCJK互換漢字4字をまとめる。chart fontだけの変更、IRG source assignment、horizontal extension、disunificationを区別する。

新規符号化されたU+2B81Eは既存文字のglyph updateではないため対象外とし、U+2EA07・U+FACB・U+2F9FFの記事からdisunificationの経緯を参照できるようにした。
