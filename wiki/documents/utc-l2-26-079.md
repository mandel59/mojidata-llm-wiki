# L2/26-079

## 要約

`L2/26-079` は、20 世紀以降の Romance dialectological atlases で使われる phonetic alphabets の basic letters / diacritics 8 文字を追加する proposal である。

対象は Rousselot-Gilliéron、Böhmer-Ascoli、Pellis-Genre、Revista de Filología Española alphabets などで、French / Italian / Spanish dialect corpora の digitization を妨げる missing symbols を扱う。

Modifier / combining letters や positional variants は別 proposal / SEW issues に分け、この文書では basic letters と diacritics に範囲を絞る。

## 提案内容

提案文字は、Combining Diacritical Marks Extended と Latin Extended-G に分かれる。

| 範囲 | 提案文字 |
| --- | --- |
| combining marks | COMBINING INVERTED LAZY S ABOVE、COMBINING COMMA ABOVE AND ACUTE。 |
| Latin letters | LATIN SMALL LETTER REVERSED SCRIPT R、B / D / G / L WITH LEFT HALF STROKE、LATIN SMALL LETTER M WITH RETROFLEX HOOK。 |

文書は各 alphabet の baseline letters / diacritics inventory と、ALF、AIS、ALI、ALPI などの atlas usage を示す。特に partial nasalization、palatal stops、reversed script r、left half stroke letters、retroflex-hook m が Unicode に未収録であることを主張する。

## 後続決定

[L2/26-100](utc-l2-26-100.md) は UTC \#187 の SEW recommendations で、Romance dialectology symbols を provisional assignment へ進める対象に含めた。Modifier / combining letters など別 issue に分けられた部分は、この proposal の外側で継続検討される。

## 論点

### digitization と missing phonetic symbols

Dialectological atlas は大規模な dialect corpus だが、Unicode にない symbols が rendering / search / reuse を妨げている。proposal は、modern use だけでなく historical atlas digitization の実務上の必要性を根拠にする。

### scope control

文書は basic letters / diacritics と positional variants を分けている。SEW pipeline では、文字集合全体を一括で進めるのではなく、encoding model と evidence が明確な subset から処理する方針が見える。
