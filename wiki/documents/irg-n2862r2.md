# IRG N2862R2

## 要約

`IRG N2862R2` は、Eiso Chan が IWDS 用の SC（Stroke Count）と FS（First Stroke）guidelines を consolidated table としてまとめた individual contribution である。`IRG N2862` への Henry Chan、CHEN Xiaoyan、CHEN Zhuang、HUANG Junliang、Ken Lunde らの feedback を受け、pending issues を解消した revision とされる。

文書は entry syntax、Shape / Variants column、FS values、SC values、future additions を整理する。後続の [IRG N2909](irg-n2909.md) M66.18 は、この文書を previous FS / SC documents の replacement として accept し、final document の準備を求めた。

## データ内容

| 区分 | 内容 |
| --- | --- |
| entry syntax | `\d{1,2}[DHOPSZ][a-z]`。数字は SC、upper Latin letter は FS、lower Latin letter は ordering。 |
| FS values | H=横、S=竪/豎/竖、P=撇、D=點/点、Z=折、O=◌。Annex K の 1/2/3/4/5 values と dotted circle を使う。 |
| SC values | 通常値は直接使い、`<value> + dotted circle` は dotted circle component を含めて数える不確定値を表す。 |
| table columns | Entry、Shape、FS、SC、Variants。N2862 から Note column を削除した。 |
| future additions | 丂、冖、廿、反、次、回/囬、耒、垂、戾/戻、面/靣、飠/𩙿/食、奐/奂、臾、乗、𥁕/昷、袞/衮、盜/盗、羨/羡、肅 などを候補として列挙する。 |

## 後続決定

[IRG N2909](irg-n2909.md) の M66.18 は、N2862R2 を FS / SC values の consolidated guideline として受け入れ、final `IRG N2951` の作成を求めた。[IRG N2951](irg-n2951.md) は、N2862R2 を含む previous documents を置き換える consolidated FS & SC Guidelines として提出され、[IRG N2935](irg-n2935.md) は 2026-04-02 に posted され IWDS Series 1 に追加されたと記録している。

## 論点

### Radical information を入れるか

[IRG N2915](irg-n2915.md) は、FS value の根拠を理解するには radical の有無を table に加えるべきだと提案した。N2862R2 はその feedback が IRG meeting で未討議であり、この revision は N2915 に基づく変更を入れていないと明記する。

N2862R2 の著者は、過去の FS documents も radicals に言及していないこと、この document は prescriptive なので radical / non-radical components を区別する必要がないこと、FS values は今後 IRG submission に必須ではなく Chief Editor が調整に使うものになることを response として述べる。

### IWDS review infrastructure

この文書は code point を追加する proposal ではなく、IRG Working Set / IWDS review で使う first stroke と stroke count の共通参照である。WS2024 などの review attributes と、Unihan `kTotalStrokes` / radical-stroke data への接続を読むための infrastructure として扱う。
