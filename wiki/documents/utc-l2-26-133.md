# L2/26-133

## 要約

`L2/26-133` は DOUBLE IDEOGRAPHIC FULL STOP を Ideographic Symbols and Punctuation block に追加し、corner-justified form と centered form の standardized variants を追加する proposal である。主な用途は Cantonese Yueju Opera / Yuequ Show の performance scripts / librettos における upper / lower sentence marking である。

提案 code point は U+16FE6 で、East Asian Width=W、Line_Break=CL、Vertical_Orientation=Tr を想定する。U+3002 IDEOGRAPHIC FULL STOP 2 文字の代用では plain text semantics を保持できないと主張する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| base character | U+16FE6 DOUBLE IDEOGRAPHIC FULL STOP を提案。 |
| variants | U+16FE6 FE00 corner-justified form、U+16FE6 FE01 centered form を提案。 |
| usage | Cantonese Yueju / Yuequ の scripts / librettos で、upper / lower sentence の境界を示す。 |
| properties | `Po`、Script=Common、East Asian Width=W、Line_Break=CL、Vertical_Orientation=Tr を想定。 |

## 論点

### repeated full stop との違い

この proposal の中心は、見た目が U+3002 を二つ並べたものに近くても、Cantonese performance text では独立した punctuation semantics を持つという主張である。base character と SVS の分担も review point になる。
