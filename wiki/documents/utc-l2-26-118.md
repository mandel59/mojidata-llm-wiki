# L2/26-118

## 要約

`L2/26-118` は、Americanist phonetic notation で使われる modifier capital Y を符号化する提案である。Kirk Miller は、modifier small y が palatalization / palatal articulation を表す既存慣用の延長として、capital Y が voiceless palatalized / palatal consonants を示す資料を挙げる。

提案は、`MODIFIER LETTER CAPITAL Y` を modifier letter として Latin Extended-G に追加し、superscript capital Y との compatibility mapping を持たせる。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | `MODIFIER LETTER CAPITAL Y` |
| general category | `Lm` |
| compatibility mapping | `<super> 0059` |
| usage | Americanist phonetic notation の voiceless palatalized / palatal consonants。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は modifier capital Y を provisional assignment へ進める SEW recommendation に含める。

## 論点

### phonetic notation の case contrast

この proposal では、capitalization が orthographic case ではなく phonetic distinction を表す。既存の modifier letters と compatibility superscript の扱いを揃えつつ、plain text phonetic transcription での区別を保つことが論点になる。
