# Proposal to change the kTotalStrokes property values of 458 ideographs

## 要約

`L2/26-148` は、458 ideographs の informative `kTotalStrokes` property values を変更する提案である。
変更の根拠は、IRG の Consolidated FS & SC Guidelines と、IRG ORT が working set metadata checking に使う `totalstrokes.txt` data との整合である。
提案は、Unicode 18.0 cycle の Unihan data updates のうち、stroke count values の大規模な整備にあたる。
添付 data file `kTotalstrokes-changes-458.txt` が 458 件の変更一覧を提供する。
文書は、この 458 件を他の `kTotalStrokes` corrections の基礎として扱う。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| 対象 property | informative Unihan property `kTotalStrokes`。 |
| 対象数 | 458 ideographs。 |
| 根拠 document | [IRG N2951](irg-n2951.md) Consolidated FS & SC Guidelines。 |
| 関連 data | IRG working set review で ORT metadata checking に使われる `totalstrokes.txt`。 |
| 添付 | `kTotalstrokes-changes-458.txt`。 |

`kTotalStrokes` は code point identity を変える property ではないが、radical / stroke based lookup、review tooling、Unihan data の consistency に影響する。文書は、IRG conventions に合わせた total strokes data を Unihan property values に反映する必要を示す。

## 後続決定

この proposal は 2026-07-05 の UTC registry 文書であり、2026-07-08 時点の wiki では UTC #188 以降の処理を未確認である。[Unicode 18.0 Change Sources](../topics/unicode-18-change-sources.md) では、Unicode 18.0 beta close 後の data correction 候補として追跡する。

## 論点

### informative property と review metadata

`kTotalStrokes` は informative property である一方、検索・indexing・IRG review metadata に広く使われる。IRG 側の FS & SC conventions と Unihan published data がずれると、working set review と implementation が別の stroke count を参照することになるため、どの source data に基づく変更かを明記する必要がある。

### IRG FS & SC guidelines との接続

[IRG N2951](irg-n2951.md) は first stroke / stroke count conventions を整理する。`L2/26-148` は、その convention を published Unihan property values へ反映する bridge として読む必要がある。詳細な rule 側の文脈は [IRG Indexing Rules](../topics/irg-indexing-rules.md) に集約する。
