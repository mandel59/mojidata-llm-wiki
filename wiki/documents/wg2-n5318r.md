# WG2 N5318R

## 要約

`WG2 N5318R` は、TCA and China による Four-Column Small Seal Script の UCS encoding proposal である。`Shuowen Jiezi` 小篆を対象に、背景、採用する source versions、glyph correction principles、mapping principles、modern CJK correspondence principles を説明する。

文書は Tenghuaxie version (THX) を主な base とし、Chenchangzhi version (CCZ)、Xiaoxu version (QJZ)、Duan Zhu version (DYC) を四欄 chart layout に組み込む model を説明する。WG2 \#72 M72.13 では、[WG2 N5306](wg2-n5306.md)、[WG2 N5317R3](wg2-n5317r3.md)、`WG2 N5341` と合わせて Small Seal future edition candidate の根拠文書にされた。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | `Shuowen Jiezi` に基づく Small Seal Script。 |
| base source | THX を主な base とし、CCZ、QJZ、DYC を比較欄として扱う。 |
| encoding model | Four-column code chart layout により、主要版本間の glyph / mapping 関係を示す。 |
| principles | [WG2 N5307R](wg2-n5307r.md) に基づく glyph mapping、unification / disunification、modern CJK correspondence の原則を提示する。 |
| WG2 \#72 での扱い | [WG2 N5304](wg2-n5304.md) M72.13 が future edition candidate の根拠文書として参照した。 |

## 後続決定

`WG2 N5318R` は proposal preamble / framework として WG2 \#72 の future candidate に接続した。後続の [WG2 N5344R2](wg2-n5344r2.md) では repertoire と range が更新され、[WG2 N5369](wg2-n5369.md) の CD.4 disposition で Small Seal names、property values、glyph correction が処理された。

## 論点

### Four-column chart model

Small Seal は一つの glyph source だけではなく、複数の `Shuowen Jiezi` versions と variant relationships を持つ。Four-column layout は、この差異を code chart / data として読めるようにするための presentation model であり、UAX \#60 data model とも接続する。
