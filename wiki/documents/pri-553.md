# PRI 553

## 要約

`PRI #553` は、UTC が Unicode 18.0.0 向けの UTS \#39 Unicode Security Mechanisms proposed update を public review に出した PRI である。PRI page の status は Closed、closing date は 2026-07-07。

review 対象は UTS \#39 Revision 33 の draft で、draft page は Version 18.0.0 (draft 2)、date 2026-05-21 を示す。PRI page は draft updated を 2026-05-20 とし、Unicode 18.0 向け update、`nonspacing mark` 用語の明確化、古い text の削除を説明する。

この PRI は CJK-specific な confusable pair の採否ではなく、UTS \#39 全体の review gate である。CJK confusables の候補・review request は [L2/26-082](utc-l2-26-082.md)、[L2/26-127](utc-l2-26-127.md)、および [CJK Security Confusables](../topics/cjk-security-confusables.md) で追う。

## 提案内容

### review window

PRI page は、UTS \#39 が Unicode 18.0 向けに更新されるため general public review and comment に出たと説明する。feedback は Unicode の PRI feedback process に従い、issue close 後は UTC meeting schedule に沿って処理される。

[PRI \#548](pri-548.md) の Unicode 18.0.0 Beta review と同じ 2026-07-07 に close しているため、UTS \#39 の最終確認は Unicode 18.0 beta package の一部として読む必要がある。

### UTS \#39 Revision 33

UTS \#39 draft の modifications section は、Revision 33 を Unicode 18.0.0 向け proposed update として記録する。主な変更は次の通りである。

- editors of record の更新。
- Section 4 Confusable Detection から、gatekeeper-confusable strings に関する段落と UTR \#36 reference を整理する変更。
- Section 4 Confusable Detection から、casefolded forms による security risk reduction の段落を整理する変更。
- Section 5.4 Optional Detection で、`nonspacing mark` が core spec と同じく `gc=Mn` と `gc=Me` を含むことを明確化する変更。

Section 7 Data Files は、`IdentifierStatus.txt` / `IdentifierType.txt`、`confusables.txt`、`confusablesSummary.txt`、`intentional.txt` を UTS \#39 の data files として示す。ただし draft 2 時点の data directory reference は 17.0.0 を指しているため、implementation review では本文の optional detection と合わせて、最終 UTS \#39 / Unicode 18.0 security data directory の差分を確認する必要がある。

## 後続決定

`PRI #553` は 2026-07-07 に closed になっている。UTC \#188 は [L2/26-102](utc-l2-26-102.md) の schedule 上、2026-07-30 に Unicode 18.0 content finalize を予定しているため、PRI feedback の処理結果は UTC \#188 minutes / action items、最終 UTS \#39、最終 security data directory で確認する必要がある。

[L2/26-096](utc-l2-26-096.md) は、UTC \#187 時点で UTS \#39 の casefolding wording、Identifier_Type、mid-priority confusables data、Cyrillic Extended-D などを properties / algorithms work として扱っている。この PRI は、そのうち公開 review に出た UTS \#39 text / data package の入口になる。

## 論点

### CJK confusables との切り分け

UTS \#39 の review gate と、CJK-specific confusables の expert review は別の粒度で扱う。`L2/26-127` は `EquivalentUnifiedIdeograph.txt`、`kSpoofingVariant`、`kZVariant`、platform glyph comparison から作った CJK pair 候補の second review request であり、`PRI #553` はそれらを含む可能性のある UTS \#39 更新全体の public review issue である。

### implementation impact

今回の text-level change は大きな algorithm rewrite ではないが、`nonspacing mark` を `gc=Mn` / `gc=Me` と明確化する点は optional detection の実装条件に関わる。confusables data、Identifier_Status / Identifier_Type data、mixed-script / mixed-number detection は、Unicode 18.0 の beta UCD と合わせて確認する。

### beta package との関係

`PRI #553` は [PRI \#548](pri-548.md) の beta package に含まれる UAX / UTS proposed updates の一つである。Unicode 18.0 の release tracking では、UTS \#39 を CJK security confusables の topic だけに閉じず、[Unicode Properties and Algorithms](../topics/unicode-properties-and-algorithms.md) と [Unicode 18.0 Change Sources](../topics/unicode-18-change-sources.md) の両方から辿れるようにする。
