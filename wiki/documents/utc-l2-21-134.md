# L2/21-134

## 要約

`L2/21-134` は、IDS で利用される ideograph components を複数の既存 collection から照合・統合した調査 data である。`L2/21-118` への応答として、IDS.TXT、`IRG N2225`、UCV、GF 3001-1997 / GF 0014-2009、HKSCS IDS、CDP 外字を比較した。

元の component sets は合計約 2,000 components、うち未符号化で一意なものは約 850 とされる。文書は一定の採択基準を適用し、119 components を、出典 index、名称、用例、unification 候補、短い IDS で表せる場合の注記とともに一覧化する。

## データ内容

### 目的と対象範囲

目的は、既存の component inventories を横断して、IDS で参照する価値がある未符号化 components の共通 collection を作ることである。対象には、WS2017 以後の IRG submission で使われた `IRG N2225` の components、IDS.TXT の components、UCV に現れる未符号化 shapes、Chinese specifications、HKSCS IDS、CDP 外字が含まれる。

候補は、IDS.TXT または `IRG N2225` に含まれるもの、あるいは UCV、GF、HKSCS、CDP のうち二つ以上に含まれ、かつ二つ以上の encoded ideographs で使われるものを基本とする。同一構造での反復利用は一回と数える。

### 一覧が保持する情報

各 row は provisional ID、representative glyph、各 source collection での index、semantic または descriptive name、最大三つの用例、既符号化 ideograph との possible unification、注記を持つ。ID は `IRG N2225` の慣例に従い、first stroke、total stroke count、serial number を組み合わせるが、検索・参照用の暫定値である。

### 限界

この文書は repertoire の比較・選別資料であり、119 components の符号化位置、character properties、正式名称、最終的な unification 判断を確定する proposal ではない。短い IDS で明瞭に表現できる候補は encoding に不適切な可能性があるとして note に回し、katakana を含む non-standard IDS や U+5F50 の disunification のような未決事項もそのまま記録する。

また、UCV は本来 IDS component collection ではなく、GF component sets も GB national standard として認定されたものではない。CDP には一つの ideograph にしか現れない components が多数あるため、source list に存在することだけでは採用根拠にならない。

## 後続決定

この文書自体は UTC の決定を記録しない。後年の [IRG N2733R](irg-n2733r.md) は GF specifications を支える China source 中心の preliminary encoding proposal を作成し、その後の CJKV Components 議論へ接続した。

## 論点

component collection を符号化 repertoire に進めるには、複数 source での利用実績だけでなく、短い IDS による代替可能性、ordinary CJK ideograph との unification、representative glyph と property data を個別に判断する必要がある。したがって本表の inclusion は、各 component の encoding 適格性を最終承認するものではない。
