# IRG N2956

## 要約

`IRG N2956` は、Extension C の GBK-source 5 characters について、current representative glyph と source reference を original evidence と照合した proposal である。対象は U+2AE99、U+2B0AF、U+2B26F、U+2B622、U+2B720 で、glyph update と source reference update が character ごとに混在する。

文書は、過去の IRG proposal と original evidence を参照し、current data のどこが source evidence とずれているかを示す。これは新規符号化提案ではなく、既存 encoded characters の G-source data correction として読む。

## 提案内容

| 対象 | 提案の方向 |
| --- | --- |
| U+2AE99 | representative glyph を original evidence に近い形へ更新する。 |
| U+2B0AF | G-source reference を `GBK-5301480` へ更新する。 |
| U+2B26F | source reference を別 reference または `GU-2B26F` へ更新する。 |
| U+2B622 | representative glyph を original evidence に合わせて更新する。 |
| U+2B720 | source reference を `GU-2B720` へ更新する。 |

## 後続決定

[IRG N2957](irg-n2957.md) が `IRG N2956` への feedback として、5 characters それぞれの処理案を評価している。Meeting \#67 agenda の [IRG N2935](irg-n2935.md) は、representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。

## 論点

### 5 characters を一括処理しない

`IRG N2956` は一つの proposal だが、5 characters の問題は同質ではない。glyph を直すべき case、source reference を直せばよい case、別 evidence へ移すべき case が含まれるため、review では character ごとの disposition が必要になる。
