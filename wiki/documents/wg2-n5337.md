# WG2 N5337

## 要約

`WG2 N5337` は、TCA が [WG2 N5327](wg2-n5327.md) の 5 comments と Michel Suignard の follow-up question に回答した文書である。文書は、pre-official Small Seal glyphs の一部を維持すること、K-source replacement の可能性、U+3961C / U+3AB02 や U+38B34 / U+39BDC の扱いを説明する。

最後に、別符号化後の characters が元 group の radical と視覚的に接続しない場合でも、Small Seal radical arrangement は視覚分析だけでなく `Shuowen Jiezi` の radical system に従うと回答する。

## フィードバック内容

| 対象 | TCA response |
| --- | --- |
| U+383C5 / C-00965 | [WG2 N5307R](wg2-n5307r.md) の General Principle 3 に従い、保持に同意する。 |
| K glyphs | WQS など別 version への source replacement で variant 差を明確化できる場合、group discussion 後に変更可能とする。 |
| U+3961C / U+3AB02 | group 全体を Guwen forms とし、TH-source / D-source が異なるため別符号化する。C / K glyph の replacement は検討余地ありとする。 |
| U+38B34 | TH-02774 を 玄 の Guwen、TH-10621 を 申 の Guwen と見て、source が異なるため別符号化すると説明する。 |
| U+39BDC | stroke count / stroke connection difference に基づく Disunification Principle 3.(3) で別符号化を提案したと説明する。 |
| radical arrangement | Shuowen radical system に従うため、視覚的な radical connection だけでは配置を決めないと説明する。 |

## 後続決定

`WG2 N5337` は `N5327` への response であり、決定文書ではない。TCA は、別符号化された 200 radicals / entries の資料を Michel Suignard に提供できると述べており、後続の disposition と code chart update は [WG2 N5317R3](wg2-n5317r3.md) と [WG2 N5341](wg2-n5341.md) に反映される。

## 論点

### radical system と visual analysis

この文書は、Small Seal の配置・別符号化判断で、glyph の見た目だけではなく `Shuowen Jiezi` の radical system を優先する場面があることを明示する。これは [WG2 N5344R2](wg2-n5344r2.md) の `kSEAL_Rad` data と block ordering を読む前提になる。
