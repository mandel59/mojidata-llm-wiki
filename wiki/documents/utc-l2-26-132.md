# L2/26-132

## 要約

`L2/26-132` は KORE SEBELI / Wakara script を Unicode に符号化する proposal である。KORE SEBELI は Soso language のために Mohamed Bentoura Bangoura が作成した contemporary Guinean script と説明される。

提案は 93 characters を扱う。38 capital letters、38 lowercase letters、10 digits、punctuation / mathematical signs / other signs を含むとするが、proposal summary の一部 field は未記入に見える箇所がある。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script scope | Soso language を書く contemporary script。Guinea-Conakry を中心とした Soso community の文脈で説明される。 |
| repertoire | 93 characters。capital / lowercase letters、digits、punctuation、mathematical signs など。 |
| community evidence | teachers、students、teaching places、scriptors / readers の情報を含む。 |
| review concern | complete proposal としての technical fields、code chart、names、properties の整備度が UTC review の焦点になる。 |

## 論点

### contemporary invented script の review

KORE SEBELI は contemporary script として提出されているため、community use、literacy / education evidence、orthographic stability、case model、digits / punctuation を同一 script block に含める妥当性が重要になる。
