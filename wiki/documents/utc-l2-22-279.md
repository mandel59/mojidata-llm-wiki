# L2/22-279

## 要約

`L2/22-279` は、Seal Script encoding model と UCS Seal Script code charts のための source mapping data specification である。関連 data file `ucs_seal_map.txt` は、30 年以上にわたり収集・校正された Seal Script information を、WG2 Seal Script Ad Hoc が共有できる multi-source mapping framework に整理する。

文書は、THX など単一 source の glyph だけに依存するのではなく、X / K / D source classes の lexical mappings と glyph variation を数量化し、抽象 character repertoire と draft multi-column code charts を設計するための共通形式を定義する。

## データ内容

| 区分 | 内容 |
| --- | --- |
| primary source classes | X = Xu Xuan / Daxu Ben 系、K = Xu Kai / Xiaoxu Ben 系、D = Duan Yucai / Duan Zhu 系として source classes を定義する。 |
| source value syntax | `S:SN` を基本形とし、delimiter `+` / `-` により major / minor glyph differences を flag する。major differences は unification / disunification 判断の入力になる。 |
| file structure | `ucs_seal_map.txt` は 5 fields を持ち、candidate UCS row number、X / K / D development PUA code points、space-delimited source mappings を記録する。 |
| scale | 32,538 source values を 11,122 data rows に mapping し、eventual candidate UCS characters の土台とする。 |
| duplicates / collation | duplicate forms は lexical mapping に基づいて同一 row にまとめ、source serial number による source order collation を保持する。 |

## 後続決定

[WG2 N5306](wg2-n5306.md) は、`L2/22-279` の source mapping data と後続の glyph evidence を前提に、Small Seal の unification / disunification issue を収束させる文書である。`L2/22-279` 自体は data specification であり、encoding repertoire の採択や range 決定は後続の WG2 documents / recommendations で扱われる。

## 論点

### lexical mapping が character identity を支える

文書は、Seal form の見た目だけでなく、head entry、semantic gloss、structural analysis を含む lexical mapping を重視する。duplicates も単純に削除するのではなく、source transmission と emendation によって character identity が変わるかを確認する必要がある。

### multi-column chart への基礎データ

この mapping data により、各 source contributor が持つ glyph data を共通の row structure に統合できる。後続の [Small Seal Script](../topics/small-seal-script.md) proposal では、この data model が multi-column code chart、source references、radical / modern CJK data の設計に接続する。
