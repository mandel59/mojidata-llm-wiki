# IRG N2910

## 要約

`IRG N2910` は、IRG Meeting \#66 で CJK Editorial Group が行った `IRG Working Set 2024 Version 4.0` の review をまとめた報告である。consolidated comments と submitter responses を candidate ごとに検討し、Version 5.0 に反映する editorial conclusions を記録する。

同文書の Appendix B は、会合中に扱った new / updated UCV and NUCV examples を収録する。会合としての採否、Version 5.0 の production schedule、UCV / NUCV list 更新の action items は `IRG N2910` 自体の決定ではなく、[IRG N2909](irg-n2909.md) の recommendations に置かれている。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| WS2024 review | `IRG N2887` Version 4.0 と `IRG N2888` consolidated comments を入力に、candidate glyph、source data、evidence、unification / withdrawal などの comments と responses を検討し、Version 5.0 用の conclusions を整理した。 |
| individual candidates | agenda で明示された KC-10175、VN-F1F1F、GZHSJ-0024、UK-30618、SAT-10292、UTC-03248、UTC-00792、GCW-00243 / GCW-00244 など、個別 candidates の disposition を扱った。GCW-00019 / GZHSJ-0006 については既存 G-source references との関係も検討対象になった。 |
| withdrawals | submitter requests や review conclusions に基づく withdrawal candidates を扱い、Version 5.0 へ残す data と除く data の編集判断を記録した。 |
| UCV / NUCV | Appendix B に new / updated UCV and NUCV examples をまとめ、working set review で用いた unification boundary の判断を次版 lists へ引き継げる形にした。 |

## 後続決定

[IRG N2909](irg-n2909.md) の M66.07 は、`IRG N2910` に記録された conclusions に基づいて WS2024 Version 5.0 を作成することを勧告し、data fix、font update、Version 5.0 release、次回 comments / responses の schedule を設定した。その成果物が `IRG N2932` である。

同じく M66.04 は Appendix B の new / updated UCV and NUCV examples を、一件を further discussion として残しつつ provisionally accept した。IWDS Editor による更新版 [IRG N2931](irg-n2931.md) の作成と、IRG Meeting \#67 に向けた expert review は、この formal recommendation から生じた action items である。

## 論点

### Editorial conclusion と formal recommendation の境界

`IRG N2910` は candidate review の結果をまとめる editorial report であり、IRG 全体の recommendation / action item 文書ではない。wiki では、個別 candidate の review conclusion をこの文書に、Version 5.0 作成の承認と期限を [IRG N2909](irg-n2909.md) に帰属させる。

### Working set review と UCV / NUCV

WS2024 の individual candidate review と UCV / NUCV list は別の成果物だが、glyph difference を unifiable variation と見るか、separate ideograph と見るかという判断で接続している。Appendix B は Meeting \#66 の個別判断を reusable reference examples へ戻す役割を持ち、後続の [IRG N2931](irg-n2931.md) で継続的な reference data になる。
