# IRG N2689

## 要約

`IRG N2689` は、China が G4K、GBK、GCH、GCY、GHC、GU の G-source reference values for 1,685 characters を更新するよう IRG と WG2 に求めた文書である。IRG M58.13、M62.12、M63.07 の follow-up と位置づけ、`IRG N2442`、`IRG N2542`、`IRG N2656` の指摘を一括して扱う。

124 charactersではsource prefix自体を別のG-sourceへ変更し、残る1,561 charactersではsourceを維持してreference valueのみを更新する。あわせて G4K / GBK / GCH / GCY / GHC のreference syntax変更を提案するが、文書のstatusは `Member's submission` であり、採択を記録する文書ではない。

## 提案内容

| 現行source | 対象数 | 提案の要点 |
| --- | ---: | --- |
| G4K | 3 | valueのない3字を GCESI 2字、GZFY 1字へ移し、G4K syntaxを `G4K-ddddd` に統一する。 |
| GBK | 175 | valueなし86字とspecific valueあり89字を見直し、31字は GU 等の別sourceへ移す。syntaxは7-digit形式へ改める。 |
| GCH | 624 | `IRG N2656`に基づき、更新値をcode pointに従う形式へ揃える。 |
| GCY | 192 | `IRG N2656`に基づき、更新値をcode pointに従う形式へ揃える。 |
| GHC | 601 | `IRG N2656`に基づき、更新値をcode pointに従う形式へ揃える。 |
| GU | 90 | GCA 4、GDM 1、GGFZ 5、GHZR 31、GKX 41、GT 8へsourceを変更する。 |

sourceを変更する124字の内訳は、G4Kから3字、GBKから31字、GUから90字である。本文は追加するGCESI、GCA、GDM referencesの一部について、刊本、ISBN、page等のpublishing sourceも示す。全1,685件の具体的な更新値はattachmentに置かれている。

## 後続決定

本文は `Actions required: To be considered by IRG and WG2` と明記されたproposalであり、IRGまたはWG2による採択・承認を本文自身は記録していない。このため、本ページでは1,685件を採択済みの変更として扱わない。

[IRG N2864R](irg-n2864r.md) は後続feedbackと追加案件を統合した別のChina member-body proposalで、G3 19、G5 15、GCH 624、GCY 192、GE 1、GHC 601、GHZ 1、GU 3、GXC 82の計1,538 charactersを対象とする。`IRG N2689`の1,685件をそのまま改訂した同一集合ではなく、G4K / GBKの大規模処理を外す一方でG3 / G5 / GXC等を加え、GCH / GCY / GHCについても辞書上の位置を表すreferenceへ戻す方向で再整理している。`IRG N2864R`もproposalであり、それだけで本提案の採択を証明するものではない。

## 論点

### reference valueとsource identity

本提案は、同じsource prefixのままreference valueを補完・置換する変更と、より具体的なpublishing sourceが判明したためsource prefix自体を変更する処理を区別する。後者は124字に限られ、残る1,561字はsource identityを維持する。

### code point型referenceの暫定性

GCH / GCY / GHCではcode pointに従う値への統一を提案したが、後続の `IRG N2835R` や `IRG N2864R` は、それでは辞書内の掲載位置を識別できないとして再変更を求めた。したがって本提案は、大規模整理の重要な中間段階ではあるが、後続議論の最終形ではない。
