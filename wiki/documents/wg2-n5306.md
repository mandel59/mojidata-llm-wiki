# WG2 N5306

## 要約

`WG2 N5306` / `L2/25-111` は、Small Seal Script の最終 encoding proposal に向けて、残る unification / disunification issue を整理した individual contribution である。前段の consideration document と同じ model を使い、主要 source glyphs が揃った段階で review すべき cases を具体化している。

文書時点の repertoire は 11,140 code points で、約 450 elements が review 対象とされた。著者は、最終 review で少数の variant disunification と若干の unification を確認すれば、Small Seal repertoire は encoding に近い mature state にあると見る。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| source model | THX / CCZ / DYC / QJZ の source sets を使い、Small Seal code chart を four-column fixed layout で提示する。 |
| database fields | `kSEAL_THXSrc`、`kSEAL_CCZSrc`、`kSEAL_DYCSrc`、`kSEAL_QJZSrc`、`kSEAL_MCJK`、`kSEAL_Rad` を使う model を示す。 |
| variant review | 既存 review で残った variant sets と追加調査を統合し、19 variant sets を確認対象として提示する。 |
| modern CJK values | modern CJK representation は useful data だが character identity を定義するものではなく、採用後も調整可能な情報として扱う。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) の Recommendation M72.13 は、`WG2 N5306`、`WG2 N5318R`、`WG2 N5317R3`、`WG2 N5341` をもとに、Small Seal characters を future edition candidate として Project Editor が検討するよう勧告した。wiki ではこの節目を [WG2 M72.13 Small Seal future edition candidate](../events/wg2-m72-13-small-seal-future-candidate.md) に集約する。

## 論点

### final proposal ではなく review convergence

`WG2 N5306` は最終符号化提案ではなく、source glyphs と data model が揃った後に、unification / disunification の残件を収束させるための文書である。後続の revised proposal では repertoire、range、chart presentation がさらに更新される。

### Small Seal は modern Hanzi の typeface ではない

文書は、Small Seal を modern Hanzi の書体差ではなく、ancient Chinese characters の historical stage に属する独立 script と説明する。modern CJK equivalent は検索や比較のための data であり、Small Seal character identity の決定根拠ではない。
