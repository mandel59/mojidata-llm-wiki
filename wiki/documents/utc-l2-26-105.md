# L2/26-105

## 要約

`L2/26-105` は、Unicode 18.0.0 向けの Proposed Update UAX \#38 Revision 40 である。Unihan database の organization、property categories、property syntax、version history を説明する UAX \#38 本体の draft で、PRI \#534 の入力文書として UTC \#187 の CJK & Unihan Working Group recommendations から参照される。

Revision 40 の変更は、provisional `kIRGDaeJaweon` / `kIRGKangXi` properties の削除、`kGB5` / `kIRG_GSource` syntax の更新、`kIRG_KSource` / `kIRG_UKSource` / `kTang` description の更新、covered ideographs table への U+2B81E 追加を含む。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| UAX \#38 本体 | Unihan database の mechanics、property categories、property syntax、Unihan.zip の構成、property history を Unicode 18.0.0 向けに更新する。 |
| IRG source properties | `kIRG_GSource` の syntax / description を更新し、`kIRG_KSource` と `kIRG_UKSource` の説明も調整する。 |
| removed properties | `kIRGDaeJaweon` と `kIRGKangXi` を provisional properties から削除し、Version 18.0.0 の removed properties として記録する。 |
| repertoire table | U+2B81E を CJK Unified Ideographs Extension D の Unicode 18.0 entry として covered ideographs table に追加する。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、PRI \#534 feedback の処理としてこの proposed update を参照し、Unihan database additions / changes / removals と UAX \#45 records の修正を UTC \#187 の CJK & Unihan Working Group recommendations に束ねている。このページでは UAX \#38 draft 自身の変更点を扱い、UTC \#187 での action item は `L2/26-099` 側を読む。

## 論点

### UAX \#38 と UAX \#45 の接点

UAX \#38 の `kIRG_USource` description は、Extension C 以降の U-source values が UAX \#45 の indices に基づくことを説明する。したがって U-source additions / status を追う場合は、Unihan source property と UAX \#45 records の両方を確認する必要がある。

### Unicode 18.0 data update の入口

`L2/26-105` は Unihan database の説明文書であり、個別 property data の更新そのものは `Unihan.zip` や `L2/26-099` の添付 data files で確認する必要がある。wiki では、UAX \#38 の説明変更は [Unihan Database Maintenance](../topics/unihan-database-maintenance.md)、release 全体の入口は [Unicode 18.0 Change Sources](../topics/unicode-18-change-sources.md) に接続する。
