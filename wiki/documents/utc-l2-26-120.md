# L2/26-120

## 要約

`L2/26-120` は、[L2/26-119](utc-l2-26-119.md) の U+06C4 glyph correction request に対する Kamal Mansour の response である。Kashmiri sources の visual survey を示し、ring を waw の tip / tail 側へ置く代表字形への修正を支持する。

文書は同時に、U+06C4 と U+06C5 が似た字形になり得るが、Kashmiri と Kyrgyz の別文字として区別されること、U+06C5 側には crossbar / loop variants があることを整理する。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| position | U+06C4 representative glyph を ring が tail / tip 側にある形へ修正することに賛成。 |
| evidence | Kashir Dictionary、Koul、Kashmiri Phonetic Reader、PASCII、TDIL Kashmiri Design Guide などを比較する。 |
| related distinction | U+06C4 と U+06C5 は字形が近くても別文字であり、U+06C5 の Kyrgyz usage も考慮する。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は、U+06C4 glyph correction に加え、U+06C4 / U+1EE85 confusables と cross-reference update を SEW recommendation の一部として扱う。

## 論点

### glyph evidence と confusables の接続

この response は、representative glyph correction を支持するだけでなく、似た Arabic letters との関係を整理する。glyph correction が security data や cross-reference に及ぶ例として、[Unicode Properties and Algorithms](../topics/unicode-properties-and-algorithms.md) 側でも参照する。
