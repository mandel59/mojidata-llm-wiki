# L2/16-358R

## 要約

`L2/16-358R` / `WG2 N4803` は、`L2/16-334` と `L2/16-354` の追加 small kana proposal に対する feedback である。文書は、`L2/16-334` の証拠が日本語版広辞苑の過去版に存在すること、`L2/16-334` の 4 文字が `L2/16-354` の subset であることを確認する。

また、日本の専門家からの feedback として 7 文字の符号化支持を記録し、UTC #149 で受理された code point / character name / property を列挙する。残る 5 文字については position がなく、HIRAGANA LETTER SMALL N の証拠は発音判断が難しく説得的でないとされた。

| subset 関係 | `L2/16-334` の 4 文字は `L2/16-354` の proposal subset。 |
| UTC #149 | 7 文字が受理されたと記録。 |

## 論点

### 受理された 7 文字

文書が UTC #149 で受理されたとする 7 文字は、HIRAGANA LETTER SMALL WI / WE / WO、KATAKANA LETTER SMALL WI / WE / WO / N である。これにより、Small Kana Extension の一部は個別 proposal と専門家 feedback を経て進んだが、未受理の文字と model 全体の問題は残った。

### Japan NB contribution への接続

[L2/17-091](utc-l2-17-091.md) は、PDAM 1.2 の 9 small kana characters に対する Japan NB の contribution であり、`L2/16-358R` が示した個別受理後に、選定基準・repertoire 調査・encoding mechanism を改めて問う。
