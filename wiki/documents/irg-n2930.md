# IRG N2930

## 要約

`IRG N2930` は、3 characters の current `GU` source references を、`Hanyu Da Cidian` evidence に対応する `GHC` source references へ変更する提案である。Boris Zhang の suggestion を Wang Xieyang が draft した individual contribution として提出された。

対象は U+236AC、U+24A1E、U+24E82 で、それぞれ `GU-236AC`、`GU-24A1E`、`GU-24E82` から `GHC-0500150`、`GHC-1109650`、`GHC-0302880` へ変更する。文書は各 source reference に対応する `Hanyu Da Cidian` の巻・ページ evidence を示す。

## 提案内容

| Character | Current reference | Suggested reference | Evidence |
| --- | --- | --- | --- |
| U+236AC | `GU-236AC` | `GHC-0500150` | `Hanyu Da Cidian` 第一版、巻五、15 頁。 |
| U+24A1E | `GU-24A1E` | `GHC-1109650` | `Hanyu Da Cidian` 第一版、巻十一、965 頁。 |
| U+24E82 | `GU-24E82` | `GHC-0302880` | `Hanyu Da Cidian` 第一版、巻三、288 頁。 |

## 後続決定

[IRG N2911](irg-n2911.md) は、IRG Meeting \#66 editorial group が `IRG N2930` の 3 G-source reference changes を受け入れたと記録している。[IRG N2909](irg-n2909.md) M66.10 もこの acceptance を formal recommendation として整理した。

[L2/26-099](utc-l2-26-099.md) は、この reference change を UTC \#187 CJK & Unihan action item に取り込み、`kIRG_GSource` changes として Unicode Version 18.0 target の data update に接続した。

## 論点

### `GU` から `GHC` への移動

この文書は representative glyph の変更ではなく、source reference の collection identity を変える提案である。current glyph が変わるとは限らず、Unihan data では `kIRG_GSource` の value update として現れる。

### G-source issue chain の中での位置

IRG Meeting \#66 では `IRG N2930` の 3 件は accept された一方、`IRG N2919R` / `IRG N2923R` の 342 reference changes は China response 待ちとされた。G-source reference issues は件数が多いため、accepted subset と pending bulk changes を分けて追う必要がある。
