# L2/26-090

## 要約

`L2/26-090` は Hebrew block に U+05F5 HEBREW PUNCTUATION ELONGATED GERESH を追加する proposal である。Tetragrammaton の graphic representations を表す minimal set のうち、Unicode 11.0 で符号化された YOD TRIANGLE に続く remaining unencoded element と位置づける。

文書は、elongated geresh を standard geresh U+05F3 の長い形として理解しつつ、medieval / early modern printed Hebrew sources で distinct graphic element として現れるため別符号化を求める。properties と collation は U+05F3 GERESH と同じにする提案である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed code point | U+05F5 HEBREW PUNCTUATION ELONGATED GERESH。 |
| usage | Tetragrammaton の graphical substitutes における long curved stroke。 |
| evidence | 13th-18th century Hebrew manuscripts / printed texts from Sephardic、Ashkenazic、Italian traditions。 |
| proposed behavior | Properties and collation は U+05F3 GERESH と同じ。 |

## 後続決定

2026-07-09 時点の local catalog では、この proposal への UTC 決定は未確認である。

## 論点

### geresh の glyph variant か distinct graphic element か

elongated geresh は ordinary geresh に置換されることもあるが、文書は YOD TRIANGLE などと組み合わさる conventionalized element として別符号化を求める。font-level variant ではなく plain text character として扱う根拠が review point になる。
