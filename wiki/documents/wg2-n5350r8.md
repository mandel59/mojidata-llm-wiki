# WG2 N5350R8

## 要約

`WG2 N5350R8` は、2026-06-22/26 に Paris, France で開催された WG2 Meeting \#73 の revised agenda である。ISO/IEC 10646 7th edition CD.4 disposition、external code charts、Amendment 1 project subdivision、IRG \#65 / \#66 reports、Small Seal、CJK / Unihan changes、SEI liaison proposals を主要議題として並べる。

WG2 \#73 の minutes / action items は `WG2 N5351` として registry に entry があるが、2026-07-08 時点の local catalog では document URL が未掲載である。そのため、この digest では agenda、recommendations、関連 input documents を中心に会議内容を整理する。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| 7th edition | CD.4 disposition [WG2 N5369](wg2-n5369.md)、external code charts [WG2 N5363R](wg2-n5363r.md)、meeting recommendations [WG2 N5354](wg2-n5354.md) が adoption item に置かれた。 |
| Amendment 1 | New amendment project [WG2 N5362](wg2-n5362.md) と provisionally assigned code points [WG2 N5361R](wg2-n5361r.md) が扱われた。 |
| IRG | IRG \#65 / \#66 recommendations、CJKV Components、`kIRG_SGSource` などが WG2 / SC2 acceptance に接続する。 |
| Unicode / CJK | `L2/26-099` CJK & Unihan recommendations と `L2/26-102` RMG report が 10646 / Unicode synchronization の文脈で載る。 |
| SEI / script proposals | SEI liaison report、Codical Maya、Maya Hieroglyphs Extended-A、[Dai Xaau](../topics/dai-xaau-script.md) supplementary materials などが future proposal review として扱われた。 |

## 後続決定

[WG2 N5354](wg2-n5354.md) が WG2 \#73 の recommendations である。CD.4 disposition は M73.01、external code charts は M73.03、DIS progression は M73.04、Amendment 1 project は M73.05、Maya review は M73.14、`kIRG_SGSource` は M73.15 に接続する。

## 論点

### agenda と minutes の差

`WG2 N5351` は WG2 \#73 minutes / action items として catalog にあるが、文書実体はまだ公開されていない。このため、会議で実際に採択された事項は [WG2 N5354](wg2-n5354.md) を一次ソースにし、agenda item は [WG2 N5350R8](wg2-n5350r8.md) で処理範囲を確認する。
