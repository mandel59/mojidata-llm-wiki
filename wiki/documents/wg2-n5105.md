# WG2 N5105

## 要約

`WG2 N5105` は、TCA and China が Small Seal Script proposal の appendix を改訂した文書である。`L2/19-228` は同じ文書を UTC register に載せた別番号なので、このページでは alias として扱う。

文書は `Shuowen Jiezi` Tenghuaxie version の 14 volumes から 11,093 Small Seal Script characters を提案し、original glyph、proposed representative glyph、modern CJK equivalent、Shuowen radical などの table fields を示す。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| repertoire | Tenghuaxie version 由来の 11,093 Small Seal characters。 |
| predecessor | `WG2 N4688` の 11,108 characters から、Chongwen で 1 つだけ受け入れる repeated characters 15 件を削除したと説明する。 |
| appendix structure | 14 volumes に対応する 14 parts。original glyphs、TrueType font、Shuowen radicals、corresponding modern characters を含む。 |
| table data | volume 内 serial number、Tenghuaxie whole-volume serial number、scanned original glyph、TTF representative glyph、modern CJK equivalent、radical equivalent、radical number、type を持つ。 |
| type values | `Zhengzhuan`、`Variant`、`Newly Added Character`。 |

文書は、2017-08-28/29 の Shuowen Seal Ad Hoc Meeting、2018 年の WG2 \#67、2019-06-01 の Shuowen Seal Informal Meeting を経た revised appendix として位置づけられている。

## 後続決定

この文書自体は proposal appendix であり、会議決定ではない。同じ WG2 \#68 では [WG2 N5108](wg2-n5108.md) が Seal Script discussion を報告し、CD / CDAM text に入れるには他 versions と taxonomy の追加議論が必要だと整理した。後続では Suzuki Toshiya による比較表、Richard Cook の [L2/22-279](utc-l2-22-279.md)、Michel Suignard の [WG2 N5209](wg2-n5209.md) などを通じて、THX だけでなく CCZ / QJZ / DYC を含む source model へ整理が進んだ。

## 論点

### Repertoire reduction

N5105 は、`WG2 N4688` からの差分として 15 repeated characters を削除した点を明示する。これは Chongwen など variant / repeated entries をどこまで disunify するかという後続の unification 論点につながる。

### Table fields

modern CJK equivalent は informative な対応値として示され、character identity そのものは Tenghuaxie source number と glyph evidence に基づく。後続の UAX \#60 / SealSources.txt では、この種の source reference と modern CJK value が data property として再整理される。
