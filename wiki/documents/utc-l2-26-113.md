# L2/26-113

## 要約

`L2/26-113` は Kodava を Kannada script で書く際に使う diacritic marks 4 文字を Kannada block に追加する proposal である。Kodava の unrounded phonemes を表すため、NCERT / CIIL の Kodava primer や辞書類に現れる上付き記号を符号化対象にする。

提案文字は U+0CD0 KANNADA SIGN OVERLINE、U+0CD1 KANNADA SIGN VERTICAL LINE ABOVE、U+0CD2 KANNADA SIGN DOT ABOVE、U+0CD3 KANNADA SIGN CANDRA ABOVE である。いずれも combining mark として、vowel letters や vowel signs と組み合わせて使う想定である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| repertoire | Kannada block に 4 combining marks を追加する。 |
| usage | Kodava の [ɨ] / [ɘ] / [ɛ] / [ɜ] 系 vowels を Kannada script で表す。 |
| evidence | Kodava dictionaries、NCERT / CIIL の Kodava primer、published examples。 |
| proposed properties | `Mn`、Indic_Syllabic_Category=Vowel_Dependent、Indic_Position_Category=Top を提案。 |

## 論点

### script-specific diacritics for Kodava

Kodava は Kannada script で書かれる一方、Kodava 固有の vowel distinction を表すために additional diacritics を使う。この proposal は新 script ではなく、existing Kannada block への combining marks 追加として扱う。既存 Kannada vowel signs との sequence model、font support、Indic property assignment が review point になる。
