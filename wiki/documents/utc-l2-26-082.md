# L2/26-082

## 要約

`L2/26-082` は、Properties and Algorithms Working Group が CJK Working Group に CJK confusable pairs の review を求めた文書である。PDF 本文の日付は 2026-03-16、UTC registry 上の日付は 2026-03-18 である。

対象は UTS \#39 `confusables.txt` に追加し得る候補で、PAWG 側では CJK の専門的判断が不足すると明記している。候補には Katakana と CJK strokes / radicals / ideographs の視覚的類似、U+3A36 / U+6440 のような CJK ideographs 同士の pair が含まれる。

## フィードバック内容

| Data target | 採用される pair は UTS \#39 security data の `confusables.txt` に追加される。 |
| 後続処理 | `L2/26-099` Section 33 で、U+3A36 / U+6440 を除く pair を追加する action item につながった。 |

## 論点

### PAWG から CJK WG への専門 review

文書は、confusables data の維持は Properties and Algorithms Working Group の担当だが、提示された CJK 系 pair を十分に review する専門性が PAWG 側にないとして CJK Working Group に判断を求めている。CJK confusables は単なる glyph 類似だけでなく、radical / component、CJK source data、variant 関係、既存の文字区別を考慮する必要がある。

### 最初の候補集合

候補には `U+30CD` / `U+793B`、`U+30D2` / `U+5315`、`U+30E0` / `U+53B6`、`U+30E1` / `U+4E44`、`U+30E8` / `U+5F50`、`U+30E8` / `U+2B739`、`U+3A36` / `U+6440` が含まれる。文書は Android 16 上の表示比較と Unicode 18.0 alpha chart の screenshot も添えている。

### U+3A36 / U+6440 の除外

後続の `L2/26-099` は、U+3A36 / U+6440 pair を除外した。CJK & Unihan Working Group は、U+9CE5 鳥と U+70CF 烏のような一画差でも区別される既存例を踏まえ、この pair を distinct と判断した。
