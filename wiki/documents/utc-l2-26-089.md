# L2/26-089

## 要約

`L2/26-089` は、National Bank of the Republic of Belarus が Belarusian Ruble sign を Currency Symbols block に追加する提案である。提案字形は Cyrillic capital letter Be に中央の parallel line を重ねた形で、Belarusian ruble の通貨記号として説明される。

文書は、National Bank board が 2026-01-27 に symbol design を承認したこと、Belarus の商業・金融実務で使用されることを根拠に、`U+20C5 BELARUSIAN RUBLE SIGN` を求める。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | `U+20C5 BELARUSIAN RUBLE SIGN` |
| block | Currency Symbols |
| general category | `Sc` |
| bidi class | `ET` |
| usage | Belarusian ruble の通貨記号として、価格表示・商業文書・金融実務で使う。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は Belarusian Ruble sign を先へ進める SEW recommendation に含める。[WG2 N5354](wg2-n5354.md) は WG2 \#73 M73.06 の first amendment additions の一部として Belarusian Ruble sign を扱う。

## 論点

### national currency symbol と Unicode encoding

この proposal は script character ではなく currency symbol の符号化である。文字名、General_Category、Bidi_Class は既存の Currency Symbols block の慣例に沿って設計されている。
