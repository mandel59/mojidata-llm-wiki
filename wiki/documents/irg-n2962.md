# IRG N2962

## 要約

`IRG N2962` は、9 G-source glyphs の revision request である。Lin Anning は、G-source glyph convention、GB/T 11460-2025、original evidence との照合を根拠に、current representative glyphs の修正を求めている。

文書は、component が G-source convention に従っていない case、機械的な拡縮に見える case、original evidence と異なる lower component を持つ case などを挙げる。単なる字形好みではなく、source convention と evidence のどちらに合わせるべきかを IRG review にかける文書である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | 9 G-source glyphs。 |
| 根拠 | G-source glyph convention、GB/T 11460-2025、original evidence。 |
| 問題 | component form、component scaling、source evidence との不一致。 |
| 作業種別 | Existing encoded characters の representative glyph revision request。 |

## 後続決定

Meeting #67 agenda の [IRG N2935](irg-n2935.md) は、representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。`IRG N2962` 自体は request document であり、採否は記録していない。

## 論点

### source convention と evidence の接点

`IRG N2962` は、current chart glyph が source convention に合っていない場合と、original evidence に合っていない場合を同時に扱う。IRG review では、どの correction が G-source convention の適用で、どれが evidence mismatch の correction なのかを分ける必要がある。
