# L2/26-131

## 要約

`L2/26-131` は Kannada script で Samavedic svara markers を表すため、12 combining marks を追加する proposal である。Samavedic svara markers は Devanagari と Grantha には既に符号化されており、Kannada publications でも同様の記法が使われるとする。

提案文字は U+0CF4..U+0CFF の範囲を想定し、combining digits 0-5 と combining letters A / U / KA / NA / RA / VI を含む。Indic_Positional_Category は Top、Indic_Syllabic_Category は Cantillation_Mark とする。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed repertoire | 12 combining Kannada marks。digits 0-5 と letters A / U / KA / NA / RA / VI。 |
| usage | Samaveda scholars と modern publications で使われる、syllables の上に置く superscript-style markers。 |
| properties | `Mn`、canonical combining class 230、Indic_Positional_Category=Top、Indic_Syllabic_Category=Cantillation_Mark を提案。 |
| related precedent | Devanagari / Grantha の Samavedic notation documents と Kannada spacing candrabindu などを参照する。 |

## 論点

### script-specific Vedic notation

同じ Samavedic notation でも、Devanagari / Grantha で符号化済みの文字を Kannada text にそのまま使うのではなく、Kannada block 内に script-specific combining marks として追加する提案である。Vedic notation の script property、rendering、既存文字との重複が review point になる。
