# L2/19-382

## 要約

`L2/19-382` は、Late Middle Japanese の transcription に用いる 1 kana character、HENTAIGANA LETTER SMALL TU-2 を提案する文書である。提案は、日葡辞書などの Portuguese data が示す final moraic -t を、現代 kana の っ / ッ や full-sized hentaigana TU-2 と区別して表す必要があると説明する。

根拠として、1980 年の邦訳日葡辞書が Entering Tone -t を促音 ㇷ と区別するため、U+1B06A HENTAIGANA LETTER TU-2 に対応する小書き形を使ったことを挙げる。提案文字は HENTAIGANA LETTER TU-2 の小書き版として位置づけられ、Kana Extended-A への追加が提案されている。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字 | HENTAIGANA LETTER SMALL TU-2。 |
| 提案 block | Kana Extended-A。 |
| 用途 | Late Middle Japanese の final -t を正確に転写する academic / dictionary use。 |
| 区別点 | full-sized U+1B06A は /tu/ を表し、っ / ッ は geminating /Q/ を表すため、final -t とは機能が異なる。 |

## 論点

### transcription 用 small hentaigana

この proposal は、small kana を現代音写の小書き文字ではなく、歴史音韻の精密転写として使う例である。small kana repertoire 論点の中でも、近代資料の実用字種ではなく、学術的 transcription convention の符号化として扱う必要がある。

### hiragana / katakana のどちらでも使う文字

文書は、提案文字が hiragana / katakana のいずれかに限定されず、両方とともに使えると説明する。この点は、Kana Extended-A に置く文字が script style にどう対応するかという model 上の論点を持つ。
