# IRG N2909

## 要約

`IRG N2909` は、2026-03-16/20 の IRG Meeting \#66 recommendations and action items である。WS2024 Version 4.0 review 後の Version 5.0 schedule、China horizontal extension、G / T / K / UK / V source reference and glyph issues、`kIRG_SGSource`、CJKV Components、FS / SC guidelines、radical assignment rules を次工程へ送っている。

この wiki では、\#66 で採択された recommendation のうち、\#67 agenda や WG2 \#73 へ直接つながる項目を追跡する入口として扱う。出来事として独立した項目は event page へ分ける。

## 決定・Action Item

| Recommendation | 内容 | 後続 |
| --- | --- | --- |
| M66.04 | Updated UCV / NUCV examples を provisionally accept し、editor に `IRG N2931` の準備を求めた。 | [UCV and NUCV Lists](../topics/ucv-nucv-lists.md) |
| M66.07 | WS2024 Version 5.0 の production schedule を決定。`IRG N2932` release、`IRG N2933` consolidated comments、submitter response を経て Meeting \#67 review へ進める。 | [IRG Working Set 2024](../topics/irg-working-set-2024.md)、[IRG Meeting \#67](../meetings/irg/irg-meeting-67.md) |
| M66.09 | China の 9156-ideograph horizontal extension `IRG N2929` を review し、revised `IRG N2929R` を Meeting \#67 で扱う方針を置いた。 | [CJK Horizontal Extensions](../topics/cjk-horizontal-extensions.md) |
| M66.10-M66.15 | G / T / K / UK / V source references、representative glyphs、Unihan property changes を個別に整理した。M66.15 は [IRG N2928](irg-n2928.md) に基づき、U+268A1 / U+268A2 の V-source glyph change は追加 evidence があれば Vietnam が提案する、とした。 | [IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md)、[V-source Representative Glyph Issues](../topics/v-source-representative-glyph-issues.md) |

| M66.17 | `IRG N2878R3` の CJK Unified Ideographs Components-A / B を WG2 へ forward することを勧告した。 | [IRG M66.17 CJKV Components forward to WG2](../events/irg-m66-17-cjkv-components-forward-to-wg2.md) |
| M66.18 | [IRG N2862R2](irg-n2862r2.md) を FS / SC values の consolidated guideline として受け入れ、final [IRG N2951](irg-n2951.md) の作成を求めた。 | [IRG Indexing Rules](../topics/irg-indexing-rules.md) |
| M66.19 | `IRG N2925` をもとに condensed rules を含む `IRG N2952` "Rules for Assigning Radicals" の作成を求めた。 | [IRG Indexing Rules](../topics/irg-indexing-rules.md) |

## 後続決定

Meeting \#67 agenda `IRG N2935` は、M66.16 `kIRG_SGSource` と M66.17 CJKV Components が WG2 \#73 / SC2 acceptance へ進んだ current status を記録している。source data / glyph issues の一部は UTC Meeting \#187 と Unicode Version 18.0 Beta review に反映された一方、G / UK / V / SAT source issues などは \#67 で継続確認が必要である。

## 論点

### Recommendation と後続 status の分離

`IRG N2909` は IRG \#66 時点の recommendation であり、WG2 / SC2 acceptance や UTC \#187 acceptance そのものは後続文書で確認する必要がある。topic page では、IRG \#66 の決定、WG2 \#73 の勧告、Unicode Version 18.0 Beta review の反映状況を body と日付で分けて読む。

### Meeting \#67 の入口

`IRG N2935` の Section 4 は `IRG N2909` の recommendation / action item follow-up で構成されている。\#66 recommendation を読むときは、\#67 agenda 側の current status と未解決項目を合わせて確認する。
