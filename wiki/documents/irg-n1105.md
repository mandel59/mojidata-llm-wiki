# IRG N1105

## 要約

`IRG N1105` は、`IRG N954AR` に対する first stroke / stroke count / radical-component counting method の amendments である。文書中では、Macao SAR の Committee for Unification of Computer Chinese Character Encoding が、Kangxi Dictionary の stroke count と HKSAR、Japan、R.O.Korea、TCA からの suggestions に基づき amendment を提出したと説明している。

この文書は、後続の stroke counting guideline 文書である [IRG N2171](irg-n2171.md)、[IRG N2221](irg-n2221.md)、[IRG N2862R2](irg-n2862r2.md)、[IRG N2951](irg-n2951.md) から predecessor として参照される。

## フィードバック内容

| stroke count | stroke count values への amendment を示す。 |
| radicals / components | radicals or components の stroke count method について、original items と変更案を並べる。 |

## 後続決定

この文書自体は IRG \#23 での group discussion 用 amendment であり、後続の [IRG N2221](irg-n2221.md) は `IRG N954AR`、N1105、[IRG N2171](irg-n2171.md) の SC entries を revisions 付きで consolidate した。さらに [IRG N2862R2](irg-n2862r2.md) と [IRG N2951](irg-n2951.md) は、これら旧 document 群を置き換える consolidated FS / SC guideline として扱われる。

## 論点

### first stroke と stroke count の分離

N1105 は、first stroke と stroke count を別々の amendment category として扱う。後続の FS / SC guideline では、この二つを entry syntax の中で組み合わせて扱うため、N1105 は後続 consolidated table の古い入力文書として読む。
