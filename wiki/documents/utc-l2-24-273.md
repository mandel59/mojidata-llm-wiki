# L2/24-273

## 要約

`L2/24-273` は、Initial Teaching Alphabet (ITA) の専用小文字13字を Latin Extended-G に追加する proposal である。

ITA は1960年代の英国で初期読字教育に使われ、Australia と United States にも広がった補助 Latin alphabet である。文書は現在も資料や ITA Foundation の活動で用例を確認できるとして、既存文字や font-level glyph variant では表せない字だけを符号化対象とする。

提案は `L2/08-428`、`L2/22-286`、`L2/23-102R` を引き継ぐ。以前のdraftに含まれた casing forms は Script Encoding Working Group (SEW) の助言に従って除外され、小文字のみを求めている。

## 提案内容

提案対象は U+1DF5A..U+1DF66 を想定した次の13字で、いずれも General_Category `Ll`、Bidi_Class `L`、decomposition なしとされる。

| code point | proposed name |
| --- | --- |
| U+1DF5A | LATIN SMALL LETTER HALF A |
| U+1DF5B | LATIN SMALL LETTER STRETCHED C-H DIGRAPH |
| U+1DF5C | LATIN SMALL LETTER LUNATE EE |
| U+1DF5D | LATIN SMALL LETTER IE |
| U+1DF5E | LATIN SMALL LETTER OI DIGRAPH |
| U+1DF5F | LATIN SMALL LETTER OU DIGRAPH |
| U+1DF60 | LATIN SMALL LETTER OMEGA WITH LOOP |
| U+1DF61 | LATIN SMALL LETTER R WITH LEFT TIE |
| U+1DF62 | LATIN SMALL LETTER ESH-H DIGRAPH |
| U+1DF63 | LATIN SMALL LETTER STRETCHED T-H DIGRAPH |
| U+1DF64 | LATIN SMALL LETTER REVERSED T-H DIGRAPH |
| U+1DF65 | LATIN SMALL LETTER WH DIGRAPH |
| U+1DF66 | LATIN SMALL LETTER REVERSED Z |

文書は ITA の全字形を別 character にするのではなく、既存 Unicode character の通常形または glyph variant として扱えるものを区別する。たとえば ITA の `d`、`g`、`eng`、`ae`、`oe` などの形状差は font で処理し、omega は Greek ではなく Latin の U+A7B7 を使う。ITA 独自の大文字は近年使われず、符号化対象から外している。

## 後続決定

このページで確認した範囲では、提案の採択を記録する UTC minutes または recommendation は未確認である。

後続の [L2/26-065](utc-l2-26-065.md) は、近年の ITA materials で旧来の U+0292 に代わって使われる zh ligature を追加提案しており、`L2/24-273` の repertoire を補う関係にある。これは別 proposal であり、`L2/24-273` 自体への採択決定ではない。

## 論点

### character と glyph variant の境界

ITA typeface では既存 Latin letters に特徴的な字形を与えるが、文書は複数の ITA publications と ITA Foundation の見解を根拠に、通常の Unicode character を font-level allograph で表示できるケースと、独立符号化が必要な13字を分離する。

### casing の扱い

初期の ITA には小文字を拡大した capital forms も見られる一方、近年の教材では capitals が使われない。proposal は SEW の助言を反映して casing forms を要求せず、歴史資料に見える大文字形の符号化は対象外としている。
