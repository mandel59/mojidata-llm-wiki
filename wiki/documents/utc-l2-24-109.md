# L2/24-109

## 要約

`L2/24-109` は、Oreen Yousuf / Daniel Yacob が Oromo 用の Sheek Bakrii Saphaloo script を UCS に符号化するために提出した初期 proposal である。文書は、Sheikh Bakri Sapalo が 1956 年に Oromo 用 abugida を作り、東部 Oromia / Dire Dawa 周辺で歴史的資料と現代教育の両方に残っていると説明する。

提案時点の script name は `SHEEK BAKRII SAPHALOO` で、33 base glyphs、vocalized / pure consonant forms、geminated set、punctuation、digits を含む 804 unique graphemes の符号化を求める。後続の [L2/26-040R2](utc-l2-26-040r2.md) は、この proposal を community feedback に基づいて `Shaaldaa` として改訂した。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script name | `SHEEK BAKRII SAPHALOO`。Oromo community では `Qubee Sheek Bakrii Saphaloo` とも呼ばれるとする。 |
| writing direction | left-to-right の horizontal lines。 |
| structure | 33 base glyphs に、10 vocalized forms と 1 pure consonant formを持たせ、同じ構造の geminated set を持つ。 |
| repertoire | 792 syllabic graphemes、2 punctuation marks、10 digits の合計 804 characters。 |
| punctuation / digits | script-specific word separator、full stop、digits 0-9 を提案する。 |
| encoding model | diacritics は使わず、Ethiopic に近い precomposed syllabic model を採る。 |

## 後続決定

[L2/26-040R2](utc-l2-26-040r2.md) は `L2/24-109` の revised proposal であり、script name を Shaaldaa に変更し、digit category、glyph variation evidence、numeric values、追加 figure などを更新した。[L2/26-100](utc-l2-26-100.md) は Shaaldaa / Cossic name changes を SEW recommendation の処理対象に含める。

## 論点

### 初期 proposal と revised proposal の差分

`L2/24-109` は初期 proposal として 804 characters を提案する。一方、[L2/26-040R2](utc-l2-26-040r2.md) は community feedback 後の名称と repertoire を反映しており、Shaaldaa topic では後者を現行 proposal として読む。

### Ethiopic 型の text model

文書は、vocalized graphemes を regular diacritics として分解しにくいこと、user community が distinct letters として把握していること、Ethiopic script と並行して使われる地域的文脈を理由に、Indic encoding model ではなく Ethiopic-style model が必要だと主張する。

### current use と preservation

提案は、Hayward and Hassen 1981 の資料、Sheikh Bakri Sapalo の manuscript、Dire Dawa での教育、現代の font / video / teaching material を根拠に、historical preservation だけでなく現在の user community の利用にも応える必要があるとする。
