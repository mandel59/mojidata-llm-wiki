# L2/26-114

## 要約

`L2/26-114` は Devanagari Extended-A block に U+11B0D DEVANAGARI VOWEL LENGTH MARK を追加する proposal である。Sunuwar / Wambule で vowel length を表す mark を対象にし、Sunuwar の教材や Wambule grammar の examples を evidence として挙げる。

提案文字は Devanagari text で vowel prolongation を表す spacing vowel-dependent mark として扱われる。文書は、Sunuwar が Devanagari と Sunuwar script の両方で書かれ、Nepal の教材でこの mark が使われることを理由にする。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| repertoire | Devanagari Extended-A に 1 character を追加する。 |
| proposed code point | U+11B0D DEVANAGARI VOWEL LENGTH MARK。 |
| usage | Sunuwar / Wambule の vowel length を Devanagari orthography で表す。 |
| evidence | Sunuwar textbooks、Sunuwar-Nepali-English dictionary、Wambule grammar。 |
| proposed properties | `Mc`、Indic_Syllabic_Category=Vowel_Dependent、Line_Break=AL を提案。 |

## 論点

### Devanagari text における vowel length mark

この proposal は Sunuwar script 自体の符号化ではなく、Devanagari orthography で Sunuwar / Wambule を書くための vowel length mark を追加する提案である。既存 Devanagari marks との区別、spacing behavior、Indic property assignment、collation position が review point になる。
