# L2/26-095

## 要約

`L2/26-095` は、2025-12-30 から 2026-03-31 まで public feedback を受け付けていた Unicode Public Review Issues の一覧である。

対象には UTS \#58 Linkification、UTR \#59 East Asian Spacing、Draft UAX \#60、UTS \#61、UTR \#25、UAX \#38、UAX \#31、Unicode 18.0 Alpha Review、Emoji 18.0 Alpha Repertoire、UAX \#57 Unikemet、UAX \#53 Arabic Mark Rendering、UTC \#187 general feedback、UTS \#37 が含まれる。

文書自体は各 PRI と feedback page への入口であり、個別仕様の変更内容は対応する proposed update / draft 文書で読む必要がある。

## データ内容

`L2/26-095` は、PRI number、issue name、feedback link を表で示す。Unicode 18.0 に向けた public review のうち、properties / algorithms、data files、emoji、release review が同じ期間に並行していたことを確認できる。

## 後続決定

UTC \#187 の [PAG report](utc-l2-26-096.md) と [meeting minutes](utc-l2-26-093.md) が、PRI feedback のうち Unicode 18.0 beta に入れる properties / algorithms changes を処理する。Emoji 18.0 や release coordination の論点は、ESR / RMG / Editorial / Charts reports に分散して記録される。

## 論点

### public review を読む入口

`L2/26-095` は仕様変更そのものではなく、review issue と feedback の目録である。個別の変更内容を追うときは、PRI page、feedback page、対応する L2 proposed update、UTC report / minutes を分けて読む必要がある。

### Unicode 18.0 の横断 review

同じ review window に UAX / UTS / UTR、UCD data、emoji、alpha review が含まれているため、Unicode 18.0 の release preparation では、文字追加だけでなく algorithm / data / documentation の public feedback がまとめて処理されたことが分かる。
