# L2/26-086

## 要約

`L2/26-086` は、David Corbett の suggestions をもとに Roozbeh Pournader が編集・更新した mid-priority confusables data の source / data listings である。PDF 本文の日付は 2026-03-09、UTC registry 上の日付は 2026-03-27 である。

文書は CJK 専用の review request ではなく、Properties and Algorithms Working Group が Unicode data files を更新する際の検討資料である。ただし Katakana、Enclosed CJK Letters and Months、CJK Compatibility、Small Form Variants、Halfwidth and Fullwidth Forms など、CJK 周辺の文字・互換記号を多数含むため、CJK Security Confusables の周辺文書として追跡する価値がある。

## 決定・Action Item

| 区分 | 内容 |
| --- | --- |
| 性質 | confusables data 更新候補の長い source / data listing。 |
| 対象 WG | Properties and Algorithms Working Group。CJK Working Group への review request ではない。 |
| CJK との関係 | Katakana、CJK Compatibility、Enclosed CJK Letters and Months、fullwidth / halfwidth forms などを含む。 |

## 論点

### CJK 専用 review sequence との違い

`L2/26-082` と `L2/26-127` は CJK Working Group に専門 review を求める文書だが、`L2/26-086` は PAWG 向けの broader data listing である。したがって CJK ideograph の source separation や variant 判断を直接扱う中心文書ではなく、UTS #39 confusables data 更新全体の周辺資料として扱う。

### CJK 周辺文字の範囲

文書には `U+30DB` / `U+6729` のような Katakana と CJK ideograph の pair、Enclosed CJK Letters and Months、CJK Compatibility、Small Form Variants、Halfwidth and Fullwidth Forms の記号・単位・互換形が含まれる。これらは CJK ideographs の専門 review とは性質が異なるが、CJK 圏の文字・記号が security data にどう現れるかを確認する入口になる。
