# IRG N2367

## 要約

`IRG N2367` は、IRG Meeting \#52 に提出された Japan NB の activity report である。JIS X 0221 を ISO/IEC 10646 5th edition と two amendments に追従させる revision project、2019-05-01 に始まった新元号「令和」の square symbol 標準化対応、IRG Working Set review への今後の参加方針を報告している。

後続の [IRG N2721](irg-n2721.md) / `WG2 N5284` は、この文書の「human resource 不足により future IRG Working Set review には参加せず、ISO/IEC SC2 balloting phase で CJK Unified Ideograph extension proposals を review し続ける」という立場を、Japan に IRG active member body への復帰を求める前提として引用した。

## 報告内容

| 項目 | 内容 |
| --- | --- |
| JIS X 0221 revision | ISO/IEC 10646 5th edition と two amendments に追従するため、JIS X 0221 revision project を開始した。draft は 2019 年末までの完成予定とした。 |
| Japanese new era | 2019-05-01 に始まった新元号「令和」について、U+32FF に reserved されていた square symbol を ISO/IEC 10646 / Unicode で標準化するため、Japan NB が SC2 chair と UTC chair に必要な対応を求めた。 |
| IRG Working Set review | Japan NB は human resource 不足を理由に、future IRG Working Set review には参加しないと報告した。一方で、ISO/IEC SC2 standardization process の balloting phase では CJK Unified Ideograph extension proposal の review を継続するとした。 |

## 後続決定

[IRG N2721](irg-n2721.md) は、Japan が 2019 年時点で IRG Working Set review から離れる方針を示したことを起点に、2024 年の大規模 JMJ horizontal extension request や Extension G / H ballot comments を踏まえ、Japan の IRG active participation 復帰を求めた。

WG2 Meeting \#72 minutes [WG2 N5301](wg2-n5301.md) は、Japan representative が復帰を検討できるが human resources / workload が課題だと述べたことを記録している。したがって `IRG N2367` は、Japan NB の review participation が 2019 年の resource constraint から 2024-2025 年の復帰要請へ接続する前史として読む。

## 論点

### IRG review と ballot phase review の分離

`IRG N2367` は、IRG Working Set review には参加しないが、SC2 balloting phase では CJK Unified Ideograph extension proposals を review し続けるという分担を示した。この分担は、後に [IRG N2721](irg-n2721.md) が問題視する「late comments が IRG work を disrupt し得る」という論点につながる。

### Japan NB の source owner role

Japan は `J-source` / `JMJ-source` の source owner として、source data、representative glyph、large horizontal extension、post-encoding correction の判断を求められる。`IRG N2367` の resource constraint は、この role と IRG review workload の緊張関係を示す。
