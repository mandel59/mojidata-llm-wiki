# L2/26-102

## 要約

`L2/26-102` は UTC Release Management Group が UTC \#187 に提出した report で、Unicode 18.0 release development、beta review、ISO/IEC 10646 7th edition との同期上の変更を扱う。RMG は、UTC \#187 で beta review 開始を authorization し、2026-05-26 に beta review を始め、2026-07-07 に public review を閉じ、UTC \#188 で 18.0 content を finalize する schedule を示した。

後続の [PRI \#548](pri-548.md) は、この schedule 上の Unicode 18.0.0 Beta public review を実際に閉じた issue であり、UTC \#188 に向けた substantive feedback deadline を示す。

この文書は、正式な変更点一覧そのものではなく、release 管理上どの資料がいつ公開されるか、beta に入れるべき調整が何かを示す資料である。Unicode 18.0 の変更点を調べる入口としては、Unicode 18.0 draft release page と beta review page を併用する必要がある。

UTC \#187 では、この RMG report に加えて Editorial WG、Charts WG、PAG、SEW、ESR、ICU4X / TC39 liaison reports が beta review の根拠資料になる。publication artifact と release process の観点は [Unicode Release Coordination and Publication](../topics/unicode-release-coordination-and-publication.md) に分けて読む。

| UAX \#60 | Title を “Data for Non Han Ideographic Scripts” から “Data for Large East Asian Scripts” に変える action を置いた。 |

## 論点

### release page と RMG schedule の日付差

`L2/26-102` は 2026-04-15 時点の RMG report であり、release date を 2026-09-15 としていた。一方、2026-07-07 に確認した Unicode 18.0 beta page は planned release date を 2026-09-16 としている。日付の最終確認には、常に Unicode 18.0 draft release page / beta page を優先する。

### beta review の範囲

RMG report は beta deliverables を core specification draft、code charts、data files、PRI cover page、blog post としている。これは、変更点を調べる際に proposal documents だけでは不十分で、UCD、delta charts、UAX / UTS proposed updates を直接確認すべきことを意味する。

### ISO/IEC 10646 との同期リスク

Unicode 18.0 は ISO/IEC 10646 7th edition と同期する必要があるが、RMG report の時点では 10646 側が committee draft stage にあり、DIS ballot は Unicode 18.0 release より後まで続く可能性があった。したがって、Unicode 18.0 beta content は WG2 / SC2 の June 2026 discussion と密接に接続する。
