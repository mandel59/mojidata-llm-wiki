# L2/26-119

## 要約

`L2/26-119` は、U+06C4 ARABIC LETTER WAW WITH RING WITHIN の representative glyph を Kashmiri usage に合わせて修正する提案である。Ayaan Farooq は、既存の code chart glyph が ring を waw の body 上に置いているのに対し、Kashmiri 資料では ring が terminus / tail stroke 側に置かれると主張する。

提案は code point、character name、property の変更ではなく、chart glyph の correction を求める。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| target character | U+06C4 ARABIC LETTER WAW WITH RING WITHIN |
| requested change | representative glyph の ring 位置を Kashmiri 資料に合う tail / terminus 側へ直す。 |
| evidence | Kashmiri dictionary、school textbook、official translation、newspaper などの printed / official sources。 |
| scope | encoding 追加ではなく glyph correction。 |

## 後続決定

[L2/26-120](utc-l2-26-120.md) は、この glyph correction request に対する response として、Kashmiri evidence を補強しつつ U+06C4 / U+06C5 の区別も整理する。[L2/26-100](utc-l2-26-100.md) は U+06C4 glyph correction、confusables、cross-reference update を SEW recommendation の一部として扱う。

## 論点

### representative glyph correction と character identity

`L2/26-119` の論点は、既存文字の identity を変えずに代表字形を直すことである。Arabic-script の似た字形や confusables data と連動するため、code chart correction だけでなく security / cross-reference 側の影響も確認が必要になる。
