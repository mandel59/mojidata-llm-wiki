# L2/20-152

## 要約

`L2/20-152` は、[L2/19-381](utc-l2-19-381.md) で提案された HIRAGANA LETTER ARCHAIC WU の base / origin character に関する短い feedback である。UTC \#162 では base character が U+7D06 である証拠を求めたが、それは encoding を止める条件ではなく、annotation として後で追加できるとされていた。

文書は、Jake Iannacone との discussion を経て、U+1B11F の origin character は U+6C59 汙だと結論し、annotation に `derived from 6C59 汙` を追加することを推奨する。

## 論点

### encoding と annotation の分離

この文書は、base character の証拠確認を encoding の blocker にせず、annotation で補う扱いを示す。missing kana の proposal では、文字そのものの需要と、母字・由来に関する注記を分けて追跡する必要がある。
