# WG2 N4755

## 要約

`WG2 N4755` は、TCA が [WG2 N4716](wg2-n4716.md) の source-based variation selector 案に反対した feedback である。TCA が招集した文字学研究者らの会合結果として、Shuowen Small Seal は古文字研究の基盤となる確立した文字集合であり、source variation としてではなく独立に符号化すべきだと主張する。

文書は、版本間の多くの差を筆写に由来する微細な glyph 差とみなし、それらすべてを別々に符号化する必要はないとする。一方、歴史資料を後から一貫した component system に作り替えることも避け、まず Tenghuaxie version (`THX`) を符号化し、その後に他版本固有の文字を追加する二段階方式を求める。

## フィードバック内容

### 独立符号化を求める理由

文書は `Shuowen Jiezi` を単なる辞書ではなく、Qin dynasty の Small Seal を Xu Shen が体系化し、後世の校訂を経た古文字研究の基盤と位置付ける。540 radicals が後世の部首体系や古文字分類の慣行につながるため、Small Seal を source-based variation selector のみによって表現する案には同意しない。

### 版本差の扱い

版本間の差の多くは、stroke の長短、傾き、交差・接触など筆写上の小差であり、構造の理解を妨げないので全 glyph を個別に符号化する必要はないとする。ただし、史料内の component variation は実際の伝承を反映するため、単一の内部整合的な glyph system に再編すべきでもないと論じる。

### `THX` を最初の source とする理由

Tang dynasty manuscript fragments は約200字に限られ、現存する Song dynasty 系版本にも欠損や粗い筆写があると説明する。比較的良好な `Jiguge`、`THX`、`Pingjingguan` (`PJG`) のうち、`THX` の glyph が明瞭であるため最初の符号化 source に選び、後に Chen Changzhi version、Duan Yucai (`DYC`)、`PJG`、XiaoXu version などとの差を拡張する案を示す。

文書末尾の要求は次の二点である。

1. Small Seal を source-based variation selector ではなく独立に符号化する。
2. 第1段階で `THX` の glyph を符号化し、第2段階で他版本にある `THX` と異なる文字を追加する。

## 後続決定

この文書は TCA の feedback であり、WG2 の決定そのものではない。直後の `WG2 N4761` は N4755 への応答、`WG2 N4855` は Small Seal 符号化への追加コメントとして登録されている。

2019 年の [WG2 N5108](wg2-n5108.md) は WG2 \#68 の discussion として、Zhuanwen / Guwen / Zhouwen の基本的な disunification、非常に似た glyph の unification、他版本と taxonomy の追加検討を報告した。後続作業は単純な `THX` 単独集合または source-based VS の二者択一には収束せず、[WG2 N5307R](wg2-n5307r.md) の mapping principles と [WG2 N5344R2](wg2-n5344r2.md) の four-column source model へ進んだ。

## 論点

N4755 は、character identity を版本から切り離して variation sequence で表す N4716 に対し、Small Seal 自体の独立した符号化価値と歴史資料への忠実さを優先する。ただし、微細な版本差をすべて別 character とするのではなく、最初の source を選んだ上で段階的に他版本を取り込むため、unification boundary と source representation の判断は後続作業に残される。
