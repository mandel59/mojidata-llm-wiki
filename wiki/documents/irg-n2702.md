# IRG N2702

## 要約

`IRG N2702` は、2024-10-21/25 に Seoul, ROK で開催された IRG Meeting \#63 の recommendations and action items である。この wiki では特に、CJK components encoding を始動した M63.17、Jianzi Musical Notation と components block を接続した M63.18、script-hybrid Han ideographs を Meeting \#64 へ継続した M63.19 を参照する。

Meeting \#63 は、新しい IRG home page / document register への移行、UCV / NUCV lists、WS2024 Version 2.0、horizontal extension、source representative glyphs も扱っており、2025 年以降の IRG work items の起点として読む必要がある。

## 決定・Action Item

| Recommendation | 内容 | 後続 |
| --- | --- | --- |
| M63.03-M63.04 | UCV / NUCV examples を確認・更新し、IWDS Editor に次版 list の準備を求めた。 | [UCV and NUCV Lists](../topics/ucv-nucv-lists.md) |
| M63.06 | WS2024 Version 1.0 review 後、Version 2.0 の production / ORT review schedule を決めた。 | [IRG Working Set 2024](../topics/irg-working-set-2024.md) |
| M63.10-M63.12 | Japan horizontal extension issues、K-source / G-source / V-source representative glyph updates を扱った。 | [CJK Horizontal Extensions](../topics/cjk-horizontal-extensions.md)、[IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md) |
| M63.17 | CJK components block の符号化は useful かつ long overdue とし、TCA と HKSAR に追加 component data の検討・提供を促した。 | [CJKV Components](../topics/cjkv-components.md) |
| M63.18 | Jianzi Musical Notation について、必要な ideographs は UNC proposal とし、six CJK components と 119 fingering letters は CJK components block proposal に merge する方向を置いた。 | [CJKV Components](../topics/cjkv-components.md) |
| M63.19 | script-hybrid Han ideographs の受け入れ可否を Meeting \#64 の agenda に継続した。 | [CJK Hybrid Characters](../topics/cjk-hybrid-characters.md) |

## 後続決定

[IRG N2799](irg-n2799.md) は、M63.17 の方針を受けて `IRG N2733R` の China source 中心の preliminary proposal を HKSAR、TCA、Vietnam、public IDS database、Jianzi Musical Notation 由来 data を含む CJKV proposal へ拡張した。

[IRG N2765](irg-n2765.md) の M64.19 は `IRG N2799` の継続開発を encourage し、TCA の CMEX CJK Components Review Tool を comments / new components の入力先として紹介した。後続の [IRG N2826](irg-n2826.md) M65.23 と [IRG N2909](irg-n2909.md) M66.17 は、updated proposal review と WG2 forwarding へ進む。

## 論点

### CJK components の立ち上げ

M63.17 は `IRG N2733R` を単に China source の preliminary proposal として受け止めるのではなく、TCA / HKSAR などの component data を加えて CJKV 全体の proposal へ広げる方向を置いた。IRG Convenor は UTC CJK & Unihan Working Group chair として UTC Meeting \#181 でも feedback を得ることになっており、IRG 内だけで閉じない review path が早い段階で設定された。

### Jianzi Musical Notation との接続

M63.18 は Jianzi Musical Notation の architecture 議論と CJK components block を接続する。Jianzi に必要な ideographs は UNC proposal とし、components / fingering letters は CJK components proposal へ merge する方向を示したため、後続の CJKV Components proposal は単なる IDS component list ではなく、音楽記譜由来の component data も含む形に広がった。

### script-hybrid Han ideographs との分岐

M63.19 は script-hybrid Han ideographs を Meeting \#64 へ継続した。CJK components は Han component collection として進む一方、Latin / Kana / Hangul / Bopomofo など非 Han script component を含む Han-like characters は別論点として分岐していく。
