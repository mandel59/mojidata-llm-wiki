# L2/26-080

## 要約

`L2/26-080` は、1 ideograph を UAX \#45 に追加する提案である。PDF の font encoding の都合で text extraction では glyph / IDS が完全には復元できないが、proposed-character table では `kMandarin guǐ`、related character 癸、first stroke `5`、total strokes `10`、`kRSUnicode` value `45.7`、evidence Fig. 1-4 が示される。

Evidence は `玉篇`、`汉字海`、`集韻`、MOE `Dictionary of Variants` 由来の page / entry を含む。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | 癸 related character として示された 1 ideograph を UAX \#45 `FutureWS` record として追加する。 |
| Data | `kMandarin guǐ`、related character 癸、FS `5`、TS `10`、RS `45.7`。 |
| Extraction note | glyph / IDS は PDF text extraction では崩れるため、正確な glyph shape は source PDF の proposed-character table と添付 font / metadata で確認する。 |
| Evidence | `玉篇`、`汉字海`、`集韻`、MOE `Dictionary of Variants`。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、`L2/26-080` を含む 2026 年 UAX \#45 additions を `FutureWS` records として受け入れるよう勧告した。採択・data update 側の summary は [UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) に集約する。

## 論点

### source PDF を一次確認する必要

この文書は PDF text extraction が glyph / IDS を破壊するため、wiki では extracted text だけで IDS を断定しない。UAX \#45 の実 data update を確認する場合は、source PDF の table と `USourceData.txt` / `USourceGlyphs.pdf` の該当 record を確認する必要がある。
