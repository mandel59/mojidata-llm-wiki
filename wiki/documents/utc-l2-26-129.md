# L2/26-129

## 要約

`L2/26-129` は Leke script を Unicode に符号化する proposal である。Leke は Pwo Eastern Karen を書く abugida で、Chicken Scratch script とも呼ばれ、Myanmar / Thailand の Eastern Pwo communities で継続的に使われている。

提案は 55 characters を扱い、consonants、medial consonants、vowel signs、final consonant signs、tone marks、digits、punctuation を含む。2013 年の [L2/13-116](utc-l2-13-116.md) を拡張した complete proposal と位置づけられる。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script model | Leke は abugida で、consonant に vowel signs、final consonant signs、tone marks、medial signs を組み合わせる。 |
| repertoire | 55 characters。提案 range は U+11B80 以降で、letters、signs、digits、punctuation を含む。 |
| tone marks | Eastern Pwo の 4 tones のうち mid tone は unmarked、他の tones は複数の marks で表す。 |
| community | Ariya millenarian buddhist sect を中心に使われるが、Eastern Pwo communities でも教育・使用が続くと説明する。 |

## 後続決定

2026-07-08 時点では、UTC \#188 agenda / minutes は未掲載であり、この proposal への UTC 決定は未確認である。Script Encoding WG の次回 recommendations で、script model、font、property values、naming が検討対象になると推定される。

[WG2 N5361R](wg2-n5361r.md) と [WG2 N5362](wg2-n5362.md) は、Leke を ISO/IEC 10646 7th edition Amendment 1 の starting repertoire / future repertoire candidate として扱う。[WG2 N5354](wg2-n5354.md) の M73.05 以降は Amendment 1 project subdivision と first amendment candidate groups を勧告しているが、これは final encoding ではない。

## 論点

### complete proposal と community use

文書は Leke を現在使用される script として扱い、complete proposal の form を備える。UTC 側での論点は、repertoire completeness、characters と stylistic variants の分離、marks の category / combining behavior、script property assignment になる。
