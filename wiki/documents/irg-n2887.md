# IRG N2887

## 要約

`IRG N2887` は、IRG Working Set 2024 Version 4.0 の main set chart / list である。候補ごとの serial number、attributes、member / source columns、discussion record をまとめた data snapshot であり、提案理由や会合決定を記す narrative document ではない。

この版は Version 4.0 review cycle の基準データとなり、[IRG N2888](https://www.unicode.org/irg/docs/n2888-WS2024v4ConsolidatedComments.pdf) の consolidated comments とともに IRG Meeting \#66 の CJK Editorial Group review へ入力された。review conclusions は [IRG N2910](irg-n2910.md)、IRG の recommendation と production schedule は [IRG N2909](irg-n2909.md)、反映後の次版は [IRG N2932](irg-n2932.md) に分かれている。

## データ内容

| 区分 | 内容 |
| --- | --- |

| 性質 | review 時点の候補、source data、attributes、discussion record を固定した chart / list。 |
| review での位置 | Version 4.0 comments、submitter responses、Meeting \#66 editorial review の参照基準。 |
| 含まないもの | Meeting \#66 における editorial conclusion、IRG recommendation、Version 5.0 production schedule。 |

## 後続決定

[IRG N2910](irg-n2910.md) は、`IRG N2887` と `IRG N2888` を入力として候補ごとの comments と submitter responses を検討し、Version 5.0 に反映する editorial conclusions をまとめた。

[IRG N2909](irg-n2909.md) の M66.07 は、その review を踏まえて Version 5.0 の作成と review schedule を勧告した。Version 4.0 の一覧自体が採否や修正を決定したのではなく、会合としての承認と action items は `IRG N2909` に帰属する。成果物である [IRG N2932](irg-n2932.md) は Version 5.0 の新しい data snapshot である。

## 論点

### data snapshot と decision record の境界

`IRG N2887` は Version 4.0 の状態を再現する一次資料として使う。個別候補について Version 5.0 に何を反映するかは `IRG N2910`、その反映を IRG がどの schedule で進めるかは `IRG N2909` を一次ソースとする。後続版 `IRG N2932` の内容を遡って Version 4.0 の決定として扱わない。
