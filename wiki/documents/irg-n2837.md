# IRG N2837

## 要約

`IRG N2837` は、CJK Extensions C / E にある 82 GXC-source characters の source reference values を整理する提案である。現行値の数値部分は元の `XC\d{6}` serial values を変形したもので、辞書の page / position として一貫して解釈できないと指摘する。

提案は《现代汉语词典》第5版を基準に、75 characters を `GXC-xxxx.yy`（page number と同一page上のhead character position）へ更新する。残る7 charactersはhead characterとして索引できないため、6 charactersをGHZR-source、1 characterをGZH-sourceへ変更する。これは個人提案であり、China NB とIRGによる検討を求めた文書である。

## 提案内容

GXC-sourceは《现代汉语词典》を表すが、文書によれば対象86 charactersの参照値は同じsyntaxでも由来が二つある。Extension Dの4 charactersは[IRG N1532R]由来で、第5版のpage numberとhead character positionを表す。一方、Extensions C / Eの82 charactersは[IRG N891]の`XC\d{6}` serial valuesから現在の形式へ変更されており、先頭4桁が辞書の総page数を超える例もある。

この不整合を解消するため、次の更新を提案する。

| 提案source | 件数 | 根拠 |
| --- | ---: | --- |
| GXC | 75 | 《现代汉语词典》第5版のhead characterを、page numberとpage内positionで索引する。 |
| GHZR | 6 | 第5版でhead characterとして索引できない文字を《汉语大字典》第2版の参照へ移す。 |
| GZH | 1 | 同様に索引できない文字を《中华字海》の参照へ移す。 |

本文のTable 2が82 charactersのcurrent / proposed source valuesを列挙する。catalogに関連リンクとして登録されたevidence ZIPは、各characterの辞書画像など、表の根拠資料を収める添付物であり、proposal本文とは別の資料である。

## 後続決定

`IRG N2868` はこの提案へのfeedbackとして、Extensions C / Eのoriginal submissionは2000年であり、2005年刊行の《现代汉语词典》第5版をその由来の基準にするのは時系列上適切でないと論じた。同文書は、当時「修订本」とされた第3版を基準にsource indexを再検討する案を示している。したがって、`IRG N2837` の第5版に基づく値は、そのまま確定した変更として扱えない。

[IRG N2923R](irg-n2923r.md) はこのfeedback chainを踏まえ、GXCを含む複数のG-source referencesをoriginal proposal時のedition / printingに基づいて再設計した後続proposalである。同文書はGXC 83 charactersを対象に《现代汉语词典》第3版をより適切なreferenceとして扱っており、`IRG N2837` の82-character proposalを置き換える方向の再検討と位置づけられる。ただしIRG Meeting \#66ではbulk reference changesは採択されず、further discussionとなった。

## 論点

### reference valueの意味

本提案は、見かけ上同じ`GXC-xxxx.yy` syntaxに、serial value由来とpage / position由来の値が混在する問題を可視化した。source referenceを機械的に利用するには、suffixが何を指すかをsourceごとに一貫させる必要がある。

### 現行辞書による再索引とoriginal submission history

`IRG N2837` は第5版で82 charactersを再索引する実務的な方法を示したが、`IRG N2868` はoriginal submission時に存在したeditionとの対応を重視した。後続議論では、現在検証しやすい証拠と、符号化提案の来歴を再現するidentifierのどちらを優先するかが論点になった。
