# L2/25-274

## 要約

`L2/25-274` は、日本古典籍で使われる 1 ideograph の追加を提案する文書である。対象文字は IDS `⿰犭華`、`kJapanese` reading `クワハ`、total strokes `13`、first stroke `4`、radical-stroke `94.10` として示される。

主な evidence は、江戸後期の日本の学者 Ikuta Kunihide（生田国秀）による `古学二千文` と、明治期資料、国立公文書館所蔵資料である。文書は、複数資料で同じ glyph が確認できるため、関連分野で一定の安定性と使用実績を持つと位置づける。

## 決定・Action Item

| 区分 | 内容 |
| --- | --- |
| 提案 | IDS `⿰犭華` の 1 ideograph を追加する。 |
| 用途 | 日本古典籍・近世近代資料での使用。 |
| Data | `kJapanese クワハ`、`TS 13`、`FS 4`、`RS 94.10`。 |
| 根拠 | `古学二千文`、明治期資料、国立公文書館資料。 |

## 論点

### 古典籍由来の未符号化文字

この文書は、地名や行政システムではなく、古典籍・教育資料・歴史資料に現れる文字を対象にする。提案の意義は、現代常用性ではなく、歴史資料を正確に転写・検索・研究するための文字同定にある。

### 読みと glyph stability

文書は、資料中の用例から reading `クワハ` を推定し、複数版・複数資料に残ることから glyph が比較的安定していると説明する。Unicode 側で扱う場合は、IDS と reading だけでなく、資料系統を含めて evidence を追跡する必要がある。
