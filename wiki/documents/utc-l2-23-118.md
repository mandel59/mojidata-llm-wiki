# L2/23-118

## 要約

`L2/23-118` は、Ryukyuan languages（Shimakutuba）用の superscript katakana letters 8 characters を Small Kana Extension に追加する proposal である。文書は、Okinawa Prefecture の Shimakutuba Orthography Review Committee が 2022-03-30 に制定した表記体系を根拠に、plain text で superscript kana を表示できる必要があると説明する。

提案文字は KATAKANA LETTER SUPERSCRIPT A / I / U / E / O / TU / HU / N で、Kunigami、Okinawan、Yonaguni、common Ryukyuan phonetic symbols などで使われるとされる。Okinawa Prefecture の資料と user community への contact を根拠として提示している。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字数 | 8 characters。 |
| 提案文字 | KATAKANA LETTER SUPERSCRIPT A / I / U / E / O / TU / HU / N。 |
| 提案 block | Small Kana Extension。 |
| 用途 | Ryukyuan languages / Shimakutuba orthography の phonetic notation。 |
| 関係者 contact | Okinawa Prefecture Cultural Promotion Division, Shimakutuba Promotion Office。 |

## 論点

### 小書きではなく superscript

提案文字は、既存 small katakana とは異なり baseline の上に置かれる superscript letters として説明される。既存 small kana と視覚的に似るが、orthography 上の機能と placement が異なるため、同一視できるかどうかが論点になる。

### 標準化主体と安定性

後続の `L2/23-123` は、根拠資料が新しく、running text・vertical layout・他 orthography との関係・政府提出として扱うべきかなどに疑問を示した。したがって `L2/23-118` は、kana usage の拡張として関連するが、地域表記政策と typographic model の確認が必要な proposal である。
