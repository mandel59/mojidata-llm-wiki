# IRG N2955

## 要約

`IRG N2955` は、[IRG N2954](irg-n2954.md) の U+2CCA3 / `GZFY-00932` issue への feedback である。文書は `IRG N2954` の問題提起に同意し、source history を補足したうえで、source reference を維持しながら glyph を original evidence に合わせて修正する方向を支持する。

文書によれば、`GZFY-00932` は Ext C の `IRG N891` で China から提出され、Ext E の `IRG N1519` では別 evidence が使われた。feedback は、wrong glyph と wrong evidence が混在した結果として current glyph issue が生じたと整理している。

## フィードバック内容

| 論点 | 内容 |
| --- | --- |
| source history | `GZFY-00932` は China submission 由来であり、U+2CCA3 の source reference として維持できる。 |
| evidence mismatch | Ext E encoding 時に `GZFY-00931` 相当の evidence が誤って使われた可能性を指摘する。 |
| suggested action | source reference は変えず、representative glyph を original evidence に合わせて修正する。 |
| separate encoding | current glyph に見える別形を別 character として符号化する必要はないとする。 |

## 後続決定

Meeting #67 agenda の [IRG N2935](irg-n2935.md) は、G / V / SAT などの representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。`IRG N2955` 自体は feedback であり、最終決定は記録していない。

## 論点

### original evidence を優先する整理

`IRG N2955` は、source reference の owner identity よりも、実際にその reference が指す original evidence を重視する。これにより、source reference を `GU` へ逃がすのではなく、G-source glyph correction として処理する理由を示している。
