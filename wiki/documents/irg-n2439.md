# IRG N2439

## 要約

`IRG N2439` / `L2/20-204` は、UTC-02993 と UTC-03009 の 2 U-source ideographs を Urgently Needed Characters (UNCs) として符号化する proposal である。文書は、Macao SAR の 5-ideograph UNC proposal と TCA の overlapping single-ideograph proposal に重なる 2 ideographs について、UTC 側の representative glyphs、attributes、evidence を補強する。

## 提案内容

対象は UTC-02993 と UTC-03009 で、source reference、representative glyph、UTCHan.ttf の PUA code point、IDS、`kRSUnicode`、`kTotalStrokes`、対応する M-source が示される。文書は、Macao proposal の MC-00137 に対する IDS と total strokes を修正すべきだと note し、2 ideographs が UNC として accepted される場合は IRG Working Set 2017 から removed されるべきだと述べる。

## 後続決定

この UNC proposal で対象にした `UTC-02993` と `UTC-03009` は、現行 `USourceData.txt` ではそれぞれ `URO` status で `U+9FFF` と `U+9FFE` に対応している。つまり、proposal の対象 2 ideographs は現在は符号化済み U-source records として追跡できる。

この wiki では、UNC としての承認 meeting / recommendation はまだ event 化していない。現時点では、proposal page では「未確認」とせず、符号化済み outcome を UAX \#45 data で確認できるものとして扱い、会合上の決定経路は後で meeting / event page に分離する。

## 論点

`IRG N2439` は、U-source record が単独で閉じたものではなく、M-source / TCA / China など他 body の overlapping submissions と接続して IRG review されることを示す。U-source は UTC の source identifier だが、actual evidence と working set handling は IRG の multi-source review の中で整理される。
