# L2/25-213

## 要約

`L2/25-213` は、Unihan Database の provisional property `kJapanese` に含まれる Katakana 表記の kun'yomi を調査する文書である。`kJapanese` の説明では、hiragana は概ね kun'yomi（訓読み）、katakana は概ね on'yomi（音読み）とされるが、文書はこの rule of thumb には例外があると指摘する。

調査は、on'yomi が一般に比較的制約された kana structure を持つこと、`kJapaneseKun` と重なる readings があることを手がかりに、Katakana 表記だが kun'yomi と見られる readings を検出する。

## 決定・Action Item

| 区分 | 内容 |
| --- | --- |
| 対象 | Unihan provisional property `kJapanese`。 |
| 指摘 | Japonic terms の Katakana 表記、taxonomic terms、non-Sino-Japanese loanwords など、Katakana 表記の kun'yomi が存在する。 |
| 提案 | `kJapanese` values の変更と、non-Japonic loanword kun'yomi に関する definition additions。 |

## 論点

### kana script と reading type は一対一ではない

`kJapanese` の説明は実用的な近似として hiragana / katakana を kun'yomi / on'yomi に対応させるが、実データには例外がある。特に分類名・外来語・単位名などでは、Japanese reading が Katakana で表記されても on'yomi とは限らない。

### `kJapanese` の保守

この文書は kana character encoding ではなく、Unihan data quality の問題である。`kJapanese` が Moji Jōhō Kiban 由来の安定した baseline として導入された後、実運用では property definition と value corrections の両方が必要になることを示す。
