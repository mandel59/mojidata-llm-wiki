# WG2 N5340

## 要約

`WG2 N5340` は、WG2 Meeting \#72 で Tai Xaau script proposal を説明した presentation である。Dao Jie / Qi Ziyuan による presentation として登録され、`WG2 N5310R` の script background、Tai Viet / Tai Dam との比較、community recognition、教育・使用例を示す。

文書は decision document ではなく、proposal presentation である。Dai Xaau / White Dai と Tai Dam / Black Tai を対比し、respondents が Tai Xaau として認識するか、Tai Dam / Tai Viet を知っているかといった survey 結果を補足資料として示す。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 位置づけ | [WG2 N5310R](wg2-n5310r.md) の meeting presentation。 |
| 主題 | Tai Xaau / White Dai と Tai Dam / Black Tai の違い、Tai Xaau script の現代使用、教育・font 作成。 |
| survey | letter comparison では「似ているが同じ type of text ではない」という回答、script recognition では Tai Xaau として認識する回答が示される。 |
| evidence | 1980 年の Tai Xaau language learning、Tai Xaau teaching、Xiaorui Font designer Luo Wenbin などの資料を提示する。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) の M72.15 は、Dai Xaau proposal の progress を note し、authors に継続作業を促した。`WG2 N5340` はその会議で proposal の背景と community evidence を説明するための資料として読む。

## 論点

### presentation evidence の扱い

`WG2 N5340` は proposal 本文ほど詳細な encoding model を持たないが、Tai Xaau と Tai Viet / Tai Dam の user recognition と教育使用を示す。script disunification の technical argument は [WG2 N5310R](wg2-n5310r.md) 側で確認し、この presentation は community evidence と meeting context を補う文書として扱う。
