# PRI 551

## 要約

`PRI #551` は、UTC が Unicode 18.0.0 向けの UAX \#14 Unicode Line Breaking Algorithm Revision 56 proposed update を public review に出した PRI である。PRI page の status は Closed、closing date は 2026-07-07。

PRI page は、Unicode 5.1 で入った en dash followed by no-break space の line breaking behavior の問題を corrected と説明する。

draft の modifications section は、U+2012 FIGURE DASH と U+2013 EN DASH の `Line_Break` assignment を `HH` から `BA` へ、U+00AD SOFT HYPHEN を `BA` から `HH` へ変え、LB12a を `BA` と `GL` の間で break しないよう変更したと記録する。

## 提案内容

### UAX \#14 Revision 56

UAX \#14 Revision 56 は Unicode 18.0.0 draft 1、date 2026-05-19 の proposed update である。Revision 56 の変更は line break class と rule LB12a の組み合わせで、EN DASH + NO-BREAK SPACE のような sequence の break opportunity を調整する。

変更は `LineBreak.txt` / derived line break data と、algorithm text の両方に影響する。単に UAX text を読むだけでなく、最終 UCD data の `Line_Break` assignments と line break test data を確認する必要がある。

## 後続決定

`PRI #551` は 2026-07-07 に closed になっている。UTC \#187 の PAG report [L2/26-096](utc-l2-26-096.md) は line breaking property の調整を Unicode 18.0 beta review の property / algorithm issue として扱っており、この PRI はその public review gate に相当する。

UTC \#188 で Unicode 18.0 content finalize が予定されているため、feedback の処理結果は UTC \#188 minutes / action items、最終 UAX \#14、最終 line break data / test files で確認する。

## 論点

### line break class と rule text

この PRI は、character property assignment と algorithm rule text が同時に変わる例である。`Line_Break` class の変更だけでは EN DASH + NBSP の behavior を説明しきれず、LB12a の no-break 条件と合わせて読む必要がある。

### East Asian typography との関係

UAX \#14 は East Asian line breaking を含む一般 algorithm である。[East Asian Spacing](../topics/east-asian-spacing.md) や [East Asian Quotation Marks](../topics/east-asian-quotation-marks.md) は別 topic だが、wide / narrow characters、punctuation、dash、space の扱いは layout implementation で交差する。
