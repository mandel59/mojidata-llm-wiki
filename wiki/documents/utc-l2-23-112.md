# L2/23-112

## 要約

`L2/23-112` は、古い日本語資料に現れる合略仮名（gōryaku gana）8 characters を Kana Extended-A に追加する proposal である。提案は、文学・古文書研究で原資料の文字情報を正確に保存するため、複数音節を表す単一 kana character を plain text で扱えるようにすることを目的とする。

文書は、単なる連綿や複数 kana の sequence を除外するための criteria を提示する。手書き資料では構成要素の筆順・配置・省略の大きさ、活字資料では単一文字として印刷されていることを重視し、十分に形態情報を保持したまま既存 kanji で代替できるものは除くとした。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字数 | 8 characters。 |
| Hiragana 側 | HIRAGANA LIGATURE KASIKO、HIRAGANA DIGRAPH KOTO、HIRAGANA DIGRAPH SAMA、HIRAGANA LIGATURE MAIRASE-SORO。 |
| Katakana 側 | KATAKANA LIGATURE TO-IU、KATAKANA DIGRAPH TOKI、KATAKANA DIGRAPH TOTE、KATAKANA DIGRAPH YORI。 |
| 提案 block | Kana Extended-A。 |

## 論点

### 合略仮名の判定基準

`L2/23-112` は、合略仮名を「複数音節の語を表す単一 kana character」としつつ、historically gōryaku gana と呼ばれたものをすべて符号化対象にしない。通常の縦書き連綿、既存 kanji で形態情報を十分保持できる abbreviation、単なる sequence を除くための基準が、後続の reduced proposal の前提になった。

### reduced set への接続

後続の `L2/24-150` は、この proposal を reduced set として改訂し、3 kana ligatures に絞った。`L2/24-279` は reduced set から漏れた KATAKANA DIGRAPH YORI を再提案する。したがって `L2/23-112` は、2024-2025 年の kana ligature 議論の親 proposal として扱う。
