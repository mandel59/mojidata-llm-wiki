# WG2 N5341

## 要約

`WG2 N5341` は、Small Seal proposal の code charts と data set をまとめる PDF attachment 文書である。[WG2 N5306](wg2-n5306.md) の initial repertoire と [WG2 N5317R3](wg2-n5317r3.md) の feedback disposition を反映し、11,339 Small Seal characters の code chart と related database (Uniseal) を提示する。

文書は、[WG2 N5318R](wg2-n5318r.md) が proposal preamble、`WG2 N5307R` が unification / disunification mapping principles、[WG2 N5317R3](wg2-n5317r3.md) が feedback disposition を担うと位置づける。WG2 \#72 M72.13 は、`WG2 N5341` を Small Seal future edition candidate の根拠文書の一つとして参照した。

## データ内容

| 区分 | 内容 |
| --- | --- |
| code charts | 11,339 Small Seal characters の chart。Multi-column code chart format を使い、THX / CCZ / QJZ / DYC などの source columns を表示する。 |
| data set | Related database (Uniseal) を PDF attachment として含む。 |
| disposition reflection | `WG2 N5317R3` による 199 characters の追加、source assignment changes、THX glyph corrections を反映する。 |
| review support | Disunifications、source assignment changes、glyph changes を highlight して review できるようにする。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) の M72.13 は、`WG2 N5306`、[WG2 N5318R](wg2-n5318r.md)、[WG2 N5317R3](wg2-n5317r3.md)、`WG2 N5341` を根拠に Small Seal characters を future edition candidate として扱った。後続の [WG2 N5344](wg2-n5344.md) では repertoire と code range が改訂され、[WG2 N5369](wg2-n5369.md) の CD.4 disposition で name / property / glyph corrections が処理された。

## 論点

### chart / data と proposal text の分担

`WG2 N5341` は proposal text ではなく、large repertoire の review と Project Editor 作業に必要な chart / database artifact である。Small Seal の rationale や principles は `WG2 N5306`、`WG2 N5307R`、[WG2 N5318R](wg2-n5318r.md)、[WG2 N5317R3](wg2-n5317r3.md) と合わせて読む必要がある。
