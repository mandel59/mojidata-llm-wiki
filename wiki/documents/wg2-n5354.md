# WG2 N5354

## 要約

`WG2 N5354` は、2026-06-22/26 に Paris, France で開催された WG2 Meeting \#73 の recommendations である。`WG2 N5369` の CD.4 disposition 受け入れ、CJK additions and changes、code charts の linked content 化、ISO/IEC 10646 7th edition DIS progression、Amendment 1 project subdivision、first amendment additions、Maya review、`kIRG_SGSource` を扱う。

この wiki では、Small Seal name / property / glyph correction を含む M73.01、`L2/26-099` などを CD の CJK additions and changes に取り込む M73.02、external code charts の M73.03、DIS 進行を示す M73.04、Amendment 1 project の M73.05、`kIRG_SGSource` を Amendment 1 target に置く M73.15 を主に参照する。

## 後続決定

M73.01 は `WG2 N5369` の disposition を受け入れる勧告であり、Small Seal の詳細な修正表は disposition document 側で確認する。M73.03 / M73.04 により、external code charts と CD.4 disposition は ISO/IEC 10646 7th edition DIS progression に入る。

M73.05 は `WG2 N5362` と `WG2 N5361R` を出発点として、7th edition Amendment 1 project を開始する勧告である。Starting repertoire は provisional であり、M73.06 以降の candidate groups や後続 review により変わり得る。

M73.15 は `kIRG_SGSource` を Amendment 1 target とする WG2 recommendation である。`IRG N2935` は、これを SC2 acceptance と Unicode Version 19.0 target に接続する current status として記録している。

## 論点

### Unicode 18.0 と ISO/IEC 10646 の同期

M73.02 は、UTC CJK & Unihan Working Group recommendations `L2/26-099` の一部を ISO/IEC 10646 CD の CJK additions and changes に反映する経路である。Unicode Version 18.0 Beta review に入った変更と、ISO/IEC 10646 7th edition DIS text に入る変更を混同しないよう、topic page では body と時点を明示する。

### Amendment 1 と provisional repertoire

M73.05 の Amendment 1 project は [WG2 N5361R](wg2-n5361r.md) の provisionally assigned code points を starting point とする。`WG2 N5361R` は encoded repertoire の確定ではないため、first amendment candidates としての扱いと、各 proposal の採否は分けて追う。

### Small Seal の状態

WG2 \#72 の `WG2 N5304` では future edition candidate だった Small Seal は、WG2 \#73 では CD.4 disposition と DIS progression の文脈で扱われる。名称、property、glyph correction は `WG2 N5369` と event page に集約する。
