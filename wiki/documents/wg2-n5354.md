# WG2 N5354

## 要約

`WG2 N5354` は、2026-06-22/26 に Paris, France で開催された WG2 Meeting #73 の recommendations である。`WG2 N5369` の CD4 disposition 受け入れ、CJK additions and changes、code charts の linked content 化、ISO/IEC 10646 7th edition DIS progression、Amendment 1 に向けた additions を扱う。

この wiki では、Small Seal name / property / glyph correction を含む M73.01、`L2/26-099` などを CD の CJK additions and changes に取り込む M73.02、DIS 進行を示す M73.04、`kIRG_SGSource` を Amendment 1 target に置く M73.15 を主に参照する。

## 決定・Action Item

| Recommendation | 内容 | wiki での扱い |
| --- | --- | --- |
| M73.01 | `WG2 N5369` の CD4 disposition を SC2 が受け入れるよう勧告。Small Seal character names を `SMALL SEAL CHARACTER-XXXXX` へ変更し、Small Seal properties 26 characters と U+3F80E glyph correction を含む。 | [WG2 M73.01 Small Seal CD4 disposition](../events/wg2-m73-01-small-seal-cd4-disposition.md) |
| M73.02 | `L2/26-099` と IRG documents に基づき、representative glyph changes、Unihan property value changes、CJK Strokes standardized variation sequences などを CD の CJK additions and changes として扱う。 | [IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md)、[Unihan Database Maintenance](../topics/unihan-database-maintenance.md) |
| M73.04 | M73.01-M73.03 と final disposition `WG2 N5369` を含む 7th edition DIS text を SC2 secretariat へ回付するよう勧告。target は DIS 2026-10-01、IS 2027-06-01。 | [Unicode 18.0 Change Sources](../topics/unicode-18-change-sources.md) |
| M73.15 | `WG2 N5359` に基づき、new normative IRG source for Singapore `kIRG_SGSource` を Amendment 1 target として追加するよう勧告。 | [IRG M66.16 kIRG_SGSource establishment](../events/irg-m66-16-kirg-sgsource-establishment.md) |

## 後続決定

M73.01 は `WG2 N5369` の disposition を受け入れる勧告であり、Small Seal の詳細な修正表は disposition document 側で確認する。M73.04 により、これらの変更は ISO/IEC 10646 7th edition DIS progression に入る。

M73.15 は `kIRG_SGSource` を Amendment 1 target とする WG2 recommendation である。`IRG N2935` は、これを SC2 acceptance と Unicode Version 19.0 target に接続する current status として記録している。

## 論点

### Unicode 18.0 と ISO/IEC 10646 の同期

M73.02 は、UTC CJK & Unihan Working Group recommendations `L2/26-099` の一部を ISO/IEC 10646 CD の CJK additions and changes に反映する経路である。Unicode Version 18.0 Beta review に入った変更と、ISO/IEC 10646 7th edition DIS text に入る変更を混同しないよう、topic page では body と時点を明示する。

### Small Seal の状態

WG2 #72 の `WG2 N5304` では future edition candidate だった Small Seal は、WG2 #73 では CD4 disposition と DIS progression の文脈で扱われる。名称、property、glyph correction は `WG2 N5369` と event page に集約する。
