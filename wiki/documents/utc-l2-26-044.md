# L2/26-044

## 要約

`L2/26-044` は、山形県東根市大字東根元東根の複数地名に使われる未符号化 ideograph を UAX #45 に追加する提案である。対象文字は IDS `⿰山入`、`kJapanese` readings `はさま` / `はざま`、radical-stroke `46.2`、total strokes `5`、first stroke `3` として示される。

文書は、`USourceData.txt` の `FutureWS` record としての追加案と、UTCHan.ttf に代表 glyph を追加するための TTF file を提示する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | IDS `⿰山入` の 1 ideograph を UAX #45 に追加する。 |
| 用途 | 山形県東根市大字東根元東根の字名群。 |
| Data | `kJapanese はさま はざま`、`RS 46.2`、`TS 5`、`FS 3`。 |
| 提案 record | UAX #45 `FutureWS` record として UTC source data に追加する候補にする。 |

## 後続決定

[UTC #187 UAX #45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) が、この提案を含む 2026 年 UAX #45 additions の decision point である。`L2/26-099` は、対象 records を `USourceData.txt` と `USourceGlyphs.pdf` に追加し、status を `FutureWS` とするよう勧告した。

## 論点

### 複数地名での使用

対象文字は単一地名ではなく、山形県東根市大字東根元東根の複数の字名に現れる。文書は「鼬はさま」「一ノはさま」「二ノはさま」「大はさま山」「小土はさま」「小はさま山」「白はさま」「田ノはさま」などの地名を列挙する。

### 代替文字との関係

旧公図の説明では、代替文字として「峡」が使われる例も示される。これは、未符号化文字が地名記録上で別字に置換されうることを示し、地名文字の同定では glyph evidence と読み・地名文脈の両方を確認する必要がある。

### UTC #187 への接続

`L2/26-044` は、`L2/26-099` の UAX #45 additions に含まれる 2026 年の提案群の一部である。したがって、個別 proposal として読むだけでなく、UTC #187 の CJK & Unihan Working Group recommendations によって `FutureWS` record としてどう処理されたかも追跡する必要がある。
