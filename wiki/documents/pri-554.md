# PRI 554

## 要約

`PRI #554` は、UTC が Unicode 18.0.0 向けの UAX \#24 Unicode Script Property Revision 40 proposed update を public review に出した PRI である。PRI page の status は Closed、closing date は 2026-07-07。

PRI page は、この draft が ISO 15924 mixed-script script codes の explanation を追加したと説明する。

UAX \#24 Revision 40 draft は Unicode 18.0.0、date 2026-05-29 を示し、modifications section は Revision 40 の変更を Unicode 18.0.0 proposed update と mixed-script script codes explanation の追加として記録する。

## 提案内容

### UAX \#24 Revision 40

UAX \#24 は `Script` と `Script_Extensions` properties を説明し、`Scripts.txt`、`ScriptExtensions.txt`、`PropertyValueAliases.txt` を data files として扱う。Revision 40 は Unicode 18.0.0 向けに出され、Section 2.2 Relation to ISO 15924 Codes に mixed-script script codes の説明を追加する。

この変更は新規 script proposal の採否そのものではなく、Script property values と ISO 15924 script codes の対応を読むための documentation update である。

## 後続決定

`PRI #554` は 2026-07-07 に closed になっている。feedback の処理結果は UTC \#188 minutes / action items、最終 UAX \#24、最終 `PropertyValueAliases.txt` / `Scripts.txt` / `ScriptExtensions.txt` で確認する必要がある。

[L2/26-110](utc-l2-26-110.md) は UAX \#31 Revision 44 の proposed update として new scripts を identifier handling に接続する文書であり、UAX \#24 の Script / Script_Extensions documentation と併読する。

## 論点

### Script property と ISO 15924

UAX \#24 は、Unicode の `Script` property values と ISO 15924 script codes が近いが目的が異なることを説明する。mixed-script script codes の説明追加は、bibliographic / ISO 15924 classification と Unicode property model を混同しないための補足である。

### script pipeline との関係

[Script Encoding Pipeline](../topics/script-encoding-pipeline.md) では新規 script proposals の progression を追うが、UAX \#24 はそれらの script が encoded 後に property value、alias、Script_Extensions、identifier / regex / rendering behavior へどう現れるかを説明する layer である。採択判断と property documentation は分けて読む必要がある。
