# L2/25-035

## 要約

`L2/25-035` は、既存 BMP small kana と Bopomofo small letters の Vertical Orientation values を見直すよう求める 1-page proposal である。対象は character addition ではなく、UAX #50 / Vertical_Orientation data に関わる data maintenance である。

small kana は縦書きで小書き位置を伴うため、単に glyph を縮小するだけでなく、vertical layout での orientation / placement が問題になる。この文書は、`L2/25-036` の small archaic YE と同時期に提出され、small kana encoding と layout property が連動することを示す。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | BMP の small Hiragana / Katakana と Bopomofo small letters。 |
| 種別 | Vertical_Orientation property change request。 |
| 関連 | KATAKANA LETTER SMALL ARCHAIC YE など、新規 small kana proposals の property 設計。 |

## 論点

### 小書き仮名は符号位置だけでは完結しない

small kana の encoding では、文字名・block・Line_Break だけでなく、縦書きでの表示方法が実用上の論点になる。`L2/25-035` はこの property 側の問題を明示的に扱う。
