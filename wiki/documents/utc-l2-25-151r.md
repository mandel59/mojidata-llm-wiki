# L2/25-151R

## 要約

`L2/25-151R` は、KATAKANA LETTER ALTERNATE NE と KATAKANA LETTER ALTERNATE WI を Kana Extended-A に追加する revised proposal である。前段の `L2/23-127` では 2 つの unencoded Katakana letters が紹介され、この revised proposal では明治期活字・教科書・出版資料などの evidence を拡充している。

提案文字は、NE が子（U+5B50）由来、WI が井（U+4E95）由来と説明される。文書は、current Katakana forms や Hentaigana とは統合できないとしつつ、子 / 井 との confusable pair として扱う必要にも触れている。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字 | `1B127; KATAKANA LETTER ALTERNATE NE; Lo`、`1B128; KATAKANA LETTER ALTERNATE WI; Lo` |
| 提案 block | Kana Extended-A |
| 用途 | 明治期を中心とする historical Katakana repertoire の plain-text representation。 |
| 関連 | 子（U+5B50）/ 井（U+4E95）との confusable pair。 |

## 論点

### Katakana と Hentaigana の境界

NE / WI の字形は漢字由来で Hentaigana とも関係するが、文書は Katakana letters として扱う。これは、母字由来の字形が Hentaigana であれば常に Hentaigana block へ行くわけではなく、用法・資料・五十音上の位置づけが encoding model に影響することを示す。

### confusables data との接続

文書は、提案文字が子 / 井 と視覚的に近いことを明示する。これは UTS #39 confusables data の topic とは別系統の proposal だが、CJK / Katakana / Han glyph の視覚的類似が security data にどう現れるかという論点に接続する。
