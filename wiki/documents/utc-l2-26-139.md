# L2/26-139

## 要約

`L2/26-139` は COMPOSE character を追加し、二つの printable characters を overstrike して新しい glyph を作る plain-text mechanism として扱う proposal である。Typewriter overstriking、APL historical characters、nonstandard punctuation、emoji-like composites などを例にする。

提案は 1 character を Combining Diacritical Marks Extended block に追加する form で、overstruck result を one grapheme cluster として扱うことを想定する。これは markup / formatting と plain text の境界、security / confusability、normalization、font shaping に広く影響する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | COMPOSE。二つの characters の間に入り、後者を前者に重ねるとする。 |
| intended behavior | resulting glyph は一つの grapheme cluster として扱われ、diacritics placement も一体として処理されるべきとする。 |
| evidence | mechanical / electric typewriters、APL historical compose / overstrike practice、nonstandard punctuation など。 |
| concern | composed glyphs が既存 Unicode characters と confusable になり得ることを認める。 |

## 論点

### plain text と markup の境界

COMPOSE は、italic / bold のような style ではなく distinct glyph を作る plain-text transformation だと主張する。一方で、実装は grapheme cluster、font composition、search、security、normalization に影響するため、Unicode text model 上の大きな論点になる。
