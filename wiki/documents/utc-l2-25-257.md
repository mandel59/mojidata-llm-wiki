# L2/25-257

## 要約

`L2/25-257` は、ESR / Jennifer Daniel による METEOR emoji proposal である。Proposal は、meteor-designs を既存 emoji で vendor がばらばらに扱っていることを vendor interoperability issue とし、METEOR を独立 emoji として追加する。

文書は、meteor を comet とは異なる astronomical / emotional symbol として扱う。Meteor は atmospheric, momentary, dramatic で、wish / impact / impending danger などに接続し、既存 comet emoji とは意味と visual tone が違うとする。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | METEOR |
| keywords | space、rock、star、falling、gravity など。 |
| interoperability | vendors with meteor-designs を新 character へ寄せることで、既存 emoji の意味衝突を減らす。 |
| distinction | comet は celestial / long-term、meteor は atmospheric / momentary として差別化する。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FA8B METEOR を Unicode 18.0 additions として受け入れ、[L2/26-008R](utc-l2-26-008r.md) の final candidate list でも維持した。

## 論点

### astronomy emoji と vendor design migration

Meteor は単なる space object addition ではなく、既存 vendor designs の意味分担を整理する proposal として読める。Emoji repertoire では、new character が既存 glyph variation の逃がし先になることがある。
