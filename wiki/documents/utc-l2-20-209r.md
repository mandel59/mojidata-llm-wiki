# Final Proposal to Encode Taiwanese Kana

## 要約

`L2/20-209R` は、Taiwanese Kana を Unicode に追加する revised final proposal である。
Fredrick R. Brennan は、Kana Extended-B block を U+1AFF0..U+1AFFF に置き、Minnan tone marks と nasalized tone marks 13 文字を提案した。
提案は [L2/20-233](utc-l2-20-233.md) と Script Ad Hoc の feedback を反映し、character names を `MINNAN` に寄せた。
Aspiration mark と overline は新規文字ではなく U+0323 / U+0305 を使い、Taiwanese Kana 用の rendering / annotation 課題として扱う。
現在の Unicode code chart では、Kana Extended-B は Minnan Chinese を annotate する furigana extensions の tone marks として掲載されている。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| block | Kana Extended-B、range U+1AFF0..U+1AFFF。 |
| assigned repertoire | Tone 2 / 3 / 4 / 5 / 7 / 8 と、nasalized tone 1 / 2 / 3 / 4 / 5 / 7 / 8 の 13 文字。 |
| names | `KATAKANA LETTER MINNAN TONE-*` と `KATAKANA LETTER MINNAN NASALIZED TONE-*`。 |
| empty slots | U+1AFF4 / U+1AFFC は tone 6 用の将来位置として空ける。 |
| unified combining marks | U+0323 COMBINING DOT BELOW、U+0305 COMBINING OVERLINE を Taiwanese Kana 用にも使う。 |

## 後続決定

この proposal は Source Document であり、会議 decision 自体ではない。現行の Unicode code chart は Kana Extended-B U+1AFF0..U+1AFFF を持ち、13 文字の Minnan tone marks を掲載しているため、proposal の主要 repertoire は標準化済みと読める。

## 論点

### Taiwanese Kana と Minnan tone marks

Taiwanese Kana は、Taiwanese Hokkien / Min Nan を kana と tone marks で注記する system として扱われる。文書は、Japanese colonial-period dictionaries や later digitization / dictionary projects を evidence とし、単なる modern font convention ではなく historical writing practice として符号化を求める。

### tone six と予約位置

Tone 6 は modern Hokkien では tone 2 と一致するため、頻繁には使われない。文書は、将来 evidence が増えた場合に論理的な位置へ追加できるよう U+1AFF4 と U+1AFFC を空ける方針を取る。

### 縦書きと combining mark

Taiwanese Kana は縦書き資料が多いが、U+0323 / U+0305 の表示位置は一般的な East Asian vertical text と同じではない。これは character repertoire の追加ではなく、NameList、UAX \#50、font / layout behavior の問題として後続確認が必要になる。
