# L2/25-131

## 要約

`L2/25-131` は、historic scientific characters proposal を分割して提出した 1 件目として、7 個の alchemical symbols を UCS に追加する提案である。

提案者は、Leibniz-Archiv / Philiumm Project などの historic science text corpora を背景に、Leibniz と同時代人の manuscript / printed sources に現れる記号を対象にする。

既存の Alchemical Symbols block に残り枠が少ないため、将来追加も見込んだ "Alchemical Symbols Extension" のような chart を想定する。

## 提案内容

提案される 7 文字は次の通りである。

| 提案名 | 論点 |
| --- | --- |
| ALCHEMICAL SYMBOL FOR OIL BOILED | common oil symbol の 180 度回転形だが、boiled oil として別意味を持ち、ordinary oil symbol と同時に現れる。 |
| ALCHEMICAL SYMBOL FOR MOON-JUPITER | silver-bearing tin を示す例がある。 |
| ALCHEMICAL SYMBOL FOR TARTAR-SALT | Leibniz correspondence の manuscript evidence に基づく。 |
| ALCHEMICAL SYMBOL ENCLOSED SUN | foliated gold を示すほか、数学 manuscript で reference mark 的にも使われる。 |
| ALCHEMICAL SYMBOL ENCLOSED MOON | foliated silver を示す。 |
| ALCHEMICAL SYMBOL FOR HORA-2 | U+1F76E とは shape が大きく異なる straight X-like form を、basic hora sign として別符号化する。 |
| ALCHEMICAL SYMBOL FOR RETORT-2 | 既存 U+1F76D とは glyph shape が異なり、homonymous definition でも別 character とする precedents があるとする。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は UTC \#187 の Script Encoding WG recommendations として、Alchemical Symbol Retort-2 を進め、crossbar annotation を付ける方向を示した。この文書全体の 7 文字すべてが同じ時点で accepted になったわけではないため、proposal page では後続決定を SEW report 側へ分離する。

## 論点

### historic scientific proposal の分割

この proposal は、2024 年に提出された 228 historic scientific characters の大きな proposal を、approval process のために character group ごとに分割する流れの一部である。個別記号の evidence と、Leibnizian / historic scientific symbols 全体の block 設計は分けて読む必要がある。

### shape distinction と同義記号

HORA-2 と RETORT-2 は、意味が既存 character と近い、または同じでも、glyph variety が editorial principle 上区別される場合があることを示す。plain text character と glyph variant / font distinction の境界が主な review point になる。
