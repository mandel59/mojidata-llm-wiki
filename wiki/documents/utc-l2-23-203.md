# L2/23-203

## 要約

`L2/23-203` は African scripts の usage と implementation status を encoded / unencoded に分けて整理する report である。Charles Riley による 2022 年 update [L2/22-073](utc-l2-22-073.md) の follow-up として、利用証拠、font / keyboard、教育・出版・online usage、encoding proposal の見込みを一覧化する。

N'ti については unencoded script として扱い、Soninke 用 left-to-right alphabet、font、books、動画、user community との接触を記録する。後続の `L2/26-028R` / `L2/26-087` N'ti proposal chain にとって、community usage evidence の早い入口になる。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| scope | African scripts の current usage と online implementation status。encoded scripts と unencoded scripts を分けて扱う。 |
| N'ti status | N'ti は Soninke 用の alphabetic LTR script とされ、proposal は Oreen Yousuf が作業予定と記録される。 |
| N'ti evidence | 2018 年 2 月以前からの font、books、TikTok / Facebook / YouTube での使用、children learning の動画、user community との early contact。 |
| KORE SEBELI | KORE SEBELI / Wakara も unencoded script として、2022 年 proposal と教育・学習状況を記録する。 |

## 後続決定

[L2/26-028R](utc-l2-26-028r.md) は N'ti の preliminary proposal として提出され、[L2/26-087](utc-l2-26-087.md) がこれを supersede した。`L2/23-203` では N'ti の作成年を 1985 年、creator を Dama Konte としている一方、2026 年 proposal は 1966 年に Adama / Dama Konte が作った script と説明するため、この点は後続 proposal で整理された履歴差として読む。

## 論点

### Usage survey と encoding proposal の接続

この report は repertoire や property model を決める proposal ではないが、unencoded scripts に対して「利用証拠があるか」「community contact があるか」「font / keyboard / publications があるか」を事前に整理する役割を持つ。N'ti と KORE SEBELI のような 2026 年時点の script proposals は、この種の usage survey と SEW review を合わせて読む必要がある。
