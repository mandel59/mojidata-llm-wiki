# L2/26-098

## 要約

`L2/26-098` は Emoji Standard & Research Working Group が UTC Meeting \#187 に提出した report である。Unicode 18.0 / Emoji 18.0 の feedback だけでなく、emoji の interoperability、inclusion criteria、CLDR keyword request handling、Emoji 19.0 intake を扱う。

Report は [monarch butterfly feedback](utc-l2-25-254.md) を例に、既存 emoji の vendor design divergence を interoperability 問題として扱う。ESR は、単に新しい emoji を追加するのではなく、既存の customer problems と実装差を減らす方向へ priorities を置いている。

## 報告内容

| 区分 | 要点 |
| --- | --- |
| Monarch butterfly feedback | U+1F98B BUTTERFLY の vendor design が morpho / monarch に分かれているため、monarch butterfly addition は interoperability 問題の解消として扱われた。 |
| Exclusion factors | Exclusion factor に該当するという feedback は、inclusion factor と interoperability 上の根拠を自動的に打ち消すものではないと整理された。 |
| CLDR keyword intake | ESR と CLDR は emoji keyword の追加・削除 request に関する best practices を整理中で、将来の document 化を予定している。 |
| Interoperability audit | 51 emoji を対象にした Q1 study では、animal emoji は full-body representation の方が vendor 間で安定しやすいという preliminary result が示された。 |
| Emoji 19.0 intake | Emoji 19.0 の review は 2026-04-02 に始まり、submission window は 2026-07-31 までとされる。 |

## 後続決定

Emoji 18.0 final candidates は [L2/25-230R](utc-l2-25-230r.md) と [L2/26-008R](utc-l2-26-008r.md) で整理される。UTS \#51 proposed update は [L2/26-104](utc-l2-26-104.md) と PRI \#543 にも分散する。UTC \#187 minutes [L2/26-093](utc-l2-26-093.md) は、Unicode 18.0 beta review authorization の一部として ESR report を扱う。

## 論点

### 新規追加より interoperability

ESR report の重要点は、emoji proposal を人気や要望の量だけでなく、既存文字の実装差・検索性・keyword・デザイン安定性の問題として扱うことである。Monarch butterfly は、既存 U+1F98B の一部実装が担っていた意味を独立させる例として説明される。

### CLDR との接続

Emoji keyword intake は Unicode character encoding だけで完結せず、CLDR keyword data と実装者の検索 UX に接続する。したがって emoji topic は UTS \#51 proposed update と CLDR process の両方から追う必要がある。
