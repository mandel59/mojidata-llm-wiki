# IRG N2860

## 要約

`IRG N2860` は、Ma Shijie による individual contribution で、25 T-source glyphs の revision を TCA に求める proposal である。文書は MOE `Dictionary of Variants`、original evidence、`Shuowen Jiezi` small seal evidence などを参照し、current T-source representative glyph が evidence や normal form と合うかを確認している。

提案は 22 numbered items で構成されるが、U+2976E / U+2278D / U+2C7AD や U+23329 / U+28765 のように複数 code points をまとめる項目があり、合計で 25 glyph issues になる。

## 提案内容

| 論点 | 内容 |
| --- | --- |
| target | 25 T-source representative glyphs。 |
| requested action | TCA に対して、glyph revision、source mapping exchange、normal form への correction を検討するよう求める。 |
| evidence | MOE `Dictionary of Variants`、original dictionary evidence、digital `Shuowen Jiezi` data など。 |
| representative examples | U+2976E / U+2278D / U+2C7AD の right part、U+43DE の T-source exchange、U+20885 の form / mapping、U+21595 の small seal evidence など。 |

## 後続決定

[IRG N2861](irg-n2861.md) は、TCA が MOE と相談したうえで `IRG N2860` に応答した feedback である。そこで 8 entries は no change、3 entries は mapping change、12 entries は modification、2 entries は pending と整理された。

pending だった U+2976E と U+20885 は、後続の [IRG N2927R](irg-n2927r.md) で再整理され、IRG Meeting \#66 recommendations と UTC \#187 CJK & Unihan recommendations に接続した。

## 論点

### evidence と member body convention

`IRG N2860` は evidence に見える形へ単純に chart glyph を合わせる提案を含むが、後続の TCA response は、current T-source glyph が TCA character table や standard form と一致する場合は no change と判断している。したがって、この文書は revision request の出発点であり、最終 disposition ではない。

### glyph change と source mapping change

U+43DE、U+4BCE、U+3862 などは、glyph を直すだけでなく T-source reference の割り当てを入れ替える提案として扱われる。これは representative glyph issue と source reference issue が同じ proposal 内に混在する例である。
