# IRG N1535

## 要約

`IRG N1535` は、UTC Extension D proposal の characters について、提出予定の complete evidence の sample を示す document である。source codes の意味は [IRG N1534](irg-n1534.md) を参照するとし、`UTC00097`、`UTC00098` などの U-source identifiers に対して dictionary references や radical / stroke data を並べる。

## データ内容

文書は、U-source identifier ごとに evidence glyph、source reference、radical / stroke value を提示する。PDF text extraction では glyph 部分が崩れる箇所があるが、document の役割は、UTC Extension D submission の evidence samples を IRG に見せ、complete evidence submission に先立って source traceability を説明することである。

## 後続決定

この wiki では、Extension D の accept/review outcome はまだ event 化していない。`IRG N1535` は、U-source record が実際の IRG submission に進むとき、UAX \#45 data とは別に evidence package が必要になることを示す文書として扱う。

## 論点

`IRG N1535` は、U-source database の identifier と source field が、IRG review でそのまま十分な evidence になるわけではないことを示す。UAX \#45 の record は candidate / source tracking の入口であり、IRG submission では glyph evidence、bibliographic source、attribute data が別途必要になる。
