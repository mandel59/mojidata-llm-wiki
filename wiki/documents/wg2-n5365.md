# WG2 N5365

## 要約

`WG2 N5365` は、WG2 Meeting \#73 向けの Script Encoding Initiative liaison contribution である。ISO/IEC 10646 CD.4 に取り込まれた script / glyph work、現在 WG2 に提出されている proposals、Unicode Script Encoding Working Group へ提出済みの proposals、今後数か月の pipeline を報告する。

WG2 \#73 では、SEI liaison report そのものは decision document ではないが、Maya、[Shaaldaa](../topics/shaaldaa-script.md)、Proto-Cuneiform、Mwangwego、Dai Xaau などの proposal pipeline を読むための入口になる。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| CD.4 に反映済みの作業 | Small Seal、Egyptian Hieroglyphs glyph updates、Sidetic naming conventions、Rejang glyph error などに触れる。 |
| WG2 submissions | Proto-Cuneiform、Sirmauri、Mwangwego、[Shaaldaa](../topics/shaaldaa-script.md) などを WG2 側の current submissions として挙げる。 |
| Unicode SEW submissions | UAX \#57 update、Minim Dag Noore、Ndiko Jonam、N'ti、Odùduwà、Codical Maya、Lampung、Vatteluttu などを挙げる。 |
| 今後の pipeline | Classic Maya、Kulitan、Masaba、Persian Siyaq、Proto-Sinaitic、Zou などの ongoing / expected proposals を示す。 |

## 後続決定

[WG2 N5354](wg2-n5354.md) の M73.14 は、Codical Maya Hieroglyphs `WG2 N5366` と Maya Hieroglyphs Extended-A `WG2 N5367` を WG2 \#74 までに review するよう National Bodies と liaison organizations に求めた。これらは [Maya Hieroglyph Encoding](../topics/maya-hieroglyph-encoding.md) で追う。

## 論点

### liaison report と proposal decision

`WG2 N5365` は SEI の活動報告であり、個別 proposal の採択ではない。WG2 \#73 recommendations では、Maya proposals の review request や first amendment candidates など、別 recommendation として扱われた事項だけを decision として読む。
