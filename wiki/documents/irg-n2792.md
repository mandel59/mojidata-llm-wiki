# IRG N2792

## 要約

`IRG N2792` は、ROK による script-hybrid Han ideographs の position paper である。議論が初期段階で consensus を欠いているため、encoding work を始める前に category、IRG scope、member body ごとの候補数を整理すべきだと述べる。

文書は、Korea の historical usage では Han と phonetic components を組み合わせた文字が Han character system の variant と見なされてきた例があるとし、作成・使用時期によって CJKUI inclusion を判断する可能性を示す。一方で、CJKUI に入れるか新 block にするかは IRG 内で議論すべきだとする。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| Han character として扱うか | Korea / Japan で 1900 年以前、または遅くとも 1945 年以前に使われた Han + phonetic script forms は、Han character system の一部と見なされた可能性があるとする。 |
| IRG scope | Han characters または Han-based chars である Kana / Bopomofo を含む script-hybrid characters は、CJKUI に入れるか新 block にするかに関わらず IRG が扱うのが望ましいとする。 |
| member body feedback | Japan は Kana を含む characters、Taiwan は Bopomofo、China は ethnic minorities 関係など、関係 member body が意見を出す必要があるとする。 |
| 候補数 | 各 member body が候補数を見積もり、数十・数百・千超のどの規模かで approach を変えるべきとする。Korea の新規候補はほぼ 0、多くても数十と見積もる。 |

## 後続決定

[IRG N2885R](irg-n2885r.md) は、`IRG N2792` を参照し、CJK Hybrid Characters の候補数見積もりと IRG P&P の扱いを再度質問した。

[IRG N2828](irg-n2828.md) と [IRG N2826](irg-n2826.md) は、IRG Meeting \#65 での議論を経て、Han-Hangul hybrid 例外を除く CJK Hybrid Characters は normal CJKUI とは別 block とする方向を記録した。

## 論点

### historical usage と CJKUI inclusion

`IRG N2792` は、script-hybrid characters を一律に CJKUI から除外するのではなく、作成・使用時期と member body の言語環境を考慮する余地を置いた。これは後続の [IRG N2893](irg-n2893.md) が示す Han-Kana / Han-Bopomofo exclusion policy より慎重な position である。

### member body ごとの responsibility

文書は、script-hybrid candidates の判断には関係 member body の書面意見が必要だとする。これは [CJK Hybrid Characters](../topics/cjk-hybrid-characters.md) の未決点である source owner / IRG gatekeeping と直結する。
