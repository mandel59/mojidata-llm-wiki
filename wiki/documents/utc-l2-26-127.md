# L2/26-127

## 要約

`L2/26-127` は、CJK Working Group に CJK confusables の追加候補 review を求める second request である。PDF 本文の日付は 2026-05-01、UTC registry 上の日付は 2026-05-04 である。

候補 pair は UCD data file `EquivalentUnifiedIdeograph.txt`、Unihan properties `kSpoofingVariant` / `kZVariant`、Windows / Android 上の glyph comparison から作られた。文書は Ken Lunde による不明確な pair の追加 review と追加提案も含むとしている。

## 論点

### 候補生成の根拠

`L2/26-127` は `EquivalentUnifiedIdeograph.txt`、`kSpoofingVariant`、`kZVariant` を候補生成の根拠にしている。これは CJK confusables の判断が、見た目だけでなく Unihan database と variant property の maintenance に接続していることを示す。

### platform glyph comparison

文書は Windows と Android で glyph comparison を行い、どちらかで十分に似て見える pair を候補にしたと説明する。一部の pair には Windows では違うが Android では同一に見える、またはその逆という注記が付くため、confusable 判定が実装 font / platform に依存する問題も含む。

### CJK WG の最終判断

候補は CJK Working Group が confusable と判断した場合に `confusables.txt` へ追加できる。`L2/26-082` と同じく、PAWG が data maintenance を担いつつ、CJK 専門判断を CJK WG に委ねる流れである。
