# IRG N2861

## 要約

`IRG N2861` は、TCA が [IRG N2860](irg-n2860.md) の 25 T-glyph issues に応答した member body contribution である。文書は、submitter が T-source glyphs と MOE `Dictionary of Variants` を比較したことを受け、TCA が MOE と協議して回答したと説明する。

結論は、8 entries は no change、3 entries は mapping change、12 entries は modification、2 entries は pending である。pending は U+2976E と U+20885 で、後続の [IRG N2927R](irg-n2927r.md) に引き継がれた。

## フィードバック内容

| 区分 | 件数 | 内容 |
| --- | --- | --- |
| No Change | 8 | name characters、TCA character table / documented source と一致する forms、standard form として維持する forms を変更しない。 |
| Mapping Change | 3 | U+43DE、U+4BCE、U+3862 について T-source references を別 code point / source value と入れ替える。 |
| Modification | 12 | U+201DC、U+2697A、U+20268 などの components を修正する。 |
| Pending | 2 | U+2976E と U+20885 はこの時点で conclusion なしとし、後続 response を出す。 |

## 後続決定

[IRG N2927R](irg-n2927r.md) は pending だった U+2976E と U+20885 に対する follow-up である。U+2976E は right part を 已 から 巳 に戻す glyph change、U+20885 は current glyph を維持する整理になった。

[IRG N2909](irg-n2909.md) M66.12 / [IRG N2911](irg-n2911.md) は、この T-source follow-up を IRG Meeting \#66 の decision として扱う。[L2/26-099](utc-l2-26-099.md) は Unicode Version 18.0 target の CJK & Unihan action item に取り込み、[WG2 N5354](wg2-n5354.md) は selected CJK additions and changes として ISO/IEC 10646 CD へ接続した。

## 論点

### no change の理由

`IRG N2861` の no change entries は、evidence に別 form があることだけでは representative glyph を変えないという判断を示す。TCA character table、documented source、standard form、name character handling が維持理由になる。

### mapping change と glyph modification の分離

TCA response は、source reference を移すべき cases と、glyph component を修正する cases を分けている。これは [IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md) で繰り返し現れる「reference を直すのか、chart glyph を直すのか」という判断枠組みの T-source 版である。
