# L2/26-117

## 要約

`L2/26-117` は、Catholic Albanian / Old Shkodran alphabet で使われた Latin letter edhe の大文字・小文字を符号化する提案である。Francesco Catoni と Kirk Miller は、1555 年の Buzuku `Meshari` から 20 世紀初頭までの Albanian Catholic printing に現れる historical letter として扱う。

提案は、modern Albanian の `dh` / `th` digraphs に対応する音価を持つが、歴史的文献では独立した文字として使われたことを根拠に、Latin Extended-G への追加を求める。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed characters | `LATIN CAPITAL LETTER EDHE`, `LATIN SMALL LETTER EDHE` |
| script / block | Latin / Latin Extended-G |
| usage | Catholic Albanian / Old Shkodran alphabet の historical text。 |
| notes | single letter は概ね /ð/、double letter は /θ/ に対応すると説明される。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は Catholic Albanian letter edhe を provisional assignment へ進める SEW recommendation に含める。

## 論点

### historical Latin letter と modern digraph

提案の焦点は、modern Albanian orthography で digraph になっている音価を、historical source では single letter として扱う必要がある点である。sorting は historical source によって揺れがあるため、文字の identity と collation behavior は分けて読む必要がある。
