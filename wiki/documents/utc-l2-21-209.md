# L2/21-209

## 要約

`L2/21-209` は、KORE SEBELI / Wakara script の 2021 年 proposal である。2020 年版と同じ 106 characters model を維持し、Soso language 用の contemporary Guinean script として repertoire、samples、community evidence を提示する。

この版は proposal chain の中間版で、後続の 2022 年版まで 106 characters model が続いたことを示す。2024 年版以降の 95 characters / 93 characters model と比較するための前段として重要である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script scope | Soso language 用の contemporary script。Koré Sébèli / Wakara として提示される。 |
| version | initial stage version 04。 |
| repertoire | 106 characters。digits、lowercase / capital letters、combining diacritics、punctuation、mathematical / other signs、vowel sequences を含む。 |
| direction | horizontal left-to-right を default とし、vertical direction を伝統的使用として説明する。 |
| evidence | teaching / literacy evidence、handwritten samples、font / syllabary samples を proposal に含める。 |

## 後続決定

この proposal 自体への UTC 決定は、この wiki では未確認である。後続の [L2/22-222](utc-l2-22-222.md) が 106 characters model を継続し、[L2/24-246](utc-l2-24-246.md) で 95 characters に再編される。

## 論点

### 106 characters model の継続

`L2/21-209` は 2020 年版からの大枠を維持している。KORE SEBELI の Unicode review では、文字集合の存在証明だけでなく、combining marks / atomic letters / sequences の境界が後続版で整理されていく点が重要である。
