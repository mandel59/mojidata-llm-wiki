# L2/26-145

## 要約

`L2/26-145` は Codical Maya Hieroglyphs を符号化する in-progress draft proposal である。対象は Postclassic codices、特に Dresden、Madrid、Paris codices に現れる standardized subset である。

提案は Codical Maya を Maya writing の最初の encoding tranche とし、496 signs を中心に、U+15500..U+156FF の 512 hieroglyph characters と U+15700..U+1571F の format controls を想定する。Classic Maya など earlier periods は future extensions として扱う。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| scope | Postclassic codical subset。Dresden、Madrid、Paris codices の hieroglyphs を対象にする。 |
| corpus size | 210 extant pages、7,122 glyph-blocks、17,147 graphemes という corpus statistics を示す。 |
| encoding model | Maya signs を abstract characters として扱い、glyph blocks / affixation に format controls を用いる。 |
| roadmap | Codical set を base とし、Classic Maya / other periods は extensions として追加する。 |
| script property | existing Maya numerals の Script=Common を Mayan_Hieroglyphs へ再割当てする必要を示す。 |

## 後続決定

2026-07-08 時点では、UTC \#188 agenda / minutes は未掲載であり、この draft proposal への UTC 決定は未確認である。Related proposal として `L2/26-146` Maya Hieroglyphs Extended-A が登録されている。

## 論点

### codical subset を base encoding にする

Codical Maya は Classic-period full repertoire より狭いが、surviving codices で standardized subset として比較しやすい。proposal は、まず codical set を符号化し、Classic Maya を extension として足す CJK / Egyptian Hieroglyphs に近い model を採る。

### format controls と glyph block composition

Maya hieroglyphs は quadrat / glyph block composition を持つため、単純な linear character sequence だけでは表現できない。format controls を導入する model は、encoded repertoire だけでなく rendering model の review を必要とする。
