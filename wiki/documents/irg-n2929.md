# IRG N2929

## 要約

`IRG N2929` は、China が 9156 encoded characters に G-source references を追加する horizontal extension proposal である。source は `GCW-Source` のみで、official geographical names、personal names、philological monographs、dictionaries、periodicals、local gazetteers などから収集した資料を根拠にする。

内訳は URO 18、Extension A 162、Extension B 3069、Extension C 421、Extension D 27、Extension E 378、Extension F 2796、Extension G 1543、Extension H 359、Extension J 383 とされる。新規 code point の追加ではなく、既存 CJK Unified Ideographs への source reference / glyph data 追加として扱う。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | G-source reference を持たない encoded Chinese characters 9156 字。 |
| source | `GCW-Source`。 |
| 根拠資料 | 中国の official geographical names、personal names、rare characters in academic databases、dialect documents、philological monographs、dictionaries、local gazetteers など。 |
| 作業種別 | Existing encoded characters への horizontal extension。 |

## 後続決定

[IRG N2909](irg-n2909.md) の M66.09 は、IRG experts が 2026-08-28 までに China へ feedback を送り、China が 2026-09-25 までに revised document `IRG N2929R` を提出する schedule を置いた。

## 論点

### horizontal extension の規模

`IRG N2929` は 9156 字規模の source data addition であり、U-source や UK-source の小規模 horizontal extension と比べて review workload が大きい。source data の追加は新規符号化ではないが、representative glyph、source evidence、Unihan source property maintenance に影響する。
