# L2/19-381

## 要約

`L2/19-381` は、現行 Unicode に欠ける Japanese Kana として、YI / WU / YE 系の historical kana と KATAKANA LETTER SMALL WU を提案した個人提案である。提案は、五十音図を埋めるために作られた文字、国語学資料で用いられた文字、外国語音写や古書 digitization で必要になる文字を扱う。

提案文字は、HIRAGANA LETTER YI、HIRAGANA LETTER WU、KATAKANA LETTER YI、KATAKANA LETTER YE、KATAKANA LETTER WU、KATAKANA LETTER SMALL WU の 6 characters である。文書は、近代国語学資料・仮名遣資料・文典類の画像を根拠として示し、一部文字については既存 hentaigana との違いや alias 案にも触れる。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字 | HIRAGANA LETTER YI / WU、KATAKANA LETTER YI / YE / WU、KATAKANA LETTER SMALL WU。 |
| 提案 block | Kana Extended-A と Small Kana Extension。 |
| 用途 | historical Japanese phonology、五十音図、近代国語学資料、外国語音写。 |
| 代替案 | HIRAGANA LETTER YI について、採用しない場合は U+1B006 HENTAIGANA LETTER I-1 への alias 案も示す。 |

## 論点

### "neat" kana と hentaigana の境界

提案は、HIRAGANA LETTER YI を U+1B006 HENTAIGANA LETTER I-1 の「整った」kana form と説明する。これは、hentaigana 由来の字形を regular kana として別符号化するか、既存 hentaigana の alias / usage として扱うかという境界問題を含む。

### small WU と youon model

KATAKANA LETTER SMALL WU は、ャ / ュ / ョ / ヮ のように他の kana に付いて consonant cluster や sound change を表す小書き文字として説明される。これは、`L2/17-091` が提示した small kana repertoire と modifier model の未決問題に接続する。
