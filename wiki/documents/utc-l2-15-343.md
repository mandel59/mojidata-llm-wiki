# L2/15-343

## 要約

`L2/15-343` は、Japan NB による HENTAIGANA revised proposal である。SC2/WG2 Meeting \#64 の議論を受け、`SC2 N4419` / `WG2 N4674` の attachment にあった Hentaigana repertoire を改訂したものとして提示された。

主な改訂点は、同じ ideograph に由来しながら異なる phonetic values を持つ HENTAIGANA を統合すること、そして character name を phonetic value と mother ideograph に基づく方式へ整理することである。提案表は、revised number、旧番号、統合対象、glyph、mother ideograph、phonetic value を対応付ける。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | HENTAIGANA repertoire の revised chart。 |
| 統合基準 | 同一 mother ideograph に由来する hentaigana を統合し、phonetic value を normative name に反映。 |
| 命名 | `HENTAIGANA LETTER A-1` のように phonetic value と serial number を基本にし、mother ideograph は informative information とする。 |

## 論点

### mother ideograph と phonetic value

この文書は、hentaigana の identity を単なる glyph variation ではなく、母字・音価・資料上の用法の組み合わせとして整理する入口である。たとえば E 系には 江、盈、縁、衣、要 由来の複数形が含まれ、YE proposal で問題になった「江 由来の字形を historical letter と見るか hentaigana と見るか」という論点にも接続する。

### 後続 revised repertoire

`L2/16-188` はこの整理をさらに進め、HENTAIGANA LETTER E-1 を withdrawn とし、U+1B001 に name alias を追加する提案と接続した。したがって `L2/15-343` は、後の Kana Supplement / Kana Extended-A における hentaigana repertoire 整理の中間段階として読む。
