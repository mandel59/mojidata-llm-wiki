# L2/24-067

## 要約

`L2/24-067` は、2024-04-05 の CJK & Unihan Working Group meeting に基づく UTC Meeting \#179 向け recommendations である。UTC \#178 以降の PRI \#483 feedback、public feedback、UAX \#38 / Unihan database documents、UAX \#45 / U-source documents、その他 CJK proposal を 44 sections で扱う。

文書は、`kMandarin` additions / changes、`kTotalStrokes` / `kRSUnicode` corrections、`kDefinition` syntax revert、`kZhuang` property、G-source reference updates、UAX \#45 FutureWS additions を UTC action に変換した。後続の [L2/26-084](utc-l2-26-084.md) は、この文書で CLDR-TC review に回された `kMandarin` items の一部に対する feedback として読める。

## 決定・Action Item

| 区分 | 決定・Action Item |
| --- | --- |
| PRI \#483 / UAX \#38 | U+2010E / U+5EF4 の `kRSUnicode` / `kTotalStrokes` corrections、`kDefinition` delimiter / syntax change の revert、複数の provisional property fixes を勧告した。 |
| `kMandarin` review | 欠落 value、5 characters の proposed value changes、152 toneless values、U+8034 / U+43B2 などを CLDR-TC に確認させる action items を置いた。 |
| Unihan properties | Zhuang readings 用 provisional property `kZhuang` を受け入れ、UAX \#38 documentation と Unihan database population を action item 化した。 |
| UAX \#45 / U-source | Putian-Xianyou Puxianxi Opera、fish / fish product、Cantonese education、Cantonese particle、All Men Are Brothers / Classic of Mountains and Rivers などの提案から、UTC-03399 から UTC-03487 までの 89 records を `FutureWS` として受け入れた。 |
| Other CJK topics | East Asian auto-spacing は Properties & Algorithms Working Group 向きとし、CJK Strokes standardized variation sequences や Suzhou numerals も CJK-adjacent issues として扱った。 |

## 後続決定

[L2/24-165](utc-l2-24-165.md) は、UTC \#180 向けにさらに `kMandarin` review requests、UAX \#45 additions、WG2 \#71 の Extension J / U+5CC0 disunification follow-up を扱った。

[L2/26-084](utc-l2-26-084.md) は、`L2/24-067` を含む 2024 年 CJK & Unihan recommendations から生じた `kMandarin` action items へ CLDR TC / Apple が返した feedback である。[L2/26-099](utc-l2-26-099.md) Section 22 は、その 169-character feedback を Unicode 18.0 target の update として進めるよう UTC \#187 に勧告した。

## 論点

### `kDefinition` は free-form property として残す

[L2/24-012](utc-l2-24-012.md) で提案された `kDefinition` delimiter / syntax change は、dictionary punctuation と delimiter の混同、parentheses 内 semicolon、既存 data の規模を理由に revert された。Unihan property の機械可読性を上げる場合でも、既存 free-form data の意味を壊さない境界が重要になる。

### `kMandarin` は CLDR review と UTC action の往復になる

`L2/24-067` は、CJK & Unihan Working Group が proposed value を妥当と判断しても、`kMandarin` は CLDR-TC check を経由して戻す運用を明確にしている。これは後の `L2/26-084` のような feedback document を生む。

### UAX \#45 intake と IRG Working Set 2024

UTC-03399 から UTC-03487 までの `FutureWS` additions は、UTC が IRG Working Set 2024 へ渡す候補 pool を形成した。UAX \#45 の `FutureWS` は immediate encoding ではなく、IRG submission 前の public tracking state である。
