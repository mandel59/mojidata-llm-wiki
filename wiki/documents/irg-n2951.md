# IRG N2951

## 要約

`IRG N2951` は、FS（First Stroke）と SC（Stroke Count）の consolidated guidelines である。Eiso Chan が複数の IRG experts / member bodies からの feedback を反映し、previous related documents と comments を統合したものとして提出した。

文書は 138 entries を持ち、entry syntax、shape、variants、FS、SC の扱いを定義する。FS values は Annex K の values と dotted circle を使い、SC では dotted circle component を含めて数える不確定値の扱いを示す。

## データ内容

| 区分 | 内容 |
| --- | --- |
| entries | 138 entries。 |
| entry syntax | `\d{1,2}[DHNPSZ][a-z]`。数字は SC、upper Latin letter は FS、lower Latin letter は ordering。 |
| FS values | H=横、S=竪/豎/竖、P=撇、D=點/点、Z=折、N=◌。 |
| replaced documents | `IRG N954AR`、`IRG N1105`、`IRG N2171`、`IRG N2221`、`IRG N2862`、`IRG N2862R2` などを参照。 |

## 後続決定

[IRG N2935](irg-n2935.md) は、`IRG N2951` が 2026-04-02 に posted され、IWDS Series 1 に追加されたと記録する。IRG Indexing Rules では、UCV / NUCV lists、radical assignment rules と並ぶ review infrastructure として扱う。

## 論点

### review infrastructure としての FS / SC

`IRG N2951` は直接の encoding proposal ではなく、IRG review で使う component shape、first stroke、stroke count の共通参照である。WS2024 や future working sets の attributes / discussion record を安定して扱うための基盤として読む。
