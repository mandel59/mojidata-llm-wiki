# L2/25-259

## 要約

`L2/25-259` は Squinting Face emoji の proposal である。Jon Magram は、squinting を suspicion、inspection、focus、glare、scrutiny、vision difficulty などを表す facial expression として提案する。

UTC \#185 では U+1FAEB FACE WITH SQUINTING EYES として short list に入ったが、[L2/26-008R](utc-l2-26-008r.md) は additional scrutiny が必要として、同 code point の name / concept を CRACKING FACE に変更した。したがってこの document は採択済み proposal ではなく、replaced candidate の背景として読む。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | SQUINTING FACE / FACE WITH SQUINTING EYES |
| keywords | accuse、concentrate、focus、glare、inspect、scrutinize など。 |
| selection rationale | existing squinting-name emoji は tightly shut eyes で、suspicion / scrutiny の squint とは違うと主張する。 |
| later status | U+1FAEB は CRACKING FACE に変更された。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FAEB FACE WITH SQUINTING EYES を Unicode 18.0 additions に入れたが、[L2/26-008R](utc-l2-26-008r.md) は U+1FAEB の name を CRACKING FACE に変更した。置き換え後の rationale は [L2/26-048](utc-l2-26-048.md) で読む。

## 論点

### short list と final candidate の差分

Emoji pipeline では、UTC が short list を受け入れた後でも name / concept が再検討されることがある。`L2/25-259` は、proposal title と final candidate name が一致しない例である。
