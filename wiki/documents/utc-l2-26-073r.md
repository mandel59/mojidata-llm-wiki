# L2/26-073R

## 要約

`L2/26-073R` は、CJK Strokes characters に standardized variation sequences を追加する revised proposal である。`L2/26-099` は、この proposal に基づく 17 standardized variation sequences を Unicode Version 18.0 target で受け入れる action item を記録している。

この文書は、CJK Unified Ideographs の source data ではなく、CJK Strokes block の presentation variant を `StandardizedVariants.txt`、code charts、Core Specification Appendix F に反映するための入力文書である。

## 決定・Action Item

| 区分 | 内容 |
| --- | --- |
| 対象 | CJK Strokes characters の standardized variation sequences。 |
| 後続処理 | `L2/26-099` により、Unicode Version 18.0 target で `StandardizedVariants.txt`、Pipeline、code charts、Core Specification Appendix F の更新対象になった。 |

## 論点

### Unicode 18.0 data update との関係

`L2/26-073R` 自体は proposal であり、UTC #187 向けの recommendation bundle である `L2/26-099` が実際の data / chart update へ接続する。Unicode 18.0 の CJK-adjacent changes を確認する場合、CJK Strokes variation sequences は Unihan database や UAX #45 とは別の data file update として確認する必要がある。
