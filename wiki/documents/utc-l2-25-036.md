# L2/25-036

## 要約

`L2/25-036` は、KATAKANA LETTER SMALL ARCHAIC YE を Small Kana Extension に追加する提案である。提案文字は U+1B121 KATAKANA LETTER ARCHAIC YE の小書き形として位置づけられ、Marshallese transliteration などに使われた昭和期資料が根拠として提示される。

文書は、2016 年の small kana proposal、Japan NB の [L2/17-091](utc-l2-17-091.md)、Small Kana Extension に確保された将来用 code points に言及し、次の空き位置として U+1B168 を提案する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字 | `1B168; KATAKANA LETTER SMALL ARCHAIC YE; Lo` |
| 提案 block | Small Kana Extension |
| 用途 | Marshallese transliteration など、昭和期の仮名による外国語音写。 |
| 関連 property | Line_Break `CJ`、Vertical_Orientation `Tu` が提案されている。 |

## 論点

### small archaic YE と既存 sequence

提案は、キェ / ジェ のような modern sequence と区別される small archaic YE usage を示す。資料では small エと small archaic YE が異なる vowel を表す例として扱われ、単なる字体差ではなく phonetic contrast の根拠として提示されている。

### Japan NB の small kana 論点との関係

[L2/17-091](utc-l2-17-091.md) は、小書き仮名の全体 repertoire と encoding model を先に検討すべきだと述べた。`L2/25-036` は個別文字提案だが、その正当性は「small kana を個別に追加し続けるのか」という大きな設計問題と切り離せない。
