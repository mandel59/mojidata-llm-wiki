# PRI 548

## 要約

`PRI #548` は、UTC が Unicode 18.0.0 Beta の public review を受け付けた Public Review Issue である。PRI page の status は Closed、closing date は 2026-07-07。

この PRI は、Unicode 18.0.0 beta information page を正式な review 対象として示し、feedback は Unicode contact form から提出する形を取る。issue 自体は変更提案ではなく、Unicode 18.0.0 beta package への review gate である。

beta page は、Unicode 18.0.0 が 2026-09-16 release planned、13,047 new characters、4 new scripts、UCD / charts / emoji charts / UAX / UTS proposed updates を含む review package であることを示す。

## 通知内容

### review 対象

PRI page は Unicode 18.0.0 beta information page を指し、そこから summary description、beta UCD、single-block delta charts、all-block charts、consolidated charts、emoji charts、auxiliary HTML charts、UAX / UTS proposed updates へ進める。

beta page は、reviewers に beta UCD data files と code charts の確認を促す。特に、新規追加文字だけでなく、既存文字の glyph regressions、property assignments、Notable Issues for Beta Reviewers を確認対象としている。

### review window と stability

comment period は 2026-07-07 に終了した。beta page は、substantive technical comments は July UTC meeting で検討するためこの日までに受け取る必要がある一方、typo など editorial comments は final editorial work 向けに後日も提出できると説明する。

同時に、Unicode 18.0.0 の character assignments は stable になっており、code points と character names への変更要求は actionable ではないと明示する。beta review の主対象は、preliminary character property assignments、data files、charts、core specification text である。

### beta reviewer の主な確認点

beta page の notable issues は、core specification text、segmentation、script-specific issues、general character properties、numeric properties、security / identifiers、Unihan、standardized variation sequences、code charts、collation、linkification、data file reorganization、stability を列挙する。

Unicode 18.0.0 の新 script は Chisoi、Jurchen、Seal、Proto-Cuneiform である。Jurchen と Seal は large siniform ideographic scripts として扱われ、Seal は 11,328 new characters を含む。CJK / Unihan 周辺では、Unihan properties の removal / addition / delimiter changes、source reference changes、horizontal extensions、CJK Strokes variation sequences、Han glyph updates が review 対象になる。

## 後続決定

`PRI #548` は 2026-07-07 に closed になっている。Unicode 18.0 content finalize は [UTC Meeting \#188](../meetings/utc/utc-meeting-188.md) で 2026-07-30 に予定されているため、この PRI で集まった substantive feedback の処理は UTC \#188 minutes / action items と最終 release data で確認する必要がある。

[L2/26-102](utc-l2-26-102.md) は、Unicode 18.0 beta review の開始、public review close、UTC \#188 content finalize を予定として示した RMG report である。この PRI は、その schedule 上の beta public review close point に対応する。

## 論点

### Source Document としての位置づけ

この page は Unicode 18.0 の変更点そのものではなく、beta review package への official entry point である。変更点の実体は beta UCD、delta charts、UAX / UTS proposed updates、UTC / WG2 / IRG 文書に分散するため、[Unicode 18.0 Change Sources](../topics/unicode-18-change-sources.md) では PRI と個別 change sources を分けて読む。

### UTC \#188 への接続

`PRI #548` の close は UTC \#188 content finalize の直前に置かれている。UTC \#188 tracking では、agenda / minutes が出るまで、beta feedback snapshot と post-beta candidate documents を分けて管理する必要がある。
