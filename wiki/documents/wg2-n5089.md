# WG2 N5089

## 要約

`WG2 N5089` は、2019-06-01 に Beijing Normal University で開かれた Shuowen Seal Informal Meeting の報告であり、`WG2 N4853`、`WG2 N4973`、`WG2 N4922` から続く8論点について参加者の合意と proposal data の修正方針を記録する。

会合は Shuowen Guwen / Zhouwen の別符号化を Shuowen 収録字に限定して認め、15組の Chongwen は各組1字のみを符号化する方針を維持した。原刻の誤り・欠損、taboo deformation、N4688 glyph や TTF font の不整合については、他版本と字理を参照した representative glyph の修正や IVS / IVD の利用を整理した。

これは informal meeting の consensus report であり、WG2 plenary の正式な disposition ではない。後続の [WG2 N5108](wg2-n5108.md) は WG2 \#68 で、他版本と taxonomy の追加検討が必要で CD / CDAM text への収録には早いと報告した。

## 報告内容

| 論点 | 会合での整理 |
| --- | --- |
| Guwen / Zhouwen | Shuowen Guwen と Shuowen Zhouwen を Zhuanwen と別に符号化する。ただしこの整理は `Shuowen` 収録字に限り、他資料の Chuanchao Guwen へ一般化しない。Suzuki Toshiya は説明後にこの方針を受け入れた。 |
| Chongwen | 専門家確認で15組を特定し、各組の一方だけを符号化する一覧を示した。 |
| possible duplicates | 非常に近い2組では、区別を明瞭にするため Tenghuaxie glyph を Duan version または Beijing Normal University Small Seal font の glyph へ差し替える。 |
| etymological errors | Tenghuaxie version の原刻誤りと判断した5字は、説明された component と他の DaXu versions を根拠に corrected representative glyph を proposal で使う。 |
| broken / inconsistent glyphs | 擦れ等による欠損・不整合11字は字理と他版本から修正し、壊れた原形を別 code point にしない。互換・教育・比較用途には IVD / IVS を推奨する。 |
| taboo deformation | 既報8字に3字を加えた11字を扱う。改字型は他版本を参照して復元し、欠筆型は欠筆しない原形を代表字形として他形を IVS / IVD で扱う。 |
| N4688 inconsistencies | source を中國書店 reprint から original Tenghuaxie version へ改めて再照合し、[WG2 N4688](wg2-n4688.md) の113件の不一致を修正したと報告する。 |
| TTF errors | original glyph との比較で見つけた TTF font error 60件を修正した一覧を付す。 |

## 後続決定

この文書自体は proposal data の作業方針を記録する informal meeting report であり、WG2 の正式な recommendation や repertoire 採択ではない。2019-06 の [WG2 N5108](wg2-n5108.md) は WG2 \#68 の議論として、Zhuanwen / Guwen / Zhouwen の基本的な disunification と非常に似た同義 glyph の unificationを報告する一方、他 versions、future extension、taxonomy / ordering を未決として残した。

## 論点

### 合意の適用範囲

Guwen / Zhouwen の扱いは `Shuowen` 内の体系に限定された合意であり、他資料に現れる古文を同じ script collection として一括符号化する結論ではない。また、壊れた原字形を保存する必要性と新規 code point の必要性を分け、前者には variation mechanism を使う方針を示した。

### Representative glyph と source fidelity

文書は original Tenghuaxie version を source evidence としつつも、誤刻、欠損、taboo deformation まで無修正で representative glyph にする立場は採らない。他版本、字義、component consistency による修正を認めており、source transcription と encoded representative glyph の境界が主要論点になっている。

### 未決事項

会合は8論点に実務上の回答を与えたが、複数版本を横断する source model、stable taxonomy / ordering、future extension、WG2 encoding text へ進める成熟度は決定していない。これらは N5108 以降の WG2 discussion に持ち越された。
