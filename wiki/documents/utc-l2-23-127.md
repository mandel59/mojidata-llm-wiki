# L2/23-127

## 要約

`L2/23-127` は、明治期の活字見本に現れる 2 つの unencoded Katakana letters を紹介する report である。Eiso Chan は、東アジア近代 typography に関する書籍編集の過程で、Komiyama Hiroshi の論文に掲載された画像から、子 由来の NE と 井 由来の WI を確認したと述べる。

文書は proposal summary form を持たず、running text の用例もまだ確認できていないと明記する。そのうえで、活字見本が多くの書籍で使用される可能性のある font samples であることを根拠に、encoding の扱いを議論すべきだと呼びかけた。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| 報告対象 | current Katakana NE / WI に対応する unencoded Katakana letters 2 文字。 |
| 字源 | NE は 子、WI は 井 として示される。 |
| 根拠 | 1903 年の『活版見本』と刊行年不明の『活版略見本』に由来する図版。 |
| 留保 | running text の用例は示されておらず、さらなる専門家の情報提供を求める。 |

## 論点

### report から proposal への接続

`L2/23-127` は formal proposal ではなく、unencoded Katakana letters の existence report として読むべき文書である。後続の `L2/25-151R` は、この report を前段として、明治期活字・教科書資料などの evidence を拡充し、KATAKANA LETTER ALTERNATE NE / WI として正式に提案した。

### Han 由来字形と Katakana identity

NE / WI は 子 / 井 と視覚的に近く、後続 proposal では confusable pair としても扱われる。一方で、文書はこれらを Katakana repertoire の historical letters として捉えるため、Han-derived glyph と Katakana encoded character の境界が論点になる。
