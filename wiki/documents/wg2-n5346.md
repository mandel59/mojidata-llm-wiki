# WG2 N5346

## 要約

`WG2 N5346` は、TCA and China が [WG2 N5341](wg2-n5341.md) の Small Seal code chart / data set を review した feedback document である。yellow-highlighted sections について 43 code point issues を整理し、さらに corresponding modern CJK unified ideographs の comprehensive review により 3,121 entries の correction を提示する。

Modern CJK correction は、Small Seal glyph に最も近い CJK character form を採用する方針によるもので、以前の corresponding modern Chinese characters よりも形が近いが、meaning は同じと説明される。後続の [WG2 N5344R2](wg2-n5344r2.md) は、この feedback を取り込んだ revised proposal として位置づけられる。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| code point adjustments | new code points は THX version の後に統一して配置するべきとして、24 code points の調整を示す。 |
| code point deletions | [WG2 N5307R](wg2-n5307r.md) Unification Principle 2.(5) に基づき、11 code points は元 code point へ戻し、1 duplicate encoding は削除すべきとする。 |
| glyph mapping adjustments | CCZ / QJZ / DYC など other versions の mapping changes 8 code points を示す。 |
| THX column update | N5317R3 の 199 additions から 12 code points を削除し、新たに disunified される THX column entries は 187 characters になると説明する。 |
| Modern CJK | Appendix 2 と XLSX で 3,121 modern CJK equivalent corrections を提示する。 |

## 後続決定

`WG2 N5346` は code chart feedback であり、会議 recommendation ではない。後続の [WG2 N5344R2](wg2-n5344r2.md) は N5346 などを取り込んだ revised proposal として 11,328 characters / `3D000..3FC3F` の block allocation を提示した。WG2 \#73 の [WG2 N5369](wg2-n5369.md) では、さらに `kSEAL_Rad`、`kSEAL_MCJK`、glyph correction が CD.4 disposition に反映された。

## 論点

### Modern CJK equivalent の性質

3,121 entries の correction は、`kSEAL_MCJK` が character identity の本体ではなく、検索・索引・照合のための data であることをよく示す。Small Seal glyph に近い modern CJK form を選ぶ方針は、一般的な現代漢字の使用頻度よりも shape correspondence を重視する。

### phono-semantic compound の統一範囲

文書は、phono-semantic compounds の variant forms について、top-bottom / left-right structure にかかわらず N5307R Unification Principle 2.(5) を適用する方向を示す。これは将来追加時の一貫した基準を作るための整理である。
