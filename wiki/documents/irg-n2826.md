# IRG N2826

## 要約

`IRG N2826` は、2025-10-13/17 に Guangzhou, China で開催された IRG Meeting #65 の recommendations and action items である。この wiki では特に、CJKV Components に関する M65.23 と、CJK Hybrid Characters に関する M65.24 を参照する。

M65.24 は、Latin、Hiragana、Katakana、Bopomofo などを部品に含む Han-like characters を CJK Hybrid Characters として扱い、原則として CJK Unified Ideographs extension blocks とは別 block に符号化する方向を示した。

## 決定・Action Item

| Recommendation | 内容 |
| --- | --- |
| M65.23 | `IRG N2878R` の CJK Unified Ideographs Components と Jianzi Musical Notation Components を review し、authors に final source references、`kRSUnicode`、`kTotalStrokes`、representative glyphs を含む R2 版の提出を求めた。 |
| M65.24 | CJK hybrid characters の encoding を WG2 に委ねつつ IRG oversight を残し、IRG が CJKUI extension block か CJK Hybrid Characters block かを判断する consensus を記録した。 |

## 論点

### CJK Hybrid Characters

M65.24 は、CJK hybrid character の定義に Latin、Hiragana、Katakana、Bopomofo を部品に含む Han-like characters を含める。これらは、部品が Han components のように manifest される場合でも対象になる。

同 recommendation は、well-attested Korean ideographs with Hangul-based phonetic components を例外とし、それ以外の CJK hybrid characters は CJKUI extension blocks とは別 block、すなわち Plane 2 / Plane 3 ではない block に置くべきとする。

### Unihan properties

UTC が CJK Hybrid Characters に Unihan properties を付ける場合、IRG review process で IRG source references、Kangxi radical assignments、stroke counts、readings などの規則を議論できると整理している。
