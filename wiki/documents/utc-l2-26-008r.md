# L2/26-008R

## 要約

`L2/26-008R` は、Emoji Standard & Research Working Group が UTC \#186 に提出した revised report である。UTC \#185 で承認された Unicode 18.0 emoji candidates のうち、U+1FAEB FACE WITH SQUINTING EYES を CRACKING FACE に変更する consensus / action item を記録する。

Report は、最終的に Unicode 18.0 に入る emoji list として、LIGHTHOUSE、METEOR、ERASER、NET WITH HANDLE、MONARCH BUTTERFLY、PICKLE、CRACKING FACE、LEFTWARDS THUMB SIGN、RIGHTWARDS THUMB SIGN を列挙する。さらに、ESR と CLDR の keyword intake、emoji review calendar、今後の proposal handling も扱う。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| name change | U+1FAEB を FACE WITH SQUINTING EYES から CRACKING FACE に変更する。 |
| final candidates | Unicode 18.0 に含める 9 emoji characters を更新後の名前で示す。 |
| CLDR keyword intake | emoji keyword の追加・削除 request が ad hoc に増えており、ESR / CLDR の処理負荷と intake process を整理する。 |
| calendar | April-July を review window とする emoji calendar と UTC submission timing を説明する。 |

## 後続決定

[L2/26-098](utc-l2-26-098.md) は、UTC \#187 ESR report として、Unicode 18.0 feedback、monarch butterfly の interoperability feedback、Emoji 19.0 intake window を扱う。[L2/26-104](utc-l2-26-104.md) は UTS \#51 Revision 30 proposed update として Emoji 18.0 draft specification を更新する。

## 論点

### final candidate name は proposal title と一致しない

`L2/25-259` は Squinting Face proposal だが、UTC \#186 の result は CRACKING FACE である。proposal page の title と最終 character name が違うため、[Emoji Repertoire Proposals](../topics/emoji-repertoire-proposals.md) では revised final list を別表として残す。

### CLDR keyword requests

Emoji search は character encoding では完結せず、CLDR short names / keywords に依存する。この report は、keyword request を単発の要望ではなく、ESR と CLDR の intake process として扱う必要を示している。
