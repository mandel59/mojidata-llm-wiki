# L2/19-282

## 要約

`L2/19-282` は、Unihan database の `kSpoofingVariant` field に入れる初期 data として 48 pairs of ideographs を提案した文書である。author は John H. Jenkins、文書本文の日付は 2019-07-18 である。

文書は、listed pairs を `kSpoofingVariants` として confidently labeled できる preliminary data set と説明し、later submission で増補され得るとしている。提案は、appropriate symmetrical relationships を持つ形で Unihan database に追加することを求める。

同日提出の [L2/19-281](utc-l2-19-281.md) は、対応する `kZVariant` field の 93-pair data set を提案している。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | 48 ideograph pairs を `kSpoofingVariant` data として Unihan database に追加する。 |
| 関係 | pair は symmetrical relationships として扱う。 |
| 後続 | `L2/19-328` が、この data の review 不足と usability issue を指摘した。 |

## 論点

### CJK security data と Unihan property の接点

`kSpoofingVariant` は、CJK ideographs の visually confusable relationship を Unihan property として持つため、UTS \#39 `confusables.txt` とは別の data layer で security / spoofing concern を表現する。2026 年の `L2/26-127` は、この `kSpoofingVariant` を CJK confusables candidate generation の根拠の一つにしている。

### data review の必要性

`L2/19-282` は初期 data proposal だが、後続の `L2/19-328` は data の validness と review process に問題があったと指摘した。この流れは、CJK confusables では専門 review と text-based data review が重要であることを示す。
