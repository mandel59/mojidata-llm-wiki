# IRG N2828

## 要約

`IRG N2828` は、IRG Meeting \#65 の CJK Editorial Group report である。disunification / unification、horizontal extension、glyph and source reference change、UCV updates、CJK Hybrid Characters、IRG P&P update などをまとめている。

この wiki では主に CJK Hybrid Characters section を参照する。report は `IRG N2792`、`IRG N2866R2`、`IRG N2867R`、`IRG N2885R`、`IRG N2893` を review し、Han-Hangul hybrid characters を除く CJK Hybrid Characters を normal CJKUI とは別 plane / 別 block に置くことを suggested と記録した。

## 報告内容

| 節 | 内容 |
| --- | --- |
| Disunifications and unifications | U+2A50D / U+21694 などについて Japan response により no change とする項目を含む。 |
| Horizontal extension | JMJ Horizontal Extension issues について Japan response により no change とし、HKSAR / China / TCA / ROK horizontal extension requests も扱う。 |
| Glyph and source reference change | G / K / T / V / SAT などの source reference and glyph changes を整理する。 |
| CJK Hybrid Characters | 関連文書を review し、Han-Hangul hybrid exceptions を除く CJK Hybrid Characters を normal CJKUI とは別 block に置くこと、property assignment rules は IRG で議論することを提案する。 |

## 後続決定

[IRG N2826](irg-n2826.md) の Recommendation M65.24 は、`IRG N2828` の editorial conclusion と接続し、CJK hybrid characters の encoding を WG2 に委ねつつ IRG oversight を残す consensus を記録した。

## 論点

### editorial conclusion と recommendation の関係

`IRG N2828` は CJK Editorial Group の report であり、M65.24 そのものではない。CJK Hybrid Characters の canonical decision point は [IRG M65.24 CJK Hybrid Characters](../events/irg-m65-24-cjk-hybrid-characters.md) に置き、`IRG N2828` はその背景 report として読む。

### CJK Hybrid と周辺 issue の同居

この report は CJK Hybrid Characters だけでなく、horizontal extension、source reference、glyph changes も同時に扱う。CJK Hybrid Characters が IRG source data / CJKUI process の境界問題として議論されていたことを示す文書である。
