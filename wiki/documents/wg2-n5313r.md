# WG2 N5313R

## 要約

`WG2 N5313R` は、TCA and China が [WG2 N5306](wg2-n5306.md) を review し、"Consolidated list of 19 variant sets" と "Consolidated table to study disunification cases" へ feedback を返した revised document である。

Appendix 1 は 19 variant sets の処理案を示し、`WG2 N5312` Part 1 を置き換える。Appendix 2 は 457 cases の disunification study table に対する expert opinions で、C / K / D sources を持つ 200 cases は別符号化を推奨するとまとめる。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| Appendix 1 | 19 variant sets について、code point change、削除候補、source duplicate、Guwen / Zhouwen / Zhengzhuan の扱いを整理する。 |
| Appendix 2 | 457 cases の disunification table に対し、separate coding、unification、mapping movement、THX glyph correction の判断を付ける。 |
| principle reference | disunification / unification principles は [WG2 N5307R](wg2-n5307r.md) を参照する。 |
| glyph correction | N5312 Appendix 2 の 4 glyphs に加え、`毒` shape に基づく 2 glyphs を含め、6 THX glyph corrections を示す。 |

## 後続決定

`WG2 N5313R` は、[WG2 N5317R3](wg2-n5317r3.md) の disposition 入力の一つである。Suzuki Toshiya は [WG2 N5327](wg2-n5327.md) でこの文書に comment し、TCA は [WG2 N5337](wg2-n5337.md) で response を返した。WG2 \#72 の M72.13 は、N5306 / N5317R3 / N5318R / N5341 を根拠に Small Seal を future edition candidate として扱う。

## 論点

### 200 separate coding cases

この文書は、N5306 の review convergence を実際の data correction に近づける資料である。200 cases の separate coding 推奨は、Small Seal が単なる font variation ではなく、source tradition と component structure の差を character identity として扱うべき場合があることを示す。

### mapping movement と別符号化の区別

Appendix 2 では、source が別 character に対応していた mapping error と、同じ source group 内で別符号化すべき structural difference を分けて扱う。この区別は後続の code chart と database update で重要になる。
