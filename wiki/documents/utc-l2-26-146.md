# L2/26-146

## 要約

`L2/26-146` は Maya Hieroglyphs Extended-A を符号化する in-progress draft proposal である。Classic Maya corpus に現れる signs のうち、Codical set と parallel representation を持たない signs を first tranche として追加する。

文書は Codical Maya proposal [L2/26-145](utc-l2-26-145.md) に依存し、Codical and Classic Maya Hieroglyphs を unified encoding として扱う。Classic / Codical の差は font level で扱い、Codical set にない signs を Extended-A で追加する方針である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| scope | Classic Maya corpus の first extension tranche。 |
| dependency | Codical set と format controls は [L2/26-145](utc-l2-26-145.md) に依存する。 |
| block | Maya Hieroglyphs Extended-A として U+15720..U+1578F 付近を想定する。 |
| unification model | Codical and Classic Maya を unified encoding とし、parallel representations は font level で扱う。 |
| future work | Classic Maya signs は今後の regular updates / tranches で拡張される見込み。 |

## 後続決定

2026-07-08 時点では、UTC \#188 agenda / minutes は未掲載であり、この draft proposal への UTC 決定は未確認である。

## 論点

### unified Maya encoding

この proposal の中心は、Codical Maya と Classic Maya を別 script / 別 encoding とせず、unified Maya Hieroglyphs として扱う点である。CJK 的な extension model と同様に、base set、extension set、font-level glyph style の分担が review point になる。
