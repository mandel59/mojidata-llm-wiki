# IRG N1133

## 要約

`IRG N1133` は、TCA が IRG に提出した CNS 11643 Planes 1 & 2 の basic components と decomposition rules の入力文書である。CNS 11643-2 の English translation として、Chinese character processing、analysis、structure description、combination、data exchange に使う basic components を定義する。

文書は、component set は decomposition rules に依存するため、IDS に取り組む前に decomposition rules と corresponding basic components が必要だと述べる。CMEX が CNS 11643-2 に基づいて Planes 1 & 2 の structural attributes に取り組んでいることも記録している。

## データ内容

| 区分 | 内容 |
| --- | --- |
| scope | CNS 11643 Planes 1 & 2 の Chinese characters を対象に、basic components を document processing / analysis / structure description / data exchange へ使う。 |
| definitions | grapheme、stroke、stroke feature、stroke sequence、component、prime / subordinate component、character formation / non-formation component、basic / compound component を定義する。 |
| analysis principles | actual writing に基づく component analysis、variable radicals、linked components、Kangxi radicals の再分析可否、overlap や minor parts の扱いなどを規定する。 |
| component table | 13,051 characters を分析し、517 characters / 384 sets of basic components を示す。 |

## 後続決定

`IRG N1133` は現行 CJKV Components proposal の直接前版ではないが、[IRG N2799](irg-n2799.md) が TCA-CNS 11643-2 を source data として含める前史にあたる。Meeting \#64 の [IRG N2765](irg-n2765.md) では TCA が CMEX CJK Components Review Tool を demo し、TCA source data と review tool が CJKV Components の development process に入っていく。

## 論点

### decomposition rule と component repertoire

`IRG N1133` の重要点は、component repertoire が decomposition rules に依存すると明示していることである。これは、単に component glyphs を列挙するだけではなく、どの規則で characters を分解するかを決めなければ component set も安定しないという CJKV Components 全体の問題につながる。

### prime / subordinate component

prime component と subordinate component の区別は、後の source references や component unification の判断に近い問題を含む。component の minor variation を同じ set に入れるか、別 component として扱うかは、CJKV Components の first batch / future batch 判断とも接続する。
