# IRG N1519

## 要約

`IRG N1519` は、China が CJK_D candidates について source、meaning、usage などの evidence を IRG editors 向けに提示した data document である。表紙は、文書を source ごとの複数 part に分け、`Ci Yuan`、`Hanyu Da Zidian`、`Hanyu Fangyan Da Cidian`、`Kangxi Zidian` などを evidence source として列挙している。

文書は evidence が見つからなかった candidate を CJK_D から withdraw する方針にも触れており、単なる glyph list ではなく、candidate の traceability を確認するための review package として機能している。

## データ内容

| 項目 | 内容 |
| --- | --- |
| target | China proposed CJK_D candidates。 |
| purpose | IRG editors の reference として、source、meaning、usage、location などを示す。 |
| source groups | `Ci Yuan`、`A Dictionary of Ancient Chinese Language`、`Hanyu Da Zidian`、`Hanyu Fangyan Da Cidian`、place-name source、Yin-Zhou inscription index、`Kangxi Zidian` など。 |
| review note | この review round で evidence が見つからなかった candidates は CJK_D から withdraw すると述べる。 |

## 後続決定

この文書自体は evidence package であり、CJK_D repertoire の最終決定は記録していない。後年の U+2CCA3 / `GZFY-00932` issue では、[IRG N2955](irg-n2955.md) が `IRG N1519` の evidence を source history の一部として参照している。

## 論点

### `Hanyu Fangyan Da Cidian` evidence

`IRG N1519` には `Hanyu Fangyan Da Cidian` を source とする entries がまとまっており、`zfy00931` と `zfy00932` も同じ page reference 周辺に現れる。これが、U+2CCA3 / `GZFY-00932` の current glyph と original evidence のずれを論じる後続 feedback の背景になる。

### source evidence と chart glyph の接続

この文書は candidate evidence の所在を示すが、後続の encoded character に対して source reference、representative glyph、radical / stroke data のどれを修正すべきかまでは判断していない。そこは [G-source Glyph and Source Reference Issues](../topics/g-source-glyph-and-reference-issues.md) で扱う。
