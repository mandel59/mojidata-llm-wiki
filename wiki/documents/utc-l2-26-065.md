# L2/26-065

## 要約

`L2/26-065` は、Initial Teaching Alphabet (ITA) で /ʒ/ を表す新しい zh ligature を 1 文字追加する proposal である。

既存の ITA proposal `L2/24-273` では古い ITA letter X が U+0292 ʒ と同一視されていたが、近年の ITA literacy materials では unsupported ligature に置き換えられているため、別 character が必要だとする。

提案名は `LATIN SMALL LETTER ZH`、code position は Latin Extended-G の U+1DF98 を想定する。

## 提案内容

提案は、ITA Foundation の 2017 / 2025 readers と Thomas Spence 1775 の資料を evidence として、zh ligature を presentation form ではなく independent character として扱う。Properties は `Ll` / `L` とされ、annotation として ITA で older letter U+0292 ʒ を置き換えることを示す。

Collation については、ITA materials は standard orthography で alphabetize されてから ITA に変換されるため、U+1DF98 は unrelated symbol ではなく `zh` digraph として扱うのが自然だと述べる。

## 後続決定

[L2/26-100](utc-l2-26-100.md) は UTC \#187 の SEW recommendations で、LATIN SMALL LETTER ZH を provisional assignment へ進める対象に含めた。採択状態は SEW report と UTC minutes 側で確認する。

## 論点

### allograph と character の境界

文書は、ITA の他の glyph adjustments は font-level allograph として扱えるが、/ʒ/ の X から zh ligature への置換は existing Unicode character で代用できないとする。教育用 alphabet における allographic accommodation と character addition の境界が論点になる。

### collation と orthography

提案文字は phonetic/teaching alphabet 用だが、collation は standard orthography の `zh` として扱うことが期待される。これは Latin Extended-G 追加文字の names / properties だけでなく、implementation guidance にも関わる。
