# IRG N2918

## 要約

`IRG N2918` は、U+3062B と U+30C68 の UK-source reference glyphs を更新する提案である。現行 reference glyphs は《南明史》に基づくが、Huáng Jùnliàng は《南明史》の該当 text が 1644 年の archival documents に由来すると推定し、より早い evidence では異なる glyph が確認できるとする。

提案は、U+3062B を IDS `⿰木聙`、`kRSUnicode` 相当の RS 75.14、total strokes 18 の形へ、U+30C68 を IDS `⿱艹𪸔`、RS 140.8、total strokes 12 の形へ更新することを求める。

## 提案内容

| Code point | Suggested IDS | RS | TS | 根拠 |
| --- | --- | --- | --- | --- |
| U+3062B | `⿰木聙` | 75.14 | 18 | 《王鰲永揭帖》などの archival evidence と printed version。 |
| U+30C68 | `⿱艹𪸔` | 140.8 | 12 | 《王鰲永揭帖》《朱朗鑅啓本》、1930 年代の《明清史料》、現代研究論文。 |

文書は、《南明史》の person names と archival documents の person names を比較し、現行 glyphs は earlier historical documents に attested ではなく、suggested glyphs の方が original characters をより正確に表すと結論する。

## 後続決定

[IRG N2911](irg-n2911.md) は、IRG Meeting \#66 editorial group がこの UK-source glyph issue について結論に至らず、China と UK が Meeting \#67 向けに response を提出すると記録している。[IRG N2909](irg-n2909.md) M66.14 も同じ扱いである。

Meeting \#67 agenda [IRG N2935](irg-n2935.md) は、representative glyph and source reference issues の follow-up として UK-source issue を継続確認対象に置いている。したがって、この文書は accepted change ではなく、追加 response 待ちの proposal として読む。

## 論点

### printed history と archival evidence

`IRG N2918` は、近代刊本《南明史》の glyph ではなく、そこに引用されたと推定される archival documents を優先根拠にする。reference glyph を更新するには、現行 source の見た目だけでなく、source text の伝来と earlier evidence の関係を評価する必要がある。

### UK-source issue の責任主体

提案者は個人だが、source owner として UK の response が必要になる。Meeting \#66 では China と UK の双方に response が求められており、IRG source-data issue では member body confirmation が重要である。
