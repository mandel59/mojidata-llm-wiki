# IRG N2799

## 要約

`IRG N2799` は `IRG N2733R` の revised version として、CJKV components を UCS に符号化するための updated proposal である。IRG #63 Recommendation M63.17 の feedback を受け、China の GF 3001-1997 / GF 0014-2009 だけでなく、HKSAR、TCA、Vietnam、public IDS database、Jianzi Musical Notation 由来の data へ scope を広げた。

文書は、CJKV ideographs の strokes / components が cognitive understanding と information processing の基本単位であり、input methods、typefaces、education products、dictionaries、IDS databases で必要になると説明する。一方で、各 source は decomposition method や audience が異なり、unification principle、conditional variants、sorting、Unihan property などが未確定であることも明示する。

## 論点

### source model

`IRG N2799` は、G-a / G-b、H-a、T-a、V-a、G-c、G-d などの provisional source labels を使い、複数 source の component data を同じ表で整理した。TCA-CNS 11643-2、HKSAR component tables、Vietnam data、Jianzi Musical Notation data を proposal に含めたことが、後の `IRG N2878R3` の source reference system につながる。

### future work

文書は future work として、repertoire formation、component unification principle、SVS または別構造での conditional variants の扱い、ideograph として使える components を別 block に置くかどうか、Unihan Database property file の提供を挙げていた。IRG #64 Recommendation M64.19 はこの plan に従って repertoire を継続開発することを encourage した。
