# L2/25-258

## 要約

`L2/25-258` は NET WITH HANDLE emoji の proposal である。Malik Adan、Theo Schear、Jennifer 8. Lee は、butterfly net / net with handle を capture、play、collecting、catching などの symbol として提案する。

Proposal は、existing GOAL NET が sports-specific であり、handheld net の tool / action meaning を表せないと主張する。Sequence examples では catching、butterfly catching、catch feelings、catch a case など多義的な用法を挙げる。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | NET WITH HANDLE |
| category | Tools |
| keywords | capture、butterfly、net、catch、sparkler、wind chime など。 |
| distinction | GOAL NET は sports context に偏り、handheld capture tool を表さないとする。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FA8D NET WITH HANDLE を Unicode 18.0 additions として受け入れ、[L2/26-008R](utc-l2-26-008r.md) の final candidate list でも維持した。

## 論点

### tool emoji と metaphor

NET WITH HANDLE は physical tool であると同時に、catch / capture の metaphor を担う。Emoji review では、existing object emoji で十分か、sequence が十分に明確か、image size で distinguishable かが論点になる。
