# L2/22-222

## 要約

`L2/22-222` は、KORE SEBELI / Wakara script の 2022 年 proposal である。文書内では February 2022 の initial stage version 05 とされ、2020 / 2021 年版と同じ 106 characters model を維持する。

提案は Soso language 用の contemporary script として、letters、digits、combining diacritics、punctuation、mathematical signs、samples、community practice を提示する。後続版では 2024 年に 95 characters、2026 年に 93 / 91 count へ整理される。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script scope | Soso language 用 KORE SEBELI / Wakara script。 |
| version | initial stage version 05。 |
| repertoire | 106 characters。2020 年版と同様に combining diacritics と vowel sequences を含む。 |
| direction | horizontal left-to-right を assigned / default とし、vertical direction の sample も示す。 |
| evidence | syllabary、student notebooks、national anthem sample、vertical writing samples、community / teaching evidence。 |

## 後続決定

この proposal 自体への UTC 決定は、この wiki では未確認である。[L2/24-246](utc-l2-24-246.md) では repertoire が 95 characters に再編され、2026 年版 [L2/26-050R](utc-l2-26-050r.md) / [L2/26-132](utc-l2-26-132.md) では version 12 として UTC \#188 の tracking 対象になっている。

## 論点

### samples と encoding model

2022 年版は community samples を増やしつつ、vowel sequences と atomic consonant letters の混在 model を維持する。2026 年版の atomic dotted-letter model と比較すると、どの marks を combining mark として残すかが重要な review point になる。
