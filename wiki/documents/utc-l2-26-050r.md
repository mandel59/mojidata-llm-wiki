# L2/26-050R

## 要約

`L2/26-050R` は、KORE SEBELI / Wakara script の 2026 年 revised proposal である。文書内では April 2026 の version 12 とされ、表紙・目次・proposal summary form は 93 characters としている。

一方、本文の proposed characters section は 91 characters と数え、38 capital letters、38 lowercase letters、3 punctuation marks、2 other mathematical signs、10 digits / numerals を列挙する。この 93 / 91 count の不一致は、同じ version 12 が [L2/26-132](utc-l2-26-132.md) として再登録された後も残っているため、review point として記録する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script scope | Soso language 用 KORE SEBELI / Wakara script。 |
| version | April 2026 version 12。 |
| repertoire | 表紙・目次・proposal summary form は 93 characters。本文 section は 91 characters とし、38 capital、38 lowercase、3 punctuation、2 mathematical / other signs、10 digits を列挙する。 |
| direction | horizontal left-to-right を default とし、vertical direction は elderly / traditional use、divinatory / talismanic context、sample として説明する。 |
| atomic model | dots は letter shape の一部で、base letter + dots を single atomic character とし、canonical decomposition しないと説明する。 |
| case model | uppercase / lowercase の使い方は Latin alphabet と同じだと説明する。 |

## 後続決定

[L2/26-132](utc-l2-26-132.md) は、UTC register で 2026-05-14 に登録された同 proposal chain の後続 entry であり、抽出テキスト上は `L2/26-050R` と同じ April 2026 version 12 の内容を持つ。2026-07-09 時点では、UTC \#188 agenda / minutes は未掲載であり、KORE SEBELI proposal への UTC 決定は未確認である。

## 論点

### 93 / 91 count

文書の front matter 相当の箇所では 93 characters、本文の proposed characters section では 91 characters になっている。採択や code chart への反映を確認するときは、どちらが意図された repertoire かを UTC / SEW disposition で確認する必要がある。

### atomic dotted letters

2026 年版では dotted letters を combining sequence ではなく atomic characters として扱う方向が明確になる。2020-2022 年版の combining diacritics / vowel sequences との関係を整理しないと、normalization、search、case mapping、font shaping の説明が曖昧になる。
