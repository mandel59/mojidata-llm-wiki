# Proposed Update UTS #37, Unicode Ideographic Variation Database

## 要約

`L2/26-112` は、Unicode 18.0.0 向けの UTS #37 Revision 15 proposed update である。
IVD の data files、collection / sequence identifiers、IVS 登録手順、collection URL の扱いを説明する。
Revision 15 では、collection URL を registrant ではなく registrar が確立・維持する扱いへ整理する。
Appendix B の U+82A6 glyph examples は画像ではなく web fonts で示す形に変更される。
UTC #187 の CJK & Unihan Working Group recommendations では PRI #541 として扱われ、`L2/26-099` に接続する。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| 対象仕様 | UTS #37, Unicode Ideographic Variation Database Revision 15。 |
| data files | `IVD_Collections.txt` と `IVD_Sequences.txt` を中心に、UTF-8 text、semicolon separated fields、comment lines、`# EOF` marker を使う。 |
| sequence record | base character と variation selector の code points、collection identifier、registrant-provided sequence identifier を field として持つ。 |
| identifier syntax | collection / sequence identifiers は letters で始まり、letters、digits、underscore、hyphen、plus sign を含められる。hyphen / plus sign は backward compatibility のため残る。 |
| Revision 15 updates | co-editor list、collection URL の管理主体、Appendix B glyph examples、references を更新する。 |

文書は、IVS が `Unified_Ideograph` を base character とする場合の unification 条件も説明する。異なる構造でも別 encode するほど significant ではない場合、または第二 level component が一般には unifiable でなくても rare / non-general circulation で strong evidence がある場合、IVD の unification rules 上は登録対象になり得る。

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、PRI #541 としてこの proposed update を UTC #187 の CJK & Unihan Working Group recommendations に取り込む。このページでは UTS #37 draft 自身の変更点を扱い、UTC #187 の action item は `L2/26-099` 側を読む。

## 論点

### IVD format と実装影響

IVD data file は plain text だが、field separator、comment line、file terminator、identifier syntax が明示されるため、implementer は parser を UTS #37 と実 data の両方で確認する必要がある。特に collection / sequence identifiers の `-` と `+` は backward compatibility のため許されるが、programmatic identifier では `_` へ fold する、または取り除く扱いが示される。

### unification boundary

IVS は encoded character の追加とは別の variation mechanism だが、base character の unification boundary と密接に関係する。UTS #37 の rule は、[UCV and NUCV Lists](../topics/ucv-nucv-lists.md) が扱う component variation の境界や、IRG review での glyph / source evidence の扱いと併せて読む必要がある。
