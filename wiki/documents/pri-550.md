# PRI 550

## 要約

`PRI #550` は、UTC が Unicode 18.0.0 向けの UAX \#41 Common References for Unicode Standard Annexes Revision 37 proposed update を public review に出した PRI である。PRI page の status は Closed、closing date は 2026-07-07。

PRI page は draft updated 2026-05-18 とし、UAX \#41 の proposed update が general public review and comment に出たことを示す。

UAX \#41 は algorithm や property values を直接定義する annex ではなく、UAX / UTS が参照する Unicode Standard、UCD data files、technical reports、外部標準の共通 references を束ねる release artifact である。Unicode 18.0 review では、versioned data URLs と technical report references が正しく更新されているかを確認する入口になる。

## 提案内容

### UAX \#41 Revision 37

Revision 37 は Unicode 18.0.0 向けの common references update であり、UCD data files、Unihan.zip、UAX / UTS / UTR references、Unicode 18.0 draft release page などを参照リストとして更新する。

UAX \#41 自体は個別 algorithm の normative rule を変更しない。ただし UAX \#14、UAX \#24、UAX \#29、UAX \#42、UAX \#44、UTS \#39 などの proposed updates は UAX \#41 の references を通じて Unicode 18.0 の versioned data / report ecosystem に接続する。

## 後続決定

`PRI #550` は 2026-07-07 に closed になっている。最終反映は、Unicode 18.0 の published UAX \#41、各 UAX / UTS の references section、versioned UCD directory の URL が一致しているかで確認する。

## 論点

### release artifact としての UAX \#41

UAX \#41 は単体で読まれることは少ないが、Unicode release ごとに references の versioning を支える文書である。draft 段階では beta / proposed update / latest version の URL が混在しやすいため、release coordination の観点では低リスクな editorial artifact ではなく、実装者が参照先を誤らないための check target になる。

### 他 PRI との関係

`PRI #550` は [PRI \#548](pri-548.md) の Unicode 18.0 beta review bundle に含まれる。UAX \#11、UAX \#14、UAX \#24、UAX \#29、UAX \#42、UAX \#44、UTS \#39 の各 proposed update を digest するとき、出典として UAX \#41 の references update も同じ close date で処理されたことを把握しておく。
