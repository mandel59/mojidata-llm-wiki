# L2/26-088

## 要約

`L2/26-088` は Khmer script に U+17DE KHMER SIGN CANDRABINDU を追加する proposal である。Khmer inscriptions と Sanskrit grammar books に attested で、vowel または semi-vowel の nasality を示すために使われると説明する。

代表 glyph は circle を持つ形とし、Nikahit / Anusvara や Reahmuk / Visarga の current forms と合わせる。ornament above を持つ variant は stylistic variant として扱う。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed code point | U+17DE KHMER SIGN CANDRABINDU。 |
| usage | Khmer inscriptions と Sanskrit grammar books における nasalization mark。 |
| glyph model | circle form を representative glyph とし、ornamental variant は stylistic variant とする。 |
| proposed properties | `Mn`、Line_Break=CM、Indic_Syllabic_Category=Bindu、Indic_Positional_Category=Top。 |

## 後続決定

2026-07-09 時点の local catalog では、この proposal への UTC 決定は未確認である。

## 論点

### Khmer script における Indic-style nasalization

この proposal は new script ではなく、Khmer script 内の historical / Sanskritic usage を表す combining mark 追加である。既存 Khmer signs との glyph consistency と、stylistic variant を別 character にしない判断が中心になる。
