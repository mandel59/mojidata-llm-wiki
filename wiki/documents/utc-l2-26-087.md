# L2/26-087

## 要約

`L2/26-087` は Soninke を書く N'ti script を Unicode に追加する proposal である。1966 年に Adama Konte が作った script と説明され、Mali、Republic of the Congo、Ivory Coast などで教育・日常使用が続くとする。

文書は `L2/26-028R` を supersede し、joining behavior の説明と GIILONDE letter extender を追加した点を改訂理由に挙げる。提案 repertoire は U+1CB80..U+1CBAF の範囲で、letters、8 diacritics、punctuation、GIILONDE、digits を含む。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script model | Left-to-right unicameral alphabet。word 内の letters は baseline で常に join / fuse する。 |
| community | Soninke communities による継続使用、official request letters、Keyman keyboard、digital media を evidence とする。 |
| repertoire | 27 letters、8 diacritics、1 punctuation、GIILONDE letter extender、10 digits。 |
| related model issue | LTR joining script として、Joining_Type の解釈問題に接続する。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は N'ti を、joining behavior や encoding model の未解決点があり SEW でさらに検討する proposal として扱う。[L2/26-137](utc-l2-26-137.md) は N'ti などの left-to-right joining scripts を背景に Joining_Type semantics を扱う。2026-07-09 時点の local catalog では採択決定は未確認である。

## 論点

### LTR joining script model

N'ti は Arabic / N'ko / Adlam のように word 内で letters が join するが、writing direction は left-to-right である。このため、representative glyph や repertoire だけでなく、Joining_Type `L` / `R` の semantic model、font shaping、line breaking が review point になる。
