# IRG N2221

## 要約

`IRG N2221` は、`IRG N954AR`、[IRG N1105](irg-n1105.md)、[IRG N2171](irg-n2171.md) の stroke count entries を revisions 付きで consolidate した guideline である。文書は、ここで定めた stroke counts を future IRG Working Set で使うと説明する。

この文書は、既存 UCS code charts と異なる stroke count を採る場合があることを明示する。例として、component 成 は UCS Ed.5 で 6 strokes と 7 strokes の両方で扱われていたが、N2221 では future IRG Working Set 用に 7 strokes として扱うと説明している。

## 後続決定

[IRG N2862R2](irg-n2862r2.md) は、N2221 を reference document として挙げ、SC / FS guideline の R2 を構成した。[IRG N2951](irg-n2951.md) は、N2221 を含む previous documents を置き換える final consolidated FS & SC Guidelines として提出された。

## 論点

### future working set 用の count

N2221 は、既存 encoded characters の chart value と future IRG Working Set 用の review value が異なり得ることを明示する。これは、IRG review infrastructure が過去の chart values をそのまま固定するのではなく、review rule としての count を整備する方向へ進んだことを示す。

### component-based examples

表は component shape と coded examples を並べることで、個別文字ではなく component-level の stroke count rule として扱えるようにしている。後続の N2862R2 / N2951 では、これが FS value と ordering letter を含む entry syntax へ再構成される。
