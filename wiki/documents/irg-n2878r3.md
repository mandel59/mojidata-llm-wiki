# IRG N2878R3

## 要約

`IRG N2878R3` は CJK Unified Ideographs Components の final proposal である。`IRG N2733R`、`IRG N2799`、`IRG N2878`、`IRG N2878R` を改訂し、IRG \#63 / \#64 / \#65 の recommendations と expert feedback を取り込んだ。

`IRG N2878` / `IRG N2878R` / `IRG N2878R2` はこの revision lineage の中間版である。2026-07-06 時点の catalog では、これらを Document Register 由来ではなく `status: referenced` の derived entry として収録し、`IRG N2878R3` の revision lineage、IRG \#65 / \#66 documents、feedback documents から位置づける。

提案の repertoire は `CJK Unified Ideographs Components-A` 119 characters と `CJK Unified Ideographs Components-B` 375 characters、合計 494 characters である。`WG2 N5358` は、この文書に Proposal Summary Form を付けて WG2 registry に提出した entry である。

## 論点

### components as CJK ideographs

文書は、CJK components と CJK ideographs の間に明確な境界はなく、textual behavior、East Asian width、vertical orientation などでも差がないと整理する。そのため、components は component usage によって group された CJK ideographs として扱われる。

### fragments の除外

Wubi、Cangjie、Zhengma などの input method mapping に現れる objects は CJK components ではなく fragments と位置づけられる。decomposition が統計的結果や人為的介入に基づき、rationale を含まない場合があるため、working group は input method 起源の fragments を長期的にも受け入れないとしている。

### feedback response

`IRG N2917` への response として、HCP-00033 と HCP-00400 の removal、secondary radical の追加、HCP source reference paragraph の修正が採用された。一方、個人提案による追加 components は first batch には採用されず、second or subsequent batches の候補として残された。

### WS2024 との関係

Final proposal は、Components-B に含まれる 11 characters が WS2024 にも含まれるため WS2024 から remove するリストを示した。`IRG N2935` はこの重複を Meeting \#67 の follow-up として再掲している。
