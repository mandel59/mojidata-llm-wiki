# L2/26-043

## 要約

`L2/26-043` は、古籍に現れる 家 variant 系の 1 ideograph を UAX \#45 に追加する提案である。対象文字は IDS `⿱宀𫡑`、variant 家、radical-stroke `40.6`、total strokes `9`、first stroke `3` として提示される。

文書は `正字通`、複数版の `康熙字典`、`玉樞經籥`、`字學正本` などを evidence とし、古籍間で glyph が安定していること、現代にも近い用例があることを根拠にする。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | 家 variant 系の 1 ideograph を UAX \#45 `FutureWS` record として追加する。 |
| Data | IDS `⿱宀𫡑`、variant 家、RS `40.6`、TS `9`、FS `3`。 |
| Evidence | `正字通`、`康熙字典`、`玉樞經籥`、`字學正本` など。 |
| 添付 | UTCHan 用の data / font support は [L2/26-099](utc-l2-26-099.md) の UAX \#45 additions bundle 側で扱われる。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、`L2/26-043` を含む 2026 年提出の UAX \#45 additions を `USourceData.txt` / `USourceGlyphs.pdf` の `FutureWS` records として受け入れるよう勧告した。採択・data update 側の summary は [UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) に集約する。

## 論点

### variant と unification

提案は、対象文字を 家 の variant と説明しつつ、古籍で安定した独自 glyph として UAX \#45 record 化する。文書中では、現代用例に見える slight glyph difference も、家 variant として同一候補に統合できると整理している。
