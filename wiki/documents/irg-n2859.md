# IRG N2859

## 要約

`IRG N2859` は、IRG Convenor が J-source / JMJ-source に関する複数の source reference・representative glyph issue を disposition 案として整理し、Japan NB に feedback を求めた文書である。対象は `kIRG_JSource` property value、J-source representative glyph、UTC が管理する provisional `kJapanese` / `kMojiJoho` / `kMorohashi` values に及ぶ。

文書は、Unicode 18.0 Alpha review と ISO/IEC 10646 Seventh Edition synchronization を背景に、2025-09-26 までの Japan NB feedback を求める。提案には U+2A50D、U+72CA、U+26B20 などの JMJ reference 変更や horizontal extension、no action recommendation が含まれる。

## 決定・Action Item

| 対象 | Recommendation |
| --- | --- |
| U+2A50D | `kIRG_JSource` を `JMJ-057575` から `JMJ-056440` へ変更し、representative glyph と関連 Unihan properties を調整する案。 |
| U+72CA / U+3094D | U+72CA の J-source を `JMJ-016917` へ変更し、U+3094D に `JMJ-016916` を horizontal extension する案。 |
| U+7BF9 / U+25CBB | JIS X 0212 由来の status と UCV #123 を理由に no action とする案。 |
| U+26B20 / U+26B07 | U+26B20 の J-source を `JMJ-046313` へ変更し、U+26B07 に `JMJ-046312` を horizontal extension する案。 |

## 後続決定

[IRG N2870](irg-n2870.md) で Japan NB は、published encoded characters には disunification を適用しないという原則を再確認し、U+2A50D、U+5CC0、U+72CA、U+7BF9、U+26B20、U+2F93B、U+21694 の current status 維持を求めた。

## 論点

### source data correction と encoded character stability

`IRG N2859` は、J-source data の整合性を高めるために source reference と representative glyph を変える案を示す。一方、これらの変更は既に published encoded characters の status と、関連する Unihan properties に波及する。後続の Japan NB feedback は、この安定性側の原則を強く示すものとして読む。

### WG2 chart glyph issue との違い

[WG2 N5304](wg2-n5304.md) の M72.07 は chart glyph stability を扱うが、`IRG N2859` は J-source reference と Unihan data maintenance に踏み込む。したがって [J-source and JMJ Source Issues](../topics/j-source.md) では、WG2 側の glyph revert と IRG 側の source disposition を分けて追跡する。
