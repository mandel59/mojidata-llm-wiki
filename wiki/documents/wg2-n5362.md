# WG2 N5362

## 要約

`WG2 N5362` は、ISO/IEC 10646 7th edition に対する new amendment project subdivision を開始する proposal である。Starting repertoire は [WG2 N5361R](wg2-n5361r.md) の provisionally assigned code points を基礎にし、Sirmauri、Leke、Proto-Cuneiform、Mwangwego、[Shaaldaa](../topics/shaaldaa-script.md) などの notable scripts を挙げる。

文書は、starting repertoire は確定版ではなく、WG2 / UTC experts の追加作業により調整されると明示する。WG2 \#73 の M73.05 は、この proposal をもとに Amendment 1 project subdivision を始めることを勧告した。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| project | ISO/IEC 10646 7th edition の amendment を新規 project として追加する。 |
| scope | Universal Coded Character Set に新しい characters を追加する。標準の scope 自体は拡張しない。 |
| starting repertoire | `WG2 N5361R` の provisionally assigned code points を開始点にする。 |
| target dates | WG2 \#73 recommendation では CDAM 2026-09-01、DAM 2028-08-01、FDAM 2029-03-01、Amd 2029-06-15 とされた。 |

## 後続決定

[WG2 N5354](wg2-n5354.md) の M73.05 は、`WG2 N5362` を supporting document とし、`WG2 N5361R` を starting repertoire として Amendment 1 project subdivision を始めるよう SC2 に勧告した。

## 論点

### starting repertoire の可変性

Amendment 1 は `WG2 N5361R` を出発点にするが、個別 script / block は WG2 recommendations、UTC review、SC2 ballot により変わり得る。特に WG2 \#73 recommendations は、Mwangwego などを個別に扱っており、provisional assignment と amendment inclusion は同一ではない。
