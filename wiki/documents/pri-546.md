# PRI 546

## 要約

`PRI #546` は、IVD Registrar が Moji_Joho collection への追加 IVS 登録を public review に出した issue である。review closing date は 2026-07-31。

提出者は Information Technology Standards Commission of Japan / Information Processing Society of Japan で、representative は Shuichi Tashiro。collection identifier は `Moji_Joho`、sequence identifier pattern は `MJ[0-9]{6}` とされる。

提案は、Japan の horizontal extension proposal [WG2 N5221](wg2-n5221.md) が ISO/IEC 10646:2020/Amd.2:2025 と Unicode 16.0 に入った過程で、Moji Jōhō Kiban database の 46 glyphs の対応 code point が更新されたことを背景にする。

結果として、5 base characters に対する 8 sequences を Moji_Joho IVD collection へ追加する必要があると説明する。うち 1 sequence は Hanyo-Denshi collection の既登録 sequence と同じ base / VS pair を共有し、7 sequences は新規登録である。

## 提案内容

### review 対象

PRI page は、reviewers に対し、proposed sequence の glyphic subset が base character の glyphic subset と言えるか、また proposed sequences が Moji_Joho collection の scope に合っているかを確認するよう求める。feedback は Unicode contact form 経由で提出され、review 期間中に submission content が調整される可能性がある。

### proposed sequences

`sequences.txt` は、header と EOF marker を除き、次の 8 lines を含む。

| Base / sequence | Collection | Identifier |
| --- | --- | --- |
| `7CA4` | Moji_Joho | `MJ058288` |
| `805A E0102` | Moji_Joho | `MJ020851` |
| `805A` | Moji_Joho | `MJ058367` |
| `20509` | Moji_Joho | `MJ057273` |
| `23AA3` | Moji_Joho | `MJ038918` |
| `23AA3` | Moji_Joho | `MJ057925` |
| `26BE7` | Moji_Joho | `MJ046446` |
| `26BE7` | Moji_Joho | `MJ046463` |

この submission は、CJK Unified Ideographs block の 2 base characters と CJK Unified Ideographs Extension B の 3 base characters を対象にする。`805A E0102; Moji_Joho; MJ020851` は、Hanyo-Denshi collection の `805A E0102; Hanyo-Denshi; JA7060` と同じ sequence を共有する。

### deprecated sequences

PRI page は、既登録の Moji_Joho sequences 9 件を deprecated とし、使用すべきではないと説明する。対象は `535A E010A` / `535A E010B`、`617A E0103`、`238A7 E0100` / `238A7 E0101`、`24138 E0100` / `24138 E0101`、`27369 E0100` / `27369 E0101` に対応する `MJ059399`、`MJ059400`、`MJ011878`、`MJ038532`、`MJ038533`、`MJ039803`、`MJ068081`、`MJ047684`、`MJ047772` である。

## 後続決定

2026-07-09 現在、この PRI は open issue であり、closing date は 2026-07-31 である。review 後に IVD data へ取り込まれたかどうかは、closed PRI page、IVD release data、または後続 UTC / IVD Registrar の記録で確認する必要がある。

## 論点

### JMJ horizontal extension から IVD への波及

この PRI は、[WG2 N5221](wg2-n5221.md) の JMJ horizontal extension そのものではなく、Unicode 16.0 / ISO/IEC 10646:2020/Amd.2:2025 への取り込み後に Moji Jōhō Kiban database 側の code point 対応が更新された結果、IVD collection の sequence registration を調整する follow-up である。

したがって、JMJ source references の horizontal extension は source-data addition に見えるが、実際には Moji_Joho IVD collection、Hanyo-Denshi collection との sequence sharing、既登録 sequence の deprecation にも影響する。

### UTS \#37 process の実例

[L2/26-112](utc-l2-26-112.md) は UTS \#37 Revision 15 の proposed update として、IVD data files、collection URL、sequence identifier、registration process を説明する。`PRI #546` はその process が具体的な Moji_Joho collection update に適用される open review であり、実装者は UTS \#37 の format と `sequences.txt` の実 data の両方を確認する必要がある。
