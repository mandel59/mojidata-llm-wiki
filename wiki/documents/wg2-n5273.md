# WG2 N5273

## 要約

`WG2 N5273` は、TCA and China experts による [WG2 N5209](wg2-n5209.md) への response である。文書は [WG2 N5211](wg2-n5211.md) の 14-column comparison chart と [WG2 N5230](wg2-n5230.md) の feedback を背景に、N5209 が示した 55 entries を再検討する。

2024-05-19 の online meeting で、experts は 55 entries を 8 duplicate entries と 47 separately encodable entries に分類した。さらに、Duan Zhu version (DYC / D) から glyph structure が異なる 2 entries を追加符号化候補として提案している。

## フィードバック内容

| --- | --- |
| background | Cook の 14-column chart は Da Xu version (X)、Xiao Xu version (K)、Duan Zhu version (D) の glyph comparison を示し、N5209 は X にない K / D entries 55 件を取り出した。 |
| Appendix I | 55 entries のうち 8 entries は duplicate glyphs なので別符号化しないと整理する。 |
| Appendix II | 47 entries は Small Seal glyphs that can be coded separately として列挙する。 |
| Appendix III | DYC 由来の 2 entries は glyph structure が異なるため、追加で符号化を提案する。 |
| data enrichment | 55 entries について Original Glyphs、corresponding modern characters、SW Radical / SW Radical Number を補ったと説明する。 |

## 後続決定

`WG2 N5273` は response / feedback であり、会議決定ではない。後続の [WG2 N5294R3](wg2-n5294r3.md) は、N5273 の結果として 55 entries のうち 8 duplicates を除き 47 entries を追加し、さらに 2 disunifications を加えた 11,140 code point repertoire を提示した。[WG2 N5306](wg2-n5306.md) は、この repertoire を前提に final proposal 前の unification / disunification review へ進む。

## 論点

### N5209 の 55 entries の扱い

N5209 では、X / THX にない K / D entries 55 件が追加 review 対象だった。N5273 は、それらを「重複なので符号化しないもの」と「source tradition 上、別符号化可能なもの」に分けたことで、N5294R3 の 11,140 code point repertoire へ直接つながる中間判断を提供した。

### duplicate と separate coding

文書は exact duplicate glyphs を別符号化しない一方、DYC や QJZ で glyph structure が異なる entries は別符号化可能とする。この線引きは、後続の [WG2 N5307R](wg2-n5307r.md) が整理する mapping / unification principles の前段階として読める。
