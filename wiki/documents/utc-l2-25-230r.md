# L2/25-230R

## 要約

`L2/25-230R` は、Emoji Standard & Research Working Group が UTC \#185 に提出した report である。Unicode 18.0 向けに 9 new emoji characters を推薦し、既に候補にあった U+1FADD APPLE CORE を Unicode 18.0 list から外す consensus / action item を記録する。

推薦された 9 characters は、LIGHTHOUSE、METEOR、ERASER、NET WITH HANDLE、MONARCH BUTTERFLY、PICKLE、FACE WITH SQUINTING EYES、LEFTWARDS THUMB SIGN、RIGHTWARDS THUMB SIGN である。thumb signs 2 件は skin tone modifiers を受け入れるため、sequence count を増やす点も明示される。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| removal | U+1FADD APPLE CORE を Unicode 18.0 approved list から外す。 |
| additions | Unicode 18.0 向けに 9 new emoji characters を受け入れる。 |
| modifier behavior | LEFTWARDS / RIGHTWARDS THUMB SIGN は Emoji_Modifier_Base として skin tone modifiers を受ける。 |
| pipeline action | RMG が Unicode 18.0 pipeline を更新し、ESR が black/white images を提供する action item が付く。 |

## 後続決定

[L2/26-008R](utc-l2-26-008r.md) は、`L2/25-230R` で受け入れられた U+1FAEB FACE WITH SQUINTING EYES を CRACKING FACE に変更し、Unicode 18.0 final emoji candidate list を更新する。個別 proposal は [Emoji Repertoire Proposals](../topics/emoji-repertoire-proposals.md) でまとめる。

## 論点

### proposal batch から short list への変換

この report は、個別 proposal そのものではなく、ESR が emoji proposal batch を UTC action に変換した decision hub である。候補の根拠は各 proposal page で読み、Unicode 18.0 に入るかどうかはこの report と後続の `L2/26-008R` で確認する。

### emoji additions と sequence count

thumb signs は code point 2 件だが、skin tone modifier sequences も発生する。Emoji repertoire の変更は単純な character count だけでなく、RGI / modifier sequence の増加として扱う必要がある。
