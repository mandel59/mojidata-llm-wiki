# L2/26-071

## 要約

`L2/26-071` は、涨 / 漲 / 涱、張 / 张、临 / 臨 に関係する common simplified forms 3 ideographs を提案する文書である。文書は 長 component の複数 simplified variants、リ 型 component の選択、PRC standard との整合を説明したうえで、normalized glyph を選ぶ。

Evidence は中国の俗簡字資料、日本の Ryakuji を集めた米軍資料、ソ連の辞書、20 世紀の教科書・辞書類にまたがる。

## 提案内容

| No. | Variant | Data の要点 |
| --- | --- | --- |
| 1 | 涨 / 漲 / 涱 | IDS は 氵 と 長 simplified component を含む形。`kMandarin zhǎng zhàng`。 |
| 2 | 張 / 张 | 長 simplified component を用いた form。`kMandarin zhāng`。 |
| 3 | 临 / 臨 | リ 型 component と 品 component を含む form。`kMandarin lín`。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、`L2/26-071` を含む 2026 年 UAX \#45 additions を `FutureWS` records として受け入れるよう勧告した。採択・data update 側の summary は [UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) に集約する。

## 論点

### component normalization

文書は、長 component と リ 型 component に複数 variants があることを認めたうえで、一般的・広く使われる形へ normalized glyph を寄せる。source evidence の glyph そのものをすべて別 candidate とするのではなく、common simplified form として統合する判断が中心である。
