# L2/08-117

## 要約

`L2/08-117` / `WG2 N3394` は、UTC / L2 が `WG2 N3388`（`L2/07-421`）を review し、Japan NB にコメントを求めた文書である。UTC / L2 は提案自体を reasonable と見たが、kana characters の追加では Japan NB involvement が必要だとした。

この文書は、HIRAGANA LETTER YE と KATAKANA LETTER ORIGINAL E をすぐに個別符号化するのか、obsolete kana / hentaigana を含むより大きな framework で扱うのかを明示的な論点にした。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| 対象 | `WG2 N3388` / `L2/07-421` の two Kana characters concerning YE。 |
| 評価 | 提案は reasonable としつつ、Japan NB review が必要と判断。 |
| 論点化 | 2 文字を早期に符号化する案と、obsolete kana / hentaigana 全体の枠組みで検討する案を比較。 |

## 論点

### 個別提案と大きな repertoire 整理

この文書の重要点は、YE proposal を isolated proposal として扱い切らず、hentaigana の複雑な repertoire 問題に接続したことである。後続の `L2/08-358` では Japan NB が、HIRAGANA LETTER YE を historical letter として扱うなら支持できるが、hentaigana 一般とは混同しないようにすべきだと回答した。
