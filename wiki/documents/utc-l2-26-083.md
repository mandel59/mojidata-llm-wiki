# L2/26-083

## 要約

`L2/26-083` は、蠱、談、綴、等、衆に関係する common simplified / abbreviated forms 5 ideographs を提案する文書である。文書は glyph / component の subtle differences を認めつつ、実用頻度や資料での出現をもとに normalized glyph を選んでいる。

Evidence は 方块瑶字、明治・大正期の日本資料、Ryakuji 研究、米軍資料、日本の国字辞典などにまたがる。

## 提案内容

| No. | Variant | Data の要点 |
| --- | --- | --- |
| 1 | 蠱 | `kMandarin gǔ`、RS `108.10`。 |
| 2 | 談 | 火 component を含む simplified form。 |
| 3 | 綴 | `kJapanese デイ`、糸と simplified components を含む form。 |
| 4 | 等 | `kMandarin děng`、`kJapanese トウ`、full-width / half-width variants のうち full-width dot-on-top form を選ぶ。 |
| 5 | 衆 | `kMandarin zhòng`、`kJapanese シュウ`。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、`L2/26-083` を含む 2026 年 UAX \#45 additions を `FutureWS` records として受け入れるよう勧告した。採択・data update 側の summary は [UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) に集約する。

## 論点

### 略字体と国字・方块瑶字 evidence

この文書は、Chinese simplified forms だけでなく、日本の Ryakuji / Kokuji 資料や 方块瑶字研究を evidence として使う。略字・簡体字形の実使用範囲は単一地域に閉じないため、UAX \#45 record では source field と comments で evidence lineage を追う必要がある。
