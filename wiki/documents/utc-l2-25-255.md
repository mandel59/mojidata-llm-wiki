# L2/25-255

## 要約

`L2/25-255` は ERASER emoji の proposal である。Karin Nakagawa は、eraser を pencil / ruler などと並ぶ daily stationery item として位置づけ、school、office、software icon、erase / cancel / forget といった metaphor で使えると説明する。

PDF text extraction は一部 garbled だが、proposal の中心は、eraser が日常物として広く認識され、stationery category を補完し、既存 emoji sequence では簡潔に表せない action / object を担うという点である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | ERASER |
| category | Things / stationery 周辺。 |
| selection rationale | stationery item、software icon、erase / cancel / forget などの metaphor、pencil / ruler との sequence usage。 |
| distinction | pencil や other stationery では「消す」機能を表せないとする。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FA8C ERASER を Unicode 18.0 additions として受け入れ、[L2/26-008R](utc-l2-26-008r.md) の final candidate list でも維持した。

## 論点

### object emoji と action metaphor

Eraser は physical object であると同時に、erase / cancel / remove の action metaphor として使える。Emoji review では、物体としての frequency と、sequence / metaphor での使いやすさの両方が評価対象になる。
