# IRG N2890

## 要約

`IRG N2890` は、Wang Xieyang による `IRG N2878` への feedback である。unencoded characters の IDS を整理した結果、5 components が first batch の CJK Unified Ideographs Components に含まれるとよいと提案した。

また、Jianzi Musical Notation components の representative glyph style について、components は CJK Unified Ideographs と同じように扱われるため、中国の CJKUI convention に合わせるべきだと提案している。音楽記譜用の style のままだと、非音楽用途で users が混乱する可能性があるという理由である。

## 論点

### first batch への追加提案

N2890 の追加提案は、encoded characters の中で有用な components として働くことを根拠にしている。これは `IRG N2878R3` の final proposal で述べられる「individual contributions」の一部に対応する。

### final proposal との関係

`IRG N2878R3` は、individual contributions を review したうえで、first batch には採用しない方針を示した。理由は、example glyphs との関係が rational decomposition / visual decomposition として十分でない、または scope 外であるためとされる。ただし second or subsequent batches への再提出可能性は残っている。
