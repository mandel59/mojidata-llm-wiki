# L2/25-221

## 要約

`L2/25-221` は、日本の地名・橋名に現れる 3 ideographs を UAX \#45 に追加する提案である。対象は IDS `⿱投石`、`⿱米𥹫`、`⿰魚久` で、それぞれ `kJapanese` reading `なげ`、`ひとぎ`、`ひさ` を持つ。

文書は、3 文字を `USourceData.txt` の `FutureWS` records として示し、UTCHan.ttf 用の representative glyph files も添付する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | 3 ideographs を UAX \#45 に追加する。 |
| 対象 1 | IDS `⿱投石`、`kJapanese なげ`、愛媛県大洲市の「逆なげ橋」に関係する橋名。 |
| 対象 2 | IDS `⿱米𥹫`、`kJapanese ひとぎ`、大分県竹田市の「ひとぎ瀬」に関係する地名。 |
| 対象 3 | IDS `⿰魚久`、`kJapanese ひさ`、長崎県五島市の「ひさ網代」に関係する地名。 |
| 提案 record | UAX \#45 `FutureWS` records として UTC source data に追加する候補にする。 |

## 後続決定

現行 catalog と既存 cache では、`L2/25-221` 自体を受け入れた UTC consensus / action item は確認していない。[UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) では、2026 年提出の [L2/26-044](utc-l2-26-044.md) が [L2/26-099](utc-l2-26-099.md) Section 24 に基づいて受理されたが、`L2/25-221` の 3 ideographs はその受理対象には含まれていない。

したがって、`L2/25-221` は現時点では UAX \#45 / FutureWS への submitted proposal として扱い、採択済み record ではなく後続の CJK & Unihan Working Group review を待つものとして追跡する。

## 論点

### 地域資料と登記資料の組み合わせ

`⿱投石` は、愛媛県の百科事典、県史、市政資料、地域文化調査など複数の国立国会図書館デジタルコレクション資料で裏付けられる。`⿱米𥹫` は、大分県竹田市の古い公図、旧土地台帳、地積測量図、荘園史料に基づく。`⿰魚久` は、長崎県公報、五島市の農用地利用計画、官報、地積測量図に基づく。

### 代替表記と未符号化文字

文書は、未符号化のため現行 official documents では片仮名表記に置き換えられる例を示す。これは、地名としての実在性と、Unicode で表せないことによる代替表記の発生を同時に示す evidence になっている。
