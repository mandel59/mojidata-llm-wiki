# L2/26-097

## 要約

`L2/26-097` は Editorial Working Group が UTC Meeting \#187 に提出した report である。Unicode 18.0 core specification の chapter updates、terminology cleanup、NamesList ownership transfer、core spec archival PDF の password protection policy などを扱う。

Unicode 18.0 の editorial work は、script / block additions の説明文だけでなく、chapter heading restructuring、West Asian terminology、chart / NamesList との ownership boundary に及ぶ。release coordination を読む際には RMG report だけでなくこの Editorial WG report も必要になる。

## 報告内容

| 区分 | 要点 |
| --- | --- |
| Core spec text | Section 6.1 abugidas、IPA / Latin Extended-G / tone diacritics / click consonants、Anatolian Hieroglyphs、Hebrew Sheva、Archaic Cuneiform Numerals、Kaithi / Devanagari、Jurchen / Seal、Leibnizian Mathematical Notation などの text updates。 |
| Chapter restructuring | Chapters 8-19 / 21 の heading と block table を整理し、“Middle Eastern” を “West Asian” に置き換える。 |
| PDF publication | Charts WG と同期し、core specification archival PDF から owner password protection を外す方向を扱う。 |
| NamesList | Names list ownership を Charts WG へ移管し、NamesList.txt の修正 flow を整理する。 |
| Public feedback | Egyptian Hieroglyphs Extended-A、Chisoi、U+1DF6E などの NamesList / annotation 修正を扱う。 |

## 後続決定

UTC \#187 minutes [L2/26-093](utc-l2-26-093.md) が、Unicode 18.0 beta review authorization と関連 publication action items を記録する。NamesList ownership と chart publication policy は [L2/26-101](utc-l2-26-101.md) Charts WG report と合わせて読む。

## 論点

### editorial report は release scope の一部

Editorial WG report は「文章の修正」に見えるが、Unicode 18.0 の beta reviewer が確認すべき core spec text、annotation、NamesList output を決める資料でもある。文字追加の digest だけでは、release で変わる読者向け説明や publication artifact の変更を見落とす。

### NamesList の ownership 境界

NamesList は core spec と charts の両方に現れるが、UTC \#187 では Charts WG 側へ ownership を移す方向が示された。この変更は、今後の NamesList 修正を Editorial WG page だけで追わない方がよいことを意味する。
