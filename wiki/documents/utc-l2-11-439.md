# L2/11-439

## 要約

`L2/11-439` / `IRG N1825R` は、Adobe-Japan1-6 character collection 由来の 27 CJK Unified Ideographs について、`kIRG_USource` の source reference を `UTC-...` から `AJ1-...` に置き換える提案である。文書は、Adobe-Japan1-6 を true source と見なせるため、language- and region-agnostic な `UTC` reference よりも `AJ1` source を立てる方が明確だとする。

## 提案内容

文書は、URO の 8 characters と Extension D の 19 characters について、current `kIRG_USource` values と proposed `AJ1-` values を対応表で示す。`AJ1-` の後ろには Adobe-Japan1-6 CID を decimal で置く。

この proposal は source reference の変更であり、representative glyphs は変更しないと明記する。採用されれば UAX #38 と UTR #45 の変更を伴う。

## 後続決定

この wiki では、この proposal の採否を示す IRG / UTC decision はまだページ化していない。U-source source reference の精密化として、[UAX #45 / U-Source Ideographs](../topics/uax45-u-source-ideographs.md) と [IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md) から追跡する。

## 論点

`L2/11-439` は、`kIRG_USource` が単に「UTC 経由」を示すだけでは不十分な場合があることを示す。source reference は、代表 glyph と別に、どの collection / evidence を source とするかを示す data であり、後からより具体的な source prefix に置き換えられることがある。
