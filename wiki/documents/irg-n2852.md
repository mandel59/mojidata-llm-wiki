# IRG N2852

## 要約

`IRG N2852` は、Night Koo による `IRG N2799` への feedback である。Traditional Chinese、Japanese kyujitai、Korean hanja で使われる traditional printing orthography の sampling が不足しているとして、4 new components の追加を提案した。

提案は、CMEX CJK Components Review Tool で使う index も併記し、Kangxi Dictionary、Adobe-Japan1、UCV / NUCV、Unicode chart、IVD、CNS 11643 などの evidence を示している。

## 論点

### scope と evidence

`IRG N2852` の主張は、China / TCA / HKSAR / Vietnam 由来の data だけでは traditional printing orthography の部件差を十分に拾えないという点にある。component としての最低レベル構造、non-cognate distinction、IVD や historical glyph component list での利用可能性を根拠に追加を求めている。

### final proposal との関係

`IRG N2878R3` は individual contributions 全体を review したうえで、first batch には採用しない整理を置いた。ただし second or subsequent batches への再提出可能性は残しているため、`IRG N2852` は future batch candidate の根拠文書として追跡する価値がある。
