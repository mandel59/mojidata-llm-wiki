# WG2 N5355

## 要約

`WG2 N5355` は、`WG2 N5348R` 末尾の Michel Suignard による note に応答し、Small Seal の modern CJK 対応値で使われた `zhèngzì`（正字）/ normalized form という説明を補足する China / TCA feedback である。

文書は、問題の中心は `zhèngzì`（正字）という訳語ではなく、`Shuowen Jiezi` における `zhèngzhuàn`（正篆）と `chóngwén`（重文）の関係だと整理する。対応する modern equivalent がない `chóngwén`（重文）では、関連する `zhèngzhuàn`（正篆）の modern equivalent を retrieval 用の modern indexing form として使う、と説明している。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| 対象 | `WG2 N5348R` で不明確とされた normalized form / standard characters の意味。 |
| 整理 | `zhèngzì`（正字）ではなく、`zhèngzhuàn`（正篆）と `chóngwén`（重文）の関係として説明する。 |
| 例 | `TH-02654` のように exact modern match がない `chóngwén`（重文）では、対応する `zhèngzhuàn`（正篆）`TH-02651` の modern equivalent `U+2A1E0` を indexing form として使う。 |

## 論点

### N5348 の feedback への補足

`WG2 N5348R` は `zhèngzì`（正字）/ standard characters という語で modern CJK 対応値の代替方針を説明したが、文末でその意味の明確化が必要だとした。`WG2 N5355` は、この説明を `Shuowen Jiezi` の内部構造、すなわち見出し的な `zhèngzhuàn`（正篆）と異体・付属形としての `chóngwén`（重文）に戻して説明する。

### M73.01 の kSEAL_MCJK correction との関係

`WG2 N5369` の CD4 disposition では、Small Seal の `kSEAL_MCJK` が 20 characters で修正された。修正表は `WG2 N5344R2` の modern CJK correspondence principle、すなわち `zhèngzhuàn`（正篆）characters で form and meaning が一致する modern CJK unified ideograph を対応づける方針を根拠にしている。`WG2 N5355` は、その方針の用語面を補う文書である。
