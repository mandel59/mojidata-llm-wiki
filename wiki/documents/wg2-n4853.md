# WG2 N4853

## 要約

`WG2 N4853` は、2017-08-28〜29 に Tatung University, Taipei で開かれた Shuowen Seal Ad Hoc の合意を、今後の UCS 符号化作業の計画として記録した文書である。`L2/17-318` は同じ文書を UTC register に載せた別番号なので alias として扱う。

参加者は Shuowen Seal を CJK Unified Ideographs と独立した separate script / new block とし、CJKUI を base とする VS 方式を採らないことで一致した。また duplicate の typology、藤花榭本を baseline とする repertoire と single-column chart、複数版本を比較する multi-column chart の作成方針を示した。

ただし、これは ad hoc 参加者の consensus を WG2 の検討に供した報告であり、WG2 plenary による repertoire 採択、符号位置確定、正式な disposition ではない。具体的な encoding methodology、properties、code-chart form は後続作業と consensus proposal に委ねられた。

## 決定・Action Item

| 論点 | Ad Hoc の合意 |
| --- | --- |
| encoding model | Shuowen Seal characters を CJKUI から独立した separate script として new block に符号化し、CJKUI を base とする VS scheme は用いない。working name は `Shuowen Seal` とする。 |
| naming / allocation | character name は `Shuowen Seal Character U+3XXXX` から始まる algorithmic name とし、Road Map 上の TIP を想定する。将来の supplemental block は `Seal Supplement` とし得る。 |
| rules | CJKUI の Annex S などをそのまま適用せず、repertoire を決める過程で Shuowen Seal 用の規則を作成・文書化する。 |
| duplicates | `exact duplicate`、`possible duplicate`、`non-duplicate` の typology を設ける。前二者は今後定める VS scheme で扱い、non-duplicate は unify しない。個別判断を表に記録する。 |
| baseline | 藤花榭本を repertoire の行数・code point 数を決める baseline、および藤花榭本 glyph style の single-column chart の基礎とする。 |
| source comparison | 最低限、藤花榭本と陳昌治本の glyph / mapping を含む multi-column chart を作り、利用可能なら他版本も加える。避諱による deformation / substitution もこの比較作業とともに検討する。 |
| follow-up | methodology、script documentation、repertoire、print source・radical assignment・collation 等の properties、code-chart form を上記作業から定め、new block の consensus proposal を WG2 に提出する。 |

## 後続決定

N4853 は `Action: For consideration by JTC1/SC2/WG2` の Ad Hoc Report であり、表中の内容は会合参加者の作業合意である。したがって separate-script model や duplicate handling を、ISO/IEC 10646 への収録を確定した WG2 plenary resolution として読むことはできない。

2019 年の [WG2 N5089](wg2-n5089.md) は、N4853 から続く論点を Beijing informal meeting で具体化し、Guwen / Zhouwen、duplicate、誤刻・欠損、taboo deformation、proposal data の修正方針を報告した。[WG2 N5108](wg2-n5108.md) はさらに WG2 \#68 の議論として、他版本と taxonomy / ordering の追加検討が必要で、CD / CDAM text に入れるには早いという評価を記録している。

## 論点

### Separate script と variation の役割分担

N4853 は script 全体を CJKUI の variation として表現する案を退ける一方、Shuowen Seal block 内部の exact / possible duplicates や taboo deformation には VS の利用余地を残した。separate encoding と glyph-level variation は排他的な二案ではなく、異なる粒度の問題として整理されている。

### Baseline は唯一の権威版本ではない

藤花榭本は initial repertoire と single-column chart の baseline だが、文書は同時に陳昌治本を最低限含む multi-column comparison を要求する。後続作業で methodology や representative glyph を決めるための起点であり、他版本の evidence を排除する決定ではない。

### 未決事項

duplicate の具体的な VS scheme、個々の unification、避諱の扱い、character properties、radical assignment、collation、最終 repertoire と code chart は未決である。文書はこれらを解決した proposal ではなく、proposal を作るための作業枠組みを定めている。
