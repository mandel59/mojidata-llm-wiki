# WG2 N5108

## 要約

`WG2 N5108` は、Suzuki Toshiya が WG2 \#68 の Seal Script discussion を報告した文書である。`L2/19-245` は同じ文書を UTC register に載せた別番号なので、このページでは alias として扱う。

文書は Beijing での 2019-06-01 informal Seal meeting と [WG2 N5105](wg2-n5105.md) の提出を受け、Zhuanwen / Guwen / Zhouwen の disunification、非常に似た同義 glyph の unification、font source の見直し、taxonomy / ordering の未決点を整理する。

結論として、参加者の多数は Seal Script 標準化には他版本の検討と taxonomy metadata を含む追加議論が必要だとし、Michel Suignard は CD / CDAM text へ入れるには早いと見た。TCA が 2019 年 10 月の次回 ad hoc meeting を host できると表明し、WG2 recommendation に ad hoc meeting endorsement を入れる方向で合意した。

## 報告内容

| 論点 | 内容 |
| --- | --- |
| written forms | Shuowen Jiezi 内の Zhuanwen、Guwen、Zhouwen は基本方針として disunify する。 |
| similar glyphs | 同じ semantics を持ち、非常に似た glyph は radical が異なっても unify する。 |
| font source | 以前の font は 中國書店 の lithograph reprints に基づいていたため、早稻田大學に保存される original Tenghuaxie version に基づく更新へ進む。 |
| glyph corrections | component consistency や etymological analysis に照らし、他版本に正しい形がある場合は font design を修正する。 |
| modern CJK cells | Andrew West は N5105 chart の Newly Added Characters で modern CJK 対応値が空欄の箇所を埋めるよう求めた。 |
| taxonomy | Michel Suignard は future extensions と code chart ordering / taxonomy を質問し、DaXu version appearance order の安定性と有用性が議論された。 |
| coordination | China、TCA、Japan、UTC experts 間の email coordination 問題が指摘され、UTC experts が document exchange を支援する方向になった。 |

## 後続決定

この文書自体は report であり、決定文書ではない。報告内では、Small Seal を CD / CDAM text に入れるには早く、他 versions と taxonomy metadata の追加検討が必要だと整理されている。後続の [WG2 N5209](wg2-n5209.md) は、Richard Cook の source mapping data と TCA / China data を比較し、multi-source database と variant review の作業へ接続した。

## 論点

### 2019 年時点の maturity

TCA / China experts は revised Small Seal character set が official submission に十分な品質だと見て [WG2 N5105](wg2-n5105.md) を提出した。一方、WG2 discussion では他 versions の扱い、future extensions、taxonomy / ordering、multi-column chart の可能性が未解決として残り、encoding text へ入れるには早いと判断された。

### 他 versions と taxonomy

N5108 は、DaXu version の heading character order は安定しているが、同一 radical 内の ordering は明確でないと記録する。Lisa Moore と Michel Suignard は、Richard Cook の関心である複数 `Shuowen` versions を考慮した taxonomy / ordering が first batch に影響し得ると述べており、後続の source mapping work への橋渡しになっている。
