# IRG N2958

## 要約

`IRG N2958` は、多数の G-source representative glyphs の revision request である。Ma Shijie は、current glyph と source evidence を照合し、normal form、taboo form、component convention、source text の形に基づく修正候補をまとめている。

文書は 70 ページ超の一覧であり、単一の code point だけを扱う proposal ではない。各 item は、current glyph、evidence、問題点、requested revision を比較する形式で、G-source glyph convention と original evidence のどちらを重視するかを review させる材料になっている。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | 複数の G-source representative glyphs。 |
| 根拠 | original evidence、normal form、taboo form、component convention、字書・文献 evidence。 |
| 例 | U+23AF9 の component normalization、U+25057 の component correction、U+21676 の component correction、U+25B9F / U+25FAC の taboo form treatment など。 |
| 作業種別 | Existing encoded characters の representative glyph revision request。 |

## 後続決定

Meeting #67 agenda の [IRG N2935](irg-n2935.md) は、G / V / SAT などの representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。`IRG N2958` 自体は request document であり、採否は記録していない。

## 論点

### 大量 revision request の扱い

`IRG N2958` は、個別の誤り修正だけでなく、G-source glyph convention の適用範囲を問い直す文書である。IRG review では、evidence に忠実な字形、member body chart convention、Unicode code chart と Unihan data の整合を item ごとに判断する必要がある。
