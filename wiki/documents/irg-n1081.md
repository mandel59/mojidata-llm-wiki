# IRG N1081

## 要約

`IRG N1081` は、China が IRG Meeting \#23 に提出した 146 ideographic components の追加提案である。1997 年に PRC State Language Committee が release した `Chinese Character Component Standard of GB 13000.1 Character Set For Information Processing` を背景に、CJK / Extension A / Extension B / CJK_C1 submission に含まれない components を ISO/IEC 10646 に追加することを求めた。

文書は、ideographic components が ideograph decomposition、analysis などに重要であるとして、各 component character、使用例、stroke count を表で示す。現行の CJKV Components proposal とは repertoire も設計も異なるが、component encoding を UCS / ISO/IEC 10646 に求める早い時期の China proposal として位置づけられる。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| source basis | GB 13000.1 character set の Chinese Character Component Standard。 |
| repertoire | CJK / Extension A / Extension B / CJK_C1 submission に未収録とされる 146 ideographic components。 |
| data fields | component character、example ideographs、stroke count。 |
| rationale | ideograph decomposition / analysis に components が重要であるため、ISO/IEC 10646 への追加を提案する。 |

## 後続決定

[IRG N1133](irg-n1133.md) は、TCA 側から CNS 11643 Planes 1 & 2 の basic components と decomposition rules を入力した。2024 年の [IRG N2733R](irg-n2733r.md) は、直接には GF 3001-1997 / GF 0014-2009 を土台にした proposal だが、`IRG N1081` と `IRG N1133` は CJKV Components topic で component encoding の先行例として扱う。

## 論点

### component encoding の初期提案

`IRG N1081` は、encoded ideographs の IDS や Unihan properties の補助 data ではなく、components そのものを UCS characters として追加する方向を示している。この点は、後の CJK Unified Ideographs Components-A/B proposal と共通する。

### 現行 proposal との距離

文書は 146 components を列挙するが、source reference model、Unicode properties、review tool、future additions procedure などはまだ整備していない。現行の [CJKV Components](../topics/cjkv-components.md) では、multi-source data、Component ORT、source references、CJKUI behavior などが後から追加される。
