# L2/08-284

## 要約

`L2/08-284` は、Draft UTR \#45 "U-Source Ideographs" の Revision 1 snapshot である。U-source ideographs を、IRG の CJK ideograph unification work で使う UTC source data として説明し、U-source database が glyph PDF と text data file から成ることを定義する。

文書は、U-source に含まれることは UTC が符号化を承認したことを意味しないと明示する。U-source records は、UTC に提出された candidates の public record と、IRG work で参照する stable identifier の両方として扱われる。

## データ内容

Draft UTR \#45 の data file は、semicolon-separated UTF-8 text として、U-source identifier、status、Unicode code point、radical-stroke index、KangXi index、IDS、source field の 7 fields を持つ。identifier は `UTC` + five decimal digits とされ、番号は再利用されず sequential に割り当てられる。

status field は当時、`C`, `D`, `E`, `U`, `V`, `W`, `X` を使っていた。`C` / `D` / `E` は Extension C / D / E proposals、`U` は既符号化、`V` は encoded ideograph の variant、`W` は encoding に不適、`X` は後続 proposal candidate を表す。後年の [L2/22-185](utc-l2-22-185.md) と [L2/25-199](utc-l2-25-199.md) では、これらがより説明的な status values へ整理される。

source field は source tag と source-specific index を持ち、URI、U-source identifier、Unihan `k*` fields、dictionary / bibliographic sources などを扱う。これは UAX \#45 が単なる glyph list ではなく、source evidence へ到達するための data model を持つことを示している。

## 後続決定

`L2/08-284` は UTR \#45 の初期 snapshot であり、後続 versions で UAX \#45 / Unicode Standard Annex に発展した。2025 年の [L2/25-199](utc-l2-25-199.md) では、prefix、status、comments、first residual stroke、IDS field、data files が更新された model として説明されている。

## 論点

この文書は、U-source を UTC submissions の backlog ではなく、IRG / WG2 から参照できる formal data source として位置づける。後続の `FutureWS` records、UK-source records、horizontal extension は、この初期 model に new status / prefix / relation を追加しながら運用されている。
