# IRG N2765

## 要約

`IRG N2765` は、2025-03-17/21 に virtual meeting として開催された IRG Meeting \#64 の recommendations and action items である。この wiki では特に、CJKV Components proposal `IRG N2799` の継続開発を encourage した M64.19 と、script-hybrid Han ideographs を将来会合へ継続した M64.20 を参照する。

M64.19 は、TCA が CMEX CJK Components Review Tool を demo し、IRG experts が comments や new CJKV components を追加できる場として位置づけた点が重要である。これにより、CJKV Components は proposal 文書だけでなく、review tool / source data / future additions の governance を伴う案件になった。

## 決定・Action Item

| Recommendation | 内容 | 後続 |
| --- | --- | --- |
| M64.02-M64.03 | UCV / NUCV examples を確認・更新し、次版 list の review schedule を決めた。 | [UCV and NUCV Lists](../topics/ucv-nucv-lists.md) |
| M64.04 | WS2024 Version 2.0 review 後、Version 3.0 の data fixing、ORT release、comments、submitter response schedule を定めた。 | [IRG Working Set 2024](../topics/irg-working-set-2024.md) |
| M64.10-M64.16 | China / TCA / Vietnam horizontal extensions と G / T / V source representative glyph and property updates を勧告した。 | [CJK Horizontal Extensions](../topics/cjk-horizontal-extensions.md)、[IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md) |
| M64.18 | IRG P&P Version 18 draft の準備と UCV から NUCV への移動案の継続議論を求めた。 | [IRG Indexing Rules](../topics/irg-indexing-rules.md)、[UCV and NUCV Lists](../topics/ucv-nucv-lists.md) |
| M64.19 | `IRG N2799` の CJKV Components proposal を review し、outlined plan に従った repertoire development の継続を encourage した。 | [CJKV Components](../topics/cjkv-components.md) |
| M64.20 | ROK contributions と UNC proposal を含む script-hybrid Han ideographs 議論を、将来会合へ継続する consensus を記録した。 | [CJK Hybrid Characters](../topics/cjk-hybrid-characters.md) |

## 後続決定

[IRG N2826](irg-n2826.md) の M65.23 は、`IRG N2878R` を review し、final source references、`kRSUnicode`、`kTotalStrokes`、representative glyphs を含む R2 version の提出を authors に求めた。CJKV Components はその後 [IRG N2878R3](irg-n2878r3.md) と [IRG N2909](irg-n2909.md) M66.17 により WG2 forwarding へ進む。

Script-hybrid Han ideographs は、M64.20 では継続議論に留まるが、Meeting \#65 の [IRG N2826](irg-n2826.md) M65.24 で CJKUI extension blocks とは別 block を原則とする policy へ整理される。

## 論点

### Review tool と future additions

M64.19 は `IRG N2799` の repertoire development を継続させるだけでなく、TCA の CMEX CJK Components Review Tool を review/comment の場として明示した。これは CJKV Components の future batch をどう扱うかという後続課題につながる。

### CJKV Components と script-hybrid の分離

M64.19 と M64.20 は隣接しているが、前者は Han component collection としての CJKV Components、後者は non-Han script components を含む Han-like characters の扱いである。Meeting \#65 以降、この 2 つは別 topic として整理される。
