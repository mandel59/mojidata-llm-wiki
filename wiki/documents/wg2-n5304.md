# WG2 N5304

## 要約

`WG2 N5304` は、2025-06-23/27 に Niigata, Japan で開催された WG2 Meeting \#72 の recommendations である。この wiki では、J-source glyph correction の revert を扱う M72.07 と、Small Seal Script を future edition candidate として扱う M72.13 を主に参照する。

M72.07 は `WG2 N5296` に基づく chart glyph stability の処理であり、J-source / JMJ-source disposition 全体を決めたものではない。M72.13 は Small Seal 11,339 characters を将来版候補として Project Editor に検討させる勧告であり、後続の `WG2 N5344` / `WG2 N5369` で repertoire と range が更新される。

## 決定・Action Item

| Recommendation | 内容 | wiki での扱い |
| --- | --- | --- |
| M72.07 | `WG2 N5296` に記載された J-source glyph changes を Project Editor が revert するよう勧告した。 | [WG2 M72.07 J-source glyph revert recommendation](../events/wg2-m72-07-j-source-glyph-revert.md) |
| M72.13 | `WG2 N5306`、`WG2 N5318R`、`WG2 N5317R3`、`WG2 N5341` をもとに、11,339 Small Seal characters を `38000..3AC4A`、block range `38000..3AC4F` で将来版候補として検討するよう勧告した。 | [WG2 M72.13 Small Seal future edition candidate](../events/wg2-m72-13-small-seal-future-candidate.md) |
| M72.21 | IPSJ/ITSCJ、Masaru Takechi、Shuichi Tashiro、Toki Messe staff など meeting host / support への appreciation。 | [Japan](../people/japan.md)、[WG2 Meeting \#72](../meetings/wg2/wg2-meeting-72.md) |

## 後続決定

Small Seal は `WG2 N5304` 時点では future edition candidate であり、符号化内容は確定していない。`WG2 N5344` で 11,328 characters、range `3D000..3FC3F` の revised proposal へ整理され、WG2 \#73 の `WG2 N5354` / `WG2 N5369` で CD4 disposition と DIS progression に接続する。

J-source glyph revert は WG2 側の chart glyph correction として処理された。IRG 側の source reference、Unihan data、disunification 境界は、`IRG N2859` / `IRG N2870` など別文書で追う。

## 論点

### recommendation の射程

M72.13 は Small Seal の最終 repertoire ではなく、将来版候補として Project Editor が扱う対象を示す勧告である。Small Seal topic では、この時点と WG2 \#73 CD4 disposition 時点を分けて記録する。

### WG2 と IRG の J-source 議論

M72.07 は `WG2 N5296` の chart glyph changes の revert に限定される。J-source owner の立場、IRG source reference changes、Unihan data maintenance は IRG / UTC 文書と突き合わせる必要がある。
