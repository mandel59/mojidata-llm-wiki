# WG2 N5304

## 要約

`WG2 N5304` は、2025-06-23/27 に Niigata, Japan で開催された WG2 Meeting \#72 の recommendations である。ISO/IEC 10646 7th edition CD.2 disposition、CJK additions、code charts、CD.3 progression、J-source glyph correction、Small Seal future edition candidate を扱う。

この wiki では、CD.2 disposition と code charts を [ISO/IEC 10646 Edition and Code Charts](../topics/iso-10646-edition-and-code-charts.md) に、J-source glyph correction の revert を扱う M72.07 を [WG2 M72.07 J-source glyph revert recommendation](../events/wg2-m72-07-j-source-glyph-revert.md) に、Small Seal Script を future edition candidate として扱う M72.13 を [WG2 M72.13 Small Seal future edition candidate](../events/wg2-m72-13-small-seal-future-candidate.md) に接続する。

## 後続決定

CD.2 disposition と code charts は WG2 \#73 でそれぞれ [WG2 N5369](wg2-n5369.md) の CD.4 disposition、[WG2 N5363R](wg2-n5363r.md) の external code charts proposal に進み、[WG2 N5354](wg2-n5354.md) M73.03 / M73.04 で DIS progression に接続する。

Small Seal は `WG2 N5304` 時点では future edition candidate であり、符号化内容は確定していない。[WG2 N5344R2](wg2-n5344r2.md) で 11,328 characters、range `3D000..3FC3F` の revised proposal へ整理され、WG2 \#73 の `WG2 N5354` / `WG2 N5369` で CD.4 disposition と DIS progression に接続する。

J-source glyph revert は WG2 側の chart glyph correction として処理された。IRG 側の source reference、Unihan data、disunification 境界は、`IRG N2859` / `IRG N2870` など別文書で追う。

## 論点

### recommendation の射程

M72.13 は Small Seal の最終 repertoire ではなく、将来版候補として Project Editor が扱う対象を示す勧告である。Small Seal topic では、この時点と WG2 \#73 CD.4 disposition 時点を分けて記録する。

### WG2 と IRG の J-source 議論

M72.07 は `WG2 N5296` の chart glyph changes の revert に限定される。J-source owner の立場、IRG source reference changes、Unihan data maintenance は IRG / UTC 文書と突き合わせる必要がある。

### Code charts と 10646 publication

M72.10 は [WG2 N5315](wg2-n5315.md) を背景に、10646 code charts を core text から分離できるかを検討する勧告である。この論点は [WG2 N5363R](wg2-n5363r.md) と WG2 \#73 M73.03 で、external code charts の normative reference へ進む。
