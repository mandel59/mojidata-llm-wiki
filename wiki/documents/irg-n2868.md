# IRG N2868

## 要約

`IRG N2868` は、`IRG N2835R`、`IRG N2837`、`IRG N2838` が提案した GCH / GXC / GCY source reference values に対する個人 contribution である。

文書は、現在入手しやすい辞書版を参照するだけでは original submission の glyph usage を再現できないため、当時使われた edition と printing を特定して source index を設計すべきだと主張する。GCH、GXC、GCY を同じ結論にまとめず、それぞれの submission history と辞書版の事情に応じた方針を示す。

## フィードバック内容

### IRG N2835R / N2838: GCH-source

`IRG N2835R` が `IRG N2838` の指摘を受け、reference source を《辞海》1999 年版の縮印本から普及本へ変更した点を確認する。一方、original proposal に使われた printing は未特定であり、proposal documents の時期が普及本の初版時期よりわずかに早いことも問題にする。

文書は、普及本だけでなく彩圖本・彩圖珍藏本も比較し、original proposal の character usage を最もよく覆う edition と first printing を確認するよう求める。GCH prefix 自体は維持し、確定した版・刷を bibliographic description に反映する方向を示す。

### IRG N2837: GXC-source

`IRG N2837` が Ext. D evidence に基づいて《現代漢語詞典》第5版を collation basis とすることに対し、第5版の初版は2005年で、Ext. C / Ext. E submission より後であると指摘する。2000年以前に「修訂本」と呼ばれていた第3版の方が `IRG N0743` の時期と記述に合い、第5版で見つからない一部の文字も第3版で確認できるとする。

そのため、GXC reference source を《現代漢語詞典》第3版修訂本（1996）へ変更し、この版に基づき source index を見直すことを提案する。ただし第3版にも多数の printing があるため、複数の適格な刷を比較して安定した suffix を設計する必要があると留保する。

### IRG N2835R / N2838: GCY-source

両文書が《辞源》建国六十周年紀念版（2009）を collation basis とする点について、Ext. B / C / E submission より大幅に新しく、Ext. F とそれ以前では使われた版が異なると指摘する。

文書は、20世紀の《辞源》修訂本を GCY の基礎資料として、四冊本・兩冊本・四冊合訂本のどれが original submission に対応するかを今後調査する方針を示す。旧版になく新版にのみ存在する文字については、`GCYR`（GCY-Revised）という別 prefix を用い、2009年版を参照する案を提示する。

## 後続決定

この文書自体は feedback であり、上記の source reference changes を採択した記録ではない。後続の [IRG N2923R](irg-n2923r.md) は GXC を含む大規模な source-reference update proposal として別に扱い、その提案内容や IRG Meeting \#66 での pending status を `IRG N2868` 自身の結論と混同しない。

## 論点

### edition と printing の識別

中心的な論点は、辞書名や edition だけでは original submission evidence を一意に再現できないことである。同じ edition でも printing によって character usage が異なりうるため、source reference の変更には proposal 当時の実物資料との照合が必要になる。

### source prefix の継続性

GCH は適切な版・刷を特定して既存 prefix を維持する方向、GXC は第3版に合わせて index を再設計する方向、GCY は旧版と2009年版を `GCY` / `GCYR` に分ける可能性を示す。単一の更新規則ではなく、各 source の submission history に応じた扱いを求めている。
