# L2/26-128

## 要約

`L2/26-128` は、Unicode Core Specification Chapter 6 の East Asian quotation marks 説明を更新する提案である。Unicode 17.0 の記述が、Korean、Japanese、Traditional Chinese、Simplified Chinese、Sibe の実際の組版慣習を十分に区別していないことを問題にする。

提案は、新規文字ではなく core spec text の更新である。comma-like quotes と bracket-like quotes を分け、horizontal / vertical layout、East Asian Width、UAX \#50、variation selector の使い方を明示する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| comma-like quotes | U+201C / U+201D / U+2019 / U+201A と U+301D / U+301F の用法を、言語・地域・縦横組に分けて説明する。 |
| Korean | South Korea は U+201C / U+201D を Latin width に近い形で使い、North Korea は U+300A / U+300B を quotation marks として使うと整理する。 |
| Japanese | U+301D / U+301F は主に縦書き内で限られて使われ、横書きの quotation marks では一般化しないとする。 |
| Chinese | PRC horizontal text では U+201C / U+201D、vertical text では U+300C / U+300D などを使う一方、Traditional Chinese outside PRC は bracket-like marks を好むとする。 |
| Sibe | Sibe の縦書きは CJK と進行方向が異なるため、quotation marks の位置も別扱いにする。 |

## 論点

### 文字追加ではなく core spec correction

この提案は新規符号位置を求めるものではなく、既存の quotation mark characters の説明を実際の East Asian layout practice に合わせるものである。したがって判断対象は repertoire ではなく、Unicode Core Specification の説明が UAX \#11 / UAX \#50 / language tagging / variation selector guidance と整合するかである。

### 地域別の quotation convention

同じ CJK 環境でも、Korean、Japanese、PRC Chinese、Traditional Chinese、Sibe で preferred marks と幅・縦書き挙動が異なる。単に East Asian quotation marks と一括りにすると、実装者向け guidance として誤解を生む。
