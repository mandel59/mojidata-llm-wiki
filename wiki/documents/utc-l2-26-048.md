# L2/26-048

## 要約

`L2/26-048` は Jennifer Daniel による Cracked Smiling Face emoji proposal である。文書は、smiling face に crack / rupture を重ねることで、public persona と private truth、resilience、tension、masking、crumbling、transformation などを表す emoji を提案する。

[L2/26-008R](utc-l2-26-008r.md) は、UTC \#185 で U+1FAEB FACE WITH SQUINTING EYES として承認された candidate を CRACKING FACE に変更する根拠として、この proposal を参照する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | CRACKING FACE / cracked smiling face |
| keywords | reveal、resilient、transformation、tension、masking、crumbling、breaking など。 |
| selection rationale | smile と crack の conflict により、inner / outer self、masking、collapse、self-acceptance などを表せるとする。 |
| relation to prior candidate | FACE WITH SQUINTING EYES の代替として U+1FAEB に接続する。 |

## 後続決定

[L2/26-008R](utc-l2-26-008r.md) は U+1FAEB の name を CRACKING FACE に変更し、Unicode 18.0 final candidate list に含めた。したがって [L2/25-259](utc-l2-25-259.md) は replaced candidate、`L2/26-048` は final rationale の文書として扱う。

## 論点

### facial expression の semantic replacement

`L2/26-048` は、同じ code point candidate の semantic target を squinting から cracking / masking に置き換える。Emoji review では、顔 emoji の微妙な表情差が expected use、visual distinctiveness、keyword design に直結する。
