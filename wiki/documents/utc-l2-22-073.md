# L2/22-073

## 要約

`L2/22-073` は African scripts の implementation status を一覧化する report である。encoded scripts については fonts、system support、keyboard、CLDR / library cataloging support を確認し、unencoded scripts については proposal の有無、community evidence、追加資料の必要性を記録する。

KORE SEBELI については、community から encoding proposal が提出済みだが、より広い usage evidence と script details が求められているとする。後続の [L2/23-203](utc-l2-23-203.md) はこの 2022 年 status report を引き継ぎ、N'ti を含む unencoded African scripts の usage / implementation status を更新した。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| purpose | African scripts の current implementation status を共有し、開発・proposal 作業を促す。 |
| encoded scripts | ADLaM、Bamum、Bassa Vah、Coptic、Egyptian Hieroglyphs、Ethiopic、N'Ko、Osmanya、Tifinagh、Vai などの font / OS / keyboard / CLDR 状況を整理する。 |
| unencoded scripts | Adinkra、Beria、Bété、Eghap、Garay、KORE SEBELI、Kpelle、Loma、Mwangwego などについて、proposal 状態と必要な追加 evidence を記録する。 |
| extended Latin / Ajami | African languages 用 extended Latin / extended Arabic では、characters 自体より language-specific keyboard layouts が大きな未解決需要だとする。 |

## 後続決定

[L2/23-203](utc-l2-23-203.md) は 2023 年版の follow-up として、usage / implementation status を encoded / unencoded 別に再整理した。N'ti は `L2/22-073` には現れず、`L2/23-203` で unencoded script として初めてこの report chain に入る。

## 論点

### KORE SEBELI の evidence 要求

`L2/22-073` は KORE SEBELI proposal が community から提出されていることを記録しつつ、widespread usage evidence と script details の追加を求める状態に置く。この観点は、後続の KORE SEBELI proposal chain で community samples、students、vertical samples、font / teaching evidence が増補されていく背景になる。
