# IRG N2911

## 要約

`IRG N2911` は、IRG Meeting \#66 で CJK editorial group が扱った miscellaneous issues の報告である。`IRG N2909` の formal recommendations と重なるが、個別文書ごとの編集上の判断、追加調査、Meeting \#67 への持ち越し理由がより細かい。

中心は、U+2E5A2 disunification request の不採択、China の G-source horizontal extension、G / T / K / UK / V source reference and representative glyph issues、UCV / NUCV lists、IRG P&P、`kIRG_SGSource`、CJKV Components、FS / SC guidelines、radical assignment rules である。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| Disunification | `IRG Disunified Ideographs` page を IWDS Series 4 replacement として受け入れ、U+2E5A2 の disunification と K-source glyph change は受け入れないとした。NUCV#363 に exception note を追加する扱い。 |
| Horizontal extension | China の `IRG N2929` 9156 characters proposal を review し、experts の feedback 後に revised version を IRG Meeting \#67 で受け入れる方向を示した。 |
| G-source references | [IRG N2930](irg-n2930.md) の 3 G-source reference changes は受け入れた。一方、[IRG N2919R](irg-n2919r.md) / [IRG N2923R](irg-n2923r.md) の 342 reference changes は China response 待ち。 |
| T-source / K-source glyphs | [IRG N2927R](irg-n2927r.md) に基づく U+2976E T-source glyph change と、[IRG N2916](irg-n2916.md) に基づく U+2DB7C K-source glyph / `kRSUnicode` change を受け入れた。 |
| UK-source / V-source | [IRG N2918](irg-n2918.md) の UK-source issue は China / UK response 待ち。V-source U+268A1 / U+268A2 は Vietnam が current glyph を意図通りと確認したが、追加 evidence があれば将来 glyph change proposal を出せるとした。 |
| UCV / NUCV | IRG Meeting \#66 で結論した UCV / NUCV examples を次版 `IRG N2931` に merge する。 |
| Other issues | IRG P&P Version 18 Draft 2、`kIRG_SGSource`、CJKV Components proposal、FS / SC guidelines replacement、radical assignment rules finalization、Second Stage Simplification Chinese characters、次期 working set submission estimates を扱った。 |

## 後続決定

[IRG N2909](irg-n2909.md) は、この editorial report の結果を Meeting \#66 recommendations / action items として正式に整理している。Meeting \#67 agenda [IRG N2935](irg-n2935.md) は、`IRG N2911` で持ち越された China horizontal extension、G / UK / V / SAT source issues、WS2024、FS / SC、radical assignment rules を follow-up agenda に置いた。

[L2/26-099](utc-l2-26-099.md) は、`IRG N2911` / `IRG N2909` のうち Unicode Version 18.0 target に入る K-source、T-source、G-source data changes を UTC CJK & Unihan Working Group の action items に変換した。

## 論点

### Formal recommendations との役割分担

`IRG N2911` は editorial group report であり、最終的な recommendation 文書ではない。wiki では、会合決定の canonical source を [IRG N2909](irg-n2909.md) に置き、`IRG N2911` は個別 issue の判断理由や持ち越しの背景を読むための source として扱う。

### Source data issues の queue

この文書は、source reference change、representative glyph change、Unihan property update、member body response、future working set submission を一つの editorial queue として扱っている。特に G-source、T-source、K-source、V-source の扱いは、IRG 内の判断から UTC / WG2 の data update へ進む案件と、Meeting \#67 に残る案件を分けて読む必要がある。
