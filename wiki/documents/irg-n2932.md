# IRG N2932

## 要約

`IRG N2932` は、IRG Working Set 2024 Version 5.0 の main set chart / list である。PDF は候補ごとに serial number、attributes、G / K / USAT / T / UK / UTC / V source columns、discussion record を並べる data document であり、proposal prose ではなく review table として読む。

Version 5.0 は、IRG #63 から #66 までの discussion record、evidence acceptance、source reference updates、radical / stroke / first stroke notes などを反映する。Meeting #67 では、この Version 5.0 review が主要議題に設定されている。
