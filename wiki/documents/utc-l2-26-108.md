# L2/26-108

## 要約

`L2/26-108` は、Draft UAX \#60 Revision 2 "Data for Non Han Ideographic Scripts"、Unicode 18.0.0 draft 1 である。

文書は Jurchen、Nüshu、Seal、Tangut など non-Han ideographic scripts の source data、radical-stroke data、readings、numeric values、other data を UAX \#38 Unihan に近い形式で扱う technical annex として説明する。

後続の RMG / CJK & Unihan WG recommendations では、title を `Data for Large East Asian Scripts` へ変更し、Small Seal data syntax / description を調整する流れが記録される。

## 提案内容

### 対象 scripts と data files

Revision 2 は、Jurchen、Nüshu、Seal、Tangut を対象にし、UCD に含まれる `JurchenSources.txt`、`NushuSources.txt`、`SealSources.txt`、`TangutSources.txt` を説明する。Khitan Small Script は、将来 data properties を追加できる候補として触れられる。

各 data file は UTF-8 / NFC / Unix line endings の text file であり、script ごとの property name prefixes を使う。たとえば Jurchen は `kJURC`、Nüshu は `kNSHU`、Seal は `kSEAL`、Tangut は `kTGT` を使う。

### property categories

文書は sources、radical-stroke counts、readings、numeric values、other data を分類する。source identity properties は normative とされ、それ以外の properties は provisional とされる。

| Script | 主な property | 内容 |
| --- | --- | --- |
| Jurchen | `kJURC_Src`, `kJURC_RSUnicode`, `kJURC_NCReading`, `kJURC_Numeric` | sources、radical-stroke、reading、numeric value。 |
| Nüshu | `kNSHU_DubenSrc`, `kNSHU_Reading` | sources と readings。 |
| Seal | `kSEAL_CCZSrc`, `kSEAL_DYCSrc`, `kSEAL_QJZSrc`, `kSEAL_THXSrc`, `kSEAL_MCJK`, `kSEAL_Rad` | Small Seal sources、modern CJK reference、radical data。 |
| Tangut | `kTGT_MergedSrc`, `kTGT_RSUnicode`, `kTGT_Numeric` | merged sources、radical-stroke、numeric value。 |

### UAX \#38 との関係

UAX \#60 は Han 以外の ideographic scripts に対して、sources や radical-stroke data を標準 annex で説明する点で UAX \#38 Unihan に近い役割を持つ。ただし UAX \#60 の properties は Unicode 18.0 draft 時点で normative / provisional status の整理が続いている。

## 後続決定

[L2/26-095](utc-l2-26-095.md) は PRI \#520 として Draft UAX \#60 を public review issue に含める。[L2/26-099](utc-l2-26-099.md) は `kSEAL_Rad` delimiter、`kSEAL_THXSrc` syntax、description strings の correction を扱い、[L2/26-102](utc-l2-26-102.md) は UAX \#60 title を `Data for Large East Asian Scripts` へ変える release-management action を記録する。

## 論点

### non-Han ideographic data の標準化

Jurchen、Nüshu、Seal、Tangut は、それぞれ source evidence、readings、radical-stroke indexing、numeric values の性質が異なる。UAX \#60 は、これらを UCD data files として同じ annex に収めることで、検索・照合・font support の基礎 data を可視化する。

### Seal data と Small Seal proposal

Seal properties は Small Seal Script の encoding proposal と密接に接続する。`kSEAL_MCJK` は対応する CJK Unified Ideograph、`kSEAL_Rad` は radical grouping、`kSEAL_*Src` は proposal で使われる source evidence への橋渡しになる。
