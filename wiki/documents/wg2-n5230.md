# WG2 N5230

## 要約

`WG2 N5230` は、TCA による [WG2 N5209](wg2-n5209.md) への feedback である。TCA は N5209 が比較対象にした [L2/22-279](utc-l2-22-279.md) を受け、THX に含まれない K / D source entries と、Small Seal に対応する modern CJK values の二点を検討している。

文書は、THX 未収録として扱われた 55 entries について、1 件の既修正 duplicate と、残る 54 entries の分類を示す。さらに TCA and China experts は、Small Seal glyph に近い modern CJK 対応値を優先する方針で 2,883 correspondences を修正した。

## フィードバック内容

| 項目 | 内容 |
| --- | --- |
| 55 entries | N5209 が示した K / D source 由来の THX 未収録 entries を検討対象にする。 |
| 既修正 duplicate | 55 entries のうち 1 件は TCA 側の既修正により duplicate となり、別符号化不要とする。 |
| category 1 | THX と order / interpretation が異なるが shape が近い 1 entry。IVS ではなく別符号化を勧める。 |
| category 2 | THX と同じ order だが glyph が異なる 3 entries。IVS ではなく別符号化を勧める。 |
| category 3 | K / D source 側で duplicate glyphs とみなされる 8 entries。IVS または no encoding を検討対象にする。 |
| category 4 | THX に記録がない 42 entries。別符号化可能とする。 |
| modern CJK | TCA and China experts が対応する modern characters を再確認し、2,883 correspondences を修正した。 |

関連 XLSX は `A1:P2884` の 1 sheet で、ヘッダを除く 2,883 行を持つ。`Type` 列の内訳は `Zhengzhuan` 1,830、`Variant` 1,017、`Newly added character` 36 である。PDF 本文は、2 件を incorrect corresponding modern characters とし、残る多くは現代の common form ではなく Seal script の字形に近い対応値へ修正したと説明する。

## 後続決定

`WG2 N5230` 自体は feedback であり、会議決定ではない。後続の [WG2 N5273](wg2-n5273.md) は、N5230 までの TCA / China feedback を踏まえて N5209 の 55 entries を再検討し、8 duplicates と 47 separately encodable entries へ整理する。さらに [WG2 N5294R3](wg2-n5294r3.md) は、この整理を 11,140 code point repertoire に反映した。

modern CJK 対応値の修正方針は、後続の [WG2 N5346](wg2-n5346.md) や [WG2 N5348R](wg2-n5348r.md) で続く `kSEAL_MCJK` / retrieval data の論点につながる。

## 論点

### IVS ではなく別符号化する境界

N5230 は、shape が近くても interpretation / pronunciation が異なるもの、または same order でも glyph structure が大きく異なるものを別符号化候補にする。一方で、K / D source 内の duplicate glyphs は IVS または no encoding の検討対象にしており、後続の unification / disunification review の前段階として読める。

### modern CJK 対応値の役割

文書は、modern CJK value を現代の一般的な字形へ単純に寄せず、Small Seal glyph に近い form を優先する。これは `kSEAL_MCJK` を検索・索引用の data として使う一方、字形・字源・現代字の対応が一対一ではないことを示す。
