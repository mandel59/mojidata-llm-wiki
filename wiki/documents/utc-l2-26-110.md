# L2/26-110

## 要約

`L2/26-110` は、Proposed Update UAX \#31 Revision 44 "Unicode Identifiers and Syntax"、Unicode 18.0.0 draft 2 である。

UAX \#31 は general-purpose identifiers、immutable identifiers、hashtag identifiers、pattern syntax、normalization / case guidance を定義する。

Revision 44 は Unicode 18.0 向けに、script reclassification の連絡先と guidelines、immutable identifiers と XML Names の関係、Unicode 18 で新規符号化される scripts の Excluded Scripts table への追加を扱う。

## 提案内容

### Revision 44 modifications

Revision 44 の modifications は次の通りである。

| 箇所 | 変更 |
| --- | --- |
| Limited Use Scripts 後 | script reclassification について PAG を primary point of contact とし、UTC 承認済み guidelines を参照する note を追加。 |
| Section 3 | default identifiers、immutable identifiers、XML Name / Nmtoken の関係に関する note を追加。 |
| Table 4 Excluded Scripts | Unicode 18 で新規符号化される Chisoi、Jurchen、Proto-Cuneiform、Seal を追加。 |

### identifier model

文書全体は、`ID_Start` / `ID_Continue` に基づく Default Identifiers、version をまたいで安定する Immutable Identifiers、hashtag identifiers、Pattern_Syntax / Pattern_White_Space、normalization / case folding、Default_Ignorable_Code_Point を扱う。

Default_Ignorable_Code_Point は identifiers で問題を起こしやすく、profile により除外する standard profile も示される。

## 後続決定

[L2/26-096](utc-l2-26-096.md) は UTC \#187 の PAG report として、UAX \#31 / UAX \#44 proposed update を properties and algorithms review の一部に置く。UAX \#44 側の Unicode 18.0.0 draft review は [PRI \#547](pri-547.md) として別に page 化する。UTC \#187 agenda [L2/26-092](utc-l2-26-092.md) でも public review issue として扱われる。

## 論点

### identifier security と script classification

UAX \#31 の Recommended / Limited / Excluded scripts は、identifier security と実装者の default policy に影響する。Revision 44 は Unicode 18 の新規 scripts をまず Excluded Scripts に入れ、再分類の窓口を PAG に明示する。

### Default_Ignorable と variation selectors

Default_Ignorable_Code_Point を identifiers から除外する profile は、variation selectors / hidden text behavior の議論と接続する。[L2/26-070R](utc-l2-26-070r.md) の variation selector conformance issue と併読する必要がある。
