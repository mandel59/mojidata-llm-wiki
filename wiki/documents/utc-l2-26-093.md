# L2/26-093

## 要約

`L2/26-093` は UTC Meeting \#187 の minutes である。Unicode 18.0 beta review authorization、working group reports、liaison reports、CJK & Unihan Working Group recommendations などを記録する。

CJK & Unihan 範囲では、[L2/26-099](utc-l2-26-099.md) に基づく Unihan database / UAX \#38 / UAX \#45 / UAX \#60 / CJK Strokes / CJK confusables / IRG follow-up の action items が並ぶ。UAX \#45 additions では、2026 年提出の proposals から 30 records を `FutureWS` status で受け入れる consensus と action items が置かれた。

この minutes は、proposal page で「後続決定があるか」を確認するための一次ソースになる。たとえば [L2/26-044](utc-l2-26-044.md) は UTC \#187 で受理されたが、2025 年提出の日本地名・古典籍 ideograph proposals はこの UAX \#45 decision set に現れない。

## 決定・Action Item

### Unicode 18.0 beta review

UTC \#187 は [L2/26-102](utc-l2-26-102.md) などの Release Management Group report に基づき、Unicode 18.0 beta review の開始を authorize した。beta review landing page、PRI page、publication coordination などの action items が RMG / PAG / UTC に置かれている。

### CJK & Unihan Working Group recommendations

minutes の E.1 は CJK & Unihan Group recommendations を扱い、[L2/26-099](utc-l2-26-099.md) の各 section を UTC \#187 の consensus / action item に展開する。主な範囲は Unihan property changes、PRI \#534 / UAX \#38 feedback、UAX \#45 record updates、CJK Strokes standardized variation sequences、CJK confusables、IRG \#66 recommendation follow-up である。

### UAX \#45 FutureWS additions

UAX \#45 additions では、2026 年提出の proposals に基づく 30 U-source records が `FutureWS` status で受け入れられた。日本地名関係では [L2/26-044](utc-l2-26-044.md) が対象になり、[UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) の canonical event に接続する。

minutes の Section 40 では、UAX \#45 additions の増加と次の IRG Working Set の処理容量も議論され、Unicode Version 19.0 までは新たな UAX \#45 additions の検討を止める方針が説明されている。したがって、UTC \#187 の decision set に入らなかった proposal は、単に wiki link がないだけでなく、少なくともこの minutes の範囲では未処理として読む必要がある。

## 後続決定

この minutes の CJK & Unihan 範囲は、[UTC Meeting \#187](../meetings/utc/utc-meeting-187.md)、[Unicode 18.0 Change Sources](../topics/unicode-18-change-sources.md)、[UAX \#45 / U-Source Ideographs](../topics/uax45-u-source-ideographs.md)、[Japanese Place-Name Ideographs](../topics/japanese-place-name-ideographs.md) から辿る。

## 論点

### proposal と decision set の切り分け

UTC \#187 minutes は、proposal document が提出された事実と、UTC が data update として受け入れた事実を分けて確認するために重要である。UAX \#45 proposals では、個別 proposal page が evidence と proposed record を扱い、UTC \#187 minutes / [L2/26-099](utc-l2-26-099.md) / event page が採択・action item 側を扱う。

### pending proposal の根拠

2025 年の日本地名・古典籍 ideograph proposals は catalog には available として存在するが、この minutes の UAX \#45 FutureWS additions には含まれていない。このため、wiki ではそれらを採択済み record とせず、現行 catalog / cache で確認できる限り pending proposal として扱う。
