# L2/24-279

## 要約

`L2/24-279` は、KATAKANA DIGRAPH YORI を Kana Extended-A に追加する提案である。提案者は、`L2/23-112` に含まれていた katakana yori ligature が `L2/24-150` の reduced proposal では外れたため、追加 attestations とともに再提出している。

文書は、既存の U+309F HIRAGANA DIGRAPH YORI と U+30FF KATAKANA DIGRAPH KOTO の関係を踏まえ、katakana yori も repertoire 上の対応関係として必要だと述べる。

| 根拠 | `L2/23-112` で示された attestations に加え、辞典・古典資料から追加例を提示。 |

## 論点

### `L2/24-150` からの継続

`L2/24-150` は missing kana ligatures の reduced set を提案したが、KATAKANA DIGRAPH YORI は含めなかった。`L2/24-279` は、CJK & Unihan WG が katakana ligatures には比較的前向きだったという前段の議論を引用し、YORI も個別に検討すべきだとする。

### 縦書き文脈

合略仮名は縦書き資料との関係が強く、単一文字として符号化する場合は Vertical Orientation や names list annotation とも関係する。この文書は character addition proposal であり、全体の model は [Kana](../topics/kana.md) 側で追跡する。
