# IRG N2959

## 要約

`IRG N2959` は、SAT-source glyphs の issue document である。Ma Shijie は、SAT-source characters の current representative glyphs を、Li Guoying `大正藏疑难字考释` などの evidence と比較し、glyph、radical、stroke data の修正候補を提示している。

この文書は G-source ではなく SAT-source を扱う。ただし、current representative glyph と source evidence の整合を確認し、必要に応じて chart glyph や indexing data を直すという意味では、G-source glyph issues と同じ IRG Section 9.1 の review queue に属する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | SAT-source characters の representative glyphs と関連 data。 |
| 根拠 | SAT-source evidence と関連字書・考証資料。 |
| 修正対象 | glyph form、radical assignment、stroke count など。 |
| 作業種別 | Existing encoded characters の source glyph / indexing data issue。 |

## 後続決定

Meeting \#67 agenda の [IRG N2935](irg-n2935.md) は、representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。`IRG N2959` 自体は issue document であり、採否は記録していない。

## 論点

### SAT-source を G-source と混同しない

`IRG N2959` は、G-source owner の correction ではなく SAT-source glyph issue である。G-source topic の隣接案件として扱えるが、source 責任主体、evidence collection、data correction path は SAT-source 側で確認する必要がある。
