# IRG N743

## 要約

`IRG N743` は、CJK Extension C の作業に向けて IRG が受領した submissions を集約した一覧型の文書である。個々の ideograph の採否を決定する文書ではなく、各 member body から出された候補を把握し、後続の統合・照合・review の入力を整理するためのスナップショットとして位置付けられる。

後年の G-source reference 議論では、Extension C submission 当時にどの資料が使われたかを推定する時点資料として参照される。ただし、この summary だけで個々の glyph、辞書の edition / printing、最終的な source reference を確定することはできない。

## データ内容

文書は Extension C submissions を集計し、IRG が次の作業段階で扱う候補群の規模と提出元を概観できるようにする。目的は submission inventory の共有であり、original evidence、character-by-character review、unification judgement、最終採択結果を置き換えるものではない。

とくに後続の GXC-source 調査では、`IRG N743` の年代と記述が original submission に使用された《現代漢語詞典》の版を考える手掛かりとして使われている。この利用は submission history の復元であり、`IRG N743` 自身が辞書版や source-reference suffix を規定したことを意味しない。

## 後続決定

この文書自体は集計資料であり、候補の accept / reject や Extension C repertoire の確定を記録する decision document ではない。China の後続 submission record は [IRG N891](irg-n891.md) に、GXC reference の edition / printing をめぐる再検討は [IRG N2868](irg-n2868.md) に、後続の大規模 source-reference update proposal は [IRG N2923R](irg-n2923r.md) に分けて読む必要がある。

## 論点

### submission summary の限界

集計値は特定時点の submissions の規模を示すが、重複候補の統合、withdrawal、追加 evidence、review 後の状態変化までは表さない。したがって、後年の encoded repertoire や現在の Unihan source data と単純に一対一対応させるべきではない。

### 後続 G-source 議論との関係

[IRG N2868](irg-n2868.md) は、GXC-source の original submission に使われた辞書版を検討する際、`IRG N743` の時期を根拠の一つにする。ここでの `IRG N743` の役割は contemporaneous inventory であり、個々の source reference correction の直接証拠には original submission、辞書実物、後続 evidence documents が別途必要になる。
