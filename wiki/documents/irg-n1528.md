# IRG N1528

## 要約

`IRG N1528` は、`IRG N1531` で言及された 3,125 G Ext D source characters のうち143字について、`Sawndip Sawndenj`（`古壮字字典`、Dictionary of Ancient Zhuang Characters）に確認できる evidence を整理した data document である。対象辞書は1989年刊で、文書は収録字を pre-1912 Zhuang characters と位置づけ、143字を Extension D にretainするよう求める。

各対象には Ext D number、G source、Zhuang reading、glyph、IDS、辞書の page number、Chinese gloss が付され、辞書entryのscanも提示される。これは候補字の実在・用例をreviewするための evidence package であり、Extension D repertoire の採否や後年のsource-reference変更を決定する文書ではない。

文書は、辞書に複数の参照箇所がある字についても evidence として1 entryだけを掲げる。このため、表は辞書内の全用例を網羅するcorpusではなく、候補字ごとの存在確認を目的とした選択的なevidence listとして読む必要がある。

## 後続決定

この文書の action は `to be discussed` であり、143字の採否を確定するrecommendationやminutesではない。Extension D repertoire にretainされたかどうかは、後続のIRG / WG2 decision recordで確認する必要がある。

後年の [IRG N2923R](irg-n2923r.md) は、GRM / GDZ / GWZ source references の再設計を検討する際に `IRG N1519` と `IRG N1528` のevidenceを再利用する。ただし同文書は、これらのevidenceがoriginal proposed source referencesと直接対応しないと明記する。そのため `IRG N1528` の再参照は、2008年のretain requestがそのままsource-reference変更として承認されたことを意味しない。

## 論点

### evidence と original source reference の違い

`IRG N1528` が示すのは、対象glyphに対応するZhuang characterが辞書にあり、reading・meaning・locationを追跡できることである。これは候補字をretainする根拠にはなるが、当初割り当てられた G source reference の由来やsyntaxを直接立証するとは限らない。

`IRG N2923R` では、この限界を踏まえつつ、`古壮字字典` のlocationを一部のGWZ-source charactersに対する代替reference候補として用いる。したがって再参照の意味は、古いevidence packageが後年のtraceable reference構築に利用可能だった点にあり、original referenceとの同一性を示す点にはない。

### evidence coverage の限界

対象は3,125字中143字に限られ、各字についても複数entryのうち1件だけを示す場合がある。また、文書はcandidate evidenceを提示するが、unification、glyph design、source mappingの妥当性を一括して判定してはいない。
