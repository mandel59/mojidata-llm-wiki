# L2/26-100

## 要約

`L2/26-100` は Script Encoding Working Group が UTC Meeting \#187 に提出した script proposal recommendations である。Unicode 18.0 向けに進める個別文字、glyph correction、script model の未解決点、roadmap policy をまとめる。

この report は、正式な script proposal そのものではなく、SEW が UTC にどれを進め、どれを保留し、どの点を再提案に戻すかを整理する hub である。Latin phonetic additions、Romance dialectology letters、Albanian letter edhe、Belarusian Ruble sign、Arabic Waw with Ring、Shaaldaa / Cossic name changes などが重要な決定候補になる。

## 決定・Action Item

| 区分 | 関連文書 | 勧告 |
| --- | --- | --- |
| Latin / phonetic additions | `L2/26-065`, `L2/26-118`, `L2/26-079`, `L2/26-117` | LATIN SMALL LETTER ZH、MODIFIER LETTER CAPITAL Y、Romance dialectology symbols、Catholic Albanian letter edhe を provisional assignment へ進める。 |
| Currency / symbols | `L2/26-089`, `L2/25-131` | Belarusian Ruble sign と Alchemical Symbol Retort-2 を進め、Retort-2 には crossbar annotation を付ける。 |
| Arabic Waw with Ring | `L2/26-119`, `L2/26-120` | U+06C4 glyph correction、U+06C4 / U+1EE85 confusables、cross-reference update を扱う。 |
| Indic script property cleanup | - | U+1CF5 / U+1CF6 から scx=Deva を除く。 |
| Shaaldaa / Cossic names | `L2/26-040R2` など | Shaaldaa と Cossic の character names / annotations を更新する。 |

## 後続決定

UTC \#187 minutes [L2/26-093](utc-l2-26-093.md) が、SEW report を Unicode 18.0 beta review の処理対象として記録する。個別 proposal は文書ごとに evidence と character repertoire を持つため、採択・保留・再提案の状態はこの report と minutes を合わせて確認する必要がある。

## 論点

### script model が未決の proposal

Ndiko Jonam / Luo Lakeside、N'ti、N'Ko Bambara、Brahmi virama minimization などは、文字の必要性だけではなく joining behavior、positional forms、productive mechanism、stability といった encoding model の未解決点があるため、SEW でさらに検討される。

### backwards compatibility と stability

Mongolian standardized variants では、既存 standardized variation sequences を消すのではなく、後方互換性を保つ revised proposal が期待されている。SEW report は、新規符号化だけでなく既存標準との整合性を script pipeline の主要条件として扱う。

### roadmap と IP clearance

Cirth / Tengwar は IP clearance と SMP space の観点から roadmap から外された。これは UTC の script pipeline が技術的妥当性だけでなく、権利関係と code space planning も含めて管理されることを示す。
