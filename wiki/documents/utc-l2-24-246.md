# L2/24-246

## 要約

`L2/24-246` は、KORE SEBELI / Wakara script の 2024 年 proposal である。文書内では June 2024 の initial stage version 08 とされ、repertoire を 95 characters に再編している。

2020-2022 年版の 106 characters model から、38 lowercase、38 capital letters、10 digits、punctuation / mathematical / other signs を中心にした model へ移る。文書の proposal summary form にはなお 106 characters と出る箇所があり、本文中の 95 characters count と食い違うため、後続 review では数え方の整合性が論点になる。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script scope | Soso language 用 KORE SEBELI / Wakara script。 |
| version | initial stage version 08。 |
| repertoire | 95 characters。10 digits、38 lowercase、38 capital letters、4 punctuation marks、5 mathematical / other signs と説明する。 |
| direction | horizontal left-to-right を default とし、vertical direction は伝統的 use / sample として示す。 |
| change from earlier drafts | 2020-2022 年版の 106 characters model から repertoire を整理する。 |
| consistency concern | proposal summary form に 106 characters と残る箇所があり、main count の 95 characters と一致しない。 |

## 後続決定

この proposal 自体への UTC 決定は、この wiki では未確認である。後続の [L2/26-050R](utc-l2-26-050r.md) / [L2/26-132](utc-l2-26-132.md) では version 12 として 93 characters が表紙・目次・summary に出る一方、本文では 91 characters と数える箇所がある。

## 論点

### repertoire reduction

2024 年版は 106 characters から 95 characters への中間段階である。初期版で combining diacritics / sequences として扱われた vowel marks が、後続の atomic dotted-letter repertoire とどう対応するかを読むための基準になる。

### proposal form の count 不整合

文書の主要箇所は 95 characters とするが、proposal summary form に 106 characters が残る。この不整合は、同じ proposal chain の更新漏れとして扱い、採択済み repertoire の根拠としては後続版を優先して確認する。
