# L2/26-064

## 要約

`L2/26-064` は、译 / 訳 / 譯 の variant 系として使われた non-standard simplified form 1 ideograph を提案する文書である。対象文字は IDS `⿰讠尺`、`kMandarin` `Yì`、total strokes `6`、first stroke `3`、radical-stroke `149.4` として示される。

根拠は 1930-1960 年代の character reform / simplified-form 資料と、1987 年の編集・出版・印刷 handbook、韓国側の漢字略字試案、関連研究である。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | 译 / 訳 / 譯 variant 系の 1 ideograph を追加する。 |
| Data | IDS `⿰讠尺`、`kMandarin Yì`、TS `6`、FS `3`、RS `149.4`。 |
| Evidence | `簡體字原`、`字庫`、`現代(上海 1932)`、`编辑出版印刷手册`、韓国の漢文略字試案、関連研究。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、`L2/26-064` を含む UAX \#45 additions を `FutureWS` records として受け入れるよう勧告した。採択・data update 側の summary は [UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) に集約する。

## 論点

### non-standard simplified form の evidence

この proposal は、official simplification ではなく、近現代に流通した non-standard simplified form を UAX \#45 record に入れる例である。関連する略字・簡体字形の扱いは [CJK Abbreviated and Simplified Ideographs](../topics/cjk-abbreviated-and-simplified-ideographs.md) に集約する。
