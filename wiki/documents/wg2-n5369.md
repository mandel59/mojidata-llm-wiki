# WG2 N5369

## 要約

`WG2 N5369` は、ISO/IEC 10646 7th edition CD.4 consultation comments の disposition である。comments は China、India、Japan、USA から出され、すべて accepted または accepted in principle とされた。

Small Seal 関連では、11,328 characters の names を `SEAL CHARACTER-XXXXX` から `SMALL SEAL CHARACTER-XXXXX` に変えること、6 characters の `kSEAL_Rad`、20 characters の `kSEAL_MCJK`、U+3F80E の glyph correction がまとめられている。WG2 Meeting \#73 の Recommendation M73.01 は、この disposition を SC2 が受け入れるよう勧告した。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| Name changes | `3D000..3FC3F` の Small Seal character names を `SMALL SEAL CHARACTER-XXXXX` に変更する。 |
| Radical corrections | 6 characters の `kSEAL_Rad` を修正する。 |
| Modern CJK corrections | 20 characters の `kSEAL_MCJK` を修正する。 |
| Glyph correction | U+3F80E の unified TH-Y version が original DYC source に合うよう glyph correction する。 |
| 進行 | dispositions 後に ISO/IEC 10646 7th edition DIS を発行する想定。 |

## 論点

### China comment CN6-025

Small Seal の property correction は China comment `CN6-025` に基づく。comment は Small Seal code charts の radical values と modern CJK values の修正を求め、secretariat observation はこれを accepted とした。さらに 3 件の modern CJK correction、すなわち U+3FB84、U+3EE45、U+3EC25、および U+3F80E glyph correction が追加された。

### N5344R2 / N5355 との接続

`kSEAL_MCJK` 修正の説明は、`WG2 N5344R2` の “Correspondence to the modern CJK characters Principle” を根拠にし、`zhèngzhuàn`（正篆）characters で form and meaning が一致する modern CJK unified ideograph を対応づけると述べる。`WG2 N5355` は、`zhèngzì`（正字）/ normalized form ではなく `zhèngzhuàn`（正篆）/ `chóngwén`（重文）の関係として説明すべきだと補足している。
