# L2/23-123

## 要約

`L2/23-123` は、`L2/23-118` の superscript katakana letters for Ryukyuan languages に対する feedback である。Eiso Chan は、現時点の evidence では安定した usage として確認するには早く、UCS / Unicode に入れる前に複数の点を確認すべきだと述べる。

主な疑問は、根拠資料が 2022 年公開の同一資料に依存していること、orthography が今後さらに新文字・記号を必要とする可能性、vertical layout / UAX #50 の情報不足、running text や signage の例がないこと、提出主体が Japan NB または Okinawa Prefecture であるべきではないか、という点である。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| 安定性 | 根拠資料の公開時期が新しく、usage の安定性を確認しにくい。 |
| typography | superscript letters は日本語・東アジア typography では新しい要素で、vertical layout と UAX #50 情報が必要。 |
| orthography scope | 他の記号や他 orthographies との関係、Script_Extensions / combining mark として扱う可能性を検討すべき。 |
| 提出主体 | 地方政府公式 orthography なら Japan NB または Okinawa Prefecture からの governmental requirement として提出すべきではないかと指摘。 |

## 論点

### kana repertoire と公的 orthography

この feedback は、proposal の文字そのものだけでなく、どの団体が evidence と requirement を担保するかを問う。Ryukyuan languages の表記は文化継承上重要だが、公的 orthography と Unicode proposal の接続には、安定性・実装・縦書き・他記号との相互作用を確認する必要がある。
