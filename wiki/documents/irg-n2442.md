# IRG N2442

## 要約

`IRG N2442` は、《中国大百科全書》を出典とする175 GBK-source charactersについて、巻・page・同一page内の位置を示すreference valuesへの更新を求めた個人提案である。新形式を `GBK-xxyyyyz` とし、全175字のlocatorと対応するevidence画像を一覧化する。

あわせて、原典と一致しないと判断したU+2A931とU+2AC00のrepresentative glyph変更、および他sourceの符号化時に同書が引用されたU+30BFCとU+30C5DへのGBK horizontal extensionを提案する。本文はIRGとChina NBに検討を求めるproposalであり、変更の採択記録ではない。

## 提案内容

### 175 GBK-source references

GBK sourceの根拠を《中国大百科全書》初版74冊とし、そのうち55のvolumeを所定の番号へ対応付ける。reference syntax `GBK-xxyyyyz` の `xx` はvolume number、`yyyy` はpage number、`z` は同一pageに複数文字がある場合のswift digitを表す。Table 2は全175字について、旧値、提案値、原典内のlocation、evidence画像名を対応させる。

この処理は単なる連番付与ではなく、各文字を刊本中の位置へ戻せるようにするbibliographic correctionである。本文と別にRegistryから公開される39MBのEvidence ZIPが、Table 2で参照する画像実体を収録する。

### representative glyphs と horizontal extension

文書は59字のglyphが疑わしいとする調査結果を述べ、そのうち正しい形が未符号化である9字と、現行glyphがevidenceに合わない2字を区別する。具体的なglyph変更提案はU+2A931とU+2AC00の2字に限られ、`IRG N2246`等の資料を根拠に原典に対応する形へ直すよう求める。

さらにU+30BFCには`GBK-2103670`、U+30C5Dには`GBK-2608790`を追加するhorizontal extensionを提案する。これは175の既存GBK-source reference更新とも2字のglyph変更とも別の処理である。

## 後続決定

本文のstatusは `Individual Contribution to IRG #55`、actionはIRGとChina NBによるconsiderationである。このため、175件のreference、2件のglyph、2件のhorizontal extensionを採択済みとは扱わない。

[IRG N2689](irg-n2689.md) は後に、この175 GBK-source charactersを1,685 G-source reference updatesの一部として再整理した。ただし31字を別sourceへ移すなど対象の扱いを変更しており、`IRG N2442`そのものの承認記録ではない。[IRG N2838](irg-n2838.md) は、本書で定義した末尾swift digitの考え方をGHC reference syntaxの検証に援用したが、GBK提案へのdecisionではない。

## 論点

### reference evidence と機械可読値

提案値はvolume・page・同一page内の位置を符号化し、Evidence ZIPの画像名とも対応する。したがって、reference valueの妥当性はPDFの表だけでなく、別添evidenceから刊本上の位置を再現できることに依存する。

### source reference とglyph identityの分離

175字のreference更新は出典位置の修正であり、59字のglyph調査や2字の代表字形変更とは範囲が異なる。さらにhorizontal extensionは既存文字へGBK sourceを追加する処理である。後続文書を読む際も、これらを一括してglyph変更として扱わない必要がある。
