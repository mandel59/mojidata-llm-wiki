# IRG N2721

## 要約

`IRG N2721` / `WG2 N5284` は、IRG Convenor が Japan に IRG active member body への復帰を求めた文書である。2019 年の Japan activity report `IRG N2367` で、Japan NB が human resource 不足を理由に IRG Working Set review から離れると述べた経緯を前提にする。

文書は、Japan が Extension G / H に ballot comments を出したこと、2023 年に `WG2 N5221` として 36,000 件超の horizontal extension を WG2 に提出したこと、Han ideograph standardization における Japan の expertise を理由に、IRG 内での早期 review participation を求める。

## 報告内容

| 論点 | 内容 |
| --- | --- |
| ballot phase comments | Japan が Extension G / H へ ballot comments を提出したことを挙げ、成熟した working set への late comments は IRG work を disrupt しうるとする。 |
| `WG2 N5221` | Japan が WG2 に提出した 36,000 件超の JMJ horizontal extension は、IRG に提出されるべきだったと述べる。 |
| IRG の review 適性 | IRG は WG2 より頻繁に会合し、horizontal extensions を review する CJK experts がいると説明する。 |
| participation barrier | COVID-19 後、spring meeting は virtual、fall meeting は hybrid になり、travel budget がなくても参加しやすくなったとする。 |

## 後続決定

[WG2 N5301](wg2-n5301.md) の Section 8.2 は、WG2 Meeting \#72 で `WG2 N5284` を議論した。Japan representative は復帰を検討できるが human resources / workload が課題だと述べ、experts は Maintenance Agency model や IRG review の効率化を踏まえて active participation を促した。

## 論点

### JMJ horizontal extension の review body

この文書は、[WG2 N5221](wg2-n5221.md) を単なる Japan NB proposal ではなく、IRG review workflow の問題として読み替える。大規模な source reference addition は、WG2 ballot phase で late comment として処理するより、IRG working set / horizontal extension review の早い段階で扱う方がよいという主張である。

### Japan NB の source owner role

Japan は `J-source` / `JMJ-source` の source owner として、source data、representative glyph、disunification / horizontal extension proposal に対する判断を求められる。後続の [IRG N2722](irg-n2722.md)、[IRG N2859](irg-n2859.md)、[IRG N2870](irg-n2870.md) は、この role が具体的な correction issue に現れた例である。
