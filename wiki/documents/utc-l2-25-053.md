# L2/25-053

## 要約

`L2/25-053` は、兵庫県淡路市黒谷字「ひよケ原」に使われる未符号化 ideograph を UAX #45 に追加する提案である。文書は対象文字を IDS `⿰向鳥`、`kJapanese` reading `ひよ`、radical-stroke `196.6`、total strokes `17`、first stroke `4` として示す。

提案 record は `USourceData.txt` の `FutureWS` entry として示され、UTCHan.ttf へ代表 glyph を追加するための TTF file も添付される。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | IDS `⿰向鳥` の 1 ideograph を UAX #45 に追加する。 |
| 用途 | 兵庫県淡路市黒谷字「ひよケ原」の地名。 |
| Data | `kJapanese ひよ`、`RS 196.6`、`TS 17`、`FS 4`。 |
| 提案 record | UTC source record として `FutureWS` に置き、将来の CJK working set submission へ渡す候補にする。 |

## 後続決定

現行 catalog と既存 cache では、`L2/25-053` 自体を受け入れた UTC consensus / action item は確認していない。[UTC #187 UAX #45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) では、同じ日本地名系の [L2/26-044](utc-l2-26-044.md) が [L2/26-099](utc-l2-26-099.md) Section 24 に基づいて `UTC-03661` として受理されたが、`L2/25-053` はその受理対象には含まれていない。

したがって、`L2/25-053` は現時点では UAX #45 / FutureWS への submitted proposal として扱い、採択済み record ではなく後続の CJK & Unihan Working Group review を待つものとして追跡する。

## 論点

### Touki System に残る地名文字

文書は、この文字が Hanyo-Denshi project（汎用電子情報交換環境整備プログラム）には含まれなかったが、Touki System（登記システム）上の地名として確実に使われていると説明する。したがって、この提案は日本の行政・登記実務に残る文字を、UAX #45 の UTC source data へ移す事例である。

### evidence の性格

主な evidence は、不動産登記記録の公図と、兵庫県小字名集である。小字名集では平仮名表記も示されるため、`kJapanese` reading `ひよ` の根拠にもなる。
