# L2/25-256

## 要約

`L2/25-256` は LIGHTHOUSE emoji の proposal である。Thomas Cappelli / Pierre Avinain は、lighthouse を maritime navigation、travel、safety、guidance、heritage、enlightenment などを表す symbol として提案する。

Proposal は、世界各地にある lighthouse の認知性、観光・歴史・建築・安全の文脈、既存 emoji では lighthouse-specific meaning を表せない点を主張する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | LIGHTHOUSE |
| keywords | lighthouse、sea、navigation、travel、light、safety、guidance など。 |
| selection rationale | maritime safety / travel / heritage の多義性、世界的な認知性、既存 emoji との非重複性。 |
| image distinctiveness | tower with light として文化差を超えて認識されやすいとする。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1F6D9 LIGHTHOUSE を Unicode 18.0 additions として受け入れ、[L2/26-008R](utc-l2-26-008r.md) の final candidate list でも維持した。

## 論点

### symbolic object と travel category

Lighthouse は literal object でありながら、guidance / safety / hope の symbol として使われる。Emoji repertoire では、object category の補完だけでなく、短い symbol sequence としての用途も評価される。
