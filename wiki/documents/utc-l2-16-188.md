# L2/16-188

## 要約

`L2/16-188` は、Japan NB による Hentaigana revised repertoire の再提出である。表紙では `SC2 N4476` / `WG2 N4732` として示され、SC2/WG2 Meeting #64 と UTC との詳細議論に沿って revised repertoire を提示すると説明する。

文書は、`L2/15-343` の流れを引き継ぎつつ、`HENTAIGANA LETTER E-1` を withdrawn と明記する。また、別文書 `SC2 N4477` で U+1B001 に new character name alias を追加することも note しており、江 由来の字形を historical YE と hentaigana のどちらで扱うかという境界整理に関係する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | Hentaigana revised repertoire と associated spreadsheet。 |
| 主な変更 | `HENTAIGANA LETTER E-1` の withdrawn。 |
| 関連 action | U+1B001 への character name alias 追加を別文書で提案。 |

## 論点

### E-1 withdrawal と HIRAGANA LETTER ARCHAIC YE

`L2/16-188` の E-1 withdrawal は、江 由来の字形を hentaigana repertoire 内の E 系として残すのではなく、historical YE の letter として別扱いする整理に接続する。これは `L2/08-358` が求めた「HIRAGANA LETTER YE は hentaigana 一般の符号化ではない」という区別を後続 repertoire に反映したものとして読める。

### repertoire chart と machine-readable review

文書は PDF chart と Excel 版の revised repertoire を attachment として示す。後続の review では、glyph、mother ideograph、phonetic value、統合対象を表形式で扱えることが重要になる。
