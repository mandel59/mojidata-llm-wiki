# L2/26-141

## 要約

`L2/26-141` は historical mathematical writings の slashed digits を表すため、2 combining marks を追加する proposal である。以前の slashed figures proposal から、separate combining elements として整理し直したものと説明される。

提案文字は U+20F1 COMBINING LONG TRIPLE SOLIDUS OVERLAY と U+20F2 COMBINING SALTIRE OVERLAY である。既存の U+0338、U+20E5、U+20EB では表せない fourth / fifth strike mode を対象にする。

## 提案内容

| Proposed code point | Character name | 用途 |
| --- | --- | --- |
| U+20F1 | COMBINING LONG TRIPLE SOLIDUS OVERLAY | digits や symbols に triple slash-like overlay を付ける。 |
| U+20F2 | COMBINING SALTIRE OVERLAY | saltire-shaped overlay を付ける。 |

## 論点

### precomposed slashed digits から combining marks へ

この proposal は、複数セットの precomposed slashed digits を追加するのではなく、combining overlays によって historical mathematical notation を表す方向へ切り替えている。既存 combining marks との coverage と normalization impact が review point になる。
