# L2/20-180

## 要約

`L2/20-180` は、KORE SEBELI / Wakara script を Unicode に符号化する最初期の UTC proposal である。文書自身は July 2020 の initial stage version 03 として、Soso language 用の contemporary Guinean script を扱う。

提案 repertoire は 106 characters で、12 digits / numerals、30 lowercase letters、29 capital letters、4 combining diacritics / tone marks、punctuation、mathematical signs、vowel + diacritic sequences を含む。後続版では repertoire と encoding model が大きく整理されるため、この文書は初期モデルの比較点になる。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script scope | Mohamed Bentoura Bangoura が Soso language 用に作成した contemporary script。Koré Sébèli / Wakara とも呼ばれる。 |
| version | initial stage version 03。 |
| repertoire | 106 characters。12 digits、30 lowercase、29 capital letters、4 combining diacritics / tone marks、4 punctuation marks、3 mathematical / other signs、24 vowel sequences を含む。 |
| direction | horizontal left-to-right を default とし、vertical direction は older / traditional use として保持する。 |
| combining model | vowels は combining diacritics との sequences を使う一方、dot を含む consonant letters は atomic character として扱う。 |
| evidence | syllabary、student notebooks、song / anthem samples、vertical samples、teaching / community use を提示する。 |

## 後続決定

この proposal 自体への UTC 決定は、この wiki では未確認である。後続の [L2/21-209](utc-l2-21-209.md)、[L2/22-222](utc-l2-22-222.md)、[L2/24-246](utc-l2-24-246.md)、[L2/26-050R](utc-l2-26-050r.md)、[L2/26-132](utc-l2-26-132.md) が同じ script の revised proposal chain を構成する。

## 論点

### 初期 combining model

2020 年版では、vowel distinctions を combining diacritics / tone marks と sequences で扱い、consonant dots は letter shape の一部として atomic に扱う。この分離は、2026 年版で dotted vowel letters も atomic repertoire に寄るため、encoding model の変化として追う必要がある。

### repertoire count

2020-2022 年版の 106 characters は、2024 年版の 95 characters、2026 年版の 93 / 91 count へ縮小・再編される。採否を読むときは、単に同名 proposal が続いたというより、repertoire の安定化過程として扱う。
