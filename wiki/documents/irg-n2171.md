# IRG N2171

## 要約

`IRG N2171` は、Chen Zhuang が UCS 内の stroke counting differences を整理し、将来の IRG work に向けて統一を提案した draft guideline である。文書は `IRG N954AR` と [IRG N1105](irg-n1105.md) の supplement として位置づけられ、IRG editors による discussion と expert feedback を求めていた。

Registry subject の通り、この PDF には bookmarked feedback も含まれる。Henry Chan は unified form / disunified shape / regional glyph の stroke count principles、`kRSUnicode` first value としての normative stroke count、vertical stroke starting indicator などを論じた。CHO Sungduk と Kim Kyongsok は U+65E2 既 / U+65E3 旣 の stroke count と radical の扱いに comments を出した。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| draft guideline | UCS に複数の stroke count が存在する component / characters を挙げ、Kangxi stroke count と recommended stroke count を比較する。 |
| expert feedback | Henry Chan、CHO Sungduk / Kim Kyongsok らの comments を含み、individual items と general principles の両方を扱う。 |
| principles | unified glyph では unified form の代表 SC、disunified shape では shape ごとの SC、regional glyph 差では最小 stroke count を使う案などが議論される。 |
| `kRSUnicode` | IRG が決めた stroke count を `kRSUnicode` の first value として扱うべきだという feedback が含まれる。 |

## 後続決定

[IRG N2221](irg-n2221.md) は、N2171 を `IRG N954AR` / [IRG N1105](irg-n1105.md) とともに consolidate し、future IRG Working Set で使う stroke count document とした。後続の [IRG N2862R2](irg-n2862r2.md) と [IRG N2951](irg-n2951.md) は、N2171 を含む旧 guideline 群を置き換える consolidated FS / SC guideline として読む。

## 論点

### actual count と agreed count

N2171 は、既存 UCS code chart や Kangxi Dictionary と apparent stroke count が一致しないケースを明示する。後続文書で重要になるのは、actual glyph count をただ追うのではなく、IRG review と indexing のための agreed / prescriptive count をどう決めるかである。

### feedback-driven guideline

この文書は draft guideline と feedback を同じ PDF 内で束ねるため、最終決定ではなく論点集として読む必要がある。N2221 以降の consolidated documents が、これらの feedback をどこまで採用したかを確認する入口になる。
