# IRG N1534

## 要約

`IRG N1534` は、UTC の Extension D proposal で使われた主要 source dictionaries の性質を IRG に説明する liaison contribution である。UTC submission の characters のうち、direct email communication に由来する 2 characters は後続の complete evidence submission で示すとし、残りは dictionaries 由来として bibliographic information、short name、abbreviation、dictionary の性質を記述する。

## 報告内容

文書は、Cheung-Bauer、Meyer-Wempe、Lau、Cowles、Mathews、Fenn、ABC2、Cihai、SBGY、XHC などの sources を列挙し、それぞれが Cantonese / Mandarin / historical Chinese dictionaries としてどのような evidence value を持つかを説明する。

また、これらの dictionaries の contents は Unihan database mapping tables のために検査済みであり、source index の意味は UAX \#38 で説明されるとする。Extension D proposal は、これらの dictionaries に載るがまだ encoded されていない characters を扱う。

## 後続決定

この wiki では、Extension D proposal 全体の meeting / event はまだページ化していない。[IRG N1535](irg-n1535.md) が、`IRG N1534` の source codes に対応する sample evidence を提示する。

## 論点

`IRG N1534` は、UAX \#45 の source field と IRG submission evidence の違いを読むための重要文書である。U-source database が source tag を持つだけでは十分ではなく、IRG に対しては third-party が追える bibliographic evidence と、source index の意味を説明する必要がある。
