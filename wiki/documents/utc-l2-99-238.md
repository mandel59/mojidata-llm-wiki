# L2/99-238

## 要約

`L2/99-238` は、JCS Committee / JSA が final draft JIS X 0213 に含まれる予定の非漢字文字を Unicode / ISO/IEC 10646 に追加するため、複数の proposal を束ねて提出した文書である。対象は linguistic educational characters、HIRAGANA、dentist symbols、enclosed numbers、KATAKANA、publishing signs に分かれる。

Kana topic から見ると、重要なのは `Addition of seven HIRAGANA characters` と `Addition of twenty five KATAKANA character` の部分である。前者は HIRAGANA LETTER SMALL KA / SMALL KE と bidakuon 用 Hiragana を提案し、後者は Ainu 表記用の extended Katakana と bidakuon 用 Katakana を提案する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 背景 | JIS X 0213 の upward compatibility を理由に、同規格へ入る予定の非漢字文字を UCS に追加する必要があるとする。 |
| Hiragana | HIRAGANA LETTER SMALL KA / SMALL KE と、BIDAKUON NGA / NGI / NGU / NGE / NGO 系の Hiragana を提案。 |
| Katakana | Ainu 表記用の extended Katakana と、BIDAKUON NGA / NGI / NGU / NGE / NGO 系の Katakana を提案。 |
| Kana 以外 | IPA 由来の phonetic symbols、dentist symbols、enclosed numbers、publishing signs も同じ束の中で扱う。 |

## 後続決定

[WG2 N2092](wg2-n2092.md) は、Japan NB が WG2 に提出した 48 characters proposal であり、`L2/99-238` の kana / punctuation / publishing signs 系の内容に接続する。

## 論点

### JIS X 0213 compatibility と kana repertoire

この文書では、追加 kana の根拠が古典資料の精密転写ではなく、JIS X 0213 と既存日本語製品との compatibility に置かれている。後年の hentaigana、small kana、alternate Katakana proposal と比べると、JIS 規格化と国内実装を入口にした早期の kana repertoire 拡張として位置づけられる。

### Ainu Katakana と bidakuon

Ainu 用 extended Katakana は、Ainu に独自の文字体系がないため Katakana が出版物で使われるという説明を根拠にする。Bidakuon 系の Hiragana / Katakana は、NHK などの発音・アクセント資料を利用者コミュニティとして挙げ、通常の五十音では表せない精密発音の記述を目的とする。
