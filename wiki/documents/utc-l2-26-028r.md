# L2/26-028R

## 要約

`L2/26-028R` は Soninke を書く N'ti script の preliminary Unicode encoding proposal である。文書は N'ti を 1966 年に Adama Konte が Mali で作った script と説明し、Mali、Republic of the Congo、Ivory Coast などでの教育・日常使用を evidence とする。

提案は left-to-right unicameral alphabet として、27 letters、8 non-spacing diacritics、script-specific punctuation、digits を示す。letters は word 内で baseline で join / fuse し、word boundaries 以外では line break しないという model が、後続の SEW review と `Joining_Type` 議論の入口になった。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| language / community | Soninke speakers を主対象とし、Mali、Mauritania、Senegal、Gambia などの Soninke community と、Mali / Congo / Ivory Coast での教育・使用を根拠にする。 |
| script name | Community name は N'ti / N’ti。Unicode script name には apostrophe を含められないため `Nti` を提案する。 |
| model | Left-to-right alphabet。word 内の letters は下部で常に join / fuse し、Arabic のような positional shapes は持たないと説明する。 |
| repertoire | 27 letters、8 combining diacritics、N'TI PUNCTUATION XUPU、digits を placeholder range `U+1X000..U+1X02D` に置く。 |
| evidence | 2018 年からの community font、Keyman keyboard、books、online media、official request letters を示す。 |

## 後続決定

[L2/26-087](utc-l2-26-087.md) はこの preliminary proposal を supersede し、joining behavior の説明強化と GIILONDE letter extender を追加した revised proposal である。[L2/26-100](utc-l2-26-100.md) は N'ti を SEW の追加検討対象に含め、[L2/26-137](utc-l2-26-137.md) は N'ti などの left-to-right joining scripts を背景に `Joining_Type` semantics を扱う。

## 論点

### Preliminary repertoire と revised proposal

`L2/26-028R` は placeholder code points を使う preliminary proposal であり、後続の `L2/26-087` で proposed range は U+1CB80..U+1CBAF へ移り、GIILONDE letter extender が追加された。文字数・repertoire の確認は、preliminary form だけでなく revised proposal の code chart と合わせて読む必要がある。

### LTR joining model

N'ti は writing direction が left-to-right である一方、letters が word 内で join する。この組み合わせは Arabic-derived joining data の `L` / `R` の意味、font shaping、line breaking、Core Specification wording に影響するため、proposal chain の中心的な review point になっている。
