# L2/26-040R2

## 要約

`L2/26-040R2` は、Oromo 用の Shaaldaa script を UCS に符号化する revised proposal である。2024 年の [L2/24-109](utc-l2-24-109.md) `Sheek Bakrii Saphaloo` proposal を改訂し、community feedback に基づいて script name を Shaaldaa に変更した。

提案は、Sheikh Bakri Sapalo が作った abugida を Ethiopic 型の syllabic repertoire として扱い、base consonant、vowel series、geminated forms、punctuation、digits を個別文字として符号化する。文書は、Oromo community 内での教育・出版・digital usage を根拠に、現代的な利用可能性も示す。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script identity | Sheikh Bakri Sapalo に由来する Oromo 用 script を `Shaaldaa` として符号化する。 |
| encoding model | Indic-style virama / combining mark ではなく、Ethiopic と同様の precomposed syllabic model を採る。 |
| repertoire | consonant-vowel series、pure consonants、geminated set、punctuation、digits を含む 824 characters を提案する。 |
| revision points | script name、digit category、glyph variation evidence、追加 figure、numeric values などを更新する。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は、Shaaldaa の character names / annotations を SEW 側の処理対象として扱う。[WG2 N5361R](wg2-n5361r.md) / [WG2 N5362](wg2-n5362.md) は Shaaldaa を Unicode 18.0 後の provisionally assigned future repertoire / Amendment 1 starting repertoire に含め、[WG2 N5354](wg2-n5354.md) は WG2 \#73 の script additions group として扱った。

## 論点

### Ethiopic 型の syllabary としての符号化

`L2/26-040R2` は、Shaaldaa の glyph system が vowel series と gemination を持つため、combining mark で分解するより、既存の Ethiopic model に近い syllabic repertoire として符号化するのが安定すると主張する。この判断は文字数を大きくする一方、既存利用者が期待する字形単位を保つ。

### revision lineage

この文書は [L2/24-109](utc-l2-24-109.md) の改訂版である。名称変更だけでなく、community feedback、glyph variation、数字の扱い、資料図版の追加を含むため、古い title だけで検索すると後続の Shaaldaa proposal を見落とす。
