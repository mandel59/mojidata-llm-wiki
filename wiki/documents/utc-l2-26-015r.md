# L2/26-015R

## 要約

`L2/26-015R` は Bambara を標準 N'ko より音韻・文法的に精密に書くため、N'ko Phonetic Extensions の一部を N'ko Extended block に追加する revised proposal である。文書は対象を Bambara additions に絞り、community name として N'ko Kakura を用いる。

提案 repertoire は 8 base letters、17 diacritics、2 modifiers と説明される。roadmap code point として U+1E960..U+1E97F 周辺を使い、N'ko の right-to-left joining behavior、tone / accent marking、collation and string matching considerations を整理する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| scope | Malian languages 全体ではなく Bambara に必要な N'ko Kakura subset。 |
| repertoire | 8 base letters、17 diacritics、2 modifiers。 |
| usage evidence | education materials、government documents、YouTube lessons、mobile / desktop input use、classes and student count。 |
| stability | 2019 以降の font / input deployment と repertoire refinement を整理する。 |
| data impact | ArabicShaping、UnicodeData、collation / string matching considerations を提示する。 |

## 後続決定

[L2/26-100](utc-l2-26-100.md) は N'ko Bambara を、joining behavior、positional forms、productive mechanism、stability などの encoding model をさらに検討する proposal 群の一つとして扱う。2026-07-09 時点の local catalog では、この revised proposal への採択決定は未確認である。

## 論点

### standard N'ko と N'ko Kakura の境界

この proposal は既存 N'ko script の置き換えではなく、Bambara に必要な phonetic / grammatical distinctions を追加する extension として構成される。標準 N'ko で既に使われる letters / marks の再利用と、N'ko Kakura 固有の new characters をどう分けるかが review point になる。

### tone marks と grammatical precision

N'ko Kakura では tone と grammatical accent / inversion marks を分けて扱う。文字追加は単なる phonetic spelling ではなく、compound words、verb / noun marking、string matching に関わるため、properties と collation が proposal の重要部分になる。
