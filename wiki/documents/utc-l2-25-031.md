# L2/25-031

## 要約

`L2/25-031` は、default identifiers で common use とみなす CJK Unified Ideographs subset を提案する文書である。文書本文の日付は 2025-01-09、UTC registry 上の日付は 2025-01-14 である。

文書は Root Zone Label Generation Rules (RZ-LGR-5) の Han Character Repertoire から、Han script だが CJK Unified Ideographs ではない U+3005 と U+3006 を除いた subset を示す。この subset 外の現在および将来の CJK Unified Ideographs には `Identifier_Type=Uncommon_Use` を与えることを提案している。

## 決定・Action Item

| 区分 | 内容 |
| --- | --- |
| 提案 | RZ-LGR-5 由来の common-use CJK unified ideographs を default identifiers の common-use subset とする。 |
| Identifier_Type | subset 外の present and future CJK Unified Ideographs を `Uncommon_Use` とする。 |
| 規模 | repertoire summary は 19,842 code points を示す。 |
| 関係資料 | Chinese / Japanese / Korean Root Zone LGR、MSR-5、Second-level Reference LGR などに基づく。 |

## 論点

### confusable pairs ではなく identifier policy

`L2/25-031` は confusable pair の提案ではない。CJK Security Confusables との関係は、UTS #39 / identifier security の側で Han characters の default identifier availability を制限する点にある。CJK の security work は、見た目が紛らわしい pair の data だけでなく、identifier に入る character set の境界設定も含む。

### RZ-LGR 由来の common-use subset

subset は East Asian ccTLD registries の common registry practice を組み合わせた Root Zone LGR 由来であり、Unicode 11.0 / MSR-5 cutoff に基づく。Unicode 11.0 以後に encoded された unified ideographs は common use として提案されていない。
