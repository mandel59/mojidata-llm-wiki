# WG2 N5361R

## 要約

`WG2 N5361R` は、Unicode 18.0 後の future repertoire として provisionally assigned code points をまとめる data document である。ISO/IEC 10646 7th edition Amendment 1 の starting repertoire の基礎資料として [WG2 N5362](wg2-n5362.md) と [WG2 N5354](wg2-n5354.md) に参照される。

文書は tentative であり、encoding / publication の acceptance そのものではない。WG2 \#73 は、この data を starting point としつつ、recommendations で個別 script / character groups を first amendment 候補として扱った。

## データ内容

| 区分 | 内容 |
| --- | --- |
| 対象 | Unicode 18.0 後の future versions と ISO/IEC 10646 7th edition amendments の候補 repertoire。 |
| 形式 | Code positions、representative glyphs、character names、block ranges を provisionally assigned data として提示する。 |
| script / block examples | Sirmauri、Leke、Proto-Cuneiform、Mwangwego、Shaaldaa、CJK Unified Ideographs Components-A/B、Leibnizian Ambiguous Signs などを含む。 |
| 注意 | Feedback を受けて調整される tentative data であり、最終 encoding ではない。 |

## 後続決定

[WG2 N5354](wg2-n5354.md) の M73.05 は、`WG2 N5362` と `WG2 N5361R` をもとに 7th edition Amendment 1 project subdivision を始めることを勧告した。M73.06 以降は、first amendment に入れる候補を group ごとに扱う。

## 論点

### provisionally assigned と encoded の違い

`WG2 N5361R` は code point を仮置きする data document であり、UTC / WG2 / SC2 の最終採択ではない。topic や proposal page では、この文書への掲載を「future candidate」として扱い、後続 recommendation や ballot result と区別する必要がある。
