# L2/26-101

## 要約

`L2/26-101` は Charts Working Group が UTC Meeting \#187 に提出した report である。Unicode 18.0 charts、per-block chart publication、PDF owner password policy、NamesList ownership transfer、auxiliary charts の整理を扱う。

この report は、Unicode 18.0 の変更点を「どの PDF / chart artifact で公開するか」を決める資料である。code charts の versioned archive、unversioned URL の扱い、NamesList.txt の生成責任が実装者・レビュアーの参照先に影響する。

## 報告内容

| 区分 | 要点 |
| --- | --- |
| PDF permissions | Unicode 18.0 beta から code charts の owner / permissions password を外し、core spec archival PDF と方針を合わせる。 |
| Versioned block charts | per-block charts を `Public/18.0.0/charts/` に versioned archive として公開し、従来の overwritten URL と区別する。 |
| Terminology | “versioned, per block delta charts”、“versioned, per block charts”、“versioned consolidated charts” へ用語を整理する。 |
| Retrospective archive | Unicode 17.0 charts を `Public/17.0.0/charts/` に遡及 archive し、pre-17 も可能なら続ける。 |
| Auxiliary charts | case mapping / collation charts は modernization を続け、script / normalization auxiliary charts は削除対象にする。 |
| NamesList ownership | NamesList を Charts WG 管理に移し、private charts repo と NamesList.txt output の flow を整理する。 |

## 後続決定

Editorial WG report [L2/26-097](utc-l2-26-097.md) と合わせて、UTC \#187 の publication process 変更を構成する。UTC \#187 minutes [L2/26-093](utc-l2-26-093.md) は Unicode 18.0 beta review authorization の一部として chart / publication action items を扱う。

## 論点

### versioned charts は review artifact になる

従来の block chart URL は最新版へ上書きされるため、beta / release の差分追跡には不向きである。Charts WG report は versioned per-block charts を public archive に置くことで、reviewer と実装者が特定 version の charts を再現できるようにする。

### NamesList は chart pipeline の一部

NamesList の ownership transfer は単なる担当変更ではなく、NamesList.txt、nomenclator、NLAnnotations.txt、chart generation の依存関係を整理する変更である。今後の annotation / cross-reference 修正は、Editorial WG と Charts WG の両方の資料を確認する必要がある。
