# WG2 N5221

## 要約

`WG2 N5221` は、Japan NB が ISO/IEC 10646 の J-column に MOJI-JOHO-KIBAN IDEOGRAPHS-2018 由来の JMJ references を追加するよう求めた horizontal extension proposal である。

対象は既存 CJK Unified Ideographs への source reference addition であり、新しい code point の追加ではない。WG2 registry subject は 36,422 horizontal extensions とし、関連 chart PDF が別リンクで提供されている。

本文は、ISO/IEC 10646 ed.5 で新規追加された一部の characters は `JMJ` ID が J-source に載るが、その他は J-column に出ていないため、空欄の J-source column へ references を追加すると説明する。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| 対象 data | MOJI-JOHO-KIBAN IDEOGRAPHS-2018 に関係する JMJ references。 |
| 作業種別 | 既存 code points の J-source column への horizontal extension。 |
| chart | `UCS` column が ISO/IEC 10646 ed.5 code chart の現在の表示、`MOJI JOHO KIBAN` column が追加対象の glyph image と reference names を示す。 |
| 付属表 | 44 件の「UCS変更予定一覧」が添付され、MJ IDs の変更先 UCS / 変更前 UCS を示す。 |

## 後続決定

[IRG N2721](irg-n2721.md) / `WG2 N5284` は、`WG2 N5221` を WG2 に直接提出された大規模 horizontal extension として取り上げ、同種の案件は CJK review に適した IRG へ提出する方がよいと述べた。

[IRG N2722](irg-n2722.md) は、Unicode 16.0 で追加された JMJ horizontal extensions を IDS / glyph shape から確認し、複数の code point 対応・glyph issue を指摘した。

[PRI \#546](pri-546.md) は、Unicode 16.0 / ISO/IEC 10646:2020/Amd.2:2025 への取り込み過程で Moji Jōhō Kiban database の code point 対応が更新されたことを背景に、Moji_Joho IVD collection へ 8 IVS を追加登録する public review として後続する。

## 論点

### source reference addition と mapping maintenance

`WG2 N5221` は、source reference を空欄の J-column に追加する horizontal extension である。一方、添付の 44 件の UCS 変更予定一覧は、MOJI-JOHO-KIBAN 側の mapping maintenance と UCS code point 対応が同時に問題になることを示す。

### review body

この proposal は WG2 document として提出されたが、後続の [IRG N2721](irg-n2721.md) は、36,000 件超の CJK horizontal extension は IRG experts による review に回すべきだったと位置づける。したがって、この文書は [JMJ Horizontal Extension Review Path](../topics/jmj-horizontal-extension-review-path.md) の起点として読む。
