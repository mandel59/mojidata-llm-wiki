# L2/24-150

## 要約

`L2/24-150` は、`L2/23-112` の missing Kana-Ligatures proposal を改訂し、合略仮名（合字的な kana ligatures）の reduced set として 3 characters を提案する文書である。対象は古い日本語資料、特に Meiji era 以前の文献・活字資料で用いられた multi-syllabic kana を plain text で保存するためのものと説明されている。

文書は、単なる縦書き連綿と encoded character の区別を重視する。手書き資料では通常の筆順とは異なる横方向配置や大きな省略を条件にし、活字資料では単一文字として印刷されていることを基準にしている。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | Kana Extended-A への 3 kana ligatures 追加。 |
| 背景 | `L2/23-112` の広い proposal に対する CJK & Unihan WG feedback を受け、reduced set として再提出。 |
| 論点 | hiragana ligatures には vertical-only implication の懸念があり、katakana ligatures は比較的受け入れやすい、という前段の議論に接続する。 |

## 論点

### 合略仮名の判定基準

文書は、合略仮名を「複数音節を表す単一 kana character」とし、単なる sequence や連綿を除外する条件を提示する。CJK letters やその abbreviation は、歴史的に合略仮名と呼ばれる場合があっても、この proposal では含めないと整理している。

### 後続の KATAKANA DIGRAPH YORI

`L2/24-279` は、この reduced set から外れた KATAKANA DIGRAPH YORI を再提案する。したがって `L2/24-150` は、近年の kana ligature 議論の分岐点として扱う。
