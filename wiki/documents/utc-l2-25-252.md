# L2/25-252

## 要約

`L2/25-252` は、thumb point / sideways thumb gesture を emoji として追加する proposal である。Daniel Nathaniel Olobayo は、index finger pointing より控えめな指示、mockery、hitchhiking、ASL などに接続する手の gesture として説明する。

UTC \#185 ESR report [L2/25-230R](utc-l2-25-230r.md) は、この proposal を LEFTWARDS THUMB SIGN / RIGHTWARDS THUMB SIGN の 2 characters として Unicode 18.0 short list に入れ、両方を skin tone modifiers 対応にした。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed characters | LEFTWARDS THUMB SIGN、RIGHTWARDS THUMB SIGN |
| category | hand-fingers-closed |
| keywords | thumb、point、mock、wrist、ASL、hitchhike など。 |
| selection rationale | pointing、mocking / jest、hitchhiking、sign-language contexts など複数用途を主張する。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FAF9 LEFTWARDS THUMB SIGN と U+1FAFA RIGHTWARDS THUMB SIGN を Unicode 18.0 additions として受け入れた。[L2/26-008R](utc-l2-26-008r.md) は final emoji candidate list でもこの 2 characters を維持する。

## 論点

### gesture emoji と skin tone sequences

thumb signs は human hand gesture であるため、単体 characters に加えて skin tone modifier sequences が増える。Emoji repertoire の実装影響は code point 2 件ではなく、RGI sequence count まで含めて見る必要がある。
