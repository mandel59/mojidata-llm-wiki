# L2/24-012

## 要約

`L2/24-012` は、2024-01-05 の CJK & Unihan Group meeting に基づく UTC Meeting \#178 向け recommendations である。UTC \#177 以降に受け取った public feedback と文書を整理し、Unihan database、UAX \#38、UAX \#45、GB 18030-2022 Amendment 1、CJK Symbols font、CJK small form characters などを扱う。

文書は recommendation bundle であり、本文とは別に `Unihan-additions` / `Unihan-changes` / `USource-changes` / `kMandarin` data files を attachment として持つ。2026 年の [L2/26-084](utc-l2-26-084.md) で扱われる 2024 年 `kMandarin` action chain の入口の一つでもある。

## 決定・Action Item

| 区分 | 決定・Action Item |
| --- | --- |
| GB 18030-2022 Amendment 1 | Unicode 15.1 の Ideographic Description Characters が Draft 3 では implementation level に属さないことを、developer 向け note として記録するよう勧告した。 |
| UAX \#38 / Unihan | `kDefinition` delimiter / syntax の変更、`kFrequency` removal、`kCantonese` / `kVietnamese` / `kIRG_JSource` / `kIRG_USource` などの updates、`RSIndex.txt` release artifact 化を扱った。 |
| `kMandarin` | U+9244 の proposed value を CLDR-TC に確認させる action item を置き、2022 年 action items 由来の `kMandarin` additions / changes を CLDR-TC review 後の形で受け入れるよう勧告した。 |
| UAX \#45 / U-source | 8 U-source records を `FutureWS` から `Variant` に変更し、U+3150D へ UTC-00420 を `kIRG_USource` として追加する horizontal extension を受け入れ、UTC-03398 を `FutureWS` record として追加した。 |
| Other CJK topics | Suzhou numerals font update、East Asian auto-spacing proposal の plenary discussion、U+16FF2 / U+16FF3 small form CJK characters の provisional assignment を扱った。 |

## 後続決定

[L2/24-067](utc-l2-24-067.md) は、`L2/24-012` でいったん提案された `kDefinition` delimiter / syntax change を feedback に基づいて revert するよう勧告した。`kDefinition` は free-form property として扱い、機械的な semicolon split を前提にしない方向へ戻った。

[L2/26-084](utc-l2-26-084.md) は、`L2/24-012`、[L2/24-067](utc-l2-24-067.md)、[L2/24-165](utc-l2-24-165.md) の 2024 年 action items から生じた 169 characters の `kMandarin` additions / changes に対する CLDR TC / Apple feedback である。後続の [L2/26-099](utc-l2-26-099.md) Section 22 は、その feedback を Unicode 18.0 target の action に接続した。

## 論点

### recommendation bundle と data attachment

`L2/24-012` は個別提案ではなく、public feedback、PRI feedback、data attachments、UTC action items を束ねる文書である。本文の consensus だけでなく、添付された Unihan / U-source / `kMandarin` data files が実データ更新の根拠になる。

### `kMandarin` と CLDR-TC review

`kMandarin` values は CJK & Unihan Group が直接確定するのではなく、CLDR-TC review を経由して UTC action へ戻る。`L2/24-012` は過去 action item 由来の review 済み data を受け入れる一方、新しい value change は CLDR-TC に確認を依頼している。

### U-source horizontal extension の小規模 precedent

U+3150D への UTC-00420 追加は、既符号化文字へ U-source reference を追加する horizontal extension である。これは後の大規模 U-source horizontal extension と同じ型の作業として、[CJK Horizontal Extensions](../topics/cjk-horizontal-extensions.md) から読むと位置づけが見えやすい。
