# PRI 545

## 要約

`PRI #545` は、UTC が Unicode 18.0.0 向けの UAX \#11 East Asian Width Revision 45 proposed update を public review に出した PRI である。PRI page の status は Closed、closing date は 2026-07-07。

review 対象は UAX \#11 Revision 45 の draft で、draft page は Unicode 18.0.0、date 2026-04-26 を示す。PRI page は、Section 6.1 の unassigned code point ranges を調整したと説明する。

変更は新しい layout algorithm の導入ではなく、Unicode 18.0 の repertoire / plane allocation に合わせて、unassigned code points の default East_Asian_Width classification を確認するための review gate である。

## 提案内容

### review window

PRI page は、UAX \#11 の proposed update が general public review and comment に出たことを示す。[PRI \#548](pri-548.md) の Unicode 18.0.0 Beta review と同じ 2026-07-07 に close しているため、final text と `EastAsianWidth.txt` は Unicode 18.0 finalization の一部として確認する。

### UAX \#11 Revision 45

UAX \#11 Revision 45 の modifications section は、Unicode 18.0.0 向け proposed update として、Section 6.1 Unassigned and Private-Use Characters の ranges を調整したことを記録する。

Section 6.1 は、CJK ideographs intended ranges の unassigned code points を Wide とし、private-use characters を context-dependent な Ambiguous、それ以外の unassigned code points を Neutral とする default rule を説明する。Revision 45 はこの default rule の対象範囲を Unicode 18.0 の codespace 状態に合わせる変更として読む。

## 後続決定

`PRI #545` は closed だが、PRI page 自体は feedback の採否を記録しない。UTC \#188 で Unicode 18.0 content finalize が予定されているため、実際の反映は UTC \#188 minutes / action items、最終 UAX \#11、最終 `EastAsianWidth.txt` で確認する。

## 論点

### East_Asian_Width と layout

UAX \#11 は East Asian legacy character sets、line layout、vertical text、font binding で使われる `East_Asian_Width` normative property を説明する。`PRI #545` の変更は Section 6.1 に限られるが、unassigned code point default は rendering fallback、terminal width assumptions、future CJK ideograph allocation の扱いに関わる。

### East Asian Spacing との切り分け

[East Asian Spacing](../topics/east-asian-spacing.md) は UTR \#59 の visible spacing algorithm を扱う別 topic である。UAX \#11 の East_Asian_Width は width classification の property であり、UTR \#59 の `East_Asian_Spacing` property と同一ではない。ただし両方とも East Asian typography と mixed-script layout の implementation input になる。
