# L2/26-143

## 要約

`L2/26-143` は Gottfried Wilhelm Leibniz の works に現れる ambiguous operator signs を符号化する proposal である。59 ambiguous signs を対象にし、そのうち 5 characters は variation sequences として提案する。

文書は ± / ∓ に関連する graphical / historical family として ambiguous signs を説明し、Leibniz manuscripts の日付推定や logical concepts の記法研究に必要だとする。Dedicated block named Leibnizian Ambiguous Signs を提案する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| repertoire | 59 ambiguous operator signs。5 は variation sequences として提案。 |
| block model | Dedicated block `Leibnizian Ambiguous Signs` を提案。 |
| naming | UTC feedback に基づき、semantic content を示す descriptive names へ改訂した。 |
| research use | Leibniz manuscript editing、date attribution、logical notation の研究に使う。 |

## 後続決定

2026-07-08 時点では、UTC \#188 agenda / minutes は未掲載であり、この proposal への UTC 決定は未確認である。WG2 \#73 recommendation M73.06 との接続は [L2/26-149](utc-l2-26-149.md) が扱う。

## 論点

### variation sequence と descriptive naming

Leibnizian ambiguous signs は、single encoded character、variation sequence、descriptive names、research catalog identifiers の関係を整理する必要がある。文字名が sign の semantics をどこまで表すかも review point になる。
