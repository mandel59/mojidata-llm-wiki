# L2/26-142

## 要約

`L2/26-142` は historical sources に現れる 6 letterlike symbols を追加する proposal である。Latin / Greek letters に stroke、slash、loop などを加えた abbreviation symbols を、facsimile editions や historical mathematical text のために符号化する。

文書は Philiumm project / Leibniz-related work の一部で、過去 revision で Greek capital Omicron candidate を落とし、character names と code point allocation を整理したと説明する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| repertoire | 6 letterlike symbols。 |
| evidence | historical mathematical / scholarly sources に現れる abbreviation-like symbols。 |
| review history | 2026-02 online discussion 後、code point allocation と variation sequence reference を修正した。 |

## 論点

### letters, symbols, abbreviations の境界

Letterlike symbols は、ordinary letters の styling ではなく、historical abbreviation / symbol として meaning を持つかが焦点になる。Leibniz / historic math 系 proposal では、glyph similarity より editor が保持したい semantic distinction が重視される。
