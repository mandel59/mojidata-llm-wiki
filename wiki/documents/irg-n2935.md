# IRG N2935

## 要約

`IRG N2935` は、2026-10-12/16 に Tokyo, Japan で予定される IRG Meeting \#67 の schedule and agenda である。last update は 2026-07-05。金曜日 2026-10-16 は no meeting とされ、実質的な議事は 2026-10-12/15 に置かれている。

この agenda は、IRG Meeting \#66 recommendations / action items の follow-up と、WS2024 Version 5.0 review を中心に構成される。さらに、WG2 \#73 / SC2 acceptance、UTC Meeting \#187、Unicode Version 18.0 Beta review に接続した current status が agenda 内に記録されている。

## 論点

### \#66 recommendation の current status

`IRG N2935` は、`kIRG_SGSource` と CJKV Components について、IRG Convenor が WG2 \#73 へ提出し、WG2 recommendation と SC2 acceptance を経て Unicode Version 19.0 target になった current status を記録している。これは `IRG N2909` の決定そのものではなく、\#67 agenda 時点の後続 status である。

### Unicode 18.0 / ISO/IEC 10646 7th edition との接続

Agenda Section 4 では、T-source U+2976E、K-source U+2DB7C、一部 G-source reference changes などが UTC CJK & Unihan Working Group、UTC Meeting \#187、Unicode Version 18.0 Beta review に反映されたと整理している。`WG2 N5354` の Recommendation M73.02 は、これらの一部を ISO/IEC 10646 CD の CJK additions and changes に接続する。

### 未決事項

Meeting \#67 では、G-source / UK-source / V-source / SAT-source glyph issues、China horizontal extension revised document、WS2024 Version 5.0 comments、radical assignment rules などが継続確認対象である。V-source では `IRG N2928` の U+268A1 / U+268A2 glyph enquiry が "Redux" として再掲され、follow-up の `IRG N2953` が入力文書になっている。agenda は決定事項ではないため、会合後は `IRG N2946` recommendations and action items で状態を更新する。
