# IRG N2916

## 要約

`IRG N2916` は、K-source KC-05216 / U+2DB7C の representative glyph correction proposal である。Eiso Chan が current glyph の右側が U+204D7（⿱兔⿰兔兔）ではなく、点が一つ欠けた ⿱兔⿰免兔 のように見えると指摘し、KR が original evidence と IDS / stroke count を確認した。

KR は、新しい evidence と Ext F submission の IDS が三つの 兔 を示すこと、stroke count も radical を除いて 24 画であることから、glyph に点を追加して三つの 兔 を含む形へ直すと結論した。`kRSUnicode` は 77.23 から 77.24 に変更し、`kTotalStrokes` 28 は correction 後に整合するため変更しない。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | U+2DB7C / KC-05216。 |
| 問題 | current glyph は二つの 兔 と一つの 免 に見えるが、IDS と stroke count は三つの 兔 を示す。 |
| evidence | 韓國古典飜譯院 韓國文集叢刊 231輯 旅菴遺稿の evidence、KR Ext F submission の IDS、`db.history.go.kr/unicode/` の stroke counts。 |
| 提案 | glyph に点を追加し、右側 component を ⿱兔⿰兔兔 に合わせる。 |
| Unihan update | `kRSUnicode` を 77.23 から 77.24 に変更する。`kTotalStrokes` 28 は維持する。 |

## 後続決定

[IRG N2911](irg-n2911.md) は、IRG Meeting \#66 editorial group が `IRG N2916` に基づく K-source glyph change と `kRSUnicode` update を受け入れたと記録している。[IRG N2909](irg-n2909.md) も M66.13 で同じ決定を formal recommendation として整理した。

[L2/26-099](utc-l2-26-099.md) は、この K-source change を UTC \#187 の CJK & Unihan action item に取り込み、Unicode Version 18.0 target の code chart / Unihan data update として扱った。WG2 \#73 の [WG2 N5354](wg2-n5354.md) Recommendation M73.02 も、selected CJK additions and changes の一部としてこの流れに接続する。

## 論点

### glyph correction と `kTotalStrokes`

この proposal は、representative glyph を直すことで既存の `kTotalStrokes` 値と整合する例である。glyph correction と同時に Unihan property が変わるが、すべての property が変わるわけではないため、`kRSUnicode` と `kTotalStrokes` を分けて追う必要がある。

### evidence と IDS の照合

KR は画像 evidence だけでなく、Ext F submission 時の IDS と stroke count を照合している。代表 glyph の見た目だけでなく、submission metadata が correction の根拠になる点が重要である。
