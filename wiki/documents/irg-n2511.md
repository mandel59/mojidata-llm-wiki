# IRG N2511

## 要約

`IRG N2511` は、U+30759 / UTC-01250 の glyph が複数の historical documents で attested されない可能性を指摘し、glyph update または正しい form の別符号化を提案した individual contribution である。対象 character は Unicode 13.0 時点の code chart glyph と、明史関連資料の evidence との関係が論点になっている。

## フィードバック内容

文書は、《南明史》、明史稿、明史藁、明史、明實錄、名山藏、1939 年の article、1974 年の中華書局版明史などを比較し、code chart glyph `U+30759` が typographic / editorial misprint に由来する可能性を論じる。

結論として、現 evidence に基づけば `U+30759` の glyph は別 form の misprint と考えられるため、ad hoc unification を追加して reference glyph を変更する、または正しい form を別に encode する、という 2 options を提示する。

## 後続決定

この wiki では、`IRG N2511` への IRG / UTC decision はまだページ化していない。U-source representative glyph が evidence review で争点になる例として、[UAX \#45 / U-Source Ideographs](../topics/uax45-u-source-ideographs.md) と [IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md) から追跡する。

## 論点

この文書は、U-source source reference と representative glyph が別々に review されうることを示す。UAX \#45 / code chart の glyph が既に encoded character に対応していても、historical evidence の再検討によって glyph correction、ad hoc unification、disunification / separate encoding が論点化する。
