# WG2 N5363R

## 要約

`WG2 N5363R` は、ISO/IEC 10646 が Unicode Consortium の生成する external code charts を normative reference するための proposal である。WG2 \#72 M72.10 の follow-up であり、10646 text 本体から code charts を分離しつつ、参照される chart set を standard の integral part として扱う方法を示す。

WG2 \#73 の M73.03 は、この proposal に基づき、UCS code charts を core text から linked content に移すことを 7th edition DIS に反映するよう勧告した。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| approach | Unicode Consortium が生成する edition-specific charts を ISO/IEC 10646 の normative reference とする。 |
| purpose | 10646 text を通常の text processor で編集できる規模にし、chart production を Unicode 側の pipeline と共有する。 |
| chart location | 10646 edition に対応する external chart resource を設け、standard text はその resource を参照する。 |
| text change | 7th edition core text は、code charts の新しい location と normative status を説明する形に変更される。 |

## 後続決定

[WG2 N5354](wg2-n5354.md) の M73.03 は、`WG2 N5363R` の内容を 7th edition DIS に入れるよう Project Editor に勧告した。M73.04 は、この変更を含む DIS text の balloting へ接続する。

## 論点

### 外部 resource だが normative

`WG2 N5363R` の要点は、chart file が standard text PDF の外に置かれても、参照された edition-specific chart set は standard の一部として normative に扱われる点である。これは、単なる web link ではなく、repertoire と code chart の版管理を標準本文と結びつける publication model である。
