# WG2 N5368

## 要約

`WG2 N5368` は、Dai Xaau Script の encoding proposal に対する supplementary materials である。`WG2 N5310R` の補足として、China の Jin Ping Dai Xaau script (White Dai) と Vietnam の Dai Viet script (Black Dai) は form、sound、meaning に大きな差があるとし、Dai Xaau を独立に符号化する必要を説明する。

文書は 76 Dai Xaau characters と、現代生活上の需要から作られた 10 numerical abbreviations を加える方向を示す。WG2 \#72 M72.15 は Dai Xaau proposal の進捗を note し、authors に継続作業を促した。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 位置づけ | `WG2 N5310R` の supplement。WG2 \#72 後の追加説明として提出された。 |
| script separation | White Dai と Dai Viet は form / sound / meaning が異なるため、Dai Xaau を独立 encoding として扱う必要があるとする。 |
| repertoire | 76 characters と 10 numerical abbreviations の追加を説明する。 |
| evidence | punctuation、teaching / use examples、現代的な数値 abbreviation の需要を補足する。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) の M72.15 は、Dai Xaau proposal の progress を note し、proposal authors に継続作業を促した。`WG2 N5368` は WG2 \#73 agenda に載る補足資料だが、[WG2 N5354](wg2-n5354.md) の recommendations では、Dai Xaau の採択 recommendation は確認できない。

## 論点

### related scripts の分離

Dai Xaau と Dai Viet の差を script distinction として扱うか、同一 script の style / orthographic variant として扱うかが中心論点である。`WG2 N5368` は、字形・音価・意味の差と追加資料により独立 encoding の根拠を補強する。
