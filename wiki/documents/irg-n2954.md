# IRG N2954

## 要約

`IRG N2954` は、U+2CCA3 の G-source representative glyph が `GZFY-00932` の original evidence と一致していない可能性を指摘する proposal である。文書は current glyph を維持するか、original evidence に合わせて直すかを、source reference の扱いと合わせて検討している。

中心問題は、current code chart glyph、`Hanyu Fangyan Dacidian` の evidence、`GZFY-00932` source reference が一貫していない点である。提案は、glyph を evidence に合わせて normalize する案と、current glyph を維持するために source reference を `GU` 系へ変更する案を示す。

## 提案内容

| 選択肢 | 内容 |
| --- | --- |
| glyph update | current representative glyph を `GZFY-00932` の original evidence に合わせる。これに伴い `kRSUnicode`、total strokes、traditional form などの関連 data も見直す。 |
| source reference update | current glyph を維持する場合、G-source reference を `GZFY-00932` から `GU-2CCA3` のような reference に変更し、original `GZFY-00932` glyph は将来の別 submission に回す。 |
| China response | どちらの処理にするか、China に source data / glyph data の更新を求める。 |

## 後続決定

[IRG N2955](irg-n2955.md) が `IRG N2954` への feedback として、source reference を維持し、representative glyph を original evidence に合わせる方向を支持している。Meeting #67 agenda の [IRG N2935](irg-n2935.md) は、representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。

## 論点

### glyph と reference のどちらを動かすか

この proposal は、source evidence mismatch を見つけたときに、representative glyph を直すのか、source reference を別 evidence に付け替えるのかという判断を明示的に分けている。G-source の一貫性を保つには、current glyph だけでなく source evidence、Unihan properties、chart data の連鎖をまとめて確認する必要がある。
