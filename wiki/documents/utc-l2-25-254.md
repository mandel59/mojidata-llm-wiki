# L2/25-254

## 要約

`L2/25-254` は MONARCH BUTTERFLY emoji の proposal である。Amy Whitaker は、monarch butterfly を migration、conservation、Día de los Muertos、rebirth / transformation などの文化的・生態的 symbol として提案する。

この proposal の重要点は、既存 U+1F98B BUTTERFLY が vendor によって monarch-like / morpho-like / generic に分かれていることを、interoperability issue として扱う点である。後続の [L2/26-098](utc-l2-26-098.md) は、この論点を ESR report の中心例として扱う。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | MONARCH BUTTERFLY |
| keywords | monarch、butterfly、migration、pollinator、conservation、milkweed など。 |
| selection rationale | 既存 butterfly emoji の vendor design divergence、monarch の文化的・生態的 recognition、既存 emoji では指定できない意味を主張する。 |
| interoperability point | U+1F98B BUTTERFLY を generic / morpho 側へ寄せる余地を作り、monarch-specific meaning を別 character にする。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FACC MONARCH BUTTERFLY を Unicode 18.0 additions として受け入れた。[L2/26-008R](utc-l2-26-008r.md) の final candidate list でも維持され、[L2/26-098](utc-l2-26-098.md) は feedback に対する ESR の考え方を整理する。

## 論点

### species-specific emoji と open-endedness

Monarch butterfly は species-specific proposal であり、overly specific / open-ended という feedback を受けやすい。Proposal と ESR report は、単に species を増やすのではなく、既存 butterfly emoji の実装差を解く interoperability case として扱う。
