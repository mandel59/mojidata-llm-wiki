# WG2 N5348

## 要約

`WG2 N5348R` は `N5343R` の Small Seal encoding proposal に対する 2025-10-07 の feedback と、2025-10-11 の TCA / China experts による回答を整理した文書である。主な対象は、Small Seal code chart / database に付随する modern CJK value の選び方である。

Kushim Jiang は modern characters に 1,700 件超の差異を見つけたとし、複数の修正候補を提示した。CheonHyeong Sim も U+3E9FC の modern CJK value について、`U+25416` ではなく `U+540E` が正しいと指摘した。

## 決定・Action Item

N5348 は決定文書ではなく feedback summary である。TCA / China experts は、Small Seal と modern CJK characters の形・発音・意味が一致する場合はその modern CJK を優先し、一致しない場合は standard characters (`zhèngzì`) を代替として使う方針を回答した。

## 論点

### `liding` と `zhèngzì`

N5348 は `liding` を、古代文字形を clerical / regular script に正規化する practice と説明し、`zhèngzì` / standard characters との対比を置いている。ただし文末の Michel note は、この文脈での `normalized form` / `standard characters` の意味がまだ十分明確でないとした。

### `zhèngzhuàn` / `chóngwén`

後続の `WG2 N5355` は、`zhèngzì` という訳語よりも `zhèngzhuàn` と `chóngwén` の関係で理解する方が正確だと補足した。modern equivalent がない `chóngwén` では、対応する `zhèngzhuàn` の modern equivalent を indexing form として使う、という整理である。

### IRG UNC との関係

N5348 では、Small Seal に対応する modern characters が未符号化の場合、IRG に UNC として申請する必要があるかが話題になっている。TCA 側の回答では、この点は後で議論できるとしつつ、historical documents に実在しない `liding` forms を符号化申請することには慎重な立場が示された。
