# L2/15-239

## 要約

`L2/15-239` / `WG2 N4674` は、Japan NB による Japanese HENTAIGANA の初期 proposal である。proposal summary form は 299 文字を対象とし、Kana Supplement と新規 block への追加を想定している。

文書は、Hentaigana を現代 Hiragana の variants としつつ、1947 年まで人名に使われた行政上の需要と、日本語文字史・日本史研究の学術需要を根拠にする。repertoire は活版印刷と digital fonts から収集した文字を基礎にし、同じ Chinese character 由来でも機能的区別を再現するために複数 variants を含めると説明する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 対象 | Japanese HENTAIGANA 299 文字。 |
| 位置づけ | 既存 Hiragana とは別に、historical / administrative use のための Hentaigana repertoire を追加する提案。 |
| 根拠 | 人名用途、歴史資料研究、活版印刷・digital font からの repertoire 収集。 |

## 後続決定

この文書の repertoire は、後続の [L2/15-343](utc-l2-15-343.md) と [L2/16-188](utc-l2-16-188.md) で revised proposal として整理される。特に [L2/16-188](utc-l2-16-188.md) では、HENTAIGANA LETTER E-1 withdrawal と U+1B001 の name alias 追加に接続する。

## 論点

### 初期 repertoire と revised repertoire

`L2/15-239` は、Hentaigana 標準化を 299 文字規模の proposal として提示した入口である。後続文書では、phonetic value と mother ideograph に基づく命名・統合の整理が進み、単純な glyph collection ではなく character identity の基準が問題になる。

### 行政需要と学術需要

文書は、Hentaigana の現在の使用頻度が低いことを認めつつ、1947 年までの人名使用と歴史資料の plain text 化を理由に、標準符号化の必要性を説明する。これは後続の kana proposal で繰り返し出る「歴史資料をどこまで encoded character として扱うか」という基準の初期例である。
