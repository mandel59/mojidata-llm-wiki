# L2/26-085

## 要約

`L2/26-085` は、5 abbreviated ideographs を UAX \#45 に追加する提案である。対象は、風、姦、磊、鑷、發 の abbreviated variants として説明される。

文書は、略字は漢字の発展過程で一般的であり、辞書や研究資料に含まれているため符号化候補として妥当だと述べる。添付として proposed glyphs 用 font `Abbr.ttf` と metadata を含む。

## 提案内容

| No. | Variant | 要点 |
| --- | --- | --- |
| 1 | 風 | complex part を 丿 で簡略化する form。 |
| 2 | 姦 | 品-like repeated components を `〓` 系 component で簡略化する form。 |
| 3 | 磊 | 同じく repeated components を簡略化する form。 |
| 4 | 鑷 | 耳 repeated component を簡略化する form。 |
| 5 | 發 | upper component を 业 に置き換える類型に属する form。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) は、`L2/26-085` を含む 2026 年 UAX \#45 additions を `FutureWS` records として受け入れるよう勧告した。採択・data update 側の summary は [UTC \#187 UAX \#45 FutureWS additions](../events/utc-187-uax45-futurews-additions.md) に集約する。

## 論点

### repeated component の簡略化

No. 2-4 は、品-like character / component の下側の repeated components を `〓` 系 component で置き換える shorthand を扱う。文書は 渋 を類例として挙げ、同種の variant form が用例に現れることを示す。

### 日本資料との接続

Evidence には `新大字典`、`漢字摘要`、`普通漢語字類大全`、国定書方教授資料、国立公文書館・国立国会図書館デジタルコレクションの資料などが含まれる。日本の略字・近代資料に現れる ideographs として [Japanese Historical Ideographs](../topics/japanese-historical-ideographs.md) とも接続する。
