# WG2 N5312

## 要約

`WG2 N5312` は、TCA and China が `N5294R2` / `N5294R3` の Small Seal code list を review し、Michel Suignard の code list modification に使える feedback としてまとめた文書である。

文書は 5 種の response を示す。中心は 17 variant sets の処理、別符号化可能な 19 glyphs、multi-column mapping errors、24 THX glyph corrections である。後続の [WG2 N5313R](wg2-n5313r.md) は、この文書の Part 1 を置き換える revised feedback として提出された。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| 17 variant sets | glyph を主たる判断材料とし、meaning を補助的に使う立場から、variant sets の削除・維持・移動候補を示す。 |
| 19 separate-code candidates | QJZ / DYC glyph が THX と大きく異なる例について、別符号化可能な glyphs を Appendix 1 に列挙する。 |
| mapping errors | 上 / 下、菿、𦼬 / 𧂚、糶 / 𦤙、䕼 などで source column の対応移動を提案する。 |
| THX glyph corrections | expert review により、interpretation と合わない 24 THX glyphs の correction を Appendix 2 に示す。 |

## 後続決定

`WG2 N5312` 自体は feedback 文書であり、会議決定ではない。後続の [WG2 N5313R](wg2-n5313r.md) は `N5306` review に対する revised feedback として、N5312 の variant-set response を置き換え、[WG2 N5317R3](wg2-n5317r3.md) が disposition 文書として WG2 \#72 の future edition candidate に接続した。

## 論点

### glyph similarity と Shuowen system

`WG2 N5312` は、見た目の重複だけで glyph を削除するのではなく、Guwen（古文）、Zhouwen（籀文）、Zhengzhuan（正篆）の関係や source tradition を考慮する。後続文書では、この判断を [WG2 N5307R](wg2-n5307r.md) の unification / disunification principles で明文化していく。
