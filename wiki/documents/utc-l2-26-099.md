# L2/26-099

## 要約

`L2/26-099` は、2026-04-03 に開催された CJK & Unihan Working Group meeting の結果を UTC Meeting \#187 へ勧告する文書である。対象は UTC \#186 以降に受け取った public feedback、PRI feedback、UTC documents、IRG documents で、Unicode Version 18.0 を target にした Unihan database、UAX \#38、UAX \#45、UAX \#60、UTS \#39、StandardizedVariants.txt、code chart glyph の更新を束ねる。

文書は 42 sections からなり、前半は public feedback と PRI \#520 / \#534 / \#536 / \#541、後半は UAX \#38 / Unihan database documents、UAX \#45 / U-source proposals、CJK Strokes variation sequences、CJK confusables、IRG Meeting \#66 recommendations を扱う。添付 data files として、Unihan property changes、UAX \#45 additions、StandardizedVariants.txt additions などの実更新データも含む。

## 論点

### recommendations は実データ更新の束ね役

`L2/26-099` は個別提案ではなく、複数の registry documents と public review feedback を UTC action に変換する hub である。特に `Unihan_IRGSources-changes-20260403.txt`、`Unihan-additions-20260403.txt`、`Unihan-changes-20260403.txt`、`USource-additions-20260403.txt`、`StandardizedVariants-20260403.txt` などの添付 data files が、本文中の consensus / action item と対応する。

### UAX \#45 additions は次期 IRG working set への前段

UAX \#45 / U-source documents では、9 documents から 30 records が `FutureWS` として accepted になった。文書末尾の Section 40 は、Unicode Version 18.0 で UAX \#45 additions が 200 ideographs を超えるため、これらを UTC の次期 IRG working set submission に含める方向を示す。一方で、CJK & Unihan Working Group は Unicode Version 19.0 までは UAX \#45 additions を新たに検討しないと整理している。

### WG2 \#73 recommendations への反映

WG2 Meeting \#73 の Recommendation M73.02 は、`L2/26-099` の Section 01、04、07、11、13、19、20、32 を参照し、ISO/IEC 10646 CD の CJK additions and changes に取り込むことを勧告した。これにより、UTC 側の CJK & Unihan recommendations は、Unicode Version 18.0 だけでなく ISO/IEC 10646 7th edition CD / DIS progression にも接続する。

### IRG \#66 recommendations との接続

`L2/26-099` は `IRG N2909` の M66.10、M66.12、M66.13 などを UTC data update に変換する役割も持つ。IRG 側の source reference / representative glyph decision が、UTC 側では Unihan property update、code chart glyph update、WG2 recommendation へ連鎖する。
