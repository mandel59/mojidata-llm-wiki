# IRG N2888

## 要約

`IRG N2888` は、`IRG Working Set 2024 Version 4.0` に対して ORT に提出された review comments を集成した文書である。candidate serial number、glyph / source、comment type、commenter と所属、description を並べ、前版から unresolved の comment も識別できる。

内容は unification、UCV、glyph・IDS・radical・stroke count などの candidate data を検討する入力であり、IRG Meeting \#66 の結論集ではない。各 comment と submitter response を検討した editorial conclusions は [IRG N2910](irg-n2910.md)、Version 5.0 作成の勧告と期限は [IRG N2909](irg-n2909.md) に帰属する。

## データ内容

| 区分 | 内容 |
| --- | --- |
| candidate identity | WS2024 serial number、representative glyph、source reference、radical / stroke data、IDS を示し、comment の対象を特定する。 |
| review metadata | comment type、commenter、member body / role、description を記録し、前版から継続する項目には unresolved の由来を付す。 |
| review issues | unification / oppose unification、UCV、glyph、IDS、radical、total / residual stroke count など、working set data の異なる検査観点をまとめる。 |
| cross-references | 既存 unified ideograph、別 candidate、ORT comment number、関連 IRG document への参照を含み、会合で比較すべき根拠を集約する。 |

## 後続決定

[IRG N2910](irg-n2910.md) は `IRG N2888` の comments と submitter responses を candidate ごとに検討し、WS2024 Version 5.0 に反映する editorial conclusions を整理した。[IRG N2909](irg-n2909.md) の M66.07 は、その conclusions に基づく data fix、font update、Version 5.0 release と次回 review schedule を正式に勧告した。成果物は `IRG N2932` である。

## 論点

### 集成の用途

この文書は、複数の reviewers が ORT に入力した指摘を会合前・会合中に追跡するための snapshot である。candidate identity と comment provenance を保ったまま、unification、evidence、glyph、attribute corrections を横断的に検討する入口になる。

### 集成の限界

comment が掲載されていることは、その提案が採用されたことを意味しない。相反する comments、前版から unresolved の項目、追加 UCV の提案、withdrawal / unification の候補が同じ集成に含まれるため、最終 disposition は `IRG N2888` 単独から確定できない。また PDF は作成時点の snapshot であり、その後の ORT discussion や submitter response を常に反映する live record ではない。

### Editorial conclusions との境界

`IRG N2888` は review input を保存する data document、`IRG N2910` は CJK Editorial Group が Meeting \#66 で検討した結果をまとめる report である。wiki では個々の主張を `IRG N2888` の決定として書かず、採否・修正・withdrawal などの editorial conclusion は `IRG N2910`、IRG 全体の recommendation と action item は `IRG N2909` に分離する。
