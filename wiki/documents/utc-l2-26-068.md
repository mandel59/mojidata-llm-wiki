# L2/26-068

## 要約

`L2/26-068` は、Vietnam が U+2B8A0 の V-source representative glyph と Unihan property values の変更を提案した文書である。

WS2024 review 中に、candidate `VN-F2079` が既に符号化済みの U+2B8A0 の original form であると判明したため、U+2B8A0 の shape / properties を `VN-F2079` に合わせ、`VN-F2079` を WS2024 から withdraw することを求める。

提案は、glyph 変更だけでなく `kRSUnicode` を `9.7` から `9.9`、`kTotalStrokes` を `8` から `11` に変更する Unihan data update を伴う。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| 対象 | U+2B8A0、V-source `V4-407A`。 |
| 現行形 | U+2B8A0 は Nôm Tày character として、読み `rầư` / `riầu`、意味 "who?" とされていた。 |
| 問題 | 現行 representative glyph の phonetic component は Unicode に見つからず、attested reading も確認できないとする。 |
| 新 evidence | `VN-F2079` は person radical と phonetic component `𱽗` からなる形で、Hoàng Triều Ân 編 `Từ Điển Chữ Nôm Tày` などに基づく。 |
| 追加 evidence | 同形は Old Zhuang script でも `lawz` と読まれ、Tày / Zhuang の近縁性から intended form を支持するとされる。 |
| 変更案 | representative glyph を `⿰亻𱽗` にし、`kRSUnicode` を `9.9`、`kTotalStrokes` を `11` に変更する。`kIRG_VSource` は `V4-407A` のまま維持する。 |
| WS2024 への影響 | `VN-F2079` を WS2024 から withdraw する。 |

## 後続決定

[L2/26-099](utc-l2-26-099.md) Section 20 は、この proposal に基づき、U+2B8A0 の informative `kRSUnicode` / `kTotalStrokes` property value changes と V-source representative glyph change を Unicode Version 18.0 target として受け入れるよう勧告した。Ken Lunde への Unihan database update と、updated TrueType font に基づく code chart glyph update が action item になった。

## 論点

### single-source character の correction

文書は U+2B8A0 が single-source Vietnamese character であることを、既存 text への影響が限定的とみなす根拠にしている。V-source issue では、source owner の evidence と既存 encoded character の意図が一致するかが、glyph change の判断材料になる。

### glyph と property の同時変更

この case は representative glyph の変更が radical / stroke data に直接波及する例である。`kRSUnicode` と `kTotalStrokes` の変更は glyph correction の副作用ではなく、変更後の字形を Unihan data と一致させるための proposal の一部である。
