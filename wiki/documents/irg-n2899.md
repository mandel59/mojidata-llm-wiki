# IRG N2899

## 要約

`IRG N2899` は、2026-03-16/20 に virtual meeting として開催された IRG Meeting \#66 の schedule and agenda である。member body activity reports、Meeting \#65 action items の follow-up、`IRG Working Set 2024 Version 4.0` review、disunification、horizontal extension、source data・representative glyph・indexing rule 関連文書を議題別に配置する。

最終更新版は各 agenda item の完了、再検討、未討議を示すが、これは議事録や formal recommendation ではない。会合の決定と action items は [IRG N2909](irg-n2909.md)、CJK Editorial Group の review conclusions は [IRG N2910](irg-n2910.md)、miscellaneous issues の editorial report は [IRG N2911](irg-n2911.md) で確認する。

## 報告内容

| 区分 | Agenda 上の範囲 |
| --- | --- |
| Schedule | 3月16日に administrative topics、activity reports、前回 action items、disunification、horizontal extension、other documents、17–18日に WS2024 review と other documents、19日に final document preparation と recommendations approval を配置した。3月20日は no meeting とされた。 |
| WS2024 | Version 4.0 の consolidated comments と responses、個別 candidates、withdrawal、source data、glyph evidenceを CJK Editorial Group でreviewする中心議題である。 |
| Proposal review | urgently needed ideographs、disunification、China の horizontal extension、G / T / K / UK / V source references と representative glyph changesを入力文書単位で列挙する。 |
| Infrastructure and policy | UCV / NUCV、FS / SC guidelines、radical assignment、IRG P&P、document register migration、`kIRG_SGSource`、CJKV Componentsなどを扱う。 |
| Status marks | agenda の最終更新時点で、完了、再検討が必要、未討議を区別する。再検討表示は持ち越しの手掛かりだが、採否そのものは recommendations と editorial reportsで確認する必要がある。 |

## 後続決定

[IRG N2909](irg-n2909.md) は、この agenda に沿って審議された事項の recommendations and action items を記録する。WS2024 Version 5.0 の schedule、source reference・glyph changes、`kIRG_SGSource`、CJKV Components、indexing rulesなどの正式な会合成果は `IRG N2899` ではなく同文書に帰属する。

[IRG N2910](irg-n2910.md) は WS2024 Version 4.0 review と UCV / NUCV examples の editorial conclusionsをまとめ、[IRG N2911](irg-n2911.md) はその他の individual issues に関する editorial review を補う。したがって `IRG N2899` は「何を、どの入力文書に基づいて審議する予定だったか」を辿る入口として用い、結論の根拠には各成果文書を用いる。

## 論点

### Agenda status と会合決定の境界

完了マークは agenda item の処理状況を示すもので、個々のproposalが承認されたことを意味しない。再検討マークも、同じ会合中または次回会合で再訪すべき項目を示すにとどまる。wikiでは schedule・入力文書・議題構造をこのページに、formal recommendation と editorial conclusionを成果文書側に分離する。

### 議題から成果物への接続

WS2024 review は `IRG N2910`、miscellaneous issues は `IRG N2911`、plenaryで承認された recommendations and action items は `IRG N2909` に収束する。この三者を併読することで、agenda上の予定、editorial groupでの検討、IRGとしての最終的な扱いを区別できる。
