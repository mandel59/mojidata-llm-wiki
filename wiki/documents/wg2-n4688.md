# WG2 N4688

## 要約

`WG2 N4688` は、TCA and China が Small Seal Script を UCS に独立 script として符号化する 2015 年 proposal である。`L2/15-281` は同じ文書を UTC register に載せた別番号なので、このページでは alias として扱う。

文書は `Shuowen Jiezi` Tenghuaxie version の 14 volumes から 11,108 Small Seal Script characters を提案する。zip package には main proposal DOCX と、14 volumes に対応する appendix PDFs が含まれ、appendix は original glyph、TTF font、corresponding modern characters、Shuowen radicals を並べる。

この文書は [WG2 N4634](wg2-n4634.md) の 10,516 characters / 799 sample glyphs から、complete proposal へ進んだ段階にあたる。後続の [WG2 N5105](wg2-n5105.md) は、N4688 の 11,108 characters から 15 repeated characters を削除した 11,093 characters の revised appendix として説明している。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| repertoire | Proposal summary form と冒頭では 11,108 Small Seal Script characters。 |
| source text | `Shuowen Jiezi` Tenghuaxie version、14 volumes。 |
| appendix | 14 parts。各 volume の original glyph、TTF glyph、SW radical、corresponding modern character を含む。 |
| ordering | Shuowen radical sequence と、Tenghuaxie version の同一 radical partition 内の arrangement order。 |
| character selection | Tenghuaxie version の Zhengzhuan、Variants / Chongwen、newly added characters、Duan Zhu / Chen Changzhi で構造差がある characters。 |
| glyph principle | Tenghuaxie original forms を保ち、追加 characters / component variants も意図的に再構成しない。 |

Section 4 は attached excel file が 11,116 entries を含むと記す一方、proposal summary form と後続文書の [WG2 N5105](wg2-n5105.md) は N4688 の repertoire を 11,108 characters として扱っている。この wiki では、文書の proposal number と後続参照に合わせて 11,108 characters を canonical な proposal count とし、11,116 entries は attached data の記述として区別する。

## 後続決定

この文書自体は proposal であり、会議決定ではない。2019 年の [WG2 N5105](wg2-n5105.md) は N4688 から repeated characters 15 件を削除した revised appendix を出し、[WG2 N5108](wg2-n5108.md) は WG2 \#68 discussion で他 versions と taxonomy の追加検討が必要だと報告した。

## 論点

### 独立 script としての提案

N4688 は [WG2 N4634](wg2-n4634.md) と同じく、Small Seal を modern Hanzi の typeface ではなく、Oracle Bone script、Bronze inscriptions、Warring States scripts と同じ古代漢字の歴史的段階として位置づける。modern standard Hanzi と一対一対応できない characters や、構造が異なる counterparts があるため、UCS では CJK Unified Ideographs の glyph variation ではなく独立 script として扱うべきだとする。

### Radical と ordering

Shuowen Jiezi の 540 radicals は、Kangxi radicals の 214 radicals よりも form / meaning の理論的体系を保つものとして説明される。N4688 では characters を Shuowen radical sequence と Tenghuaxie order に従って並べるが、後続の [WG2 N5108](wg2-n5108.md) では taxonomy / ordering の有用性と multi-column future extension が追加論点になった。

### Modern Hanzi 対応値

N4688 は old Hanzi から modern Hanzi への transfer principles として、component position、line-to-stroke conversion、Li glyph form などを保つ descriptive glyph を第一対応とし、自然な script evolution による standard script form がある場合は secondary glyph として扱う。後続の UAX \#60 / `kSEAL_MCJK` では、対応値は検索・索引用の data property として再整理される。
