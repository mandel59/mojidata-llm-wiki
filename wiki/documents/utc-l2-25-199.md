# L2/25-199

## 要約

`L2/25-199` は、Unicode 17.0.0 向けの UAX \#45 Revision 31 proposed update である。editor は Ken Lunde。文書は U-source ideographs の current model として、identifier prefixes、classes of U-source ideographs、data files、submission process、text file fields、status values、U-source additions table を説明する。

## データ内容

Revision 31 は、U-source identifiers の prefixes として `UTC`、obsolete な `UCI`、UK submission の `UK` を説明する。U-source database は、UTC に提出された candidates、placeholder records、IRG source information を補うための records、UK 経由の WS2015 submission records などを含む。

Data files は `USourceGlyphs.pdf`、`USourceRSChart.pdf`、`USourceData.txt` の 3 つとして説明される。`USourceData.txt` は identifier、status、Unicode code point、radical-stroke index、KangXi index、IDS、source field、comments、total strokes、first residual stroke を持つ。

status values は、`Comp`、`ExtA` から `ExtJ`、`FutureWS`、`NoAction`、`Rejected`、`URO`、`Variant`、`WS-2024`、duplicate identifier reference などを含む。Revision 31 の modification note は、`WS-2021` records を `ExtJ` / `Variant` / `Rejected` へ整理し、WS2024 submission records を `FutureWS` から `WS-2024` へ変えたことを記録する。

## 後続決定

この proposed update は、UAX \#45 / U-source data の current model を読む入口である。2026 年の [L2/26-099](utc-l2-26-099.md) では、さらに 30 records が `FutureWS` として accepted され、Unicode Version 19.0 までは新規 UAX \#45 additions を検討しないという intake pause が示された。

## 論点

`L2/25-199` は、U-source database が単なる UTC candidate list から、encoded status、working set status、source prefix、comments、stroke data を持つ structured registry へ発展したことを示す。[L2/10-198](utc-l2-10-198.md) で強調された UTC source と IRG evidence の区別は、source field を符号化承認や evidence sufficiency と同一視しないための前提として残っている。特に `WS-2024` status は、UAX \#45 records が IRG Working Set submission pipeline と直接結びつくことを示している。
