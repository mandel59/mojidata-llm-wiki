# L2/26-149

## 要約

`L2/26-149` は Leibniz Unicode Characters Project からの Unicode 18.0 delta review feedback である。内容は、Leibnizian ambiguous variation sequence characters の descriptive names に関する options と、Unicode 18.0 delta charts の wording / glyph corrections である。

文書は [L2/26-143](utc-l2-26-143.md) / `N5329R4` と WG2 \#73 recommendation M73.06 に接続する。特に 5 VS characters の description で “horizontal strokes” と “crossbars” のどちらを使うかを提案する。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| Leibnizian VS names | B-13 / B-14 / B-15 / B-17 / B-18 の descriptive names について two options を提示する。 |
| Unicode 18.0 delta charts | Mathematical Operators / Supplemental Mathematical Operators などで VS descriptions の wording consistency を指摘する。 |
| relation | `N5329R4`、WG2 \#73 M73.06、Unicode 18.0 delta charts に接続する。 |

## 後続決定

2026-07-08 時点では、UTC \#188 agenda / minutes は未掲載であり、この feedback への UTC 決定は未確認である。

## 論点

### proposal と delta review の接続

`L2/26-149` は新規 proposal ではなく、既に進行している Leibnizian ambiguous signs と Unicode 18.0 delta chart review を接続する feedback である。variation sequence descriptions は、正式 data / charts に残るため、terminology の consistency が重要になる。
