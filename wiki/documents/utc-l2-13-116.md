# L2/13-116

## 要約

`L2/13-116` は、Eastern Pwo Karen が使う Leke script を SMP に追加する 2013 年の revised proposal である。UTC registry 上の source は Erich Fickle / Martin Hosken、同じ文書は WG2 側では `WG2 N4438` として扱われる。

文書は、Leke を Thailand / Myanmar 周辺の Eastern Pwo Karen community で使われる left-to-right script とし、consonants、vowels / tones、digits、punctuation を含む 55 characters を提案した。Preliminary proposal 後の community testing と追加調査に基づき、character names を user community の呼称に合わせて変更したと説明する。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| script | Leke。Eastern Pwo Karen を書く script で、Buddhists、Christians、Ariya religion members に使われると説明する。 |
| repertoire | 55 characters。25 consonants、vowels / tones、10 digits、LEKE END OF SECTION を含む。 |
| encoding order | syllable は consonant first、LEKE VOWEL RI YAH、LEKE VOWEL DWIA THWAI、その他 vowels、after-vowel / vowelized consonant、tone の順で encoded する案。 |
| typography | rotated forms、ornamental dots、handwriting の whitespace-oriented placement は semantic distinction ではないと説明する。 |
| 後続 | [L2/26-129](utc-l2-26-129.md) が、この 2013 年案を拡張した current complete proposal として提出された。 |

## 後続決定

この 2013 年案そのものに対する採択 decision は、この wiki の現行 digest 範囲では確認していない。2026 年の [L2/26-129](utc-l2-26-129.md) は、`L2/13-116` を先行 proposal として参照し、medials、final consonant signs、tone marks、Brahmi model などを整理し直した complete proposal として提出された。

## 論点

### syllable model の再整理

`L2/13-116` は、Leke には final consonants がないとしつつ、LEKE VOWEL LONG PUU / LONG DUU が consonant の初期音を kill して別の vowel に変えると説明する。2026 年の [L2/26-129](utc-l2-26-129.md) は、medial consonants、final consonant signs、tone marks をより明確に分け、Brahmi model に沿った encoding model を提示する。

### handwritten variation と encoding boundary

文書は、90 度回転した characters、ornamental dots、tone mark の handwritten placement variation は encoding distinction ではないと述べる。一方、dots が meaning を持つ例や digit zero との混同を避ける typography もあるため、glyph variation と character identity の境界が重要になる。
