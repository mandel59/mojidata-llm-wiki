# IRG N2931

## 要約

`IRG N2931` は、IWDS Editor による UCV / NUCV lists の data document である。PDF は 193 pages あり、前半から page 185 付近までが `UCS Ideograph Unifiable Component Variations Summary List (UCV)`、page 186 以降が `UCS Ideograph Non-Unifiable Component Variations Summary List (NUCV)` になっている。

UCV は、stroke / dot、接触、交差、折れ、突出、rooftop modification、extra / reduced stroke などの差を category ごとに整理し、unifiable component variation の例を示す。NUCV は、source code separations、disunified ideographs、compatibility ideographs、review system references などを含み、unification しない境界を示す。

## 論点

### UCV / NUCV は unification policy の参照表

`IRG N2931` は proposal prose ではなく reference data である。UCV は component variation を unifiable と見るための positive reference、NUCV は source separation / disunification を維持するための negative reference として働く。

### indexing rules との分離

UCV / NUCV は [IRG Indexing Rules](../topics/irg-indexing-rules.md) と同じ IRG review infrastructure に属するが、FS / SC や radical assignment よりも unification boundary に近い。したがって、この wiki では [UCV and NUCV Lists](../topics/ucv-nucv-lists.md) に分割して追う。
