# WG2 N5327

## 要約

`WG2 N5327` は、Suzuki Toshiya による [WG2 N5313R](wg2-n5313r.md) への individual contribution である。TCA and China が QJZ / DYC glyphs の多くを separate encoding とする提案に対し、pre-official Small Seal glyphs の微妙な style difference と、source selection / documentation の必要性を論じる。

文書は 5 comments を置く。C / K entries を残すべき例、U+3961C と U+3AB02 の識別、U+38B34、U+39BDC の disunification、same components but different structures の別符号化方針である。

## フィードバック内容

| Comment | 内容 |
| --- | --- |
| 1 | pre-official Small Seal と official Small Seal の subtle stylistic difference を理由に、C-00965 や複数 K glyphs を維持する案を示す。 |
| 2 | U+3961C / U+3AB02 は semantics が異なるが glyph 区別が難しいため、glyph replacement、C glyph removal、documentation の選択肢を示す。 |
| 3 | U+38B34 について、glyph similarity だけでなく non-cognate relationship と source treatment の一貫性を問う。 |
| 4 | U+39BDC の追加 stroke 差を別符号化する場合、将来 extension で cognate-but-slightly-different glyphs が増える懸念を示す。 |
| 5 | same components but different structures を Shuowen Seal では別符号化する余地を認めつつ、Old Hanzi encoding 一般へ拡張しすぎないよう Annex で範囲を記録すべきとする。 |

## 後続決定

[WG2 N5337](wg2-n5337.md) は `WG2 N5327` の 5 comments に対する TCA の response である。最終的な future edition candidate への整理は [WG2 N5317R3](wg2-n5317r3.md)、[WG2 N5341](wg2-n5341.md)、後続の [WG2 N5344R2](wg2-n5344r2.md) に集約される。

## 論点

### glyph 差をどこまで character identity にするか

`WG2 N5327` の重要点は、Small Seal では source tradition の差を重く見る一方で、あまりに微妙な style difference を別符号化に使うと extension の境界が膨らむという懸念である。文書は、必要なら source replacement や font modification を使い、標準本文や annex で方針を明文化することを求めている。
