# IRG N1531

## 要約

`IRG N1531` は、China が CJK_D に提出していた G-source candidates のうち3,215字をwithdrawするよう IRG \#31 に求めた proposal である。IRG が各candidateに求めるsource、meaning、pronunciation等の evidence を、対象すべてについて期限内に収集できないことを理由とする。

文書は、このwithdrawalが当初の提出時にcharactersを未確認だったことを意味しないと明記する。本文後の25ページには、withdraw対象のglyph、CJK_D index、G source identifierが列挙される。proposal自体はrepertoire決定文書ではないが、IRG \#31 minutesはChinaが3,215字のwithdrawalを決めたと記録している。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| 対象 | CJK_Dの G-source candidates 3,215字。 |
| 要求 | 対象をCJK_Dの作業集合からwithdrawする。 |
| 主な理由 | source、meaning、pronunciation等、IRGが求めるevidenceを全対象について期限内に揃えられない。 |
| 作業上の事情 | 元の提案者から後任者へ十分な資料が残されていない例があり、調査を継続中だった。また、人手不足のため期限内に収集できないevidenceもあった。 |
| 意図の限定 | 当初のCJK_D提出時に対象字が未確認だった、または文字として不適格だったという主張ではない。 |
| 添付一覧 | glyph、CJK_D index、`G_BK`、`G_CYY`、`G_CH`、`G_WZ`等のG source identifierを掲載する。 |

したがってwithdrawal理由は、3,215字すべてを不要・重複・非漢字と判定したことではなく、IRG reviewに必要なtraceable evidenceを所定の時間と体制で用意できなかったことである。

## 後続決定

`IRG N1531` のheaderは `Member's submission`、actionを `To be considered by IRG` としており、文書単独ではwithdrawalを採択する決定ではない。一方、IRG \#31 minutesである `IRG N1511` は、Chinaのactivity reportの説明として、3,215 Extension D charactersのevidenceを見つけられず、Chinaがこれらをwithdrawすることを決めたと記録する。

この記録からChina側のwithdrawal方針は確認できるが、このwikiで確認した範囲では、後述する143字を含む最終的な個別candidateの扱いを確定するresolutionまでは特定していない。最終repertoireへのretain / removalは、CJK Editorial Groupの後続working setとIRG / WG2のdecision recordで別途確認する必要がある。

## 論点

### `IRG N1528` の143字retain request

[IRG N1528](irg-n1528.md) は、`Sawndip Sawndenj`（`古壮字字典`）から辞書掲載箇所、Zhuang reading、IDS、Chinese gloss、scanを示せる143字を取り出し、Extension Dにretainするよう求めた後続のindividual contributionである。これは `IRG N1531` の一括withdrawal理由だったevidence不足に対して、限定されたsubsetへ追加evidenceを提示する関係にある。

ただし `IRG N1528` は対象母数を「3,125」と記すのに対し、`IRG N1531` のtitle、本文、添付見出し、および `IRG N1511` minutesはいずれも「3,215」とする。一次文書間に数値の不一致があるため、143字を算術的に差し引いた残数を確定値として扱わない。また `IRG N1528` のactionも `to be discussed` であり、evidenceの提示と143字の最終retain決定は区別する必要がある。

### evidence不足とcharacter validity

このproposalが示すのは、IRGのcandidate reviewでbibliographic・linguistic evidenceを追跡可能にする必要性である。withdrawalは作業集合の品質管理上の措置であり、列挙されたglyphが実在しないという一括判定ではない。後から追跡可能なevidenceが見つかった字については、`IRG N1528` のように個別にretainを求める余地が残されていた。
