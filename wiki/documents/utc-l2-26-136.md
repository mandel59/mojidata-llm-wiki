# L2/26-136

## 要約

`L2/26-136` は WHITE IDEOGRAPHIC COMMA を U+16FE5 として符号化する proposal である。対象は日本語の shiroten / shiro-goma ten / goma-shiro ten で、Meiji-period Japanese punctuation と modern scholarly / literary contexts に現れる inline punctuation mark と説明される。

提案は、U+3001 IDEOGRAPHIC COMMA、U+FE46 WHITE SESAME DOT、semicolon とは機能が異なると主張する。特に翻訳・校訂・辞書・デジタルアーカイブで、source text の punctuation distinction を保持する必要があるとする。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | U+16FE5 WHITE IDEOGRAPHIC COMMA。 |
| usage | 明治期日本語資料、翻訳、文学・辞書・政府資料、後年の scholarly editions。 |
| semantics | Japanese comma と full stop の中間、または European semicolon に対応する punctuation value と説明される。 |
| distinction | U+3001、U+FE46、semicolon では代用できない inline punctuation とする。 |

## 論点

### 日本語 punctuation history と plain text

この proposal は、現在一般的ではない historical punctuation を、検索・引用・校訂・digital text preservation のために plain text character として保持する必要があるとする。字体差ではなく punctuation function として扱えるかが review point になる。
