# WG2 N5339

## 要約

`WG2 N5339` は、ISO/IEC 10646 7th edition CD.2 consultation comments の disposition である。Canada、China、India、USA などからの comments を扱い、accepted / accepted in principle / not accepted の理由を整理する。

WG2 \#72 recommendations では、M72.01 がこの disposition を SC2 に受け入れるよう勧告し、M72.11 が disposition と M72.01-M72.10 の変更を含む CD.3 text の準備に接続した。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| removals | Sidetic characters U+1095A..U+1095C を CD.2 から外す。 |
| name / alias changes | Bamum correction aliases、Arabic / music / Latin Extended-G などの name changes を扱う。 |
| CJK additions | Extension C の KP-source disunification、Extension E の urgent G-source ideographs、T-source / G-source / Taiwan source additions などに接続する。 |
| ballot disposition | Comments ごとに accepted / accepted in principle / not accepted を示し、次の CD text へ送る変更を整理する。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) の M72.01 は `WG2 N5339` の disposition を受け入れる recommendation である。M72.11 は、この disposition を含む ISO/IEC 10646 7th edition CD.3 text を SC2 secretariat へ送るよう勧告した。

## 論点

### ballot disposition と proposal decision の分離

`WG2 N5339` は proposal そのものではなく、CD.2 に対する national body comments の disposition である。したがって、個別文字の採否や修正は、対応する recommendation や元 proposal と合わせて読む必要がある。

### CJK additions の経路

CJK additions / changes は、IRG source documents、UTC / Unihan recommendations、WG2 recommendations を経由して CD text に入る。`WG2 N5339` はその ballot disposition 側の記録であり、source owner response や Unihan data の詳細は関連 topic で追う。
