# IRG N2917

## 要約

`IRG N2917` は、Henry Chan による `IRG N2878R2` への feedback である。主な指摘は、variant forms を CJK Unified Ideographs Components として符号化すべきではないこと、HCP-00033 / HCP-00400 の扱い、radical data、HCP source reference の修正である。

## 論点

### variant と component の境界

N2917 は U+2FAEF について、`屯` の variant form であり、component ではなく proper character として normal CJK Unified Ideograph にすべきだと主張した。この点は `IRG N2878R3` が明示的に応答し、past meetings の consensus として、対象 characters は CJKUI behavior と整合し、component collection の目的で提案されていると整理した。

### HCP source と glyph distinctions

N2917 は、HCP-00033 と TCP-01014 の hook 有無、HCP-00400 の必要性、HCP source reference が Kai Style table ではなく Song Style (Print Style) table であるべき点を指摘した。`IRG N2878R3` は HCP-00033 と HCP-00400 の removal、secondary radical の追加、HCP source reference paragraph の修正に同意した。
