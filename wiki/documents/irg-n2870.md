# IRG N2870

## 要約

`IRG N2870` は、Japan NB が [IRG N2859](irg-n2859.md) に回答した短い feedback である。Japan NB は、published encoded characters には disunification を適用しないという原則を再確認し、ISO/IEC 10646:2020/Amd 2:2025 が既に published であるため、この原則を対象文字にも適用すべきだと述べる。

具体的には、U+2A50D、U+5CC0、U+72CA、U+7BF9、U+26B20、U+2F93B、U+21694 の current status を維持すべきだとする。

| 維持対象 | U+2A50D、U+5CC0、U+72CA、U+7BF9、U+26B20、U+2F93B、U+21694。 |

## 論点

### current status 維持の意味

`IRG N2870` は、J-source data の整合性改善よりも published encoded characters の stability を優先する立場を明示する。このため、[IRG N2859](irg-n2859.md) が示した property value / representative glyph changes は、Japan NB の承認を前提にそのまま進められるものではない。
