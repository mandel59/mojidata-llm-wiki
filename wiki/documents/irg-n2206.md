# IRG N2206

## 要約

`IRG N2206` は、CJK Unified Ideographs と extensions に含まれる easily confused ideographs について、rare ideographs に informative note と cross reference を付けることを提案した個人寄書である。提出先は IRG \#48 と UTC で、author は Henry Chan である。

この文書は UTS \#39 `confusables.txt` の data proposal ではない。対象は、非常によく似た形を持つが semantic meaning を区別すべき ideographs であり、Source Code Separated characters は ISO/IEC 10646 の Annex で扱われるため対象外とする。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案 | rare ideograph に informative note と easily confused character への cross reference を付ける。 |
| Scope | 形が非常に似ており、意味を区別すべき文字に限定する。 |
| 除外 | 同形に近くても meaning が同一の pair、Source Code Separated characters は対象外。 |
| 依頼 | IRG member に、この考え方への feedback と追加 group の提案を求める。 |

## 論点

### Security confusables 以前の CJK 内部注記案

`IRG N2206` は、security data ではなく code chart / standard text 上の annotation を提案する。CJK ideographs では、形が似ていても同一視すべき variant と、意味を区別すべき別字が混在する。この区別を利用者に示すため、rare character 側へ note と cross reference を付ける発想である。

### Source separation との切り分け

文書は Source Code Separated characters を対象外とする。これは、source separation により分離符号化された文字は ISO/IEC 10646 の Annex で説明されるため、easily confused ideographs の annotation proposal とは別問題だと整理している。
