# L2/25-253

## 要約

`L2/25-253` は PICKLE emoji の proposal である。Elizabeth Scopel は、pickle を food item としてだけでなく、tricky situation、pickleball、good luck、transfemininity、pop culture など複数の意味を持つ symbol として説明する。

提案は、既存 cucumber emoji とは brine / pickled food としての形・意味が異なり、food、idiom、sport、culture の複数領域で使えると主張する。UTC \#185 / \#186 の ESR reports は PICKLE を Unicode 18.0 final candidate list に残した。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| proposed character | PICKLE |
| category | Food-vegetable / prepared food 周辺。 |
| keywords | gherkin、brine、relish、predicament、pickleball、intoxicated など。 |
| selection rationale | cucumber とは異なる pickled item としての視覚性、idiom / pop culture / sport での多義性を主張する。 |

## 後続決定

[L2/25-230R](utc-l2-25-230r.md) は U+1FADD PICKLE を Unicode 18.0 additions として受け入れ、[L2/26-008R](utc-l2-26-008r.md) の final candidate list でも維持した。

## 論点

### 既存 emoji との representability

主な review point は cucumber emoji で代表できるかどうかである。Proposal は、pickle が food transformation、idiom、pickleball などを持つため、cucumber では表現できないと整理する。
