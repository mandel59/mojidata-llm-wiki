# WG2 N5300R9

## 要約

`WG2 N5300R9` は、2025-06-23/27 に Niigata, Japan で開催された WG2 Meeting \#72 の revised agenda である。ISO/IEC 10646 7th edition CD.2 disposition、IRG reports、CJK additions、code charts、Small Seal、J-source glyph issues、Khitan / Jurchen / Tai Xaau などを会議の処理対象として並べる。

この agenda は、WG2 \#72 を Small Seal や J-source だけでなく、7th edition CD progression と future repertoire selection の会合として読む入口になる。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| SC2 / 10646 progression | ISO/IEC 10646 6th edition DAM2 / FDAM2、7th edition CD / CD.2、CD.2 disposition draft / final を扱う。 |
| WG2 documents | CD.2 disposition [WG2 N5339](wg2-n5339.md)、code charts [WG2 N5315](wg2-n5315.md)、meeting recommendations [WG2 N5304](wg2-n5304.md) が adoption item に置かれた。 |
| IRG reports | IRG \#63 / \#64 recommendations、Japan active IRG participation request、CJKV Components、WS2024 などを扱う。 |
| CJK / Unihan | Additional repertoires、source data、JMJ-source glyph issues、Tangut data などが 10646 CD と IRG review に接続する。 |
| Script proposals | Small Seal、Khitan Small Script、Jurchen naming、Tai Xaau、Leibnizian proposal 群などが agenda に入る。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) は、この agenda の adoption / recommendation 結果である。CD.2 disposition は [WG2 N5339](wg2-n5339.md)、code charts は [WG2 N5315](wg2-n5315.md) から M72.10 へ、Small Seal は M72.13 へ接続する。

## 論点

### WG2 \#72 の主軸

WG2 \#72 は、ISO/IEC 10646 7th edition CD.2 を進める会合であると同時に、Unicode / ISO の chart publication model、Small Seal、CJK source data、script proposal pipeline を次の CD / future edition に振り分ける会合でもある。
