# WG2 N5301

## 要約

`WG2 N5301` は、2025-06-23/27 に Niigata, Japan で開催された WG2 Meeting \#72 の minutes and action items である。この wiki では、ISO/IEC 10646 7th edition CD.2 progression、Section 8.2 の Japan IRG active participation request、Section 11 / M72.07 の J-source glyph revert を主に参照する。

Section 8.2 は `WG2 N5284` を入力文書として、Japan が IRG に active member body として戻るべきかを議論した。minutes は、36,000 件超の horizontal extension `WG2 N5221` は WG2 より IRG に提出した方がよいという文脈を記録する。

Section 11 は `WG2 N5296` を入力文書として、JK-source / J-source chart glyphs が font change により HeiseiMincho / MS Mincho から IPAmjMincho へ変わって見える問題を扱い、Project Editor が J-source glyph changes を revert する disposition を記録する。

会議全体の decisions は [WG2 N5304](wg2-n5304.md) に集約される。CD.2 disposition は [WG2 N5339](wg2-n5339.md)、code charts は [WG2 N5315](wg2-n5315.md) を参照する。

## 後続決定

[WG2 N5304](wg2-n5304.md) は、M72.07 を formal recommendation として記録している。wiki では、J-source glyph revert の canonical event を [WG2 M72.07 J-source glyph revert recommendation](../events/wg2-m72-07-j-source-glyph-revert.md) に集約する。

Japan IRG active participation については、`WG2 N5301` 時点では復帰決定ではなく、human resources / workload が課題として残る。`IRG N2859` / `IRG N2870` は、その後も Japan NB の source owner response が必要になることを示す。

## 論点

### review path と correction issue の接続

Section 8.2 は、large JMJ horizontal extension の review body を IRG に寄せる必要を扱う。一方、Section 11 と M72.07 は、JMJ-source expansion 後の code chart font selection による glyph stability issue を扱う。両者は関連するが、前者は review workflow、後者は post-extension chart correction である。

### CD.2 progression と code charts

Minutes は CD.2 disposition と code charts を WG2 \#72 recommendations へ接続する会議記録である。詳細な disposition は [WG2 N5339](wg2-n5339.md)、publication model の検討は [WG2 N5315](wg2-n5315.md) と後続の [WG2 N5363R](wg2-n5363r.md) で追う。
