# L2/26-138

## 要約

`L2/26-138` は Indic scripts における consecutive anusvaras の occurrence と rendering support を説明する FYI 文書である。一般に repeated combining marks は不正・不自然な組み合わせと見なされがちだが、Sanskrit grammar では valid forms に two consecutive anusvaras が現れる場合があるとする。

文書は、Devanagari で `<U+0938, U+0902, U+0902>` が dotted circle になったり、単一 anusvara と同じに表示されたりする実装差を指摘する。一方、Kannada、Oriya、Saurashtra など spacing anusvara を持つ scripts では support が比較的自然であると説明する。

## 論点

### invalid sequence と valid grammatical form

Indic rendering では repeated combining marks が generic に invalid と扱われやすい。しかし、この文書は Sanskrit grammar に基づく valid textual forms として consecutive anusvaras を扱う。encoding というより、font / shaping / specification guidance の問題である。
