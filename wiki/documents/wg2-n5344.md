# WG2 N5344

## 要約

`WG2 N5344R2` は Small Seal Script を UCS に符号化する revised proposal である。対象は `Shuowen Jiezi` に基づく 11,328 characters で、主 source は `THX`、補助 source として `CCZ`、`QJZ`、`DYC` を含む。

提案は four-column code chart と database を持ち、source references、modern CJK equivalents、radical data、Unicode properties、collation、proposal summary form をまとめている。過去文書として `N4688`、`N5105`、`N5211`、`L2/22-279`、`N5306`、`N5317`、`N5346` を取り込んだ集約版である。

## 提案内容

この文書自体は proposal であり、決定文書ではない。後続の [WG2 meeting #73](../meetings/wg2/wg2-meeting-73.md) では、CD4 disposition の significant changes として Small Seal character names と 26 characters の property correction が記録された。

## 論点

### Script / block name

N5344 は block allocation として `3D000..3FC3F` を提案し、script property と character names には ISO 15924 との整合から `Seal` を使う案を示していた。一方、WG2 #73 では character names を `SMALL SEAL CHARACTER-XXXXX` に変更する方向になった。

### Four-column chart

提案は `THX` を primary source とし、`CCZ`、`QJZ`、`DYC` を並べる four-column layout を採用する。THX ordering を基本にし、source ごとの差異を unification / disunification criteria で整理する。

### Modern CJK equivalents

modern CJK equivalent は検索・索引用に重要だが、normative identity は source reference が担う。N5344 は modern CJK equivalent が今後更新され得る work in progress であることを明示している。
