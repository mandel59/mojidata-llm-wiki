# Feedback on L2/20-209 and some information for the shakuhachi musical notation

## 要約

`L2/20-233` は、Taiwanese Kana proposal `L2/20-209` に対する Eiso Chan の feedback である。
Tone marks の character names では、Hokkien ではなく ISO 639-3 の language name に合わせた Min Nan を使う案、または language name を入れない案を示す。
Combining dot below と combining overline は、既存の U+0323 / U+0305 へ統合する方向を確認しつつ、Taiwanese Kana 用の縦書き表示が一般的な East Asian text と異なる点を指摘する。
U+0305 は shakuhachi musical notation でも縦書きで使われ得るため、Taiwanese Kana 用の glyph behavior と区別する必要があるとする。

## フィードバック内容

| 区分 | 内容 |
| --- | --- |
| tone mark names | `Hokkien` より `Min Nan` を使う naming、または language name を含めない naming を提案。 |
| combining marks | `COMBINING DOT BELOW` と `COMBINING OVERLINE` は既存文字に統合し、Taiwanese Kana 用の注記を追加する方向を示す。 |
| vertical layout | Taiwanese Kana では U+0323 / U+0305 を横書きと同じ上下位置で扱う必要があるが、一般の縦書きでは左右に出る。 |
| shakuhachi notation | Shakuhachi notation では overline が右側に出る例があるため、同じ combining mark でも用途別の glyph behavior が必要になる。 |

## 後続決定

[L2/20-280](utc-l2-20-280.md) と [L2/20-209R](utc-l2-20-209r.md) は、この feedback を受けて character names を Min Nan に寄せ、U+0323 / U+0305 の扱いと shakuhachi notation を proposal 内で説明する。`L2/20-233` 自体は feedback 文書であり、Taiwanese Kana の repertoire proposal ではない。

## 論点

### language name と character names

Feedback の中心は、historical / regional label としての Hokkien より、ISO 639-3 の name に合わせた Min Nan を character names に使う方が安定するという点である。最終的な Kana Extended-B chart では `KATAKANA LETTER MINNAN TONE-*` / `KATAKANA LETTER MINNAN NASALIZED TONE-*` の形になっている。

### combining marks と縦書き

Taiwanese Kana は既存の U+0323 / U+0305 を使うが、縦書き時の mark position は通常の East Asian text と同じではない。文書は NameList.txt、UAX #50、OpenType feature など、文字追加ではなく rendering / data 側で扱うべき課題を示す。
