# L2/26-147

## 要約

`L2/26-147` / `IRG N2961` は、40 encoded CJK Unified Ideographs に新しい `kIRG_USource` property values と U-source representative glyphs を追加する U-source horizontal extension proposal である。対象は URO、Extension A、Extension B、Extensions F through J にまたがる。

文書は、code point、Unicode Version 18.0 code chart excerpt、U-source source reference、representative glyph を表で示す。Unicode Version 18.0 は ISO/IEC 10646 Seventh Edition と synchronized とされる。

## 提案内容

この proposal は新規符号化ではなく、既に encoded された 40 characters へ U-source data を追加する作業である。`UTC-00492`、`UTC-00156`、`UTC-00477` などの source references が、対応する code points に付けられる。

文書末尾の source-specific notes は、UCV number、ad hoc unification、analogous unification などを code point order で説明する。`kIRG_USource` property value additions は PDF attachment の `data.txt` に含まれるとされる。

## 後続決定

[IRG N2935](irg-n2935.md) は、IRG Meeting #67 agenda Section 8 に `IRG N2961` を U-source horizontal extension proposal として載せている。会合後の decision は `IRG N2946` recommendations / action items が available になった時点で確認する必要がある。

## 論点

`L2/26-147` は、[IRG N2369R](irg-n2369r.md) に続く U-source horizontal extension の current example である。UAX #45 additions が未符号化 ideographs を `FutureWS` records として蓄積するのに対し、この proposal は encoded ideographs へ `kIRG_USource` と representative glyph を追加する。

source-specific notes は、horizontal extension が単なる property addition ではなく、IRG の UCV / ad hoc unification rule と接続していることを示す。このため [IRG Indexing Rules](../topics/irg-indexing-rules.md) と一緒に読む必要がある。
