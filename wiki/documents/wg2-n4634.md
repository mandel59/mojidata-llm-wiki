# WG2 N4634

## 要約

`WG2 N4634` は、TCA and China が Small Seal Script を UCS に独立 script として符号化する初期 proposal である。`L2/14-242` は同じ文書を UTC register に載せた別番号なので、このページでは alias として扱う。

文書は `Shuowen Jiezi` の Tenghuaxie version を主 reference とし、10,516 Small Seal characters を提案する。提出時点では 799 characters の TrueType font sample を示し、残りの font は後続提供とされていた。

## 提案内容

| 項目 | 内容 |
| --- | --- |
| repertoire | `Shuowen Jiezi` Tenghuaxie version の 14 volumes に基づく 10,516 characters。 |
| sample data | Volume 1 由来の 799 characters について glyph samples と TrueType font を提示。 |
| ordering | Shuowen radical sequence と Tenghuaxie 内部順序を基準にする。 |
| character selection | 正篆、variants / Chongwen、Tenghuaxie の newly added characters、Duan Zhu / Chen Changzhi で構造差があるもの、関連 components / variants を含める。 |
| glyph principle | Tenghuaxie original forms を基準にし、追加 characters / components も意図的な再構成ではなく原形を保つ。 |

文書は、Small Seal が modern Hanzi の calligraphic typeface ではなく、Oracle Bone script、Bronze inscriptions、Warring States scripts などと同じ古代漢字の発展段階を表すと説明する。modern standard Hanzi と一対一対応しない characters や、modern form と構造が異なる characters があるため、CJK Unified Ideographs の glyph variation としてではなく独立符号化すべきだというのが初期 proposal の中心である。

## 後続決定

この文書自体は proposal であり、会議決定ではない。後続では [WG2 N4688](wg2-n4688.md) の大容量 proposal、[WG2 N5105](wg2-n5105.md) の revised appendix、[WG2 N5209](wg2-n5209.md) の multi-source comparison、[WG2 N5344R2](wg2-n5344r2.md) の revised proposal へ整理が進んだ。

## 論点

### 独立 script の根拠

N4634 は、Small Seal を modern Han の書体差ではなく、古代漢字研究で使われる別 script として扱う根拠を早い段階で明示した。これは後続 proposal でも維持され、Small Seal topic の中心論点になっている。

### Shuowen Jiezi version

初期 proposal は Daxu Ben 系の Tenghuaxie version を major reference とし、Duan Zhu や Chen Changzhi との差分も取り込む設計だった。ただし 2014 年時点では source model はまだ後続 proposal の four-column THX / CCZ / QJZ / DYC model ほど明確ではない。
