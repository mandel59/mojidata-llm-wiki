# IRG N2957

## 要約

`IRG N2957` は、[IRG N2956](irg-n2956.md) の Extension C GBK-source issues への feedback である。文書は、5 characters を同じ処理に寄せるのではなく、glyph correction、source reference correction、または新しい source reference 提供の必要性を character ごとに分けて述べる。

feedback は、source evidence mismatch がある場合でも、必ず representative glyph を変えるとは限らないことを示している。誤字形として扱えるものは source reference の修正だけで足りる場合があり、逆に glyph correction が必要な場合もある。

## フィードバック内容

| 対象 | feedback |
| --- | --- |
| U+2AE99 | `IRG N2956` の proposed glyph へ直すことには慎重で、別形への直接修正と GZHSJ 側の新 source reference 提供を提案する。 |
| U+2B0AF | source reference modification に同意する。 |
| U+2B622 | representative glyph を original evidence に近い形へ修正することに同意する。 |
| U+2B26F | 誤字形として扱い、source reference の修正だけでよいとする。 |
| U+2B720 | U+2CE65 の誤字形として扱い、source reference の修正だけでよいとする。 |

## 後続決定

Meeting #67 agenda の [IRG N2935](irg-n2935.md) は、G-source / SAT-source などの representative glyph and source reference issues を Section 9.1 の継続確認対象に置く。`IRG N2957` 自体は feedback であり、最終 disposition は記録していない。

## 論点

### 誤字形と glyph correction の境界

`IRG N2957` は、original evidence と current glyph の差を、単純に glyph correction へ変換しない。誤字形、source reference mismatch、representative glyph mismatch を分けることで、encoded character の identity と chart glyph の更新を混同しないようにしている。
