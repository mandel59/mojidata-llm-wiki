# L2/26-130

## 要約

`L2/26-130` は Chinese folk music、Chinese Xiqu Opera、Chinese Quyi Show で使われる 1 musical symbol と 2 format controls を Musical Symbols Supplement block に追加する提案である。提案文字は breath mark 1 文字と free repeat の begin / end controls である。

提案は、音楽記譜の plain text representation と format control の境界に関わる。U+1D282 は visible symbol、U+1D283 / U+1D284 は format controls として `Cf` / `BN` property を提案する。

## 提案内容

| Proposed code point | Character name | 性質 |
| --- | --- | --- |
| U+1D282 | MUSICAL SYMBOL DOWN ARROWHEAD-SHAPED BREATH MARK | 中国民間音楽で使う breath mark。U+1D112 と用法が近いとする。 |
| U+1D283 | MUSICAL SYMBOL BEGIN FREE REPEAT | free repeat の開始を示す format control。 |
| U+1D284 | MUSICAL SYMBOL END FREE REPEAT | free repeat の終了を示す format control。 |

## 論点

### musical symbol と format control の組み合わせ

この proposal は visible symbol だけでなく、music notation の範囲指定を plain text control として表す。既存の Musical Symbols controls との整合、line break / East Asian Spacing / script property、format controls を追加する妥当性が review point になる。
