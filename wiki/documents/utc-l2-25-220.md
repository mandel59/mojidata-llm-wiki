# L2/25-220

## 要約

`L2/25-220` は、前回の cumulative document 以後、UTC Meeting \#185 前後に受け取った Public Review Issue と一般 public feedback をまとめた記録である。冒頭では PRI \#509、\#510、\#520、\#523、\#532 と各 permanent feedback document への入口を示し、それ以外の報告を CJK & Unihan、Script Encoding、Properties & Algorithms、Emoji、Editorial の working group 別に振り分けている。

内容の中心は、Unihan readings / variants / radical-stroke / representative glyph / source mapping、UAX \#14 line breaking、UCD casing、UAX \#42、Core Specification と code chart の誤記である。この文書は feedback の採否を決めるものではなく、担当 working group が評価するための入力を保存する。

表題の収集期間は 2025-07-03 から 2025-10-06 だが、CJK section には 10 月 7 日・14 日付の報告も収録されているため、個々の報告日は本文の timestamp を基準に読む必要がある。

## 後続決定

この文書自身は disposition や UTC decision を記録しない。後続の [L2/26-099](utc-l2-26-099.md) は CJK & Unihan Working Group がこの public feedback の一部を評価した recommendation 文書であり、U+2B158 の `kRSUnicode` と U+2B25D の `kRSUnicode` / `kTotalStrokes` を current T-source representative glyph に合わせる変更などを Unicode Version 18.0 target としてまとめている。

PRI \#510、\#520、\#523 に対応する規格案は、それぞれ [L2/25-100](utc-l2-25-100.md)、[L2/26-108](utc-l2-26-108.md)、[L2/26-111](utc-l2-26-111.md) から追跡できる。ただし `L2/25-220` に集約された各コメントが最終規格へ採用されたかは、各 PRI の resolution と後続 UTC documents を個別に確認する必要がある。

## 論点

### PRI feedback と一般報告の区別

冒頭の5件は open PRI と permanent feedback documents への索引である。一方、working group 別 sections は publication / data error や general feedback の本文を直接収録する。したがって、このページを単一 PRI の disposition としてではなく、複数 review channel の cumulative intake document として扱う必要がある。

### CJK data の相互依存

CJK section では representative glyph、source mapping、radical-stroke、stroke count、reading、variant relation が同時に現れる。glyph correction が `kRSUnicode` や `kTotalStrokes` の変更を必要とする例、過去の CJK & Unihan / IRG decision を根拠に再修正を求める例があり、個別 property だけでなく source evidence と code chart の整合を確認する必要がある。
