# L2/07-421

## 要約

`L2/07-421` / `WG2 N3388` は、historical Japanese の /e/ と /ye/ の区別を plain text で表すため、2 kana characters を追加する提案である。提案者は Nozomu Kato で、Proposal Summary Form では既存 Hiragana / Katakana repertoire への追加として扱っている。

提案は、現代の Hiragana / Katakana に YE 専用字がないのは、10 世紀ごろまでに /e/ と /ye/ が合流したためだと説明する。古典文学・国語史資料で区別を示すため、Hiragana 側に HIRAGANA LETTER YE、Katakana 側に KATAKANA LETTER ORIGINAL E を置くことを求めた。

## 提案内容

| 区分 | 内容 |
| --- | --- |
| 提案文字 | HIRAGANA LETTER YE、KATAKANA LETTER ORIGINAL E。 |
| 根拠 | HIRAGANA LETTER YE は 江 由来、KATAKANA LETTER ORIGINAL E は 衣 の一部由来と説明される。 |
| 提案位置 | HIRAGANA LETTER YE は U+3097、KATAKANA LETTER ORIGINAL E は Katakana block に空きがないため別位置が必要とする。 |
| 用途 | 古典日本語で /e/ と /ye/ を区別する学術・辞書・古典文学用途。 |

## 論点

### HIRAGANA LETTER YE と既存 hentaigana

提案時点では、江 由来の字形を historical /ye/ の letter として扱うが、同じ字形は hentaigana の文脈でも現れる。この点は後続の `L2/08-117` と `L2/08-358` で、個別文字として先に符号化するか、obsolete kana / hentaigana 全体の枠組みで扱うかという論点になった。

### KATAKANA LETTER ORIGINAL E の符号化位置

提案は、既存 U+30A8 KATAKANA LETTER E が本来 /ye/ を表していたため、/e/ 側の historical letter として新字を置くという convention を採る。この convention は後続の Japan NB feedback で「専門家にも混乱しやすい」と評価され、同時点での支持は得られなかった。
