# WG2 N5315

## 要約

`WG2 N5315` は、ISO/IEC 10646 の code charts を Unicode Standard と共通化し、10646 text 本体を巨大な chart PDF から分離する可能性を検討する文書である。前回会合で議論された [WG2 N5253](wg2-n5253.md) を踏まえ、formal recommendation を求めるとともに、共通 code chart resource の作成を Project Editor へ勧告する方向を示す。

文書は、UAX \#38 / UAX \#60 など Unicode 側の data / annex を normatively reference する方向と、Unicode が生成する archived / consolidated charts を ISO/IEC 10646 の publication artifact として使う方向を接続する。

## 報告内容

| 区分 | 内容 |
| --- | --- |
| 背景 | ISO/IEC 10646 text に code charts を含めると、標準本文が数千ページ規模になり編集負荷が高い。 |
| 提案 | Unicode と ISO/IEC 10646 が共通の code chart resource を使えるかを検討し、Project Editor に具体化を促す。 |
| data reference | CJK / Tangut / Small Seal など large East Asian scripts の source description は UAX \#38 / UAX \#60 のような Unicode data annex との整合が必要になる。 |
| 後続 | WG2 \#72 M72.10 は code charts の電子的 attachment / linked content 化を ITTF と検討するよう勧告した。 |

## 後続決定

[WG2 N5304](wg2-n5304.md) の M72.10 は、Project Editor が ITTF と code charts の core text 外部化を検討することを勧告した。[WG2 N5363R](wg2-n5363r.md) は、その follow-up として external code charts を normative reference する具体案を示し、[WG2 N5354](wg2-n5354.md) の M73.03 で DIS へ反映する勧告になった。

## 論点

### ISO と Unicode の publication artifact

code charts は Unicode と ISO/IEC 10646 の repertoire synchronization を読者が確認する主要 artifact である。外部 chart resource 化は、単なる PDF サイズ削減ではなく、どの版の Unicode / 10646 charts がどの standard text の integral part になるかを明示する publication model の変更である。
