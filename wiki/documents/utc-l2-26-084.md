# L2/26-084

## 要約

`L2/26-084` は、2024 年の UTC action items から生じた 169 characters の `kMandarin` additions / changes について、CLDR TC と Apple Localization and Release Engineering が確認結果を返した feedback document である。対象 action は UTC \#178 から \#180 までの複数 action items にまたがり、関連文書は [L2/24-012](utc-l2-24-012.md)、[L2/24-067](utc-l2-24-067.md)、[L2/24-165](utc-l2-24-165.md) である。

feedback は、proposed value に同意するもの、proposed / existing value に同意せず alternative を示すもの、zh-Hans / zh-Hant で preferred reading を分けて combined value とするものに分類される。文書本体は character-by-character table で、dictionary / online reference links を根拠として添える。

## フィードバック内容

| 分類 | 内容 |
| --- | --- |
| Agree / No Change | proposed `kMandarin` value または existing value を支持する。 |
| Disagree | proposed value または existing value に同意せず、別の reading を提案する。 |
| Combined | zh-Hans と zh-Hant で preferred reading が異なる場合に、複数値を combined form として提案する。 |

代表例として、U+941F は proposed `tì` ではなく `zān` を維持すること、U+4680 は `jiàn` ではなく `chēn` を提案すること、U+4E48 は zh-Hans では `me`、zh-Hant では `yāo` を併記することなどが示されている。

## 後続決定

[L2/26-099](utc-l2-26-099.md) Section 22 は、`L2/26-084` の feedback を受けて 169 characters の `kMandarin` additions / changes を Unicode Version 18.0 target で進めるよう UTC \#187 に勧告した。したがって `L2/26-084` は、個々の readings の evidence review を示す source であり、UTC 側の action item は `L2/26-099` で追う。

## 論点

### `kMandarin` と locale preference

文書は単一の Mandarin reading を機械的に選ぶだけでなく、zh-Hans / zh-Hant の preferred reading 差を `Combined` として扱う。Unihan data の reading property は、source references と同様に複数地域・複数辞書の evidence を調整する必要がある。

### CLDR / localization review の役割

この feedback は CLDR TC と Apple Localization and Release Engineering の review として提出されており、Unicode data maintenance が UTC 内の action item だけで完結しないことを示す。実装上は CLDR と Unihan data の alignment が重要になる。
