# L2/07-160

## 要約

`L2/07-160` は、CJK Extension C の文書 `N3267` にある U+2B151 の U-source indication が誤っていると指摘し、正しい U-source は `UTC00029` であるとする短い correction document である。US comments にこの correction を入れるべきだと述べ、`UTC00029` の glyph を示している。

## フィードバック内容

文書は、U-source attribution correction を WG2 / UTC comment process に入れるための材料である。単独では UAX #45 の data model を定義しないが、[L2/07-159](utc-l2-07-159.md) が指摘した「public document として source attribution を参照できる必要性」を具体例として示す。

## 後続決定

この wiki では、この correction を取り上げた meeting / event はまだページ化していない。U-source attribution correction の一般論は [UAX #45 / U-Source Ideographs](../topics/uax45-u-source-ideographs.md) と [IRG Source Data and Representative Glyphs](../topics/irg-source-data-and-representative-glyphs.md) で追跡する。
