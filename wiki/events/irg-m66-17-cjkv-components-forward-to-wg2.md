# IRG M66.17 CJKV Components forward to WG2

## 概要

IRG Meeting \#66 Recommendation M66.17 は、`IRG N2878R3` の CJK Unified Ideographs Components-A / B を WG2 へ forward することを勧告した出来事である。Proposal Summary Form を付けて WG2 \#73 へ提出することも後続として示された。

## 背景

`IRG N2878R3` は CJKV Components の final proposal であり、IRG \#66 はこれを WG2 / SC2 の標準化 pipeline へ送るかを判断する会合だった。会合では、将来の components 追加方法も authors に検討させることになった。

## 結果

M66.17 により、CJK Unified Ideographs Components-A / B は IRG から WG2 へ進んだ。Meeting \#67 agenda `IRG N2935` は、その後 WG2 が受け入れを勧告し、SC2 が ISO/IEC 10646 Seventh Edition Amendment 1 として accept した current status を記録している。

## 影響範囲

この event は、CJKV Components が IRG 内の proposal から WG2 / SC2 pipeline へ移った節目である。proposal 内容の詳細は [IRG N2878R3 CJKV Components final proposal](irg-n2878r3-cjkv-components-final-proposal.md) に集約する。
