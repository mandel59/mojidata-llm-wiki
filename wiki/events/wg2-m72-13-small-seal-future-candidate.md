# WG2 M72.13 Small Seal future edition candidate

## 概要

WG2 Meeting #72 は Recommendation M72.13 で、11,339 Small Seal characters を `38000..3AC4A` に置き、新 block `38000..3AC4F` として将来版で検討するよう Project Editor に勧告した。これは Small Seal Script が WG2 の future edition candidate として明示的に扱われた節目である。

## 背景

`WG2 N5306` は、Small Seal の unification / disunification issue を整理し、最終 proposal に向けた review を進める文書だった。`WG2 N5317` は TCA / China feedback の disposition、`WG2 N5318` は Four-Column Small Seal Script proposal を示し、WG2 #72 の recommendation の根拠になった。

## 結果

M72.13 は、Small Seal を最終符号化として確定したのではなく、将来版で検討する候補として Project Editor に扱わせる勧告である。後続の `WG2 N5344` では repertoire と range が整理され、11,328 characters、`3D000..3FC3F` へ更新された。

## 影響範囲

この event は、Small Seal Script が長期の proposal / feedback 段階から WG2 の標準化 pipeline に入った時点として扱う。後続の CD4 disposition と DIS 進行は別 event として追跡する。
