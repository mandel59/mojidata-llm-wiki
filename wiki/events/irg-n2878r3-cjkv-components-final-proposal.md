# IRG N2878R3 CJKV Components final proposal

## 概要

`IRG N2878R3` は、CJK Unified Ideographs Components-A / B の final proposal として、Components-A 119 characters、Components-B 375 characters、合計 494 characters を提示した出来事である。WG2 registry では同じ提案が `WG2 N5358` として扱われる。

## 背景

この proposal は、`IRG N2733R`、`IRG N2799`、`IRG N2878`、`IRG N2878R`、`IRG N2878R2` の revision lineage と、IRG \#63 / \#64 / \#65 の recommendations、個別 feedback を取り込んだものとして位置づけられる。HKSAR、TCA、Vietnam、Jianzi Musical Notation などの source data も整理された。

## 結果

`IRG N2878R3` は、input method fragments を scope 外とし、rational decomposition / visual decomposition によって説明できる components を first batch の対象とした。individual contributions の一部は first batch には採用せず、second or subsequent batches への再提出可能性を残した。

## 影響範囲

この event は、CJKV Components の proposal 内容そのものの最終化を表す。IRG が WG2 へ forward する決定は [IRG M66.17 CJKV Components forward to WG2](irg-m66-17-cjkv-components-forward-to-wg2.md) として別に扱う。
